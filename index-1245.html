安装 apk 时 INSTALL_FAILED_NO_MATCHING_ABIS

Arya

2023-02-18 04:59

我尝试将我的应用程序安装到 Android L Preview Intel Atom 虚拟设备中，但失败并出现错误：

安装失败_NO_MATCHING_ABIS

这是什么意思？



解答

INSTALL_FAILED_NO_MATCHING_ABIS是当您尝试安装具有本机库但没有适用于您的 cpu 架构的本机库的应用程序时。例如，如果您为 armv7 编译了一个应用程序并试图将其安装在使用 Intel 架构的模拟器上，它将无法运行。


更多建议

INSTALL_FAILED_NO_MATCHING_ABIS 是当您尝试安装具有本机库但没有适用于您的 cpu 架构的本机库的应用程序时。例如，如果您为 armv7 编译了一个应用程序并试图将其安装在使用 Intel 架构的模拟器上，它将无法运行。

在 Visual Studio 2015 上使用 Xamarin。 通过以下方式解决此问题：

  1. 打开你的 xamarin .sln
  2. 右键单击您的android项目
  3. 单击属性
  4. 点击安卓选项
  5. 单击“高级”选项卡

  6. 在“支持的体系结构”下，检查以下内容：

    1. armeabi-v7a
    2. x86
    3. 节省

  7. F5（建造）

编辑： 据报道，此解决方案也适用于 Visual Studio 2017 。

编辑 2： 据报道，此解决方案也适用于 Visual Studio 2017 for Mac 。

如何使用 ImageMagick 将 SVG 转换为 PNG？

Arya

2023-02-18 04:58

我有一个定义大小为 16x16 的 SVG 文件。当我使用 ImageMagick 的转换程序将其转换为 PNG 时，我得到一个 16x16 像素的 PNG，这太小了：

convert test.svg test.png

我需要指定输出 PNG 的像素大小。-size参数似乎被忽略，-scale参数在将 PNG转换为 PNG 后对其进行缩放。 迄今为止我使用-density参数获得的最佳结果：

convert -density 1200 test.svg test.png

但我不满意，因为我想在不进行数学运算来计算密度值的情况下以像素为单位指定输出大小。所以我想做这样的事情：

convert -setTheOutputSizeOfThePng 1024x1024 test.svg test.png

那么我必须在这里使用的神奇参数是什么？



解答

在这种情况下，我无法从 ImageMagick 获得好的结果，但 Inkscape 在 Linux 和 Windows 上缩放 SVG 做得很好：

# Inkscape v1.0+
inkscape -w 1024 -h 1024 input.svg -o output.png



# Inkscape older than v1.0
inkscape -z -w 1024 -h 1024 input.svg -e output.png

请注意，您可以省略其中一个宽度/高度参数，以使另一个参数根据输入图像尺寸自动缩放。

更多建议

尝试 svgexport ：

svgexport input.svg output.png 64x
svgexport input.svg output.png 1024:1024

svgexport 是我制作的一个简单的跨平台命令行工具，用于将 svg 文件导出为 jpg 和 png，请参见此处了解更多选项。要安装 svgexport install npm，然后运行：

npm install svgexport -g
