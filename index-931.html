将 Flask 开发服务器配置为在网络上可见

Arya

2023-02-17 20:01

我不确定这是否是特定于 Flask 的，但是当我以开发模式 ( http://localhost:5000) 运行应用程序时，我无法从网络上的其他机器访问它 ( 使用http://[dev-host-ip]:5000)。例如，在开发模式下使用 Rails，它工作正常。我找不到任何关于 Flask 开发服务器配置的文档。知道应该配置什么来启用它吗？



解答

虽然这是可能的，但您不应在生产环境中使用 Flask 开发服务器。Flask 开发服务器并不是特别安全、稳定或高效的。请参阅有关部署的文档以获取正确的解决方案。

--host的选项或flask run参数host控制app.run()开发服务器监听的地址。默认情况下它运行于localhost，将其更改为flask run --host=0.0.0.0（或app.run(host="0.0.0.0")）以在您计算机的所有 IP 地址上运行。

0.0.0.0是一个你不能直接在浏览器中使用的特殊值，你需要导航到网络上机器的实际 IP 地址。您可能还需要调整防火墙以允许外部访问该端口。

Flask快速入门文档在“外部可见服务器”部分对此进行了解释：

如果您运行该服务器，您会注意到该服务器只能从您自己的计算机访问，而不能从网络中的任何其他计算机访问。这是默认设置，因为在调试模式下，应用程序的用户可以在您的计算机上执行任意 Python 代码。

如果您禁用了调试器或信任您网络上的用户，则只需在--host=0.0.0.0命令行中添加以下命令即可使服务器公开可用：

$ flask run --host=0.0.0.0

这告诉您的操作系统监听所有公共 IP。


更多建议

如果您使用flask可执行文件来启动您的服务器，请使用flask run --host=0.0.0.0更改默认值 127.0.0.1 并将其打开到非本地连接。

read_csv可以选择encoding处理不同格式的文件。我主要使用read_csv('file', encoding = "ISO-8859-1"), 或者用于encoding = "utf-8"阅读，通常utf-8用于to_csv.

您还可以使用多个alias选项之一，例如'latin'或'cp1252'(Windows) 而不是'ISO-8859-1'（请参阅python 文档，以及您可能遇到的许多其他编码）。

请参阅相关的 Pandas 文档、csv 文件上的 python 文档示例，以及此处关于 SO 的大量相关问题。What every developer should know about unicode and character sets是一个很好的背景资源。

要检测编码（假设文件包含非 ascii 字符），您可以使用enca（参见手册页）或file -i（linux）或file -I（osx）（参见手册页）。


更多建议

所有解决方案中最简单的：

import pandas as pd
df = pd.read_csv('file_name.csv', engine='python')

替代解决方案：

崇高的文字：

  • 在Sublime 文本编辑器 或 VS Code 中打开 csv 文件。
  • 以 utf-8 格式保存文件。
  • 在 sublime 中，点击 File -> Save with encoding -> UTF-8

VS代码：

在 VSCode 的底部栏中，您会看到标签 UTF-8。点击它。弹出窗口打开。单击使用编码保存。您现在可以为该文件选择一个新的编码。

然后，您可以像往常一样阅读您的文件：

import pandas as pd
data = pd.read_csv('file_name.csv', encoding='utf-8')

其他不同的编码类型是：

encoding = "cp1252"
encoding = "ISO-8859-1"

HTML 中是否有最小长度验证属性？

Arya

2023-02-17 20:00

这似乎minlength是一个属性<input>字段的属性不起作用。

HTML 中是否有任何其他属性可以帮助我设置字段值的最小长度？



解答

您可以使用该pattern属性。还需要该required属性，否则具有空值的输入字段将被排除在约束验证之外。

<input pattern=".{3,}"   required title="3 characters minimum">
<input pattern=".{5,10}" required title="5 to 10 characters">

如果要创建选项以将模式用于“空或最小长度”，则可以执行以下操作：

<input pattern=".{0}|.{5,10}" required title="Either 0 OR (5 to 10 chars)">
<input pattern=".{0}|.{8,}"   required title="Either 0 OR (8 chars minimum)">


更多建议

现在HTML5 规范 中有 一个属性minlength，以及接口。validity.tooShort

两者现在都在所有现代浏览器的最新版本中启用。有关详细信息，请参阅https://caniuse.com/#search=minlength。
