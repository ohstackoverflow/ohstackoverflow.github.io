Android Studio中如何查看方法信息

Arya

2023-02-18 02:11

在 Eclipse 中，当您将鼠标悬停在一个方法上时，会出现一个窗口，其中描述了该方法的作用、参数的含义以及它返回的内容。有没有办法让 Android Studio 做同样的事情？



解答

最简单直接的方法：

要激活：菜单 文件 → 设置 → 编辑器 → 常规

对于 Mac OS X， Android Studio → Preferences → Editor → General 并选中 Show quick documentation on mouse move ：

其他方法：

  • 您可以进入IntelliJ IDEA 的 bin 文件夹并搜索 idea.properties 。将此行添加到文档中：

auto.show.quick.doc=true

现在您将拥有与Eclipse中相同的浮动文档窗口。

  • 您必须按Ctrl+Q才能查看 Javadoc。

不过，您可以固定窗口并在每次使用鼠标选择方法时显示文档。

Android Studio 1.0： 如果要按住文档窗口，则必须按住Ctrl，例如滚动文档。否则，当您将鼠标移离该方法时，文档窗口将消失。


更多建议

是的你可以。转到菜单 文件 → 设置 → 编辑器 → 显示鼠标移动的快速文档

或者，在 Mac OS X 中，转到 Android Studio → Preferences → Editor → General → Show quick doc on mouse move 。

使用 jQuery 转义键的哪个键码

Arya

2023-02-18 02:10

我有两个功能。当按下 enter 时，函数运行正常，但当按下 escape 时，它​​不会。转义键的正确数字是多少？

$(document).keypress(function(e) { 
    if (e.which == 13) $('.save').click();   // enter (works as expected)
    if (e.which == 27) $('.cancel').click(); // esc   (does not work)
});



解答

尝试使用keyup 事件：

$(document).on('keyup', function(e) {
  if (e.key == "Enter") $('.save').click();
  if (e.key == "Escape") $('.cancel').click();
});


更多建议

与其在函数中对键码值进行硬编码，不如考虑使用命名常量来更好地传达您的意思：

var KEYCODE_ENTER = 13;
var KEYCODE_ESC = 27;

$(document).keyup(function(e) {
  if (e.keyCode == KEYCODE_ENTER) $('.save').click();
  if (e.keyCode == KEYCODE_ESC) $('.cancel').click();
});

一些浏览器（比如 FireFox，其他的不确定）定义了一个全局KeyEvent对象来为您公开这些类型的常量。This SO question展示了一种在其他浏览器中定义该对象的好方法。

列出 Git 别名

Arya

2023-02-18 02:10

如何打印我的 git 别名列表，即类似于 bashalias命令的东西？



解答

Just adding this because it's so simple and I didn't see it in previous answers (sorry if I missed it).

git help -a

You'll have to scroll to the bottom (use > as ma11hew28 pointed out) to see the list, e.g.:

Command aliases
   restore-deleted      !git restore $(git ls-files -d)

If you forget even this switch, a simple git help will help you remember:

'git help -a' and 'git help -g' list available subcommands and some concept guides. See 'git help ' or 'git help ' to read about a specific subcommand or concept.


更多建议

这个答案建立在johnny的答案之上。如果您不使用from则它适用。git-alias``git-extras

在 Linux 上，运行一次：

git config --global alias.alias "! git config --get-regexp ^alias\. | sed -e s/^alias\.// -e s/\ /\ =\ /"

这将创建一个名为的永久 git 别名，alias该别名将存储在您的~/.gitconfig文件中。使用它将列出你所有的 git 别名，其格式与文件中的格式几乎相同~/.gitconfig。要使用它，请键入：

$ git alias
loga = log --graph --decorate --name-status --all
alias = ! git config --get-regexp ^alias\. | sed -e s/^alias\.// -e s/\ /\ =\ /

以下注意事项适用：

  • 为防止别名alias如上所示，| grep -v ^'alias '在结束双引号之前附加。我不推荐这样做，这样用户就不会忘记该命令alias只是一个别名，而不是 git 的功能。

  • 要对列出的别名进行排序，请| sort在结束双引号之前附加。或者，您可以对别名进行~/.gitconfig排序。

  • 要将别名添加为系统范围的别名，请将--global(for current user) 替换为--system(for all users)。这通常放在/etc/gitconfig文件中。
