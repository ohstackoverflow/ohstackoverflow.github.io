更改 hr 元素的颜色

Arya

2023-02-17 09:47

我想使用 CSS 更改标签的颜色hr。我在下面尝试过的代码似乎不起作用：

hr {
    color: #123455;
}



解答

如果您打算更改标签生成的线条的颜色，我认为您应该使用border-color而不是, 。color``<hr>

虽然，评论中指出，如果您更改线条的大小，边框仍将与您在样式中指定的一样宽，并且线条将填充默认颜色（这不是大多数人想要的效果时间）。因此，在这种情况下，您似乎还需要指定background- color（正如@Ibu 在他的回答中所建议的那样）。

HTML 5 Boilerplate 项目在其默认样式表中指定了以下规则：

hr { display: block; height: 1px;
    border: 0; border-top: 1px solid #ccc;
    margin: 1em 0; padding: 0; }

*SitePoint 最近发表的 *一篇 名为“12 Little-Known CSS Facts”的文章提到，如果您指定 ，<hr>可以将其设置border-color为其父级。color``hr { border-color: inherit }


更多建议

  • border-color适用于 Chrome 和 Safari 。
  • background-color适用于 Firefox 和 Opera。
  • color适用于 IE7+ 。

如果文件夹不存在，则创建它

Arya

2023-02-17 09:46

我FileUploader在我的应用程序中使用了一个控件。我想将文件保存到指定的文件夹。如果这个文件夹不存在，我想先创建一个，然后把我的文件保存到这个文件夹中。如果该文件夹已经存在，则只需将文件保存在其中即可。

我怎样才能做到这一点？



解答

使用 System.IO.Directory.CreateDirectory 。

根据官方“.NET”文档，您无需先检查它是否存在。

System.io > 目录 > Directory.CreateDirectory

中指定的任何和所有目录path都会被创建，除非它们已经存在或除非其中的某些部分path无效。如果目录已经存在，则此方法不会创建新目录，但会返回DirectoryInfo现有目录的对象。

— learn.microsoft.com/dotnet/api/


更多建议

根据如何使用文件上传服务器控件动态创建文件夹， 使用以下代码？:

string subPath ="ImagesPath"; // Your code goes here

bool exists = System.IO.Directory.Exists(Server.MapPath(subPath));

if(!exists)
    System.IO.Directory.CreateDirectory(Server.MapPath(subPath));

在字典中按值获取键

Arya

2023-02-17 09:46

我做了一个函数，它将在 a 中查找年龄Dictionary并显示匹配的名称：

dictionary = {'george' : 16, 'amber' : 19}
search_age = raw_input("Provide age")
for age in dictionary.values():
    if age == search_age:
        name = dictionary[age]
        print name

我知道如何比较和查找我只是不知道如何显示人名的年龄。KeyError此外，由于第 5 行，我得到了一个。我知道它不正确，但我不知道如何让它向后搜索。



解答

空无一人。dict不打算以这种方式使用。

dictionary = {'george': 16, 'amber': 19}
search_age = input("Provide age")
for name, age in dictionary.items():  # for name, age in dictionary.iteritems():  (for Python 2.x)
    if age == search_age:
        print(name)
