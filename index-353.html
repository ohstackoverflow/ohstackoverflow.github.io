如何将命令行参数传递给 rake 任务

Arya

2023-02-17 04:47

我有一个 rake 任务需要将一个值插入多个数据库。

我想从命令行或 另一个 rake 任务将这个值传递给 rake 任务。

我怎样才能做到这一点？



解答

选项和依赖项需要在数组中：

namespace :thing do
  desc "it does a thing"
  task :work, [:option, :foo, :bar] do |task, args|
    puts "work", args
  end

  task :another, [:option, :foo, :bar] do |task, args|
    puts "another #{args}"
    Rake::Task["thing:work"].invoke(args[:option], args[:foo], args[:bar])
    # or splat the args
    # Rake::Task["thing:work"].invoke(*args)
  end

end

然后

rake thing:work[1,2,3]
=> work: {:option=>"1", :foo=>"2", :bar=>"3"}

rake thing:another[1,2,3]
=> another {:option=>"1", :foo=>"2", :bar=>"3"}
=> work: {:option=>"1", :foo=>"2", :bar=>"3"}

注意：变量task是任务对象，除非您了解/关心 Rake 内部结构，否则它不是很有用。

轨道注意：

如果从 Rails 运行任务，最好通过添加预加载环境，=> [:environment]这是设置 依赖 任务的一种方式。

  task :work, [:option, :foo, :bar] => [:environment] do |task, args|
    puts "work", args
  end

获取 Oracle 中所有表的列表？

Arya

2023-02-17 04:46

如何查询 Oracle 数据库以显示其中所有表的名称？



解答

SELECT owner, table_name
  FROM dba_tables

这是假设您有权访问DBA_TABLES数据字典视图。如果您没有这些权限但需要它们，您可以请求 DBA 明确授予您对该表的权限，或者 DBA 授予您权限SELECT ANY DICTIONARY或SELECT_CATALOG_ROLE角色（其中任何一个都允许您查询任何数据字典表). 当然，您可能希望排除某些模式，例如SYS具有SYSTEM大量您可能不关心的 Oracle 表的模式。

或者，如果您没有访问权限DBA_TABLES，您可以通过视图查看您的帐户可以访问的所有表ALL_TABLES：

SELECT owner, table_name
  FROM all_tables

虽然，这可能是数据库中可用表的一个子集（ALL_TABLES向您显示您的用户已被授予访问权限的所有表的信息）。

如果你只关心你拥有的表，而不是你有权访问的表，你可以使用USER_TABLES：

SELECT table_name
  FROM user_tables

因为USER_TABLES只有关于您拥有的表的信息，所以它没有列OWNER——根据定义，所有者就是您。

Oracle 还有许多可以使用的遗留数据字典视图—— 、TAB、和。一般来说，我不建议使用这些遗留视图，除非您绝对需要将您的脚本向后移植到 Oracle 6。Oracle 已经很长时间没有更改这些视图，因此它们经常在处理较新类型的对象时出现问题。例如，和视图都显示有关用户回收站中的表的信息，而视图都将这些信息过滤掉。还显示有关带有“TABLE”的物化视图日志的信息，这不太可能是您真正想要的。组合表和同义词，但不会告诉您谁拥有该对象。DICT``TABS``CAT``TAB``CAT``[DBA|ALL|USER]_TABLES``CAT``TABLE_TYPE``DICT


更多建议

查询user_tables并dba_tables没有工作。
这个做了：

select table_name from all_tables

如何使用 JavaScript 获取文本输入字段的值？

Arya

2023-02-17 04:45

我正在使用 JavaScript 进行搜索。我会使用一个表单，但它会弄乱我页面上的其他内容。我有这个输入文本字段：

<input name="searchTxt" type="text" maxlength="512" id="searchTxt" class="searchField"/>

这是我的 JavaScript 代码：

<script type="text/javascript">
  function searchURL(){
    window.location = "http://www.myurl.com/search/" + (input text value);
  }
</script>

如何从文本字段中获取值到 JavaScript 中？



解答

有多种方法可以直接获取输入文本框值（无需将输入元素包装在表单元素中）：

方法一

document.getElementById('textbox_id').value获得所需框的值

例如

document.getElementById("searchTxt").value;

注意： 方法 2、3、4 和 6 返回元素的集合，因此使用 [whole_number] 来获取所需的出现次数。对于第一个元素，使用[0]，对于第二个元素，使用[1]，依此类推...

方法二

使用document.getElementsByClassName('class_name')[whole_number].value它返回一个 Live HTMLCollection

例如

document.getElementsByClassName("searchField")[0].value;如果这是您页面中的第一个文本框。

方法三

使用document.getElementsByTagName('tag_name')[whole_number].value它还返回一个实时 HTMLCollection

例如

document.getElementsByTagName("input")[0].value;，如果这是您页面中的第一个文本框。

方法四

document.getElementsByName('name')[whole_number].value这也 > 返回一个实时的 NodeList

例如

document.getElementsByName("searchTxt")[0].value;如果这是您页面中名称为“searchtext”的第一个文本框。

方法五

使用强大的document.querySelector('selector').valueCSS选择器来选择元素

例如

  • document.querySelector('#searchTxt').value;由 id 选择
  • document.querySelector('.searchField').value;选课
  • document.querySelector('input').value;由标记名选择
  • document.querySelector('[name="searchTxt"]').value;按名称选择

方法六

document.querySelectorAll('selector')[whole_number].value它也使用 CSS 选择器来选择元素，但它将所有带有该选择器的元素作为静态节点列表返回。

例如

  • document.querySelectorAll('#searchTxt')[0].value;由 id 选择
  • document.querySelectorAll('.searchField')[0].value;选课
  • document.querySelectorAll('input')[0].value;由标记名选择
  • document.querySelectorAll('[name="searchTxt"]')[0].value;按名称选择

//creates a listener for when you press a key
window.onkeyup = keyup;

//creates a global Javascript variable
var inputTextValue;

function keyup(e) {
  //setting your input text to the global Javascript Variable for every key press
  inputTextValue = e.target.value;

  //listens for you to press the ENTER key, at which point your web address will change to the one you have input in the search box
  if (e.keyCode == 13) {
    window.location = "http://www.myurl.com/search/" + inputTextValue;
  }
}
