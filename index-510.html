在 Python 中将十六进制字符串转换为整数

Arya

2023-02-17 08:50

如何将十六进制字符串转换为整数？

"0xffff"   ⟶   65535
"ffff"     ⟶   65535



解答

没有 前缀0x，您需要明确指定基数，否则无法分辨：

x = int("deadbeef", 16)

通过 前缀0x，Python 可以自动区分十六进制和十进制：

>>> print(int("0xdeadbeef", 0))
3735928559
>>> print(int("10", 0))
10

（您 必须 指定0为基数才能调用此前缀猜测行为；如果您省略第二个参数，int()将假定基数为 10。）


更多建议

int(hexstring, 16)可以解决问题，并且可以使用和不使用 0x 前缀：

>>> int("a", 16)
10
>>> int("0xa", 16)
10

如何播放音频？

Arya

2023-02-17 08:49

我正在用 HTML5 和 JavaScript 制作游戏。

我如何通过 JavaScript 播放游戏音频？



解答

如果你不想弄乱 HTML 元素：

var audio = new Audio('audio_file.mp3');
audio.play();

显示代码片段

function play() {
  var audio = new Audio('https://interactive-examples.mdn.mozilla.net/media/cc0-audio/t-rex-roar.mp3');
  audio.play();
}


<button onclick="play()">Play Audio</button>

这使用了接口，它以与元素HTMLAudioElement相同的方式播放音频。<audio>

如果您需要更多功能，我使用了howler.js库，发现它简单实用。

显示代码片段

<script src="https://cdnjs.cloudflare.com/ajax/libs/howler/2.1.1/howler.min.js"></script>
<script>
    var sound = new Howl({
      src: ['https://interactive-examples.mdn.mozilla.net/media/cc0-audio/t-rex-roar.mp3'],
      volume: 0.5,
      onend: function () {
        alert('Finished!');
      }
    });
    sound.play()
</script>


更多建议

这很简单，只需获取您的audio元素并调用该play()方法即可：

document.getElementById('yourAudioTag').play();

看看这个例子： http: //www.storiesinflight.com/html5/audio.html

该站点揭示了您可以做的一些其他很酷的事情，例如load()、pause()和元素的一些其他属性audio。

Visual Studio Code 中的多行编辑

Arya

2023-02-17 08:49

是否可以像Sublime Text那样启用多行编辑？

例如，按下Ctrl以放置额外的光标插入符号，并能够一次在文档的多个位置写入/删除。



解答

在 Windows 上，按住Ctrl+键Alt的同时按向上↑或向下↓箭头键可添加光标。

麦克：⌥ Opt+ ⌘ Cmd+ ↑/↓

Linux：Shift+ Alt+ ↑/↓

请注意，第三方软件可能会干扰这些快捷方式，使它们无法按预期工作（特别是 Windows 上的英特尔高清显卡软件；有关详细信息，请参阅评论）。

如果遇到此问题，您可以禁用 Intel/其他软件热键，或修改 VS Code 快捷方式（如下所述）。

按Esc重置为单个光标。

或者，正如 Isidor Nikolic 指出的那样，您可以按住Alt并左键单击以任意放置光标。

您可以通过以下方式查看和编辑键盘快捷键：

文件 → 首选项 → 键盘快捷键

文档：

https://code.visualstudio.com/docs/customization/keybindings

官方 VS Code 键盘快捷键备忘单：

https://code.visualstudio.com/shortcuts/keyboard-shortcuts-windows.pdf
https://code.visualstudio.com/shortcuts/keyboard-shortcuts-macos.pdf
https://code.visualstudio.com/shortcuts/键盘快捷键-linux.pdf


更多建议

两步解决！

  1. Ctrl+F

  2. Alt+Enter

它等于 Sublime Text 的Alt+ F3。

注意：对于 macOS，⌘+F后跟⌥+Enter等同于 Sublime Text 的⌃+ ⌘+ G。
