MySQL 查询 GROUP BY 日/月/年

Arya

2023-02-17 14:17

是否可以进行一个简单的查询来计算我在确定的时间段（如年、月或日）中有多少条记录，有一个字段TIMESTAMP，如：

SELECT COUNT(id)
FROM stats
WHERE record_date.YEAR = 2009
GROUP BY record_date.YEAR

甚至：

SELECT COUNT(id)
FROM stats
GROUP BY record_date.YEAR, record_date.MONTH

每月统计一次。

谢谢！



解答

GROUP BY YEAR(record_date), MONTH(record_date)

查看MySQL 中的日期和时间函数。


更多建议

GROUP BY DATE_FORMAT(record_date, '%Y%m')

注意 （主要是针对潜在的反对者）。目前，这可能不如其他建议有效。尽管如此，我还是把它作为一种替代方案，也是一种可以用来了解其他解决方案有多快的替代方案。（因为在看到差异之前，您无法真正分辨快慢。）此外，随着时间的推移，可以在优化方面对 MySQL 引擎进行更改，以便在某些情况下（也许不是这样）做出这个解决方案遥远的）未来点，在效率上与大多数其他人相当。

编程中的声明式范式和命令式范式有什么区别？

Arya

2023-02-17 14:16

我一直在网上搜索 声明式 和 命令式 编程的定义，这对我有所启发。然而，我发现的一些资源所使用的语言令人生畏——例如维基百科。有没有人有一个真实的例子可以给我看，这可能会为这个主题带来一些观点（可能在 C# 中）？



解答

声明式编程与命令式编程的一个很好的 C# 示例是 LINQ。

使用 命令式 编程，您可以一步一步地告诉编译器您想要发生什么。

例如，让我们从这个集合开始，并选择奇数：

List<int> collection = new List<int> { 1, 2, 3, 4, 5 };

使用命令式编程，我们将一步步完成，并决定我们想要什么：

List<int> results = new List<int>();
foreach(var num in collection)
{
    if (num % 2 != 0)
          results.Add(num);
}

在这里，我们说：

  1. 创建结果集合
  2. 遍历集合中的每个数字
  3. 检查数字，如果是奇数，将其添加到结果中

另一方面，使用 声明式 编程，您编写的代码描述了您想要什么，但不一定是如何获得它（声明您想要的结果，但不是一步一步）：

var results = collection.Where( num => num % 2 != 0);

在这里，我们说的是“给我们所有奇怪的地方”，而不是“遍历集合。检查这个项目，如果它是奇怪的，将它添加到结果集合中。”

在许多情况下，代码也将是两种设计的混合体，因此它并不总是黑白分明的。


更多建议

声明式编程是说你想要 什么 ，命令式语言是说 如何 得到你想要的。

Python 中的一个简单示例：

# Declarative
small_nums = [x for x in range(20) if x < 5]

# Imperative
small_nums = []
for i in range(20):
    if i < 5:
        small_nums.append(i)

第一个示例是声明性的，因为我们没有指定构建列表的任何“实现细节”。

结合 C# 示例，通常，使用 LINQ 会产生声明式样式，因为您没有说明 如何 获得所需内容；你只是在说你 想要 的。你可以对 SQL 说同样的话。

声明式编程的一个好处是它允许编译器做出决定，这些决定可能会产生比您手工编写的代码更好的代码。使用 SQL 示例运行，如果您有类似的查询

SELECT score FROM games WHERE id < 100;

SQL“编译器”可以“优化”这个查询，因为它知道这id是一个索引字段——或者它可能没有索引，在这种情况下，无论如何它都必须遍历整个数据集。或者，也许 SQL 引擎知道这是利用所有 8 个内核进行快速并行搜索的最佳时机。 作为程序员，您 不关心这些条件中的任何一个，并且您不必编写代码来以那种方式处理任何特殊情况。

如何使用 JavaScript 从字符串中删除空格？

Arya

2023-02-17 14:16

如何去除字符串中的空格？例如：

输入：

'/var/www/site/Brand new document.docx'

输出：

'/var/www/site/Brandnewdocument.docx'



解答

这？

str = str.replace(/\s/g, '');

例子

var str = '/var/www/site/Brand new document.docx';

document.write( str.replace(/\s/g, '') );

更新： 基于这个问题，这个：

str = str.replace(/\s+/g, '');

是一个更好的解决方案。它产生相同的结果，但速度更快。

正则表达式

\s是“空白”的正则表达式，g是“全局”标志，意思是匹配所有\s（空白）。

可以在此处+找到一个很好的解释。

作为旁注，您可以将单引号之间的内容替换为您想要的任何内容，这样您就可以将空格替换为任何其他字符串。


更多建议

var a = b = " /var/www/site/Brand new   document.docx ";

console.log( a.split(' ').join('') );
console.log( b.replace( /\s/g, '') );

有两种方法可以做到这一点！
