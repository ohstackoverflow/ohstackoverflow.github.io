使用反射从字符串中获取属性值

Arya

2023-02-17 05:13

我正在尝试在我的代码中使用 Reflection 1示例实现数据转换。

该GetSourceValue函数有一个比较各种类型的开关，但我想删除这些类型和属性，并GetSourceValue只使用一个字符串作为参数来获取属性的值。我想在字符串中传递一个类和属性并解析属性的值。

这可能吗？

1 个 原始博客文章的 Web 存档版本



解答

 public static object GetPropValue(object src, string propName)
 {
     return src.GetType().GetProperty(propName).GetValue(src, null);
 }

当然，您会想要添加验证等等，但这就是它的要点。


更多建议

这样的事情怎么样：

public static Object GetPropValue(this Object obj, String name) {
    foreach (String part in name.Split('.')) {
        if (obj == null) { return null; }

        Type type = obj.GetType();
        PropertyInfo info = type.GetProperty(part);
        if (info == null) { return null; }

        obj = info.GetValue(obj, null);
    }
    return obj;
}

public static T GetPropValue<T>(this Object obj, String name) {
    Object retval = GetPropValue(obj, name);
    if (retval == null) { return default(T); }

    // throws InvalidCastException if types are incompatible
    return (T) retval;
}

这将允许您使用单个字符串下降到属性，如下所示：

DateTime now = DateTime.Now;
int min = GetPropValue<int>(now, "TimeOfDay.Minutes");
int hrs = now.GetPropValue<int>("TimeOfDay.Hours");

您可以将这些方法用作静态方法或扩展。

如何在 .NET 的格式字符串中转义大括号（花括号）

Arya

2023-02-17 05:12

如何在使用中转义括号string.Format？

例如：

String val = "1,2,3"
String.Format(" foo {{0}}", val);

此示例不会引发异常，但会输出 string foo {0}。

有没有办法逃避括号？



解答

为了让你输出foo {1, 2, 3}你必须做这样的事情：

string t = "1, 2, 3";
string v = String.Format(" foo {{{0}}}", t);

要输出一个{你使用{{和输出一个}你使用}}。

或者现在，您还可以像这样使用 C# 字符串插值（C# 6.0 中提供的功能）

转义括号：字符串插值 $("") 。这是 C# 6.0 中的新特性。

var inVal = "1, 2, 3";
var outVal = $" foo {{{inVal}}}";
// The output will be:  foo {1, 2, 3}


更多建议

是的，要输出{你string.Format必须像这样转义它：{{

所以下面会输出"foo {1,2,3}".

String val = "1,2,3";
String.Format(" foo {{{0}}}", val);

但是 你必须知道 C# 中的一个设计错误，即通过执行上述逻辑你会假设下面的代码将打印 {24.00}：

int i = 24;
string str = String.Format("{{{0:N}}}", i); // Gives '{N}' instead of {24.00}

但这会打印 {N}。这是因为 C# 解析转义序列和格式字符的方式。要在上述情况下获得所需的值，您必须改用它：

String.Format("{0}{1:N}{2}", "{", i, "}") // Evaluates to {24.00}

如何根据对象的属性对对象列表进行排序？

Arya

2023-02-17 05:12

我有一个 Python 对象列表，我想按每个对象的特定属性对其进行排序：

>>> ut
[Tag(name="toe", count=10), Tag(name="leg", count=2), ...]

.count如何按降序对列表进行排序？



解答

# To sort the list in place...
ut.sort(key=lambda x: x.count, reverse=True)

# To return a new list, use the sorted() built-in function...
newlist = sorted(ut, key=lambda x: x.count, reverse=True)

有关按键排序的更多信息。


更多建议

一种最快的方法是使用operator.attrgetter("count"). 但是，这可能会在 Python 的预运算符版本上运行，因此最好有一个回退机制。那么您可能想要执行以下操作：

try: import operator
except ImportError: keyfun= lambda x: x.count # use a lambda if no operator module
else: keyfun= operator.attrgetter("count") # use operator since it's faster than lambda

ut.sort(key=keyfun, reverse=True) # sort in-place
