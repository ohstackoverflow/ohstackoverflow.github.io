如何检查 Ruby 中的数组中是否存在一个值

Arya

2023-02-17 01:54

我有一个值'Dog'和一个数组['Cat', 'Dog', 'Bird']。

如何在不循环遍历的情况下检查它是否存在于数组中？有没有一种简单的方法来检查值是否存在，仅此而已？



解答

您正在寻找include?：

>> ['Cat', 'Dog', 'Bird'].include? 'Dog'
=> true


更多建议

正如@campeterson 所指出的，自 v3.1 以来（Rails 的一部分）中有一种in?方法。ActiveSupport所以在 Rails 中，或者如果你require 'active_support'，你可以写：

'Unicorn'.in?(['Cat', 'Dog', 'Bird']) # => false

OTOH，Ruby 本身没有in运算符或#in?方法，即使之前已经提出过，特别是 Yusuke Endoh是 ruby​​-core 的顶级成员。

正如其他人所指出的，对于包括, , ,在内的所有sinclude?都存在反向方法：Enumerable``Array``Hash``Set``Range

['Cat', 'Dog', 'Bird'].include?('Unicorn') # => false

请注意，如果您的数组中有许多值，它们将一个接一个地检查（即O(n)），而哈希查找将是恒定时间（即O(1)）。因此，例如，如果您的数组是常量，则最好改用Set。例如：

require 'set'
ALLOWED_METHODS = Set[:to_s, :to_i, :upcase, :downcase
                       # etc
                     ]

def foo(what)
  raise "Not allowed" unless ALLOWED_METHODS.include?(what.to_sym)
  bar.send(what)
end

快速测试显示调用include?10 元素Set比调用等效元素Array（如果未找到该元素）快 3.5 倍。

include?最后的结束语：在 a 上使用时要小心Range，有一些微妙之处，所以请参考文档并与cover?...进行比较

如何检查目录是否存在于 Python 中？

Arya

2023-02-17 01:52

如何检查目录是否存在？



解答

仅用于 os.path.isdir 目录：

>>> import os
>>> os.path.isdir('new_folder')
True

用于 os.path.exists 文件和目录：

>>> import os
>>> os.path.exists(os.path.join(os.getcwd(), 'new_folder', 'file.txt'))
False

或者，您可以使用 pathlib ：

 >>> from pathlib import Path
 >>> Path('new_folder').is_dir()
 True
 >>> (Path.cwd() / 'new_folder' / 'file.txt').exists()
 False


更多建议

Python 3.4 将该模块引入标准pathlib库，它提供了一种面向对象的方法来处理文件系统路径。对象的和方法可以用来回答这个问题is_dir()：exists()``Path

In [1]: from pathlib import Path

In [2]: p = Path('/usr')

In [3]: p.exists()
Out[3]: True

In [4]: p.is_dir()
Out[4]: True

路径（和字符串）可以与运算符连接在一起/：

In [5]: q = p / 'bin' / 'vim'

In [6]: q
Out[6]: PosixPath('/usr/bin/vim')

In [7]: q.exists()
Out[7]: True

In [8]: q.is_dir()
Out[8]: False

Pathlib 也可以通过PyPi 上的 pathlib2 模块在 Python 2.7 上使用。

如何使用 jQuery 按名称选择元素？

Arya

2023-02-17 01:52

我有一个要扩展和隐藏的表列。当我通过而不是通过元素的 .jQuery<td>选择它时，jQuery 似乎隐藏了元素。class``name

例如：

$(".bold").hide(); // Selecting by class works.
$("tcol1").hide(); // Selecting by name does not work.

请注意下面的 HTML。name第二列对所有行都相同。我如何使用该name属性创建此集合？

<tr>
  <td>data1</td>
  <td name="tcol1" class="bold"> data2</td>
</tr>
<tr>
  <td>data1</td>
  <td name="tcol1" class="bold"> data2</td>
</tr>
<tr>
  <td>data1</td>
  <td name="tcol1" class="bold"> data2</td>
</tr>



解答

您可以使用jQuery 属性选择器：

$('td[name="tcol1"]')   // Matches exactly 'tcol1'
$('td[name^="tcol"]' )  // Matches those that begin with 'tcol'
$('td[name$="tcol"]' )  // Matches those that end with 'tcol'
$('td[name*="tcol"]' )  // Matches those that contain 'tcol'


更多建议

可以使用方式选择任何属性[attribute_name=value]。在此处查看示例：

var value = $("[name='nameofobject']");
