为网站开发禁用 Chrome 缓存

Arya

我正在修改网站的外观（CSS 修改），但由于烦人的持久缓存而无法在 Chrome 上看到结果。我试过Shift+refresh 但它不起作用。

如何暂时禁用缓存或以某种方式刷新页面以查看更改？



解答

Chrome DevTools 可以禁用缓存。

  1. 右键单击 并选择Inspect Element打开 DevTools。或使用以下键盘快捷键之一：

  2. F12

  3. Control+ Shift+i

  4. Command+ Shift+i

  5. 单击 Network工具栏中的 以打开网络窗格。

  6. 选中Disable cache顶部的复选框。

请记住，正如来自@ChromiumDev 的推文所述，此设置 仅在开发工具打开时有效 。 ****

请注意，这将导致重新加载 所有资源。 如果您只想为某些资源禁用缓存，您可以修改服务器与文件一起发送的HTTP 标头。

如果你不想使用复选框， 在 DevTools 打开的Disable cache情况下长按刷新按钮将显示一个菜单，其中包含选项或应该具有类似效果。阅读选项之间的区别以了解选择哪个选项。可以使用以下快捷方式： ****Hard Reload``Empty Cache and Hard Reload

  • Command+ Shift+R在 Mac 上
  • Control+ Shift+R在 Windows 或 Linux 上


更多建议

制作一个带有参数的 Bash 别名？

Arya

2023-02-17 00:35

我曾经使用 CShell (csh)，它可以让你创建一个带有参数的别名。符号是这样的

alias junk="mv \\!* ~/.Trash"

在 Bash 中，这似乎不起作用。鉴于 Bash 有许多有用的特性，我认为这个已经实现了，但我想知道如何实现。



解答

Bash 别名不直接接受参数。您将必须创建一个函数。

alias不接受参数，但可以像别名一样调用函数。例如：

myfunction() {
    #do things with parameters like $1 such as
    mv "$1" "$1.bak"
    cp "$2" "$1"
}


myfunction old.conf new.conf #calls `myfunction`

顺便说一下，在您的.bashrc文件和其他文件中定义的 Bash 函数在您的 shell 中可以作为命令使用。因此，例如，您可以像这样调用较早的函数

$ myfunction original.conf my.conf


更多建议

细化上面的答案，您可以获得像别名一样的 1 行语法，这对于 shell 或 .bashrc 文件中的临时定义更方便：

bash$ myfunction() { mv "$1" "$1.bak" && cp -i "$2" "$1"; }

bash$ myfunction original.conf my.conf

不要忘记右括号前的分号。同样，对于实际问题：

csh% alias junk="mv \\!* ~/.Trash"

bash$ junk() { mv "$@" ~/.Trash/; }

或者：

bash$ junk() { for item in "$@" ; do echo "Trashing: $item" ; mv "$item" ~/.Trash/; done; }
