GUID 是 100% 唯一的吗？

Arya

GUID 是 100% 唯一的吗？

它会在多个线程中保持唯一性吗？



解答

虽然不能保证每个生成的 GUID 都是唯一的，但唯一键的总数（2 128或 3.4×10 38）是如此之大，以至于相同的数字被生成两次的概率非常小。例如，考虑包含大约 5×10 22颗恒星的可观测宇宙；每颗星都可以有 6.8×10 15 个普遍唯一的 GUID。

来自维基百科。

获取实现接口的所有类型

使用反射，我怎样才能用最少的代码获得所有使用 C# 3.0/.NET 3.5 实现接口的类型，并最大限度地减少迭代次数？

这就是我想重写的：

foreach (Type t in this.GetType().Assembly.GetTypes())
    if (t is IMyInterface)
        ; //do stuff



解答

我的在 c# 3.0 中就是这个 :)

var type = typeof(IMyInterface);
var types = AppDomain.CurrentDomain.GetAssemblies()
    .SelectMany(s => s.GetTypes())
    .Where(p => type.IsAssignableFrom(p));

基本上，最少的迭代次数总是：

loop assemblies  
 loop types  
  see if implemented.


更多建议

这对我有用。它遍历类并检查它们是否派生自 myInterface

 foreach (Type mytype in System.Reflection.Assembly.GetExecutingAssembly().GetTypes()
                 .Where(mytype => mytype .GetInterfaces().Contains(typeof(myInterface)))) {
    //do stuff
 }

如何将 cURL 输出捕获到文件？

Arya

2023-02-17 20:46

我有一个包含一堆这种格式的 URL 的文本文档：

URL = "sitehere.com"

我想要做的是运行curl -K myfile.txt，并将响应 cURL 返回的输出获取到一个文件中。

我怎样才能做到这一点？



解答

curl -K myconfig.txt -o output.txt

将收到的 第一个 输出写入您指定的文件中（如果存在旧文件则覆盖）。

curl -K myconfig.txt >> output.txt

将您收到的所有输出附加到指定文件。

注意：-K 是可选的。


更多建议

对于单个文件，您可以使用-O而不是-o filename使用 URL 路径的最后一段作为文件名。例子：

curl http://example.com/folder/big-file.iso -O

会将结果保存到当前文件夹中名为 big-file.iso 的新文件中。通过这种方式，它的工作方式类似于wget，但允许您指定使用 wget 时不可用的其他curl 选项。
