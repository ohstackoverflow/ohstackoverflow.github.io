如何垂直对齐div中的元素？

Arya

2023-02-17 07:19

我有一个div有两个图像和一个h1. 所有这些都需要在 div 中垂直对齐，彼此相邻。其中一张图片需要放置absolute在div.

在所有常见浏览器上工作需要什么 CSS？

<div id="header">
  <img src=".." ></img>
  <h1>testing...</h1>
  <img src="..."></img>
</div>



解答

哇，这个问题很流行。这是基于对财产的误解vertical-align。这篇优秀的文章对此进行了解释：

vertical-alignGavin Kistner 的理解或“如何（不）垂直居中内容” 。

“How to center in CSS” 是一个很棒的网络工具，可以帮助找到不同情况下必要的 CSS 居中属性。

简而言之（并防止链接失效）：

  • 内联元素 （并且 只有 内联元素）可以通过vertical-align: middle. 但是，“上下文”不是整个父容器的高度，而是它们所在的文本行的高度。jsfiddle示例
  • 对于块元素，垂直对齐更难并且在很大程度上取决于具体情况：
    • 如果内部元素可以有一个 固定的 height ，你可以设置它的 positionabsolute并指定它的height,margin-top和topposition。jsfiddle 示例
    • 如果居中元素 由单行组成 并且 其父高度是固定的， 您可以简单地设置容器的line-height高度来填充它的高度。根据我的经验，这种方法非常通用。jsfiddle 示例
    • ......还有更多这样的特殊情况。


更多建议

现在 Flexbox 支持越来越多，这个应用于包含元素的 CSS 将垂直居中所有包含的项目（除了那些自己指定对齐方式的项目，例如align- self:start）

.container {
    display: flex;
    align-items: center;
}

如果您还需要针对Internet Explorer 10和更旧的（< 4.4 (KitKat)）Android 浏览器，请使用带前缀的版本：

.container {
    display: -ms-flexbox;
    display: -webkit-flex;
    display: flex;

    -ms-flex-align: center;
    -webkit-align-items: center;
    -webkit-box-align: center;
    align-items: center;
}

如何查看哪些 Git 分支正在跟踪哪个远程/上游分支？

Arya

2023-02-17 07:18

我知道我能做到git branch --all，这向我展示了本地和远程分支机构，但它在向我展示它们之间的关系方面没有多大用处。

如何以显示哪个本地分支正在跟踪哪个远程分支的方式列出分支？



解答

非常瓷器命令，如果你想用它来编写脚本就不好了：

git branch -vv   # doubly verbose!

请注意，对于 git 1.8.3，上游分支显示为 蓝色 （请参阅“什么是 git 中的分支跟踪（如果有的话）？ ”）

如果您想要干净的输出，请参阅Carl Suster 的回答- 它使用了一个瓷器命令，我认为在我最初写这个答案时不存在，所以它更简洁一些并且适用于为变基配置的分支，而不仅仅是合并。


更多建议

git remote show origin

将“来源”替换为您的遥控器的任何名称。

如何从数组中获取前 N 个元素

Arya

2023-02-17 07:18

我正在使用 Javascript(ES6) /FaceBook 做出反应并尝试获取大小不同的数组的前 3 个元素。我想做相当于 Linq take(n) 的事情。

在我的 Jsx 文件中，我有以下内容：

var items = list.map(i => {
  return (
    <myview item={i} key={i.id} />
  );
});

然后得到我试过的前 3 个项目

  var map = new Map(list);
    map.size = 3;
    var items = map(i => {
      return (<SpotlightLandingGlobalInboxItem item={i} key={i.id} />);
    });

这不起作用，因为地图没有设置功能。

你能帮忙吗？



解答

我相信你要找的是：

// ...inside the render() function

var size = 3;
var items = list.slice(0, size).map(i => {
    return <myview item={i} key={i.id} />
});                       
return (
  <div>
    {items}
  </div>   
)
