在 Python 中解压缩文件

Arya

2023-02-17 13:26

我通读了zipfile文档，但无法理解如何 解压缩 文件，只能理解如何压缩文件。如何将 zip 文件的所有内容解压缩到同一目录中？



解答

import zipfile
with zipfile.ZipFile(path_to_zip_file, 'r') as zip_ref:
    zip_ref.extractall(directory_to_extract_to)

差不多就是这样！


更多建议

如果您使用的是 Python 3.2 或更高版本：

import zipfile
with zipfile.ZipFile("file.zip","r") as zip_ref:
    zip_ref.extractall("targetdir")

您不需要对此使用 关闭 或 尝试/捕获，因为它使用 上下文管理器构造。

如何将小数值显示为小数点后两位？

Arya

2023-02-17 13:25

当前用 显示小数的值时.ToString()，精确到小数点后 15 位，因为我用它来表示美元和美分，所以我只希望输出为小数点后 2 位。

.ToString()我为此使用变体吗？



解答

decimalVar.ToString("#.##"); // returns ".5" when decimalVar == 0.5m

或者

decimalVar.ToString("0.##"); // returns "0.5"  when decimalVar == 0.5m

或者

decimalVar.ToString("0.00"); // returns "0.50"  when decimalVar == 0.5m


更多建议

我知道这是一个老问题，但令我惊讶的是似乎没有人发布答案；

  1. 没有使用银行四舍五入
  2. 将值保留为小数。

这就是我要使用的：

decimal.Round(yourValue, 2, MidpointRounding.AwayFromZero);

wget命令下载文件并另存为不同的文件名

Arya

2023-02-17 13:25

我正在使用命令下载文件wget。但是当它下载到我的本地机器时，我希望它被保存为一个不同的文件名。

例如：我正在从www.examplesite.com/textfile.txt

我想使用wget将文件保存textfile.txt在我的本地目录中作为newfile.txt. 我正在使用wget如下命令：

wget www.examplesite.com/textfile.txt



解答

使用该-O file选项。

例如

wget google.com
...
16:07:52 (538.47 MB/s) - `index.html' saved [10728]

对比

wget -O foo.html google.com
...
16:08:00 (1.57 MB/s) - `foo.html' saved [10728]


更多建议

还要注意命令行中参数的顺序。至少在某些系统上（例如 CentOS 6）：

wget -O FILE URL

作品。但：

wget URL -O FILE

不起作用。
