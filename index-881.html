拉伸和缩放 CSS 背景

Arya

2023-02-17 18:40

有没有办法让 CSS 中的背景拉伸或缩放以填充其容器？



解答

对于现代浏览器，您可以使用以下方法完成此操作background-size：

body {
    background-image: url(bg.jpg);
    background-size: cover;
}

cover意味着垂直或水平拉伸图像，使其永远不会平铺/重复。

这适用于 Safari 3（或更高版本）、Chrome、Opera 10+、Firefox 3.6+ 和 Internet Explorer 9（或更高版本）。

要使其与较低版本的 Internet Explorer 一起使用，请尝试以下 CSS：

filter: progid:DXImageTransform.Microsoft.AlphaImageLoader(src='.myBackground.jpg', sizingMethod='scale');
-ms-filter: "progid:DXImageTransform.Microsoft.AlphaImageLoader(src='myBackground.jpg', sizingMethod='scale')";

TypeScript 中的“不可分配给永不类型的参数”错误是什么？

Arya

2023-02-17 18:39

代码是：

const foo = (foo: string) => {
  const result = []
  result.push(foo)
}

我收到以下 TS 错误：

[ts]“字符串”类型的参数不可分配给“从不”类型的参数。

我究竟做错了什么？这是一个错误吗？



解答

您所要做的就是将您的定义result为字符串数组，如下所示：

const result : string[] = [];

在不定义数组类型的情况下，它默认为never. 所以当你试图向它添加一个字符串时，它是一个类型不匹配，所以它抛出了你看到的错误。


更多建议

另一种方法是：

const result: any[] = [];

在 R 中的同一图中绘制两个图

Arya

2023-02-17 18:39

我想在同一个图中绘制 y1 和 y2。

x  <- seq(-2, 2, 0.05)
y1 <- pnorm(x)
y2 <- pnorm(x, 1, 1)
plot(x, y1, type = "l", col = "red")
plot(x, y2, type = "l", col = "green")

但是当我这样做时，它们并没有一起绘制在同一个情节中。

在 Matlab 中可以做到hold on，但是有人知道如何在 R 中做到这一点吗？



解答

lines()或points()将添加到现有图形，但不会创建新窗口。所以你需要做

plot(x,y1,type="l",col="red")
lines(x,y2,col="green")


更多建议

par您还可以在同一图表但不同的轴上使用和绘制。内容如下：

plot( x, y1, type="l", col="red" )
par(new=TRUE)
plot( x, y2, type="l", col="green" )

如果您详细阅读par中的内容R，您将能够生成非常有趣的图表。另一本书是 Paul Murrel 的 R Graphics。
