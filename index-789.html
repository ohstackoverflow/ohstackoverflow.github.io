过滤字典以仅包含某些键？

Arya

2023-02-17 16:11

我有一个dict有一大堆条目的。我只对其中的少数几个感兴趣。有没有一种简单的方法可以修剪掉所有其他的？



解答

构建一个新的字典：

dict_you_want = {key: old_dict[key] for key in your_keys}

使用词典理解。

如果您使用缺少它们的版本（即 Python 2.6 和更早版本），请将其设为dict((key, old_dict[key]) for ...). 它是一样的，虽然更丑陋。

请注意，这与 jnnnnn 的版本不同，对于任何大小的 s 都具有稳定的性能（仅取决于 your_keys 的数量）old_dict。无论是速度还是内存。由于这是一个生成器表达式，它一次处理一个项目，它不会遍历 old_dict 的所有项目。

就地删除所有内容：

unwanted = set(keys) - set(your_dict)
for unwanted_key in unwanted: del your_dict[unwanted_key]


更多建议

稍微更优雅的字典理解：

foodict = {k: v for k, v in mydict.items() if k.startswith('foo')}

如何让我的程序休眠 50 毫秒？

Arya

2023-02-17 16:09

如何让我的 Python 程序休眠 50 毫秒？



解答

使用time.sleep()

from time import sleep
sleep(0.05)


更多建议

请注意，如果您依赖 sleep 正好 花费50 毫秒，您将无法得到。它只是关于它。
