如何使用 jQuery 获取元素的 ID？

Arya

2023-02-17 01:02 

<div id="test"></div>
<script>
  $(document).ready(function() {
    alert($('#test').id);
  });  
</script>

为什么上面的方法不起作用，我应该怎么做？



解答

jQuery 方式：

$('#test').attr('id')

在你的例子中：

$(document).ready(function() {
  console.log($('#test').attr('id'));
});


<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
<div id="test"></div>

或者通过 DOM：

$('#test').get(0).id;

甚至 ：

$('#test')[0].id;

在 JQuery 中使用的背后的原因$('#test').get(0)甚至$('#test')[0]是它$('#test')是一个 JQuery 选择器并 返回一个结果数组（） 而不是其默认功能的单个元素

jquery 中 DOM 选择器的替代方法是

$('#test').prop('id')

这不同于.attr()并$('#test').prop('foo')获取指定的 DOMfoo属性，同时$('#test').attr('foo')获取指定的 HTML属性，您可以在此处foo找到有关差异的更多详细信息。


更多建议

$('selector').attr('id')将返回第一个匹配元素的 id。参考。

如果您的匹配集包含多个元素，您可以使用传统的.each 迭代器返回一个包含每个 ID 的数组：

var retval = []
$('selector').each(function(){
  retval.push($(this).attr('id'))
})
return retval

或者，如果你愿意变得更坚韧一点，你可以避免包装器并使用.map 快捷方式。

return $('.selector').map(function(index,dom){return dom.id})

如何强制刷新网站图标？

Arya

2023-02-17 01:01

我有一个使用自己的 tomcat 在本地运行的 Grails 应用程序，我刚刚更改了一个新的图标。问题是我无法在任何浏览器中看到它。旧的图标出现了，或者我根本没有得到任何图标，但没有看到我的新图标。我不认为这本身是一个 Grails 问题，更多的是网站图标的问题。

网站图标应该发生什么？他们应该如何工作？我的浏览器中有很多书签，它们的图标都是错误的，而且它们似乎永远不会刷新。我如何强制服务器/浏览器停止缓存它们？鉴于它们通常只有 16x16，总是缓存它们似乎很愚蠢。为什么不在每次访问该页面时上传它们？这并不是一个巨大的开销。



解答

要刷新站点的图标，您可以强制浏览器使用链接标记和文件名上的查询字符串下载新版本。
这在生产环境中特别有用，可确保您的用户获得更新。

<link rel="icon" href="http://www.yoursite.com/favicon.ico?v=2" />


更多建议

改编自lineofbird 的回答 below，您可以执行以下操作：

  1. 通过输入地址直接转到地址栏中的图标 url， 例如

如何从 Git 存储库中删除 .DS_Store 文件？

Arya

2023-02-17 01:00

如何.DS_Store从 Git 存储库中删除那些恼人的 Mac OS X 文件？



解答

.DS_Store从存储库中删除现有文件：

find . -name .DS_Store -print0 | xargs -0 git rm -f --ignore-unmatch

添加此行：

.DS_Store

到文件.gitignore，可以在存储库的顶层找到（如果文件不存在，则创建该文件）。您可以使用顶级目录中的以下命令轻松完成此操作：

echo .DS_Store >> .gitignore

然后将文件提交到仓库：

git add .gitignore
git commit -m '.DS_Store banished!'


更多建议

结合 benzado 和 webmat 的答案，使用 更新git rm，发现不在 repo 中的文件不会失败，并使其可以为任何用户通用粘贴：

# remove any existing files from the repo, skipping over ones not in repo
find . -name .DS_Store -print0 | xargs -0 git rm --ignore-unmatch
# specify a global exclusion list
git config --global core.excludesfile ~/.gitignore
# adding .DS_Store to that list
echo .DS_Store >> ~/.gitignore
