如何向数组追加一些东西？

Arya

2023-02-16 20:28

这个问题的答案是 社区的努力。编辑现有答案以改进这篇文章。它目前不接受新的答案或互动。

如何在 JavaScript 中将对象（例如字符串或数字）附加到数组？



解答

使用该Array.prototype.push方法将值追加到数组的末尾：

// initialize array
var arr = [
  "Hi",
  "Hello",
  "Bonjour"
];

// append new value to the array
arr.push("Hola");

console.log(arr);

您可以使用该push()函数在一次调用中将多个值附加到数组：

// initialize array
var arr = ["Hi", "Hello", "Bonjour", "Hola"];

// append multiple values to the array
arr.push("Salut", "Hey");

// display all values
for (var i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}

更新

如果你想将一个数组的项目添加到另一个数组，你可以使用firstArray.concat(secondArray)：

var arr = [
  "apple",
  "banana",
  "cherry"
];

// Do not forget to assign the result as, unlike push, concat does not change the existing array
arr = arr.concat([
  "dragonfruit",
  "elderberry",
  "fig"
]);

console.log(arr);

更新

如果您想将任何值添加到数组的开头（即第一个索引），那么您可以将其用于Array.prototype.unshift此目的。

var arr = [1, 2, 3];
arr.unshift(0);
console.log(arr);

它还支持一次附加多个值，就像push.

更新

ES6 语法的另一种方法是使用 _ 扩展语法_ 返回一个新数组。这使原始数组保持不变，但返回一个附加了新项目的新数组，符合函数式编程的精神。

const arr = [
  "Hi",
  "Hello",
  "Bonjour",
];

const newArr = [
  ...arr,
  "Salut",
];

console.log(newArr);


更多建议

如果您只附加一个变量，那么就push()可以正常工作。如果您需要附加另一个数组，请使用concat()：

var ar1 = [1, 2, 3];
var ar2 = [4, 5, 6];

var ar3 = ar1.concat(ar2);

alert(ar1);
alert(ar2);
alert(ar3);

concat 不会影响ar1并且ar2除非重新分配，例如：

var ar1 = [1, 2, 3];
var ar2 = [4, 5, 6];

ar1 = ar1.concat(ar2);
alert(ar1);

JavaScript Reference 上有很多重要信息。

将字符串“Jun 1 2005 1:33PM”转换为日期时间

Arya

2023-02-16 20:26

如何将以下字符串转换为datetime对象？

"Jun 1 2005  1:33PM"



解答

datetime.strptime将用户指定格式的输入字符串解析为 timezone-naive datetime对象：

>>> from datetime import datetime
>>> datetime.strptime('Jun 1 2005  1:33PM', '%b %d %Y %I:%M%p')
datetime.datetime(2005, 6, 1, 13, 33)

要date使用现有对象获取对象datetime，请使用以下方法转换它.date()：

>>> datetime.strptime('Jun 1 2005', '%b %d %Y').date()
date(2005, 6, 1)

链接：

  • strptime文档：Python 2，Python 3

  • strptime/strftime格式化字符串文档：Python 2、Python 3

  • strftime.org格式字符串速查表

笔记：

  • strptime= "字符串解析时间"
  • strftime= "字符串格式化时间"


更多建议

使用第三方dateutil库：

from dateutil import parser
parser.parse("Aug 28 1999 12:00AM")  # datetime.datetime(1999, 8, 28, 0, 0)

strptime它可以处理大多数日期格式并且比它通常猜测正确的格式更方便。它对于编写测试也非常有用，因为可读性比性能更重要。

安装它：

pip install python-dateutil

如何将 Git 存储库克隆到特定文件夹中？

Arya

2023-02-16 20:25

该命令git clone git@github.com:whatever创建一个名为的目录whatever，其中包含一个 Git 存储库：

./
    whatever/
        .git

我想将 Git 存储库的内容克隆到我当前的目录./中：

./
    .git



解答

选项A：

git clone git@github.com:whatever folder-name

因此，right here使用：

git clone git@github.com:whatever .

选项B：

.git也移动文件夹。请注意，该.git文件夹在大多数图形文件资源管理器中都是隐藏的，因此请务必显示隐藏文件。

mv /where/it/is/right/now/* /where/I/want/it/
mv /where/it/is/right/now/.* /where/I/want/it/

第一行抓取所有普通文件，第二行抓取点文件。shopt -s dotglob也可以通过启用 dotglob（即）在一行中完成此操作，但如果您要问这个答案回答的问题，那可能是一个不好的解决方案。

更好的是：

将您的工作副本保存在其他地方，并创建一个符号链接。像这样：

ln -s /where/it/is/right/now /the/path/I/want/to/use

对于您的情况，这将类似于：

ln -sfn /opt/projectA/prod/public /httpdocs/public

如果您需要，可以轻松更改以测试它，即：

ln -sfn /opt/projectA/test/public /httpdocs/public

无需移动文件。添加-fn以防有人复制这些行（-f强制，-n避免与已经存在和不存在的链接进行一些通常不需要的交互）。

如果您只是想让它工作，请使用选项 A，如果其他人要查看您所做的，请使用选项 C。


更多建议

我想很多人问这个问题的例子就是这个。如果您位于要将 git 存储库的内容转储到的目录 中，请运行：

git clone git@github.com:whatever .

这 ”。” 最后将当前文件夹指定为结帐文件夹。
