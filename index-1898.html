How to auto-format code in Eclipse?

Arya

2023-03-03 12:02

How do you auto-format code in Eclipse?



Answer

On Windows and Linux : Ctrl + Shift + F

On Mac : ⌘ + ⇧ + F

(Alternatively you can press Format in Main Menu > Source)


Suggest

Another option is to go to Window- >Preferences->Java->Editor->SaveActions and check the Format source code option. Then your source code will be formatted truly automatically each time you save it.

Replacing spaces with underscores in JavaScript?

Arya

2023-03-03 12:01

I'm trying to use this code to replace spaces with _, it works for the first space in the string but all the other instances of spaces remain unchanged. Anybody know why?

function updateKey()
{
    var key=$("#title").val();
    key=key.replace(" ","_");
    $("#url_key").val(key);
}



Answer

Try .replace(/ /g,"_");

Edit : or .split(' ').join('_') if you have an aversion to REs

Edit : John Resig said:

If you're searching and replacing through a string with a static search and a static replace it's faster to perform the action with .split("match").join("replace") - which seems counter-intuitive but it manages to work that way in most modern browsers. (There are changes going in place to grossly improve the performance of .replace(/match/g, "replace") in the next version of Firefox - so the previous statement won't be the case for long.)


Suggest

try this:

key=key.replace(/ /g,"_");

that'll do a global find/replace

How can I change the current running project to another project in GCP ( Google Cloud Platform ) account using cli commands other than using gcloud init manually?

gcloud projects list will list the projects running on my account. I want to change the current project to any other project from the list using a cli command.



Answer
