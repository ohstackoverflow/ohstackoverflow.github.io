如何访问我的 SSH 公钥？

Arya

2023-02-17 10:37

我刚刚生成了我的 RSA 密钥对，我想将该密钥添加到 GitHub。

我试过cd id_rsa.pub了id_rsa.pub，但没有运气。如何访问我的 SSH 公钥？



解答

cat ~/.ssh/id_rsa.pub或者cat ~/.ssh/id_dsa.pub

您可以通过以下方式列出您拥有的所有公钥：

$ ls ~/.ssh/*.pub


更多建议

将密钥复制到剪贴板。

$ pbcopy < ~/.ssh/id_rsa.pub
# Copies the contents of the id_rsa.pub file to your clipboard

警告：在不添加换行符或空格的情况下准确复制密钥很重要。值得庆幸的是，pbcopy 命令可以轻松完美地执行此设置。

并将其粘贴到您需要的任何位置。

有关该过程的更多详细信息，请查看： Generating SSH Keys 。

如何注释掉 XML 中的标签块？

Arya

2023-02-17 10:36

如何注释掉 XML 中的标签块？

即如何在下面的代码中注释掉<staticText>其中的所有内容？

  <detail>
    <band height="20">
      <staticText>
        <reportElement x="180" y="0" width="200" height="20"/>
        <text><![CDATA[Hello World!]]></text>
      </staticText>
    </band>
  </detail>

我可以使用<!-- staticText-->，但这仅适用于单个标记（据我所知），就像//在 Java 和 C 中一样。我想要更像如何/** comment **/在 Java 和 C 中使用的东西，这样我就可以注释掉更长的 XML 代码块。



解答

您可以在多行中使用该注释样式（HTML 中也存在）

<detail>
    <band height="20">
    <!--
      Hello,
         I am a multi-line XML comment
         <staticText>
            <reportElement x="180" y="0" width="200" height="20"/>
            <text><![CDATA[Hello World!]]></text>
          </staticText>
      -->
     </band>
</detail>


更多建议

您可以使用不存在的处理指令包装文本，例如：

<detail>
<?ignore
  <band height="20">
    <staticText>
      <reportElement x="180" y="0" width="200" height="20"/>
      <text><![CDATA[Hello World!]]></text>
    </staticText>
  </band>
?>
</detail>

不允许嵌套处理指令，'?>' 结束处理指令（参见http://www.w3.org/TR/REC-xml/#sec-pi）

如何在 PHP 中将字符串转换为数字？

Arya

2023-02-17 10:36

我想将这些类型的值、'3'、'2.34'、'0.234343'等转换为数字。在 JavaScript 中我们可以使用Number()，但是在 PHP 中有没有类似的方法可用？

Input             Output
'2'               2
'2.34'            2.34
'0.3454545'       0.3454545



解答

您通常不需要这样做，因为在大多数情况下 PHP 会为您强制转换类型。对于确实要显式转换类型的情况， 请强制转换 它：

$num = "3.14";
$int = (int)$num;
$float = (float)$num;


更多建议

有几种方法可以做到这一点：

  1. 将字符串转换为数字原始数据类型：

    $num = (int) "10";

    $num = (double) "10.12"; // same as (float) "10.12";

  2. 对字符串执行数学运算：

    $num = "10" + 1;

    $num = floor("10.1");

  3. 使用intval()或floatval()：

    $num = intval("10");

    $num = floatval("10.1");

  4. 使用settype()。
