如何在 jQuery 中更改按钮的文本？

Arya

2023-02-18 00:03

你如何在 jQuery 中更改按钮的文本值？目前，我的按钮的文本值为“添加”，单击后我希望它更改为“保存”。我在下面尝试过这种方法，但到目前为止没有成功：

$("#btnAddProfile").attr('value', 'Save');



解答

取决于您使用的按钮类型

<input type='button' value='Add' id='btnAddProfile'>
$("#btnAddProfile").attr('value', 'Save'); //versions older than 1.6

<input type='button' value='Add' id='btnAddProfile'>
$("#btnAddProfile").prop('value', 'Save'); //versions newer than 1.6

<!-- Different button types-->

<button id='btnAddProfile' type='button'>Add</button>
$("#btnAddProfile").html('Save');

您的按钮也可以是一个链接。您需要发布一些 HTML 以获得更具体的答案。

编辑 ：当然，假设您已将其包装在.click()电话中，这些将起作用

编辑 2 ：较新的 jQuery 版本（从 > 1.6）使用.prop而不是.attr

编辑 3 ：如果您使用的是 jQuery UI，则需要使用 DaveUK 的调整文本属性的方法（如下）


更多建议

对于在 jQuery 中使用 .Button() 创建的按钮......

虽然其他答案会更改文本，但它们会弄乱按钮的样式，但事实证明，当呈现 jQuery 按钮时，按钮的文本嵌套在一个范围内，例如

<button id="thebutton">
  <span class="ui-button-text">My Text</span>
</button>

如果您删除 span 并将其替换为文本（如其他示例中所示）- 您将失去 span 和关联的格式。

所以你实际上需要更改 SPAN 标记中的文本而不是 BUTTON！

$("#thebutton span").text("My NEW Text");

或者（如果像我一样，它是在点击事件上完成的）

$("span", this).text("My NEW Text");

如何将 docker 镜像推送到私有仓库

Arya

2023-02-18 00:02

我有一个标记为 的 docker 镜像me/my-image，并且我在 dockerhub 上有一个名为 的私人仓库me-private。
当我推动我的时me/my-image，我最终总是访问公共回购。

专门将我的图像推送到我的私人仓库的确切语法是什么？



解答

您需要首先使用以下内容正确标记您的图像registryhost：

docker tag [OPTIONS] IMAGE[:TAG] [REGISTRYHOST/][USERNAME/]NAME[:TAG]

然后 docker push 使用相同的标签。

docker push NAME[:TAG]

例子：

docker tag 518a41981a6a myRegistry.com/myImage
docker push myRegistry.com/myImage


更多建议

只需三个简单的步骤：

  1. docker login --username username

    • 如果您省略，则提示输入密码--password，这是推荐的，因为它不会将其存储在您的命令历史记录中
    • docker tag my-image username/my-repo

  2. docker push username/my-repo

在 Python 中获取当前脚本的名称

Arya

2023-02-18 00:02

我正在尝试获取当前正在运行的 Python 脚本的名称。

我有一个名为的脚本foo.py，我想做这样的事情来获取脚本名称：

print(Scriptname)



解答

您可以使用__file__获取当前文件的名称。在主模块中使用时，这是最初调用的脚本的名称。

如果您想省略目录部分（可能存在），您可以使用os.path.basename(__file__).


更多建议

import sys
print(sys.argv[0])

这将打印foo.pyfor python foo.py、dir/foo.pyforpython dir/foo.py等。这是 的第一个参数python。（请注意，在 py2exe 之后它将是foo.exe.）
