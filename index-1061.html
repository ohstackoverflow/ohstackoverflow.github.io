如何在命令行上为差异着色

Arya

2023-02-18 00:25

当我有差异时，我怎样才能给它上色以使其看起来不错？

我想要它用于命令行，所以请不要使用 GUI 解决方案。



解答

手册页diff不建议从内部进行着色的解决方案。请考虑使用colordiff. 它是一个包装器diff，产生与 diff 相同的输出，除了它使用彩色语法突出显示来增加输出以提高可读性：

diff old new | colordiff

要不就：

colordiff old new

安装：

  • Ubuntu/Debian：sudo apt-get install colordiff
  • OS X：brew install colordiff或port install colordiff


更多建议

使用维姆：

diff /path/to/a /path/to/b | vim -R -

或者更好的是，VimDiff（或vim -d，输入更短）将并排显示两个、三个或四个文件之间的差异。

例子：

vim -d /path/to/[ab]

vimdiff file1 file2 file3 file4

给 UIView 圆角

Arya

2023-02-18 00:23

我的登录视图有一个子视图，其中有一个UIActivityView和一个UILabel说法“正在登录......”。这个子视图有不圆角的角。我怎样才能使它们变圆？

在我的 xib 中有什么办法可以做到吗？



解答

尝试这个

#import <QuartzCore/QuartzCore.h> // not necessary for 10 years now  :)

...

view.layer.cornerRadius = 5;
view.layer.masksToBounds = true;

注意：如果您尝试将圆角应用于 aUIViewController的视图，则不应在视图控制器的构造函数中应用，而应在实际实例化-viewDidLoad之后应用。view


更多建议

您还可以使用界面构建器的 用户定义的运行时属性 功能将键路径设置layer.cornerRadius为一个值。不过，请确保你包括了QuartzCore图书馆。

这个技巧也适用于设置 layer.borderWidth 但是它不会工作，layer.borderColor因为它期望CGColor不是UIColor.

您将无法在故事板中看到效果，因为这些参数是在运行时评估的。
