什么是自举？

Arya

2023-02-17 13:52

我不断看到在应用程序开发的讨论中提到“自举”。它似乎既普遍又重要，但我还没有遇到过关于什么是自举的糟糕解释；相反，似乎每个人都应该知道这意味着什么。不过我没有。据我所知，它与启动时应用程序所需的初始化任务有关，但我对此可能完全错了。谁能帮助我理解这个想法？



解答

“Bootstrapping”来自“通过自己的引导将自己拉起来”一词。你可以从维基百科得到这么多。

在计算中，引导加载程序是机器启动时运行的第一段代码，负责加载操作系统的其余部分。在现代计算机中，它存储在 ROM 中，但我记得 PDP-11 上的引导过程，您可以通过前面板开关插入位以将特定磁盘段加载到内存中，然后运行它。不用说，引导加载程序通常非常小。

“Bootstrapping”也用作使用自身构建系统的术语——或者更准确地说，是一个前身版本。例如，ANTLR版本 3 是使用在 ANTLR 版本 2 中开发的解析器编写的。


更多建议

引导的一个例子是在一些网络框架中。您调用 index.php（引导程序），然后它加载框架助手、模型、配置，然后加载控制器并将控制传递给它。

如您所见，这是一个启动大型进程的简单文件。

在 C# 中，我应该使用 string.Empty 还是 String.Empty 或 "" 来初始化一个字符串？

Arya

2023-02-17 13:52

在 C# 中，我想用一个空字符串初始化一个字符串值。

我应该怎么做？什么是正确的方法，为什么？

string willi = string.Empty;

或者

string willi = String.Empty;

或者

string willi = "";

或者是什么？



解答

使用您和您的团队认为最易读的内容。

其他答案建议每次使用时都会创建一个新字符串""。这不是真的——由于字符串驻留，它将在每个程序集或每个 AppDomain 中创建一次（或者可能在整个过程中创建一次——在这方面不确定）。这种差异可以忽略不计——非常 非常 微不足道。

但是，您发现哪个更具可读性是另一回事。这是主观的，因人而异——所以我建议你找出你团队中大多数人喜欢什么，并且所有人都喜欢它以保持一致性。我个人觉得""更容易阅读。

""和很容易被误认为彼此的论点" "并没有真正洗掉我。除非您使用的是比例字体（而且我还没有与 任何 开发人员合作过），否则很容易分辨出差异。


更多建议

从性能和代码生成的角度来看，确实没有区别。在性能测试中，他们在哪一个比另一个更快之间来回切换，而且只有几毫秒。

在查看幕后代码时，您也确实看不出任何区别。唯一的区别在于 IL，它string.Empty使用 opcodeldsfld和""使用 opcode ldstr，但这只是因为它string.Empty是静态的，并且两条指令做同样的事情。如果您查看生成的组件，它完全相同。

C#代码

private void Test1()
{
    string test1 = string.Empty;    
    string test11 = test1;
}

private void Test2()
{
    string test2 = "";    
    string test22 = test2;
}

如何使用 SVN 忽略目录？

Arya

2023-02-17 13:51

我刚开始使用 SVN，在源代码管理下我有一个不需要的缓存目录。如何使用 SVN 忽略整个目录/文件夹？

我在 OS X 和命令行上使用Versions和TextMate。



解答

设置svn:ignore父目录的属性：

svn propset svn:ignore dirname .

如果您有多个要忽略的东西，请在属性值中用换行符分隔。在这种情况下，使用外部编辑器编辑属性值会更容易：

svn propedit svn:ignore .


更多建议

这是一个示例目录结构：

\project
    \source
    \cache
    \other

当project您看到您的缓存目录未添加并显示时。

> svn status
M  source
?  cache

要设置忽略属性，请执行

svn propset svn:忽略缓存。

其中svn:ignore是您要设置的属性的名称，cache是属性的值，.是您要设置此属性的目录。cache它应该是需要该属性的目录的父目录。

要检查设置了哪些属性：

> svn proplist
Properties on '.':
  svn:ignore

查看 的值svn:ignore：

> svn propget svn:ignore
cache

要删除以前设置的属性：

svn propdel svn:ignore
