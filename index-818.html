如何删除使用 Python 的 easy_install 安装的包？

Arya

2023-02-17 16:57

蟒蛇的easy_install使得安装新包极其方便。但是，据我所知，它没有实现依赖项管理器的其他常见功能——列出和删除已安装的包。

找出已安装内容的最佳方法是什么，删除已安装软件包的首选方法是什么？rm /usr/local/lib/python2.6/dist- packages/my_installed_pkg.egg如果我手动删除软件包（例如通过或类似方式），是否有任何文件需要更新？



解答

pip是 setuptools/easy_install 的替代品，提供了一个“卸载”命令。

根据安装说明安装 pip ：

$ wget https://bootstrap.pypa.io/get-pip.py
$ python get-pip.py

然后你可以使用pip uninstall删除安装的包easy_install


更多建议

要卸载一个.egg你需要rm -rfegg（它可能是一个目录）并从中删除匹配的行site-packages/easy-install.pth

查找数组字段不为空的 MongoDB 记录

Arya

2023-02-17 16:56

我所有的记录都有一个名为“图片”的字段。该字段是一个字符串数组。

我现在想要这个数组不为空的最新 10 条记录。

我四处搜索，但奇怪的是我没有找到太多关于这个的信息。我已经阅读了 $where 选项，但我想知道它对本机函数有多慢，以及是否有更好的解决方案。

即便如此，这也行不通：

ME.find({$where: 'this.pictures.length > 0'}).sort('-created').limit(10).execFind()

什么都不返回。没有长度位的离开this.pictures确实有效，但当然它也会返回空记录。



解答

如果您还有没有密钥的文档，您可以使用：

ME.find({ pictures: { $exists: true, $not: {$size: 0} } })

$size如果涉及到MongoDB 不使用索引，那么这里有一个更好的解决方案：

ME.find({ pictures: { $exists: true, $ne: [] } })

如果您的财产可能具有无效值（例如或其他），那么您可以按照此答案中的建议null boolean添加额外的检查：$types

使用 mongo >= 3.2：

ME.find({ pictures: { $exists: true, $type: 'array', $ne: [] } })

使用 mongo < 3.2：

ME.find({ pictures: { $exists: true, $type: 4, $ne: [] } })

自 MongoDB 2.6 版本开始，您可以与 operator 进行比较$gt，但这可能会导致意想不到的结果（您可以在这个答案中找到详细解释）：

ME.find({ pictures: { $gt: [] } })


更多建议

经过更多的查看，尤其是在 mongodb 文档中，并将一些令人费解的部分放在一起，这就是答案：

ME.find({pictures: {$exists: true, $not: {$size: 0}}})

JavaScript 窗口调整大小事件

Arya

2023-02-17 16:56

如何挂接到浏览器窗口调整大小事件？

有一种 jQuery 方法可以侦听调整大小事件，但我不希望仅出于这一要求将其引入我的项目中。



解答

最佳做法是添加到调整大小事件，而不是替换它：

window.addEventListener('resize', function(event) {
    ...
}, true);

另一种方法是为 DOM 事件创建一个处理程序（但只能有一个），例如。

window.onresize = function(event) {
    ...
};

jQuery 可能会 做一些工作来确保在所有浏览器中一致地触发调整大小事件，但我不确定是否有任何浏览器不同，但我鼓励您在 Firefox、Safari 和 IE 中进行测试。


更多建议

首先，我知道addEventListener上面的评论中提到了该方法，但我没有看到任何代码。由于这是首选方法，因此它是：

window.addEventListener('resize', function(event){
  // do stuff here
});

这是一个工作示例。
