如何在 TextView 中水平和垂直居中文本？

Arya

2023-02-16 21:54

如何将 a 中的文本水平和垂直居中TextView，使其恰好出现在TextViewin的中间Android？



解答

我假设您正在使用 XML 布局。

<TextView  
    android:layout_width="match_parent" 
    android:layout_height="match_parent" 
    android:gravity="center"
    android:text="@string/**yourtextstring**"
/>

您也可以使用重力center_vertical或center_horizontal根据您的需要。

正如@stealthcopter 评论的那样，在 java 中：.setGravity(Gravity.CENTER);.

对于 Kotlin 用户，.gravity = Gravity.CENTER


更多建议

android:gravity="center"

这会成功的

当用户将鼠标悬停在列表项上时，如何将光标变为手形？

Arya

2023-02-16 21:54

我有一个列表，我有一个用于其项目的点击处理程序：

<ul>
  <li>foo</li>
  <li>goo</li>
</ul>

如何将鼠标指针更改为手形指针（例如将鼠标悬停在按钮上时）？现在，当我将鼠标悬停在列表项上时，指针变成了文本选择指针。



解答

鉴于时间的流逝，正如人们所提到的，您现在可以安全地使用：

li { cursor: pointer; }

使用 .format（或 f 字符串）时如何转义字符串中的大括号 ({}) 字符？

Arya

2023-02-16 21:53

非工作示例：

print(" \{ Hello \} {0} ".format(42))

期望的输出：

 {Hello} 42



解答

你需要加倍{{and }}：

>>> x = " {{ Hello }} {0} "
>>> print(x.format(42))
' { Hello } 42 '

以下是格式字符串语法的 Python 文档的相关部分：

格式字符串包含用花括号包围的“替换字段” {}。大括号中未包含的任何内容都被视为文字文本，将原封不动地复制到输出中。如果您需要在文字文本中包含大括号字符，可以通过加倍来转义：{{和}}。


更多建议

Python 3.6+ (2017)

在最新版本的 Python 中，人们会使用f 字符串（另请参阅PEP498）。

对于 f-strings，应该使用 double{{或}}

n = 42  
print(f" {{Hello}} {n} ")

产生所需的

 {Hello} 42

如果您需要解析括号中的表达式而不是使用文字文本，则需要三组括号：

hello = "HELLO"
print(f"{{{hello.lower()}}}")

产生

{hello}
