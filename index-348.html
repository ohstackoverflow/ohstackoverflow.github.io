按值设置选择选项“已选择”

Arya

2023-02-17 04:39

我有一个select字段，里面有一些选项。现在我需要使用 jQuery 选择其中一个options。但是，当我只知道必须选择的value那个时，我该怎么做呢？option

我有以下 HTML：

<div class="id_100">
  <select>
    <option value="val1">Val 1</option>
    <option value="val2">Val 2</option>
    <option value="val3">Val 3</option>
  </select>
</div>

我需要选择 value 的选项val2。如何才能做到这一点？

这是一个演示页面：http://jsfiddle.net/9Stxb/



解答

有一种更简单的方法不需要您进入选项标签：

$("div.id_100 select").val("val2");

查看此jQuery 方法。

注意 ：以上代码不会触发更改事件。为了完全兼容，你需要这样称呼它：

$("div.id_100 select").val("val2").change();


更多建议

要选择值为“val2”的选项：

$('.id_100 option[value=val2]').attr('selected','selected');

如何让'git log'显示像'svn log -v'这样的文件名

Arya

2023-02-17 04:39

SVN 的日志有一个“-v”模式，输出每次提交中更改的文件的文件名，如下所示：

jes5199$ svn 日志-v
---------------------------------------------- ----------------------
r1 | jes5199 | 2007-01-03 14:39:41 -0800（2007 年 1 月 3 日，星期三）| 1 行
更改路径：
   A /作者
   A /正在复制
   A /更改日志
   A /进化
   /安装
   /MacOSX

有没有一种快速的方法来获取 Git 中每次提交中更改文件的列表？



解答

对于更改文件的完整路径名：

git log --name-only

对于已更改文件的完整路径名和状态：

git log --name-status

对于缩写的路径名和更改文件的 diffstat：

git log --stat

还有很多选择。查看文档。


更多建议

鼓励新用户改用git-log[1]。该命令与git-log[1]whatchanged基本相同，但默认显示原始格式差异输出并跳过合并。

该命令主要出于历史原因而保留；许多在很久以前git log就通过阅读 Linux 内核邮件列表学习 Git 的人的手指被训练来打字。

您可以使用该命令git whatchanged --stat获取每次提交中更改的文件列表（以及提交消息）。

如何修剪空格？

Arya

2023-02-17 04:38

是否有一个 Python 函数可以从字符串中删除空格（空格和制表符）？

所以给定的输入" \t example string\t "变成了"example string"。



解答

对于两边的空白，使用str.strip：

s = "  \t a string example\t  "
s = s.strip()

对于右侧的空白，请使用str.rstrip：

s = s.rstrip()

对于左侧的空白，请使用str.lstrip：

s = s.lstrip()

您可以提供一个参数来为这些函数中的任何一个去除任意字符，如下所示：

s = s.strip(' \t\n\r')

这将从字符串的两侧去除任何空格\t、、、\n或字符。\r

上面的示例仅从字符串的左侧和右侧删除字符串。如果您还想从字符串中间删除字符，请尝试re.sub：

import re
print(re.sub('[\s+]', '', s))

那应该打印出来：

astringexample


更多建议

在 Python 中， 修剪 方法被命名为strip：

str.strip()  # trim
str.lstrip()  # left trim
str.rstrip()  # right trim
