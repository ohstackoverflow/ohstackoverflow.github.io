如何在 bash 中对整数序列进行零填充，以便所有整数都具有相同的宽度？

Arya

2023-02-18 03:36

我需要循环一些值，

for i in $(seq $first $last)
do
    does something here
done

对于$first和$last，我需要它的长度固定为 5。因此，如果输入是1，我需要在前面添加零，使其变为00001。例如，它循环直到99999，但长度必须为 5。

例如：00002、00042、00212等等12312。

关于我该怎么做的任何想法？



解答

在您的特定情况下，尽管使用-f标志来seq让它在输出列表时格式化数字可能是最简单的。例如：

for i in $(seq -f "%05g" 10 15)
do
  echo $i
done

将产生以下输出：

00010
00011
00012
00013
00014
00015

更一般地说，bash有printf一个内置的，所以你可以用零填充输出，如下所示：

$ i=99
$ printf "%05d\n" $i
00099

您可以使用-v标志将输出存储在另一个变量中：

$ i=99
$ printf -v j "%05d" $i
$ echo $j
00099

请注意，printf支持的格式略有不同，seq因此您需要使用%05d而不是%05g.


更多建议

更简单的你可以做

for i in {00001..99999}; do
  echo $i
done

无法截断表，因为它正被 FOREIGN KEY 约束引用？

Arya

2023-02-18 03:35

使用MSSQL2005，如果先截断子表（外键关系主键的表），是否可以截断带外键约束的表？

我知道我可以

  • 使用DELETE不带 where 子句的 a，然后使用RESEED身份（或）
  • 删除 FK，截断表，然后重新创建 FK。

我认为只要我在父表之前截断子表，我就可以不执行上述任何一个选项，但我收到此错误：

无法截断表“TableName”，因为它正被 FOREIGN KEY 约束引用。



解答

正确的; 您不能截断具有 FK 约束的表。

通常我的过程是：

  1. 删除约束
  2. 截断表
  3. 重新创建约束。

（当然，所有这些都在交易中。）

当然，这仅适用于 子项已被截断的情况。 否则我会走不同的路线，完全取决于我的数据是什么样子。（变量太多，无法进入此处。）

如何以编程方式创建基本的 UIButton？

我怎样才能UIButton以编程方式创建一个基本的？例如在我的视图控制器中，当执行该viewDidLoad方法时，将动态创建三个UIButtons 并设置其布局或属性。



解答

这是一个：

UIButton *button = [UIButton buttonWithType:UIButtonTypeCustom];
[button addTarget:self 
           action:@selector(aMethod:)
 forControlEvents:UIControlEventTouchUpInside];
[button setTitle:@"Show View" forState:UIControlStateNormal];
button.frame = CGRectMake(80.0, 210.0, 160.0, 40.0);
[view addSubview:button];


更多建议

目标-C

UIButton *but= [UIButton buttonWithType:UIButtonTypeRoundedRect];
[but addTarget:self action:@selector(buttonClicked:) forControlEvents:UIControlEventTouchUpInside];
 [but setFrame:CGRectMake(52, 252, 215, 40)];
[but setTitle:@"Login" forState:UIControlStateNormal];
[but setExclusiveTouch:YES];

NSLog(@"you clicked on button %@", sender.tag);

    let myButton = UIButton() // if you want to set the type use like UIButton(type: .RoundedRect) or UIButton(type: .Custom)

    myButton.setTitleColor(UIColor.blueColor(), forState: .Normal)

    myButton.addTarget(self, action: "pressedAction:", forControlEvents: .TouchUpInside)

    self.view.addSubview( myButton)

}

Swift3及以上

  let myButton = UIButton() // if you want to set the type use like UIButton(type: .RoundedRect) or UIButton(type: .Custom)
    myButton.setTitle("Hi, Click me", for: .normal)
    myButton.setTitleColor(UIColor.blue, for: .normal)
    myButton.frame = CGRect(x: 15, y: 50, width: 300, height: 500)

    myButton.addTarget(self, action: #selector(pressedAction(_:)), for: .touchUpInside)
    self.view.addSubview( myButton)



func pressedAction(_ sender: UIButton) {
   // do your stuff here 
  print("you clicked on button \(sender.tag)")
