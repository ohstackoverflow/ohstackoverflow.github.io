如何使用 git config 删除全局配置中的条目？

Arya

2023-02-18 03:54

我在git 中运行了一个全局配置命令来使用一个文件排除某些文件.gitignore_global：

git config --global core.excludesfile ~/.gitignore_global

有没有办法在全球范围内撤消此设置的创建？



解答

我不确定“撤消”更改是什么意思。core.excludesfile您可以像这样删除设置：

git config --global --unset core.excludesfile

当然你可以简单地编辑配置文件：

git config --global --edit

...然后手动删除设置。


更多建议

您可以像这样使用--unset标志来执行此操作：git config

git config --global --unset user.name
git config --global --unset user.email

如果一个配置有更多变量，您可以使用：

git config --global --unset-all user.name

如何获得一周开始的 DateTime？

Arya

2023-02-18 03:54

如何在 C# 中仅知道当前时间来查找一周的开始时间（星期日和星期一）？

就像是：

DateTime.Now.StartWeek(Monday);



解答

使用扩展方法：

public static class DateTimeExtensions
{
    public static DateTime StartOfWeek(this DateTime dt, DayOfWeek startOfWeek)
    {
        int diff = (7 + (dt.DayOfWeek - startOfWeek)) % 7;
        return dt.AddDays(-1 * diff).Date;
    }
}

可以按如下方式使用：

DateTime dt = DateTime.Now.StartOfWeek(DayOfWeek.Monday);
DateTime dt = DateTime.Now.StartOfWeek(DayOfWeek.Sunday);


更多建议

我能想到的最快方法是：

var sunday = DateTime.Today.AddDays(-(int)DateTime.Today.DayOfWeek);

如果您希望一周中的任何一天作为您的开始日期，您需要做的就是将 DayOfWeek 值添加到末尾

var monday = DateTime.Today.AddDays(-(int)DateTime.Today.DayOfWeek + (int)DayOfWeek.Monday);

var tuesday = DateTime.Today.AddDays(-(int)DateTime.Today.DayOfWeek + (int)DayOfWeek.Tuesday);

在 Java 中从哪里获取“UTF-8”字符串文字？

Arya

2023-02-18 03:53

我试图在这段代码中使用常量而不是字符串文字：

new InputStreamReader(new FileInputStream(file), "UTF-8")

"UTF-8"经常出现在代码中，而是引用一些static final变量会更好。你知道在 JDK 中哪里可以找到这样的变量吗？

顺便说一句，再想一想，这样的常量是糟糕的设计：Public Static Literals ... Are Not a Solution for Data Duplication



解答

在 Java 1.7+ 中，java.nio.charset.StandardCharsets定义了Charset包含UTF_8.

import java.nio.charset.StandardCharsets;

...

StandardCharsets.UTF_8.name();

对于安卓：minSdk 19


更多建议

现在我使用commons-langorg.apache.commons.lang3.CharEncoding.UTF_8中的常量。
