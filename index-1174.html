在 SQL Server 中对结果进行分页的最佳方法是什么

Arya

2023-02-18 03:14

如果您还想获得结果总数（在分页之前），那么在 SQL Server 2000、2005、2008、2012 中对结果进行分页的最佳方法（性能方面）是什么？



解答

获取结果总数和分页是两种不同的操作。为了这个例子，我们假设你正在处理的查询是

SELECT * FROM Orders WHERE OrderDate >= '1980-01-01' ORDER BY OrderDate

在这种情况下，您将使用以下方法确定结果总数：

SELECT COUNT(*) FROM Orders WHERE OrderDate >= '1980-01-01'

...这可能看起来效率低下，但实际上性能非常好，假设所有索引等都已正确设置。

接下来，要以分页方式返回实际结果，以下查询将是最有效的：

SELECT  *
FROM    ( SELECT    ROW_NUMBER() OVER ( ORDER BY OrderDate ) AS RowNum, *
          FROM      Orders
          WHERE     OrderDate >= '1980-01-01'
        ) AS RowConstrainedResult
WHERE   RowNum >= 1
    AND RowNum < 20
ORDER BY RowNum

这将返回原始查询的第 1-19 行。这里很酷的一点是，特别是对于 Web 应用程序，您不必保留任何状态，除了要返回的行号。

如何检查密钥库文件中的证书名称和别名？

Arya

2023-02-18 03:13

我有一堆 .keystore 文件，需要找到一个具有特定 CN 和别名的文件。有没有办法用 keytool、jarsigner 或其他工具来做到这一点？我找到了一种方法来检查是否使用了特定的密钥库来签署特定的 apk，但我还需要在每个文件中获取别名和证书名称。



解答

您可以运行以下命令来列出密钥库文件的内容（和别名）：

keytool -v -list -keystore .keystore

如果你正在寻找一个特定的别名，你也可以在命令中指定它：

keytool -list -keystore .keystore -alias foo

如果找不到别名，则会显示异常：

keytool 错误：java.lang.Exception：别名不存在


更多建议

为了获得所有细节，我必须将 -v 选项添加到 romaintaz 答案中：

keytool -v -list -keystore <FileName>.keystore

如何将列表项添加到现有的无序列表

Arya

2023-02-18 03:13

我有这样的代码：

<div id="header">
    <ul class="tabs">
        <li><a href="/user/view"><span class="tab">Profile</span></a></li>
        <li><a href="/user/edit"><span class="tab">Edit</span></a></li>
    </ul>
</div>

我想使用 jQuery 将以下内容添加到列表中：

<li><a href="/user/messages"><span class="tab">Message Center</span></a></li>

我试过这个：

$("#content ul li:last").append("<li><a href="/user/messages"><span class="tab">Message Center</span></a></li>");

但这会将新li 内容 添加到最后一个li（就在结束标记之前），而不是在它之后。添加这个的最佳方法是什么li？



解答

这样做：

$("#header ul").append('<li><a href="/user/messages"><span class="tab">Message Center</span></a></li>');

两件事情：

  • 您可以将其附加<li>到<ul>自身。
  • 您需要使用与您在 HTML 中使用的引号类型相反的引号。因此，由于您在属性中使用了双引号，因此请用单引号将代码括起来。


更多建议

您还可以以更“对象”且仍然易于阅读的方式进行操作：

$('#content ul').append(
    $('<li>').append(
        $('<a>').attr('href','/user/messages').append(
            $('<span>').attr('class', 'tab').append("Message center")
)));

那时你不必用引号打架，但你必须保留大括号的痕迹:)
