在 Java 中获取文件的 MD5 校验和

Arya

2023-02-18 03:43

我正在寻找使用 Java 来获取文件的 MD5 校验和。我真的很惊讶，但我没能找到任何显示如何获取文件的 MD5 校验和的信息。

它是如何完成的？



解答

有一个输入流装饰器 ，java.security.DigestInputStream因此您可以像往常一样在使用输入流的同时计算摘要，而不必额外传递数据。

MessageDigest md = MessageDigest.getInstance("MD5");
try (InputStream is = Files.newInputStream(Paths.get("file.txt"));
     DigestInputStream dis = new DigestInputStream(is, md)) 
{
  /* Read decorated stream (dis) to EOF as normal... */
}
byte[] digest = md.digest();


更多建议

使用Apache Commons Codec库中的DigestUtils：

try (InputStream is = Files.newInputStream(Paths.get("file.zip"))) {
    String md5 = org.apache.commons.codec.digest.DigestUtils.md5Hex(is);
}

在 bash 脚本中创建时间戳变量

Arya

2023-02-18 03:42

我正在尝试在 shell 脚本中创建一个时间戳变量，以使日志记录更容易一些。我想在脚本的开头创建变量，并让它在我发出时打印出当前时间echo $timestamp。事实证明比我想象的要难。以下是我尝试过的一些事情：

timestamp="(date +"%T")"回声打印出来(date +"%T")

timestamp="$(date +"%T")"echo 打印变量初始化的时间。

我尝试过的其他方法只是细微的变化，效果并不好。有谁知道如何完成我想要做的事情？



解答

为了获取当前时间戳而不是定义固定变量的时间，诀窍是使用函数而 不是 变量：

#!/bin/bash
# Define a timestamp function
timestamp() {
  date +"%T" # current time
}
# do something...
timestamp # print timestamp
# do something else...
timestamp # print another timestamp
# continue...

如果您不喜欢说明符给出的格式，%T您可以组合date. 对于 GNU date，您可以在此处的官方文档中找到这些说明符的完整列表： https: //www.gnu.org/software/coreutils/manual/html_node/Time-conversion- specifiers.html#Time-conversion-specifiers
