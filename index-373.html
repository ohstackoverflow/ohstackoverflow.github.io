如何解析 XML 并获取特定节点属性的实例？

Arya

2023-02-17 05:16

我在 XML 中有很多行，我正在尝试获取特定节点属性的实例。

<foo>
   <bar>
      <type foobar="1"/>
      <type foobar="2"/>
   </bar>
</foo>

如何访问属性的值foobar？在这个例子中，我想要"1"和"2"。



解答

我建议ElementTree。相同 API 的其他兼容实现，例如lxml, 和cElementTreePython 标准库本身；但是，在这种情况下，他们主要添加的是更快的速度——编程部分的易用性取决于ElementTree定义的 API。

root首先从 XML构建一个 Element 实例，例如使用XML函数，或者通过使用如下内容解析文件：

import xml.etree.ElementTree as ET
root = ET.parse('thefile.xml').getroot()

或 中显示的许多其他方式中的任何一种ElementTree。然后做类似的事情：

for type_tag in root.findall('bar/type'):
    value = type_tag.get('foobar')
    print(value)

输出：

1
2


更多建议

minidom是最快和非常直接的。

XML：

<data>
    <items>
        <item name="item1"></item>
        <item name="item2"></item>
        <item name="item3"></item>
        <item name="item4"></item>
    </items>
</data>

Python：

from xml.dom import minidom

dom = minidom.parse('items.xml')
elements = dom.getElementsByTagName('item')

print(f"There are {len(elements)} items:")

for element in elements:
    print(element.attributes['name'].value)

输出：

There are 4 items:
item1
item2
item3
item4

您如何在 TypeScript 的“window”上显式设置新属性？

Arya

2023-02-17 05:15

我通过在 上显式设置属性来为我的对象设置全局命名空间window。

window.MyNamespace = window.MyNamespace || {};

TypeScript 强调MyNamespace并抱怨说：

属性“MyNamespace”在类型“window”的值上不存在任何“

MyNamespace我可以通过声明为环境变量并放弃显式来使代码工作window，但我不想这样做。

declare var MyNamespace: any;

MyNamespace = MyNamespace || {};

我怎样才能留在那里window并让 TypeScript 开心？

作为旁注，我发现 TypeScript 的抱怨特别有趣，因为它告诉我这是一种绝对可以包含任何东西的window类型。any



解答

我刚刚在另一个 Stack Overflow 问题的答案中找到了这个问题的答案。

declare global {
    interface Window { MyNamespace: any; }
}

window.MyNamespace = window.MyNamespace || {};

基本上，您需要扩展现有window界面以告知它您的新属性。


更多建议

要保持动态，只需使用：

(<any>window).MyNamespace

请注意，这可能不适用于 TSX，因为编译器可能认为这<any>是一个 TSX 元素。查看此答案以了解与 TSX 兼容的类型断言。

整数的最大值和最小值

Arya

2023-02-17 05:15

如何在 Python 中表示整数的最小值和最大值？在 Java 中，我们有Integer.MIN_VALUE和Integer.MAX_VALUE。

另请参阅：Python 中的最大浮点数是多少？.



解答

蟒蛇3

在 Python 3 中，这个问题不适用。普通int类型是无界的。

但是，您实际上可能正在寻找有关当前解释器的 字数的 信息，在大多数情况下，它与机器的字数相同。该信息在 Python 3 中仍然可用，如sys.maxsize，它是有符号字可表示的最大值。等效地，它是最大可能列表或内存序列的大小。

通常，无符号字可表示的最大值为sys.maxsize * 2 + 1，字中的位数为math.log2(sys.maxsize * 2 + 2)。有关详细信息，请参阅此答案。

蟒蛇2

在 Python 2 中，普通值的最大值int可用作sys.maxint：

>>> sys.maxint  # on my system, 2**63-1
9223372036854775807

您可以计算最小值，如文档中-sys.maxint - 1所示。

一旦超过这个值，Python 就会从普通整数无缝切换到长整数。所以大多数时候，你不需要知道它。


更多建议

如果您只需要一个比其他所有数字都大的数字，您可以使用

float('inf')

以类似的方式，一个比其他所有数字都小的数字：

float('-inf')

这适用于 python 2 和 3。
