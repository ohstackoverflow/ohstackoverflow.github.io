makefile:4: *** 缺少分隔符。停止

Arya

2023-02-17 13:48

这是我的生成文件：

all:ll

ll:ll.c   
  gcc  -c  -Wall -Werror -02 c.c ll.c  -o  ll  $@  $<

clean :
  \rm -fr ll

当我尝试make cleanor时make make，出现此错误：

:makefile:4: *** missing separator.  Stop.

我该如何解决？



解答

make 定义了启动每个 recipe 所必需的选项卡。每个规则的所有操作都由选项卡标识。如果您更喜欢使用制表符以外的字符为您的食谱添加前缀，您可以将 .RECIPEPREFIX 变量设置为替代字符。

为了检查，我使用了命令cat -e -t -v makefile_name。

它显示了带有 的制表符^I和带有 的行结尾$。两者对于确保依赖关系正确结束和制表符标记规则的操作都是至关重要的，以便它们很容易被 make 实用程序识别。

例子：

Kaizen ~/so_test $ cat -e -t -v  mk.t
all:ll$      ## here the $ is end of line ...                   
$
ll:ll.c   $
^Igcc  -c  -Wall -Werror -02 c.c ll.c  -o  ll  $@  $<$ 
## the ^I above means a tab was there before the action part, so this line is ok .
 $
clean :$
   \rm -fr ll$
## see here there is no ^I which means , tab is not present .... 
## in this case you need to open the file again and edit/ensure a tab 
## starts the action part


更多建议

在 VS Code 上，只需单击右下角的“空格：4”，然后在编辑 Makefile 时将其更改为制表符。

将列表中的所有字符串转换为 int

Arya

2023-02-17 13:47

如何将列表中的所有字符串转换为整数？

['1', '2', '3']  ⟶  [1, 2, 3]



解答

鉴于：

xs = ['1', '2', '3']

然后使用mapthenlist获取整数列表：

list(map(int, xs))

在 Python 2 中，list没有必要，因为map它返回了一个列表：

map(int, xs)


更多建议

在列表上使用列表理解xs：

[int(x) for x in xs]

例如

>>> xs = ["1", "2", "3"]
>>> [int(x) for x in xs]
[1, 2, 3]

将数组转换为对象

Arya

2023-02-17 13:47

什么是最好的转换方式：

['a','b','c']

到：

{
  0: 'a',
  1: 'b',
  2: 'c'
}



解答

ECMAScript 6 引入了易于填充的Object.assign：

该Object.assign()方法用于将所有可枚举自身属性的值从一个或多个源对象复制到目标对象。它将返回目标对象。

Object.assign({}, ['a','b','c']); // {0:"a", 1:"b", 2:"c"}

length数组的自身属性不会被复制，因为它不可枚举。

此外，您可以在对象上使用 ES8扩展语法来获得相同的结果：

{ ...['a', 'b', 'c'] }

对于自定义键，您可以使用reduce：

['a', 'b', 'c'].reduce((a, v) => ({ ...a, [v]: v}), {}) 
// { a: "a", b: "b", c: "c" }


更多建议

具有这样的功能：

function toObject(arr) {
  var rv = {};
  for (var i = 0; i < arr.length; ++i)
    rv[i] = arr[i];
  return rv;
}

您的数组已经或多或少只是一个对象，但数组确实具有一些关于整数命名属性的“有趣”和特殊行为。上面会给你一个普通的对象。

编辑 哦，你也可能想考虑数组中的“漏洞”：

function toObject(arr) {
  var rv = {};
  for (var i = 0; i < arr.length; ++i)
    if (arr[i] !== undefined) rv[i] = arr[i];
  return rv;
}

在现代 JavaScript 运行时中，您可以使用以下.reduce()方法：

var obj = arr.reduce(function(acc, cur, i) {
  acc[i] = cur;
  return acc;
}, {});

那一个也避免了数组中的“漏洞”，因为这就是.reduce()工作原理。
