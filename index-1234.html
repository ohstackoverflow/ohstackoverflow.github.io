如何将两个字段“唯一”定义为一对

Arya

2023-02-18 04:42

有没有办法将几个字段定义为 Django 中的唯一字段？

我有一个卷表（期刊），我不希望同一期刊的卷号超过一个。

class Volume(models.Model):
    id = models.AutoField(primary_key=True)
    journal_id = models.ForeignKey(Journals, db_column='jid', null=True, verbose_name = "Journal")
    volume_number = models.CharField('Volume Number', max_length=100)
    comments = models.TextField('Comments', max_length=4000, blank=True)

我试图将unique = Trueas 属性放入字段中journal_id，volume_number但它不起作用。



解答

有一个名为unique_together的简单解决方案，它可以满足您的需求。

例如：

class MyModel(models.Model):
  field1 = models.CharField(max_length=50)
  field2 = models.CharField(max_length=50)

  class Meta:
    unique_together = ('field1', 'field2',)

在你的情况下：

class Volume(models.Model):
  id = models.AutoField(primary_key=True)
  journal_id = models.ForeignKey(Journals, db_column='jid', null=True, verbose_name = "Journal")
  volume_number = models.CharField('Volume Number', max_length=100)
  comments = models.TextField('Comments', max_length=4000, blank=True)

  class Meta:
    unique_together = ('journal_id', 'volume_number',)


更多建议

姜戈 2.2+

使用这些constraints功能UniqueConstraint优于 unique_together 。

来自 Django 文档unique_together：

请改用带有约束选项的 UniqueConstraint。
UniqueConstraint 提供了比 unique_together 更多的功能。
unique_together 将来可能会被弃用。

例如：

class Volume(models.Model):
    id = models.AutoField(primary_key=True)
    journal_id = models.ForeignKey(Journals, db_column='jid', null=True, verbose_name="Journal")
    volume_number = models.CharField('Volume Number', max_length=100)
    comments = models.TextField('Comments', max_length=4000, blank=True)

    class Meta:
        constraints = [
            models.UniqueConstraint(fields=['journal_id', 'volume_number'], name='name of constraint')
        ]

如何将参数传递给 PowerShell 脚本？

Arya

2023-02-18 04:42

有一个名为itunesForward.ps1使 iTunes 快进 30 秒的 PowerShell 脚本：

$iTunes = New-Object -ComObject iTunes.Application

if ($iTunes.playerstate -eq 1)
{
  $iTunes.PlayerPosition = $iTunes.PlayerPosition + 30
}

它使用提示行命令执行：

powershell.exe itunesForward.ps1

是否可以从命令行传递参数并将其应用于脚本而不是硬编码的 30 秒值？



解答

测试工作：

#Must be the first statement in your script (not counting comments)
param([Int32]$step=30)

$iTunes = New-Object -ComObject iTunes.Application

if ($iTunes.playerstate -eq 1)
{
  $iTunes.PlayerPosition = $iTunes.PlayerPosition + $step
}

调用它

powershell.exe -file itunesForward.ps1 -step 15

多参数语法（注释是可选的，但允许）：

<#
    Script description.

    Some notes.
#>
param (
    # height of largest column without top bar
    [int]$h = 4000,

    # name of the output image
    [string]$image = 'out.png'
)

还有一些高级参数的例子，例如 Mandatory ：

<#
    Script description.

    Some notes.
#>
param (
    # height of largest column without top bar
    [Parameter(Mandatory=$true)]
    [int]$h,

    # name of the output image
    [string]$image = 'out.png'
)

Write-Host "$image $h"

默认值不适用于强制参数。您可以省略=$trueboolean 类型的高级参数[Parameter(Mandatory)]。


更多建议

您也可以使用$args变量（就像位置参数）：

$step = $args[0]

$iTunes = New-Object -ComObject iTunes.Application

if ($iTunes.playerstate -eq 1)
{
  $iTunes.PlayerPosition = $iTunes.PlayerPosition + $step
}

然后它可以这样调用：

powershell.exe -file itunersforward.ps1 15

正则表达式：匹配一个字符的第一次出现

Arya

2023-02-18 04:42

我正在寻找一种匹配所有内容的模式， 直到 第一次出现特定字符，比如“;” - 一个 分号 。

我写了这个：

/^(.*);/

但它实际上匹配所有内容（包括分号），直到最后一次出现分号。



解答

你需要

/^[^;]*/

这[^;]是一个 字符类 ，它匹配除分号以外的所有内容。

^（行锚点的开始）被添加到正则表达式的开头，因此仅捕获每行的第一个匹配项。这可能需要也可能不需要，具体取决于是否需要可能的后续匹配。

引用perlre联机帮助页：

您可以通过在 [] 中包含一个字符列表来指定一个字符类，它将匹配列表中的任何字符。如果“[”之后的第一个字符是“^”，则该类匹配任何不在列表中的字符。

这应该适用于大多数正则表达式方言。


更多建议

会;

/^(.*?);/

工作？

是?惰性运算符，因此正则表达式在匹配;.
