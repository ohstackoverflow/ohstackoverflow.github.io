如何将小时数添加到 Date 对象？

Arya

2023-02-18 03:24

令我惊讶的是 JavaScript 的 Date 对象没有实现任何类型的添加函数。

我只是想要一个可以做到这一点的功能：

var now = Date.now();
var fourHoursLater = now.addHours(4);

function Date.prototype.addHours(h) {
  // How do I implement this?
}

我只是想要一些方向上的指示。

  • 我需要做字符串解析吗？

  • 我可以使用 setTime 吗？

  • 毫秒呢？

像这样：

new Date(milliseconds + 4*3600*1000 /* 4 hours in ms */)?

不过，这看起来真的很骇人听闻——它真的有效吗？



解答

JavaScript 本身有糟糕的日期/时间 API。尽管如此，您可以使用纯 JavaScript 执行此操作：

Date.prototype.addHours = function(h) {
  this.setTime(this.getTime() + (h*60*60*1000));
  return this;
}


更多建议

Date.prototype.addHours= function(h){
    this.setHours(this.getHours()+h);
    return this;
}

测试：

alert(new Date().addHours(4));

使用文本对齐中心居中图像？

Arya

2023-02-18 03:23

该属性是text-align: center;使用 CSS 使图像居中的好方法吗？

img {
    text-align: center;
}



解答

这将不起作用，因为该text-align属性适用于块容器，而不是内联元素，并且img是内联元素。请参阅W3C 规范。

改用这个：

img.center {
    display: block;
    margin: 0 auto;
}


<div style="border: 1px solid black;">
<img class="center" src ="https://cdn.sstatic.net/Sites/stackoverflow/company/img/logos/so/so-icon.png?v=c78bd457575a">

</div>


更多建议

这并不总是有效......如果没有，请尝试：

img {
    display: block;
    margin: 0 auto;
}

在 Android 的 EditText 视图中允许多行？

Arya

2023-02-18 03:23

如何在 Android 的视图中允许多行EditText？



解答

默认情况下，EditTextAndroid 中的所有小部件都是多行的。

这是一些示例代码：

<EditText
    android:inputType="textMultiLine" <!-- Multiline input -->
    android:lines="8" <!-- Total Lines prior display -->
    android:minLines="6" <!-- Minimum lines -->
    android:gravity="top|start" <!-- Cursor Position -->
    android:maxLines="10" <!-- Maximum Lines -->
    android:layout_height="wrap_content" <!-- Height determined by content -->
    android:layout_width="match_parent" <!-- Fill entire width -->
    android:scrollbars="vertical" <!-- Vertical Scroll Bar -->
/>


更多建议

您可能会发现使用它更好：

<EditText 
...
android:inputType="textMultiLine"
/>

这是因为android:singleLine已弃用。
