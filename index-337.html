如何在 Python 中将字典键作为列表返回？

Arya

2023-02-17 04:22

使用 Python 2.7，我可以获得字典 键 、 值 或 项 作为list：

>>> newdict = {1:0, 2:0, 3:0}
>>> newdict.keys()
[1, 2, 3]

使用 Python >= 3.3，我得到：

>>> newdict.keys()
dict_keys([1, 2, 3])

我如何list使用 Python 3 获得密钥的简单信息？



解答

这会将dict_keys对象转换为list：

list(newdict.keys())

另一方面，你应该问问自己这是否重要。假设鸭子类型是 Pythonic——如果它看起来像鸭子并且叫起来像 鸭子，那么它就是鸭子 。该对象dict_keys可以像list. 例如：

for key in newdict.keys():
    print(key)

请注意，它dict_keys不支持插入newdict[k] = v，尽管您可能不需要它。


更多建议

Python >= 3.5 替代方案：解压为列表文字 [*newdict]

Python 3.5 引入了新的解包泛化 (PEP 448)，使您现在可以轻松地执行以下操作：

>>> newdict = {1:0, 2:0, 3:0}
>>> [*newdict]
[1, 2, 3]

解包*适用于 任何 可迭代的对象，并且由于字典在迭代时会返回它们的键，因此您可以通过在列表文字中使用它来轻松创建列表。

添加.keys()ie[*newdict.keys()]可能有助于使您的意图更加明确，尽管这会花费您查找和调用函数的费用。（老实说，这不是您 真正 应该担心的事情）。

语法*iterable类似于 doing list(iterable)，其行为最初记录在Python 参考手册的调用部分。在 PEP 448 中，放宽了对出现位置的限制，*iterable允许它也可以放置在列表、集合和元组文字中，表达式列表的参考手册也已更新以说明这一点。

虽然等同于list(newdict)它更快的区别（至少对于小型词典而言）因为实际上没有执行函数调用：

%timeit [*newdict]
1000000 loops, best of 3: 249 ns per loop

%timeit list(newdict)
1000000 loops, best of 3: 508 ns per loop

%timeit [k for k in newdict]
1000000 loops, best of 3: 574 ns per loop

对于较大的字典，速度几乎相同（遍历大型集合的开销胜过函数调用的小成本）。

以类似的方式，您可以创建元组和字典键集：

>>> *newdict,
(1, 2, 3)
>>> {*newdict}
{1, 2, 3}

当心元组大小写中的尾随逗号！

如何获取 foreach 循环当前迭代的索引？

Arya

2023-02-17 04:21

在 C# 中是否有一些我没有遇到过的罕见语言构造（比如我最近学到的一些，一些在 Stack Overflow 上）来获取表示 foreach 循环当前迭代的值？

例如，我目前根据情况做这样的事情：

int i = 0;
foreach (Object o in collection)
{
    // ...
    i++;
}



解答

用于foreach迭代实现IEnumerable. 它通过调用GetEnumerator集合来做到这一点，集合将返回一个Enumerator.

这个 Enumerator 有一个方法和一个属性：

  • MoveNext()
  • Current

Current返回 Enumerator 当前所在的对象，MoveNext更新Current到下一个对象。

索引的概念对于枚举的概念来说是陌生的，是做不到的。

因此，大多数集合都可以使用索引器和 for 循环结构进行遍历。

与使用局部变量跟踪索引相比，我更喜欢在这种情况下使用 for 循环。

如何将方法标记为过时或已弃用？

Arya

2023-02-17 04:21

如何使用 C# 将方法标记为过时或弃用？



解答

最短的方法是将ObsoleteAttribute作为属性添加到方法中。确保包括适当的解释：

[Obsolete("Method1 is deprecated, please use Method2 instead.")]
public void Method1()
{ … }

您还可以导致编译失败，将方法的使用视为 错误而不是警告 ，如果该方法是从代码中的某处调用的，如下所示：

[Obsolete("Method1 is deprecated, please use Method2 instead.", true)]


更多建议

要标记为已过时并发出警告：

[Obsolete]
private static void SomeMethod()
