解答

键盘快捷键是⌘+]缩进和⌘+[取消缩进。

  • 在 Xcode 的首选项窗口中，单击 Key Bindings 工具栏按钮。Key Bindings 部分是您自定义键盘快捷键的地方。

编辑器 -> 结构 -> 重新缩进

如何使用 pip 和需求文件升级特定包？

Arya

2023-02-17 23:07

我在 virtualenv 中将 pip 与需求文件一起用于我的 Django 项目。我正在尝试升级一些包，尤其是 Django 本身，但我收到有关源代码冲突的错误消息：

/build/Django中的源<virtualenv具有与 Django==1.2.4 冲突的版本 1.2.3（来自 -r requirements/apps.txt（第 3 行））

那是在我的需求文件中将 Django 的版本号从 1.2.3 更新到 1.2.4 之后。我正在使用此命令实际进行升级：

pip --install --upgrade -E `<virtualenv dir`> --requirement `<requirements file`>

我找不到任何触发整个包重新下载的标志。我什至尝试先运行卸载命令，然后再安装，但没有成功。我错过了什么吗？



解答

首先确保您已经检查了投票最多的答案。

我不确定这是否正是你的问题，但就我而言，我无法将 Django 升级到 1.2.4 - 我总是以 1.2.3 版本结束，所以我卸载了 Django：

<virtualenv>/bin/pip uninstall Django

然后我删除了<virtualenv>/build/Django目录，最后我安装了正确的版本：

<virtualenv>/bin/pip install Django
