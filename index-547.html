在 MySQL 中查找重复值

Arya

2023-02-17 09:45

我有一个包含 varchar 列的表，我想找到该列中所有具有重复值的记录。我可以用来查找重复项的最佳查询是什么？



解答

SELECT用一个子句做一个GROUP BY。假设 名称 是您要在其中查找重复项的列：

SELECT name, COUNT(*) c FROM table GROUP BY name HAVING c > 1;

这将返回一个结果，其中第一列为 名称 值，第二列为该值出现次数的计数。


更多建议

SELECT varchar_col
FROM table
GROUP BY varchar_col
HAVING COUNT(*) > 1;

如何仅使用 CSS 禁用链接

Arya

2023-02-17 09:45

有没有办法使用 CSS 禁用链接？

我有一个名为的类current-page，并希望禁用与此类的链接，以便在单击它们时不会发生任何操作。



解答

从这个解决方案：

[aria-current="page"] {
  pointer-events: none;
  cursor: default;
  text-decoration: none;
  color: black;
}


<a href="link.html" aria-current="page">Link</a>

有关浏览器支持，请参阅https://caniuse.com/#feat=pointer-events。如果您需要支持 Internet Explorer，则有一个解决方法；看到这个答案。

pointer-events警告：在 CSS 中对非 SVG 元素的使用是实验性的。该功能曾经是 CSS 3 UI 草案规范的一部分，但由于存在许多未解决的问题，已推迟到 CSS 4。


更多建议

.disabled {
  pointer-events: none;
  cursor: default;
  opacity: 0.6;
}


<a href="#" class="disabled">link</a>

渲染后如何将焦点设置在输入字段上？

Arya

2023-02-17 09:44

呈现组件后将焦点设置在特定文本字段上的反应方式是什么？

文档似乎建议使用 refs，例如：

在渲染函数中设置ref="nameInput"我的输入字段，然后调用：

this.refs.nameInput.getInputDOMNode().focus();

但是我应该在哪里称呼它呢？我已经尝试了几个地方，但我无法让它工作。



解答

你应该在里面做componentDidMount，refs callback而不是。像这样的东西

componentDidMount(){
   this.nameInput.focus(); 
}



class App extends React.Component{
  componentDidMount(){
    this.nameInput.focus();
  }
  render() {
    return(
      <div>
        <input 
          defaultValue="Won't focus" 
        />
        <input 
          ref={(input) => { this.nameInput = input; }} 
          defaultValue="will focus"
        />
      </div>
    );
  }
}

ReactDOM.render(<App />, document.getElementById('app'));
