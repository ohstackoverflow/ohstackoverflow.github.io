通过 Wi-Fi 运行/安装/调试 Android 应用程序？

Arya

2023-02-17 03:42

我认为有一种方法可以通过 Wi-Fi 测试开发中的应用程序。这可能吗？

我希望能够解开手机的束缚并进行无线开发。



解答

请参阅论坛帖子 无需 root 即可远程查看 Android 屏幕的任何方法？ - 发布#9。

  1. 通过USB连接设备并确保调试正常；
  2. adb tcpip 5555. 这使得设备开始监听端口 5555 上的连接；
  3. 查找6.0及以上版本的设备IPadb shell netcfg地址adb shell ifconfig；
  4. 您现在可以断开 USB；
  5. adb connect <DEVICE_IP_ADDRESS>:5555. 这将连接到我们在第 2 步中在设备上设置的服务器；
  6. 现在您在网络上有了一个设备，您可以像往常一样使用它进行调试。

要将服务器切换回 USB 模式，请运行adb usb，这会将手机上的服务器切换回 USB 模式。如果您有多个设备，您可以使用以下选项指定设备-s：adb -s <DEVICE_IP_ADDRESS>:5555 usb。

无需root！

要查找设备的 IP 地址：运行adb shell，然后运行netcfg​​。你会在那里看到它。要在使用 OSX 时查找 IP 地址，请运行命令adb shell ip route。

警告 ：启用该选项是危险的，您网络中的任何人都可以在调试中连接到您的设备，即使您在数据网络中也是如此。仅在连接到受信任的 Wi-Fi 时执行此操作，并记得在完成后断开连接！

@Sergei 建议修改第 2 行，评论道：“-d 选项需要在其他连接持续存在时连接到 USB 设备（例如，模拟器连接或其他 Wi-Fi 设备）”。

这些信息可能对未来的读者有价值，但我回滚到获得 178 票赞成票的原始版本。

在某些设备上，即使 没有 USB 数据线 也可以执行相同的操作：

  1. 在开发人员设置中通过网络启用 ADB它应该显示 IP 地址
  2. adb connect <DEVICE_IP_ADDRESS>:5555
  3. 完成后禁用设置

使用 Android Studio 有一个插件允许您连接 USB 调试，而无需从终端使用任何 ADB 命令。

如何在 JavaScript 中进行整数除法并分别取余？

Arya

2023-02-17 03:41

在 JavaScript 中，我如何获得：

  1. 给定整数进入另一个整数的总次数？
  2. 其余的？



解答

对于一些数字y和一些除数x计算商 ( quotient) [1]和余数 ( remainder) 为：

const quotient = Math.floor(y/x);
const remainder = y % x;

例子：

const quotient = Math.floor(13/3); // => 4 => the times 3 fits into 13  
const remainder = 13 % 3;          // => 1

[1] 一个数除以另一个数得到的整数


更多建议

我不是按位运算符方面的专家，但这是获取整数的另一种方法：

var num = ~~(a / b);

这也适用于负数，但Math.floor()会朝错误的方向舍入。

这似乎也是正确的：

var num = (a / b) >> 0;

获取数组的第一个元素

Arya

2023-02-17 03:40

我有一个数组：

array( 4 => 'apple', 7 => 'orange', 13 => 'plum' )

我想得到这个数组的第一个元素。预期结果： 字符串 apple

一个要求： 它不能通过引用传递来完成 ，所以array_shift不是一个好的解决方案。

我怎样才能做到这一点？



解答

原始答案，但代价高昂（O（n））：

array_shift(array_values($array));

在 O(1) 中：

array_pop(array_reverse($array));

其他用例等...

如果修改（在重置数组指针的意义上）$array不是问题，您可以使用：

reset($array);

如果需要数组“副本”，这在理论上应该更有效：

array_shift(array_slice($array, 0, 1));

使用 PHP 5.4+（但如果为空可能会导致索引错误）：

array_values($array)[0];


更多建议

正如 Mike 指出的（最简单的方法）：

$arr = array( 4 => 'apple', 7 => 'orange', 13 => 'plum' );
echo reset($arr); // Echoes "apple"

如果要获取key：（reset后执行）

echo key($arr); // Echoes "4"

来自PHP 的文档：

混合 重置 （数组 | 对象 & $数组）；

描述：

reset() 将 数组的 内部指针倒回第一个元素并返回第一个数组元素的值，如果数组为空则返回 FALSE。
