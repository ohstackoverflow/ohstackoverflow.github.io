如何在 Android 上创建透明 Activity？

Arya

2023-02-17 07:45

我想在另一个活动之上创建一个透明的活动。

我怎样才能做到这一点？



解答

在您的文件中添加以下样式res/values/styles.xml（如果您没有，请创建它。）这是一个完整的文件：

<?xml version="1.0" encoding="utf-8"?>
<resources>
  <style name="Theme.Transparent" parent="android:Theme">
    <item name="android:windowIsTranslucent">true</item>
    <item name="android:windowBackground">@android:color/transparent</item>
    <item name="android:windowContentOverlay">@null</item>
    <item name="android:windowNoTitle">true</item>
    <item name="android:windowIsFloating">true</item>
    <item name="android:backgroundDimEnabled">false</item>
  </style>
</resources>

（该值是我在文件中输入的@color/transparent颜色值。您也可以在以后的Android版本中使用。）#00000000``res/values/color.xml``@android:color/transparent

然后将样式应用于您的活动，例如：

<activity android:name=".SampleActivity" android:theme="@style/Theme.Transparent">
...
</activity>

如何找到 MySQL 中包含特定列名的所有表？

Arya

2023-02-17 07:44

我有 2-3 个不同的列名，我想在整个数据库中查找它们并列出所有包含这些列的表。有没有简单的脚本？



解答

要获取所有包含列columnA或ColumnB数据库中的表YourDatabase：

SELECT DISTINCT TABLE_NAME 
    FROM INFORMATION_SCHEMA.COLUMNS
    WHERE COLUMN_NAME IN ('columnA','ColumnB')
        AND TABLE_SCHEMA='YourDatabase';


更多建议

SELECT TABLE_NAME, COLUMN_NAME
FROM INFORMATION_SCHEMA.COLUMNS
WHERE COLUMN_NAME LIKE '%wild%';

是否可以从另一个 git 存储库中挑选一个提交？

Arya

2023-02-17 07:44

我正在使用一个 git 存储库，它需要来自另一个对第一个一无所知的 git 存储库的提交。

HEAD@{x}通常我会在 reflog 中使用 cherry-pick ，但是因为它对.git这个 reflog 条目（不同的物理目录）一无所知，所以我怎么能 cherry-pick 这个，或者我可以吗？

我正在使用git-svn。我的第一个分支使用git-svn的是trunkSubversion 存储库，下一个分支使用的是git- svnSubversion 分支。



解答

您需要将另一个存储库添加为远程存储库，然后获取其更改。从那里你可以看到提交，你可以挑选它。

像那样：

git remote add other https://example.link/repository.git
git fetch other

现在您拥有了所有的信息，可以简单地做git cherry-pick。

完成后，如果您不再需要它，您可能想再次移除遥控器，使用

git remote remove other

有关在此处使用遥控器的更多信息：https ://git-scm.com/book/en/v2/Git-Basics-Working-with- Remotes
