ngFor 以索引作为属性中的值

Arya

2023-02-17 11:16

我有一个简单的ngFor循环，它也跟踪当前的index. 我想将该index值存储在一个属性中，以便打印它。但我无法弄清楚这是如何工作的。

我基本上有这个：

<ul *ngFor="#item of items; #i = index" data-index="#i">
    <li>{{item}}</li>
</ul>

我想将 的值存储#i在属性中data-index。我尝试了几种方法，但都没有用。

我在这里有一个演示：http://plnkr.co/edit/EXpOKAEIFlI9QwuRcZqp ?p=preview

如何将index值存储在data-index属性中？



解答

我将使用此语法将索引值设置为 HTML 元素的属性：

角度 >= 2

您必须使用let来声明值而不是#.

<ul>
    <li *ngFor="let item of items; let i = index" [attr.data-index]="i">
        {{item}}
    </li>
</ul>

角度 = 1

<ul>
    <li *ngFor="#item of items; #i = index" [attr.data-index]="i">
        {{item}}
    </li>
</ul>

这是更新的 plunkr：http://plnkr.co/edit/LiCeyKGUapS5JKkRWnUJ ?p=preview 。


更多建议

在 Angular 5/6/7/8 中：

<ul>
  <li *ngFor="let item of items; index as i">
    {{i+1}} {{item}}
  </li>
</ul>

在旧版本中

<ul *ngFor="let item of items; let i = index">
  <li>{{i+1}} {{item}}</li>
</ul>

C# 循环 - 中断与继续

Arya

2023-02-17 11:16

break在 C#（请随意回答其他语言）循环中，和continue作为离开循环结构并进入下一次迭代的手段之间有什么区别？

例子：

foreach (DataRow row in myTable.Rows)
{
    if (someConditionEvalsToTrue)
    {
        break; //what's the difference between this and continue ?
        //continue;
    }
}



解答

break将完全退出循环，continue将 跳过 当前迭代。

例如：

for (int i = 0; i < 10; i++) {
    if (i == 0) {
        break;
    }

    DoSomeThingWith(i);
}

break 将导致循环在第一次迭代时退出 -DoSomeThingWith永远不会执行。这里：

for (int i = 0; i < 10; i++) {
    if(i == 0) {
        continue;
    }

    DoSomeThingWith(i);
}

不会执行DoSomeThingWithfor i = 0，但循环会 继续 并DoSomeThingWith会执行 for i = 1to i = 9。


更多建议

理解这一点的一个真正简单的方法是在每个关键字之后放置“循环”一词。如果这些术语像日常用语一样阅读，那么它们现在就有意义了。

break loop - 循环中断并停止。

continue loop - 循环在下一次迭代中继续执行。

CSS强制图像调整大小并保持纵横比

Arya

2023-02-17 11:15

我正在处理图像，但遇到了纵横比问题。

<img src="big_image.jpg" width="900" height="600" alt="" />

如您所见，height和width已经指定。我为图像添加了一个 CSS 规则：

img {
  max-width: 500px;
}

但是对于big_image.jpg，我收到width=500和height=600。如何设置要调整大小的图像，同时保持其纵横比。



解答

img {
  display: block;
  max-width:230px;
  max-height:95px;
  width: auto;
  height: auto;
}


<p>This image is originally 400x400 pixels, but should get resized by the CSS:</p>
<img width="400" height="400" src="http://i.stack.imgur.com/aEEkn.png">

如果图像对于指定区域来说太大，这将使图像缩小（不利的是，它不会放大图像）。


更多建议

这是一个解决方案：

object-fit: cover;
width: 100%;
height: 250px;

您可以调整宽度和高度以满足您的需要，object-fit 属性将为您进行裁剪。

有关该属性的可能值object-fit和兼容性表的更多信息，请参见此处： https: //developer.mozilla.org/en- US/docs/Web/CSS/object-fit
