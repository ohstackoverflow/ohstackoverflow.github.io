private const int THE_ANSWER = 42;

或者

private const int theAnswer = 42;

我个人认为对于现代 IDE，我们应该使用驼峰命名法，因为 ALL_CAPS 看起来很奇怪。你怎么认为？



解答

推荐的命名和大写约定是对常量使用 Pascal C asing （Microsoft 有一个名为StyleCop的工具，它记录了所有首选约定，并且可以检查您的来源是否合规 - 尽管对于许多人的口味来说它有点过分 保留 ） . 例如

private const int TheAnswer = 42;

Pascal 大写约定也记录在 Microsoft 的Framework Design Guidelines中。


更多建议

在视觉上，大写是可行的方法。这种方式非常有辨识度。为了唯一性和不给猜测的机会，我投票给 UPPER_CASE！

const int THE_ANSWER = 42;

  • 方法 3 - table-cell/ vertical-align: middle：

此处示例 / 全屏示例

在某些情况下，您需要确保html/body元素的高度设置为100%。

对于垂直对齐，将父元素的width/设置height为100%并添加display: table. 然后对于子元素，将 更改为display并table-cell添加vertical-align: middle。

对于水平居中，您可以添加text-align: center以居中文本和任何其他inline子元素。或者，您可以使用margin: 0 auto，假设元素是block水平的。

html, body {
    height: 100%;
}
.parent {
    width: 100%;
    height: 100%;
    display: table;
    text-align: center;
}
.parent > .child {
    display: table-cell;
    vertical-align: middle;
}

  • 方法 4 -50%从顶部绝对定位位移：

此处示例 / 全屏示例

如果 CSS3 是一个选项（或者你有后备），你可以使用转换：

.center {
    right: 50%;
    bottom: 50%;
