删除、删除和弹出列表的区别

Arya

2023-02-17 04:50

这三种从列表中删除元素的方法有什么区别吗？

>>> a = [1, 2, 3]
>>> a.remove(2)
>>> a
[1, 3]

>>> a = [1, 2, 3]
>>> del a[1]
>>> a
[1, 3]

>>> a = [1, 2, 3]
>>> a.pop(1)
2
>>> a
[1, 3]



解答

从列表中删除元素的三种不同方法的效果：

remove删除 第一个 匹配 值 ，而不是特定索引：

>>> a = [0, 2, 3, 2]
>>> a.remove(2)
>>> a
[0, 3, 2]

del删除特定索引处的项目：

>>> a = [9, 8, 7, 6]
>>> del a[1]
>>> a
[9, 7, 6]

并pop删除特定索引处的项目并将其返回。

>>> a = [4, 3, 5]
>>> a.pop(1)
3
>>> a
[4, 5]

它们的错误模式也不同：

>>> a = [4, 5, 6]
>>> a.remove(7)
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: list.remove(x): x not in list
>>> del a[7]
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
>>> a.pop(7)
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: pop index out of range


更多建议

用于del按索引删除元素，pop()如果需要返回值则按索引删除元素，remove()按值删除元素。最后一个需要搜索列表，ValueError如果列表中没有出现这样的值则引发。

i从元素列表中删除索引时n，这些方法的计算复杂度是

del     O(n - i)
pop     O(n - i)
remove  O(n)

定义带或不带导出的变量

Arya

2023-02-17 04:50

是export为了什么？

有什么区别：

export name=value

和

name=value



解答

export使变量可用于子流程。

那是，

export name=value

意味着该变量名可用于您从该 shell 进程运行的 任何进程 。如果您希望某个进程使用此变量，请使用export并从该 shell 运行该进程。

name=value

意味着变量范围仅限于 shell，并且不可用于任何其他进程。您可以将其用于（比如说）循环变量、临时变量等。

请务必注意，导出变量不会使其对父进程可用。也就是说，在派生进程中指定和导出变量不会使其在启动它的进程中可用。


更多建议

为了说明其他答案在说什么：

$ foo="Hello, World"
$ echo $foo
Hello, World
$ bar="Goodbye"
$ export foo
$ bash
bash-3.2$ echo $foo
Hello, World
bash-3.2$ echo $bar

bash-3.2$

如何从命令行重新加载 .bash_profile

Arya

2023-02-17 04:49

如何从命令行重新 加载文件.bash_profile？ __

我可以通过退出并重新登录让 shell 识别对 .bash_profile 的更改，但我希望能够按需执行此操作。



解答

只需键入source ~/.bash_profile。

或者，如果您喜欢保存击键，您可以键入. ~/.bash_profile.


更多建议

. ~/.bash_profile

只要确保你对那里的当前状态没有任何依赖。
