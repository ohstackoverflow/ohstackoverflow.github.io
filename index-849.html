如何将文件读入shell中的变量？

Arya

2023-02-17 17:48

我想读取一个文件并将其保存在变量中，但我需要保留变量而不仅仅是打印出文件。我怎样才能做到这一点？我写了这个脚本，但它并不是我所需要的：

#!/bin/sh
while read LINE  
do  
  echo $LINE  
done <$1  
echo 11111-----------  
echo $LINE

在我的脚本中，我可以将文件名作为参数，因此，如果文件包含“aaaa”，例如，它会打印出：

aaaa
11111-----

但这只是将文件打印到屏幕上，我想将它保存到一个变量中！是否有捷径可寻？



解答

sh在跨平台中，您使用的最低公分母：

#!/bin/sh
value=`cat config.txt`
echo "$value"

在bashor中zsh，在不调用的情况下将整个文件读入变量cat：

#!/bin/bash
value=$(<config.txt)
echo "$value"

调用cat或吞噬文件将被视为对bashCat的无用使用。zsh

请注意，不必引用命令替换来保留换行符。

请参阅：Bash Hacker's Wiki - 命令替换 - 专业。


更多建议

如果你想将整个文件读入一个变量：

#!/bin/bash
value=`cat sources.xml`
echo $value

如果你想逐行阅读它：

while read line; do    
    echo $line    
done < file.txt

如何删除 Flutter 中的调试横幅？

Arya

2023-02-17 17:47

我正在使用flutter screenshot并且我希望屏幕截图没有横幅，但它有。

请注意，我收到一条not supported for emulator关于配置文件和发布模式的消息。



解答

在你的MaterialApp设置debugShowCheckedModeBanner上false。

MaterialApp(
  debugShowCheckedModeBanner: false,
)

调试标语也将在发布版本 _ 中_ 自动删除。


更多建议

过时的

  • 如果您使用的是 Android Studio，则可以在 Flutter Inspector 选项卡 → More Actions 中找到该选项。

  • 或者，如果您使用的是 Dart DevTools，您也可以在右上角找到相同的按钮。

如何从一个表中选择另一个表中不存在的所有记录？

Arya

2023-02-17 17:47

表 1 (id, 名称)
表 2 (id, 名称)

询问：

SELECT name   
FROM table2  
-- that are not in table1 already



解答

SELECT t1.name
FROM table1 t1
LEFT JOIN table2 t2 ON t2.name = t1.name
WHERE t2.name IS NULL

问 ：这里发生了什么？

A ：从概念上讲，我们从中选择所有行table1，并为每一行尝试找到table2具有相同name列值的行。如果没有这样的行，我们就将该table2行的结果部分留空。然后我们通过仅选择结果中不存在匹配行的那些行来限制我们的选择。最后，我们忽略结果中的所有字段，除了列name（我们确定存在的列，来自table1）。

虽然它可能不是在所有情况下都可能是性能最高的方法，但它应该适用于基本上所有尝试实施ANSI 92 SQL 的数据库引擎


更多建议

你可以做

SELECT name
FROM table2
WHERE name NOT IN
    (SELECT name 
     FROM table1)

或者

SELECT name 
FROM table2 
WHERE NOT EXISTS 
    (SELECT * 
     FROM table1 
     WHERE table1.name = table2.name)

请参阅此问题以了解实现此目的的 3 种技术
