从命令行列出所有环境变量

Arya

2023-02-17 05:32

是否可以从 Windows 的命令提示符中列出 所有环境变量？

等同于 PowerShell 的gci env:（或ls env:或dir env:）。



解答

做就是了：

SET

您还可以SET prefix查看名称以 开头的所有变量prefix。

例如，如果您只想从环境变量中读取 derbydb，请执行以下操作：

set derby

...你会得到以下信息：

DERBY_HOME=c:\Users\amro-a\Desktop\db-derby-10.10.1.1-bin\db-derby-10.10.1.1-bin


更多建议

乔恩有正确的答案，但要用一些语法糖来详细说明......

SET | more

使您能够一次一页地查看变量，而不是全部，或者

SET > output.txt

将输出发送到文件 output.txt 中，您可以在记事本或其他任何工具中打开该文件...

PHP数组按值删除（不是键）

Arya

2023-02-17 05:32

我有一个 PHP 数组，如下所示：

$messages = [312, 401, 1599, 3, ...];

我想删除包含该值的元素$del_val（例如，$del_val=401），但我不知道它的键。这可能会有所帮助： 每个值只能存在一次 。

我正在寻找执行此任务的最简单的功能。



解答

使用array_search()和unset，尝试以下操作：

if (($key = array_search($del_val, $messages)) !== false) {
    unset($messages[$key]);
}

array_search()返回它找到的元素的键，可用于使用从原始数组中删除该元素unset()。它会在失败时返回FALSE，但它可以在成功时返回 false-y 值（0例如，您的键可能是），这就是!==使用严格比较运算符的原因。

该if()语句将检查是否array_search()返回了一个值，如果返回了则只执行一个操作。


更多建议

好吧，从数组中删除一个元素基本上只是与一个元素设置差异。

array_diff( [312, 401, 15, 401, 3], [401] ) // removing 401 returns [312, 15, 3]

它概括得很好，如果你愿意，你可以同时删除任意数量的元素。

免责声明： 请注意，我的解决方案生成数组的新副本，同时保持旧副本完好无损，这与接受的变异答案形成对比。选择你需要的那个。

如何使用逗号作为千位分隔符打印数字

Arya

2023-02-17 05:31

如何使用逗号作为千位分隔符打印整数？

1234567   ⟶   1,234,567

在句点和逗号之间做出决定不需要特定于语言环境。



解答

语言环境未知

'{:,}'.format(value)  # For Python ≥2.7
f'{value:,}'          # For Python ≥3.6

语言环境感知

import locale
locale.setlocale(locale.LC_ALL, '')  # Use '' for auto, or force e.g. to 'en_US.UTF-8'

'{:n}'.format(value)  # For Python ≥2.7
f'{value:n}'          # For Python ≥3.6

参考

根据格式规范迷你语言，

该','选项表示对千位分隔符使用逗号。对于区域设置感知分隔符，请改用'n'整数表示类型。


更多建议

我很惊讶没有人提到您可以像这样简单地在 Python 3.6+ 中使用 f-strings 做到这一点：

>>> num = 10000000
>>> print(f"{num:,}")
10,000,000

...冒号后的部分是格式说明符。逗号是您想要的分隔符，因此f"{num:_}"使用下划线而不是逗号。此方法只能使用“,”和“_”。

这相当于使用format(num, ",")旧版本的 python 3。

当您第一次看到它时，这可能看起来很神奇，但事实并非如此。它只是语言的一部分，并且通常需要足够的东西来提供快捷方式。要了解更多信息，请查看组子组件。
