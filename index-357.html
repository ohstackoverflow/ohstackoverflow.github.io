如何在 Bash 中比较字符串

Arya

2023-02-17 04:53

如何将变量与字符串进行比较（如果它们匹配则执行某些操作）？



解答

在 if 语句中使用变量

if [ "$x" = "valid" ]; then
  echo "x has the value 'valid'"
fi

如果您想在它们不匹配时执行某些操作，请替换=为!=. 您可以在各自的文档中阅读有关字符串运算和算术运算的更多信息。

为什么我们使用引号$x？

你想要引号$x，因为如果它是空的，你的 Bash 脚本会遇到语法错误，如下所示：

if [ = "valid" ]; then

==运算符使用不规范

请注意，Bash 允许==用于与 相等[，但这不是标准的。

使用第一种情况，其中引号$x是可选的：

if [[ "$x" == "valid" ]]; then

或使用第二种情况：

if [ "$x" = "valid" ]; then


更多建议

或者，如果您不需要 else 子句：

[ "$x" == "valid" ] && echo "x has the value 'valid'"

如何按日期属性对对象数组进行排序？

Arya

2023-02-17 04:52

假设我有一些对象的数组：

var array = [{id: 1, date: Mar 12 2012 10:00:00 AM}, {id: 2, date: Mar 8 2012 08:00:00 AM}];

如何从最接近当前日期和时间的日期开始按日期元素对该数组进行排序？请记住，数组可能有很多对象，但为了简单起见，我使用了 2 个。

我会使用排序功能和自定义比较器吗？



解答

最简单的答案

array.sort(function(a,b){
  // Turn your strings into dates, and then subtract them
  // to get a value that is either negative, positive, or zero.
  return new Date(b.date) - new Date(a.date);
});

更通用的答案

array.sort(function(o1,o2){
  if (sort_o1_before_o2)    return -1;
  else if(sort_o1_after_o2) return  1;
  else                      return  0;
});

或者更简洁：

array.sort(function(o1,o2){
  return sort_o1_before_o2 ? -1 : sort_o1_after_o2 ? 1 : 0;
});

通用的，强大的答案

在所有数组上sortBy使用Schwartzian 变换定义自定义不可枚举函数：

(function(){
  if (typeof Object.defineProperty === 'function'){
    try{Object.defineProperty(Array.prototype,'sortBy',{value:sb}); }catch(e){}
  }
  if (!Array.prototype.sortBy) Array.prototype.sortBy = sb;

  function sb(f){
    for (var i=this.length;i;){
      var o = this[--i];
      this[i] = [].concat(f.call(o,o,i),o);
    }
    this.sort(function(a,b){
      for (var i=0,len=a.length;i<len;++i){
        if (a[i]!=b[i]) return a[i]<b[i]?-1:1;
      }
      return 0;
    });
    for (var i=this.length;i;){
      this[--i]=this[i][this[i].length-1];
    }
    return this;
  }
})();

像这样使用它：

array.sortBy(function(o){ return o.date });

如果您的日期不能直接比较，请从中制作一个可比较的日期，例如

array.sortBy(function(o){ return new Date( o.date ) });

如果返回值数组，也可以使用它按多个条件排序：

// Sort by date, then score (reversed), then name
array.sortBy(function(o){ return [ o.date, -o.score, o.name ] };

有关详细信息， 请参阅http://phrogz.net/JS/Array.prototype.sortBy.js 。


更多建议

@Phrogz 的答案都很好，但这里有一个很棒的、更简洁的答案：

array.sort(function(a,b){return a.getTime() - b.getTime()});

使用箭头函数方式

array.sort((a,b)=>a.getTime()-b.getTime());

如何在 Node.js 中进行 Base64 编码？

Arya

2023-02-17 04:52

Node.js 是否内置了 Base64 编码？

之所以这么问是因为final()fromcrypto只能输出十六进制、二进制或ASCII数据。例如：

var cipher = crypto.createCipheriv('des-ede3-cbc', encryption_key, iv);
var ciph = cipher.update(plaintext, 'utf8', 'hex');
ciph += cipher.final('hex');

var decipher = crypto.createDecipheriv('des-ede3-cbc', encryption_key, iv);
var txt = decipher.update(ciph, 'hex', 'utf8');
txt += decipher.final('utf8');

根据文档，update()可以输出Base64编码的数据。但是，final()不支持 Base64。我试过了，它会坏的。

如果我这样做：

var ciph = cipher.update(plaintext, 'utf8', 'base64');
    ciph += cipher.final('hex');

那我应该用什么来解密呢？十六进制还是 Base64？

因此，我正在寻找一个函数来对我的加密十六进制输出进行 Base64 编码。



解答

缓冲区可用于获取字符串或数据片段并对结果进行 Base64 编码。例如：

> console.log(Buffer.from("Hello World").toString('base64'));
SGVsbG8gV29ybGQ=
> console.log(Buffer.from("SGVsbG8gV29ybGQ=", 'base64').toString('ascii'))
Hello World

缓冲区是一个全局对象，因此不需要 require。使用字符串创建的缓冲区可以采用可选的编码参数来指定字符串的编码方式。可用编码toString和Buffer构造函数编码如下：

'ascii' - 仅适用于 7 位 ASCII 数据。这种编码方式速度非常快，设置后会去掉高位。

'utf8' - 多字节编码的 Unicode 字符。许多网页和其他文档格式都使用 UTF-8。

'ucs2' - 2 字节，小端编码的 Unicode 字符。它只能编码 BMP（基本多语言平面，U+0000 - U+FFFF）。

'base64' - Base64 字符串编码。

'binary' - 一种通过仅使用每个字符的前 8 位将原始二进制数据编码为字符串的方法。此编码方法已弃用，应尽可能避免使用 Buffer 对象。这种编码将在 Node.js 的未来版本中被删除。


更多建议

The accepted answer previously contained new Buffer()，这在大于 6 的 Node.js 版本中被认为是一个安全问题（尽管对于这个用例来说输入似乎总是可以被强制转换为字符串）。

Buffer根据文档，构造函数已弃用。

代码片段应为：

console.log(Buffer.from("Hello World").toString('base64'));
console.log(Buffer.from("SGVsbG8gV29ybGQ=", 'base64').toString('ascii'));

写完这个答案后，它已经更新，现在匹配这个。
