2023-02-17 03:03

有没有一种简单的方法可以在 C# 中创建多行字符串文字？

这是我现在拥有的：

string query = "SELECT foo, bar"
+ " FROM table"
+ " WHERE id = 42";

我知道 PHP 有

<<<BLOCK

BLOCK;

C#有类似的东西吗？



解答

您可以使用@a 前面的符号string来形成逐字字符串文字：

string query = @"SELECT foo, bar
FROM table
WHERE id = 42";

使用此方法时，您也不必转义特殊字符，但 Jon Skeet 的回答中所示的双引号除外。


更多建议

它在 C# 中称为 逐字字符串文字 ，只需将 @ 放在文字之前即可。这不仅允许多行，而且还关闭了转义。所以例如你可以这样做：

string query = @"SELECT foo, bar
FROM table
WHERE name = 'a\b'";

但是，这包括字符串中的换行符（使用您的源中的任何换行符）。对于 SQL，这不仅无害，而且可能会 提高 您看到字符串的任何地方的可读性 - 但在其他地方可能不需要它，在这种情况下，您要么不需要使用多行逐字字符串文字，或从结果字符串中删除它们。

唯一的转义是，如果你想要一个双引号，你必须添加一个额外的双引号：

string quote = @"Jon said, ""This will work,"" - and it did!";

如何获取当前文件目录的完整路径？

Arya

2023-02-17 03:03

如何获取当前文件的目录路径？我试过：

>>> os.path.abspath(__file__)
'C:\\python27\\test.py'

但我想要：

'C:\\python27\\'



解答

特殊变量__file__包含当前文件的路径。pathlib从那里我们可以使用任一模块获取目录os.path。

蟒蛇3

对于正在运行的脚本的目录：

import pathlib
pathlib.Path(__file__).parent.resolve()

对于当前工作目录：

import pathlib
pathlib.Path().resolve()

Python 2 和 3

对于正在运行的脚本的目录：

import os
os.path.dirname(os.path.abspath(__file__))

如果您指的是当前工作目录：

import os
os.path.abspath(os.getcwd())

注意before和afterfile是两个下划线，而不是一个。

另请注意，如果您以交互方式运行或从文件以外的其他内容（例如：数据库或在线资源）加载代码，则__file__可能无法设置，因为没有“当前文件”的概念。上面的答案假设了运行文件中的 python 脚本的最常见场景。

参考

  1. python文档中的pathlib 。
  2. os.path - Python 2.7，os.path - Python 3
  3. os.getcwd - Python 2.7，os.getcwd - Python 3
  4. file 变量是什么意思/做什么？


更多建议

从 Python 3 开始，推荐使用Pathfrom ：pathlib

from pathlib import Path
print("File      Path:", Path(__file__).absolute())
print("Directory Path:", Path().absolute()) # Directory of current working directory, not __file__

注意： 如果使用 Jupyter Notebook，则__file__不会返回预期值，因此Path().absolute()必须使用。

在 JavaScript 中修剪字符串

Arya

2023-02-17 03:02

如何删除字符串开头和结尾的所有空格？



解答

自 IE9+ 以来的所有浏览器都有trim()字符串方法：

" \n test \n ".trim(); // returns "test" here

对于那些不支持的浏览器，您可以使用来自MDNtrim()的这个 polyfill ：

if (!String.prototype.trim) {
    (function() {
        // Make sure we trim BOM and NBSP
        var rtrim = /^[\s\uFEFF\xA0]+|[\s\uFEFF\xA0]+$/g;
        String.prototype.trim = function() {
            return this.replace(rtrim, '');
        };
    })();
}

也就是说，如果使用jQuery,$.trim(str)也可用并处理未定义/空。

看到这个：

String.prototype.trim=function(){return this.replace(/^\s+|\s+$/g, '');};

String.prototype.ltrim=function(){return this.replace(/^\s+/,'');};

String.prototype.rtrim=function(){return this.replace(/\s+$/,'');};

String.prototype.fulltrim=function(){return this.replace(/(?:(?:^|\n)\s+|\s+(?:$|\n))/g,'').replace(/\s+/g,' ');};


更多建议

如果您已经在使用该框架，则jQuery的 trim会很方便。

$.trim('  your string   ');
