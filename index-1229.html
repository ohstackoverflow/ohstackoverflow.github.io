如何在正则表达式中匹配多行中的任何字符？

Arya

2023-02-18 04:36

例如，这个正则表达式

(.*)<FooBar>

将匹配：

abcde<FooBar>

但是我如何让它匹配多行呢？

abcde
fghij<FooBar>



解答

这取决于语言，但应该有一个可以添加到正则表达式模式的修饰符。在 PHP 中是：

/(.*)<FooBar>/s

末尾的 s 使点匹配所有 字符 ， 包括换行符。 __

从值中枚举字符串名称

Arya

2023-02-18 04:35

我有一个这样的枚举结构：

public enum EnumDisplayStatus
{
    None    = 1,
    Visible = 2,
    Hidden  = 3,
    MarkedForDeletion = 4
}

在我的数据库中，枚举是按值引用的。我的问题是，如何将枚举的数字表示形式转回字符串名称。

例如，给定的2结果应该是Visible。



解答

您可以int通过简单的强制转换将 back 转换为枚举成员，然后调用ToString()：

int value = GetValueFromDb();
var enumDisplayStatus = (EnumDisplayStatus)value;
string stringValue = enumDisplayStatus.ToString();


更多建议

"Visible"如果您需要在不获取实例的情况下获取字符串，EnumDisplayStatus您可以这样做：

int dbValue = GetDBValue();
string stringValue = Enum.GetName(typeof(EnumDisplayStatus), dbValue);

如何在 jQuery 中定义多个 CSS 属性？

Arya

2023-02-18 04:35

jQuery 中是否有任何句法方式来定义多个 CSS 属性，而无需像这样将所有内容都串到右边：

$("#message").css("width", "550px").css("height", "300px").css("font-size", "8pt");

如果您有 20 个这样的代码，您的代码将变得难以阅读，有什么解决方案吗？

例如，从 jQuery API 中，jQuery 理解并为两者返回正确的值

.css({ "background-color": "#ffe", "border-left": "5px solid #ccc" })

和

.css({backgroundColor: "#ffe", borderLeft: "5px solid #ccc" }).

请注意，对于 DOM 表示法，属性名称周围的引号是 可选的 ，但对于 CSS 表示法，由于名称中的连字符，引号是 必需的。



解答

最好只使用.addClass()，.removeClass()即使您有 1 种或多种样式要更改。它更具可维护性和可读性。

如果你真的有做多个 CSS 属性的冲动，那就用下面的：

.css({
   'font-size' : '10px',
   'width' : '30px',
   'height' : '10px'
});

注意！ 任何带有连字符的
CSS 属性都需要被引用。我已经放置了引号，因此没有人需要澄清这一点，并且代码将 100% 起作用。 __


更多建议

将其作为对象传递：

$(....).css({
    'property': 'value', 
    'property': 'value'
});

http://docs.jquery.com/CSS/css#properties
