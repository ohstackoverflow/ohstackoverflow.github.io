在 LaTeX 中插入 PDF 文件

Arya

2023-02-17 20:18

我想在我的 LaTeX 文件中插入一个 PDF 或 doc 文件作为附录。你知道我该怎么做吗？



解答

使用pdfpages包。

\usepackage{pdfpages}

要在 PDF 文件中包含所有页面：

\includepdf[pages=-]{myfile.pdf}

仅包含 PDF 的第一页：

\includepdf[pages={1}]{myfile.pdf}

在 shell 中运行texdoc pdfpages以查看pdfpages.


更多建议

要将整个 pdf 文件放入您的文件中，而不仅仅是一页，请使用：

\usepackage{pdfpages}

\includepdf[pages=-]{myfile.pdf}

如何更改 MySQL 中列的数据类型？

Arya

2023-02-17 20:18

我想将多列的数据类型从 float 更改为 int。最简单的方法是什么？

目前还没有数据值得担心。



解答

http://dev.mysql.com/doc/refman/5.1/en/alter-table.html

ALTER TABLE tablename MODIFY columnname INTEGER;

这将更改给定列的数据类型

根据您希望修改的列数，最好生成一个脚本，或使用某种 mysql 客户端 GUI


更多建议

alter table table_name modify column_name int(5)

在 Razor 视图引擎中转义 @ 字符

Arya

2023-02-17 20:18

我正在使用 Razor 作为视图引擎创建一个示例 ASP.NET MVC 3 站点。razor 语法以@字符 eg开头@RenderBody()。如果我在我的 cshtml 页面上写 @test 它会给我解析错误

CS0103：当前上下文中不存在名称“test”

如何转义“@”字符？



解答

@@应该这样做。


更多建议

Razor @ 将字符转义为符号...

<img src="..." alt="Find me on twitter as @("@username")" />

或者

<img src="..." alt="Find me on twitter as @("@")username" />
