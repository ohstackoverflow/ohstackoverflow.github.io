C 中的 size_t 是什么？

Arya

2023-02-17 13:51

我在 C 中感到困惑。size_t我知道它是由sizeof操作员返回的。但它到底是什么？它是一种数据类型吗？

假设我有一个for循环：

for(i = 0; i < some_size; i++)

我应该使用int i;还是size_t i;？



解答

来自维基百科：

根据 1999 ISO C 标准 (C99)，size_t是至少 16 位的无符号整数类型（请参阅第 7.17 和 7.18.3 节）。

size_t是由多个 C/C++ 标准定义的无符号数据类型，例如 C99 ISO/IEC 9899 标准，在stddef.h. 1它可以通过包含进一步导入，stdlib.h因为该文件内部包含 sub includes stddef.h。

此类型用于表示对象的大小。采用或返回大小的库函数期望它们的类型或返回类型为size_t. 此外，最常用的基于编译器的运算符 sizeof 的计算结果应为与兼容的常量值size_t。

言外之意，size_t是一种保证可以保存任何数组索引的类型。


更多建议

size_t是无符号类型。因此，它不能表示任何负值（<0）。您在数数时使用它，并确保它不会为负数。例如，strlen()返回 asize_t因为字符串的长度必须至少为 0。

在您的示例中，如果您的循环索引将始终大于 0，则使用size_t, 或任何其他无符号数据类型可能是有意义的。

当你使用一个size_t对象时，你必须确保在所有使用它的上下文中，包括算术，你需要非负值。例如，假设您有：

size_t s1 = strlen(str1);
size_t s2 = strlen(str2);

str2并且您想找出和的长度之差str1。你不能这样做：

int diff = s2 - s1; /* bad */

这是因为分配给的值diff始终为正数，即使在 时也是如此s2 < s1，因为计算是使用无符号类型完成的。在这种情况下，根据您的用例，您最好使用int(or long long) for s1and s2。

C/POSIX 中有一些函数可以/应该使用size_t，但由于历史原因而不使用。例如，fgets理想情况下，第二个参数应该是size_t，但实际上是int。

如何将命令的输出直接复制到剪贴板？

Arya

2023-02-17 13:50

如何将命令的输出通过管道传输到剪贴板并在使用终端时将其粘贴回去？例如：

cat file | clipboard



解答

一种方法如下：

  1. 安装xclip，例如：

sudo apt-get 安装 xclip

  1. 通过管道xclip将输出复制到剪贴板：

猫文件 | 剪辑

  1. 将刚刚复制的文本粘贴到X应用程序中：

xclip -o

要粘贴应用程序以外的其他位置X，例如浏览器窗口中网页的文本区域，请使用：

cat file | xclip -selection clipboard

考虑创建别名：

alias "c=xclip"
alias "v=xclip -o"

要了解这有多大用处，假设我想在一个新的终端窗口中打开我当前的路径（在某些系统上可能还有其他方法，比如Ctrl+ T，但这只是为了说明目的）：

Terminal 1:
pwd | c

Terminal 2:
cd `v`

注意周围v。这首先作为命令执行v，然后就地替换它以供cd使用。

仅将内容复制到X剪贴板

cat file | xclip


更多建议

在 OS X 上，使用pbcopy; pbpaste朝相反的方向前进。

pbcopy < .ssh/id_rsa.pub

重命名列 SQL Server 2008

Arya

2023-02-17 13:48

我正在使用 SQL Server 2008 和 Navicat。我需要使用 SQL 重命名表中的列。

ALTER TABLE table_name RENAME COLUMN old_name to new_name;

这个说法是行不通的。



解答

使用sp_rename

EXEC sp_RENAME 'TableName.OldColumnName' , 'NewColumnName', 'COLUMN'

请参阅：SQL SERVER – 如何重命名列名或表名

文档： sp_rename (Transact-SQL)

对于您的情况，它将是：

EXEC sp_RENAME 'table_name.old_name', 'new_name', 'COLUMN'

请记住使用单引号将您的值括起来。
