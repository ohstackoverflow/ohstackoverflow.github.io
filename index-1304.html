展平不规则（任意嵌套）的列表列表

Arya

2023-02-18 06:24

是的，我知道之前已经讨论过这个主题：

  • Python 成语链接（展平）有限迭代的无限迭代？
  • 在 Python 中展平浅列表
  • 对序列序列展平的理解？
  • 如何从列表列表中制作平面列表？

但据我所知，除一个解决方案外，所有解决方案都在列表中失败[[[1, 2, 3], [4, 5]], 6]，其中所需的输出是[1, 2, 3, 4, 5, 6]（或者甚至更好，一个迭代器）。

我看到的唯一适用于任意嵌套的解决方案是在这个问题中找到的：

def flatten(x):
    result = []
    for el in x:
        if hasattr(el, "__iter__") and not isinstance(el, basestring):
            result.extend(flatten(el))
        else:
            result.append(el)
    return result

这是最好的方法吗？我忽略了什么吗？任何问题？



解答

使用生成器函数可以使您的示例更易于阅读并提高性能。

蟒蛇2

使用2.6中添加的IterableABC：

from collections import Iterable

def flatten(xs):
    for x in xs:
        if isinstance(x, Iterable) and not isinstance(x, basestring):
            for item in flatten(x):
                yield item
        else:
            yield x

蟒蛇3

在 Python 3 中，basestring不再有，但元组(str, bytes)具有相同的效果。此外，yield from操作员一次从生成器返回一个项目。

from collections.abc import Iterable

def flatten(xs):
    for x in xs:
        if isinstance(x, Iterable) and not isinstance(x, (str, bytes)):
            yield from flatten(x)
        else:
            yield x


更多建议

我的解决方案：

import collections


def flatten(x):
    if isinstance(x, collections.Iterable):
        return [a for i in x for a in flatten(i)]
    else:
        return [x]

更简洁一点，但几乎相同。

如何将来自其他分支的未合并的上游拉取请求应用到我的分支中？

Arya

2023-02-18 06:24

我有一个分支的 GitHub 上的一个项目有一个新的拉取请求，我想将其拉入我的分支，但作者尚未拉入。

有没有一种简单的方法可以将来自其他分叉的拉取请求应用到我的分叉中？这里还有什么我想念的吗？



解答

您可以很容易地手动完成：

  • 添加另一个分支作为你的仓库的远程：

    git remote add otherfork git://github.com/request-author/project.git

  • 获取他的 repo 的提交

    git fetch otherfork

  • 然后，您有两个选项来应用拉取请求（如果您不想选择选择 1。）

    1. 如果您不关心还应用在源和拉取请求之间添加的最终提交，您可以只重新设置拉取请求形成的分支

          git rebase master otherfork/pullrequest-branch

    2. 如果您只想要拉取请求中的提交，请识别他们的 SHA1 并执行

          git cherry-pick <first-SHA1> <second-SHA1> <etc.>
