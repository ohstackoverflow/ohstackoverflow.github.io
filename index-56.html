如何使用 将文件夹从远程主机复制到本地主机scp？

我用来ssh登录我的服务器。
然后，我想将远程文件夹复制foo到本地/home/user/Desktop。

我如何实现这一目标？



解答

scp -r user@your.server.example.com:/path/to/foo /home/user/Desktop/

通过不包括 foo 末尾的尾随 '/'，您将复制目录本身（包括内容），而不仅仅是目录的内容。

来自man scp（参见在线手册）

-r 递归复制整个目录


更多建议

要使用scp 的全部功能，您需要完成以下步骤：

  1. 公钥授权
  2. 创建 SSH 别名

然后，例如，如果你有这个 ~/.ssh/config ：

Host test
    User testuser
    HostName test-site.example
    Port 22022

Host prod
    User produser
    HostName production-site.example
    Port 22022

您将免于输入密码并简化 scp 语法，如下所示：

scp -r prod:/path/foo /home/user/Desktop   # copy to local
scp -r prod:/path/foo test:/tmp            # copy from remote prod to remote test

此外，您将能够使用远程路径补全：

scp test:/var/log/  # press tab twice
Display all 151 possibilities? (y or n)

要启用远程 bash-completion，您需要在主机<source>和<target>主机上都有 bash-shell，并且 bash- completion 可以正常工作。有关更多信息，请参阅相关问题：

使用 scp 时如何为远程路径启用自动完成？

如何以易于阅读（对于人类读者）的格式显示 JSON？我主要寻找缩进和空格，甚至可能有颜色/字体样式/等。



解答

漂亮打印在JSON.stringify(). 第三个参数启用漂亮的打印并设置要使用的间距：

var str = JSON.stringify(obj, null, 2); // spacing level = 2

如果你需要语法高亮，你可以像这样使用一些正则表达式魔术：

function syntaxHighlight(json) {
    if (typeof json != 'string') {
         json = JSON.stringify(json, undefined, 2);
    }
    json = json.replace(/&/g, '&amp;').replace(/</g, '&lt;').replace(/>/g, '&gt;');
    return json.replace(/("(\\u[a-zA-Z0-9]{4}|\\[^u]|[^\\"])*"(\s*:)?|\b(true|false|null)\b|-?\d+(?:\.\d*)?(?:[eE][+\-]?\d+)?)/g, function (match) {
        var cls = 'number';
        if (/^"/.test(match)) {
            if (/:$/.test(match)) {
                cls = 'key';
            } else {
                cls = 'string';
            }
        } else if (/true|false/.test(match)) {
            cls = 'boolean';
        } else if (/null/.test(match)) {
            cls = 'null';
        }
        return '<span class="' + cls + '">' + match + '</span>';
    });
}

如何在 Python 中连接两个列表？

Arya

2023-02-16 19:52

这个问题的答案是 社区的努力。编辑现有答案以改进这篇文章。它目前不接受新的答案或互动。

如何在 Python 中连接两个列表？

例子：

listone = [1, 2, 3]
listtwo = [4, 5, 6]

预期结果：

>>> joinedlist
[1, 2, 3, 4, 5, 6]



解答

使用+运算符组合列表：

listone = [1, 2, 3]
listtwo = [4, 5, 6]

joinedlist = listone + listtwo

输出：

>>> joinedlist
[1, 2, 3, 4, 5, 6]


更多建议

Python >= 3.5 替代方案：[*l1, *l2]

通过接受PEP 448引入了另一种替代方案，值得一提。

PEP，标题为 _ Additional Unpacking Generalizations_* ，在 Python 中使用星号表达式时通常减少了一些句法限制；有了它，加入两个列表（适用于任何可迭代的）现在也可以通过以下方式完成：

>>> l1 = [1, 2, 3]
>>> l2 = [4, 5, 6]
>>> joined_list = [*l1, *l2]  # unpack both iterables in a list literal
>>> print(joined_list)
[1, 2, 3, 4, 5, 6]

此功能 是 为 Python 3.5 定义的，但尚未向后移植到 3.x 系列中的先前版本。在不受支持的版本中，aSyntaxError将被提升。

与其他方法一样，这也会创建相应列表中元素的 浅表副本。

这种方法的好处 是你真的不需要列表来执行它；任何可迭代的都可以。正如 PEP 中所述：

这作为一种将可迭代对象汇总到列表中的更具可读性的方式也很有用，例如my_list + list(my_tuple) + list(my_range)which 现在等同于 just [*my_list, *my_tuple, *my_range]。

因此，虽然添加 with会由于类型不匹配+而引发：TypeError

l = [1, 2, 3]
r = range(4, 7)
res = l + r

以下不会：

res = [*l, *r]

因为它会首先解压 iterables 的内容，然后简单地list从内容中创建一个。
