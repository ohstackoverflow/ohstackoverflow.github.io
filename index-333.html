使用值列表从 Pandas 数据框中选择行

Arya

2023-02-17 04:16

假设我有以下 Pandas 数据框：

df = DataFrame({'A' : [5,6,3,4], 'B' : [1,2,3, 5]})
df

     A   B
0    5   1
1    6   2
2    3   3
3    4   5

我可以根据特定值进行子集化：

x = df[df['A'] == 3]
x

     A   B
2    3   3

但是我如何根据值列表进行子集化？- 是这样的：

list_of_values = [3,6]

y = df[df['A'] in list_of_values]

要得到：

     A    B
1    6    2
2    3    3



解答

您可以使用以下isin方法：

In [1]: df = pd.DataFrame({'A': [5,6,3,4], 'B': [1,2,3,5]})

In [2]: df
Out[2]:
   A  B
0  5  1
1  6  2
2  3  3
3  4  5

In [3]: df[df['A'].isin([3, 6])]
Out[3]:
   A  B
1  6  2
2  3  3

并获得相反的用途~：

In [4]: df[~df['A'].isin([3, 6])]
Out[4]:
   A  B
0  5  1
3  4  5


更多建议

您可以使用方法查询：

df.query('A in [6, 3]')
# df.query('A == [6, 3]')

或者

lst = [6, 3]
df.query('A in @lst')
# df.query('A == @lst')

在 Java 中将数组转换为列表

Arya

2023-02-17 04:16

如何将数组转换为 Java 中的列表？

我使用了，Arrays.asList()但是行为（和签名）以某种方式从 Java SE 1.4.2（文档现在在存档中）更改为8，我在网上找到的大多数片段都使用 1.4.2 行为。

例如：

int[] numbers = new int[] { 1, 2, 3 };
Arrays.asList(numbers)
  • 在 1.4.2 上返回包含元素 1、2、3 的列表
  • 在 1.5.0+ 上返回一个包含数组“numbers”的列表

在许多情况下，它应该很容易被发现，但有时它可能会被忽视：

Assert.assertTrue(Arrays.asList(numbers).indexOf(4) == -1);



解答

在您的示例中，这是因为您不能拥有原始类型的列表。换句话说，List<int>是不可能的。

但是，您可以使用List<Integer>包装Integer原语的类int。List使用实用程序方法将数组转换为 a Arrays.asList。

Integer[] numbers = new Integer[] { 1, 2, 3 };
List<Integer> list = Arrays.asList(numbers);

在 IdeOne.com 上查看此代码的实时运行。


更多建议

在 Java 8 中，您可以使用流：

int[] numbers = new int[] { 1, 2, 3 };
Arrays.stream(numbers)
      .boxed()
      .collect(Collectors.toList());

原始答案

您可以通过简单地浏览设备列表并检查useragent匹配项是否像这样来做到这一点：

  function detectMob() {
    const toMatch = [
        /Android/i,
        /webOS/i,
        /iPhone/i,
        /iPad/i,
        /iPod/i,
        /BlackBerry/i,
        /Windows Phone/i
    ];

    return toMatch.some((toMatchItem) => {
        return navigator.userAgent.match(toMatchItem);
    });
}

然而，由于您认为这种方法不可靠，您可以假设任何分辨率为 800x600 或更小的设备也是移动设备，从而进一步缩小您的目标范围（尽管如今许多移动设备的分辨率比这大得多）

IE

  function detectMob() {
    return ( ( window.innerWidth <= 800 ) && ( window.innerHeight <= 600 ) );
  }
