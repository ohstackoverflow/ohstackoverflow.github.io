获取触发事件的元素的 ID

Arya

2023-02-17 06:11

有没有办法获取触发事件的元素的 ID？

我在想这样的事情：

$(document).ready(function() {
  $("a").click(function() {
    var test = caller.id;
    alert(test.val());
  });
});


<script type="text/javascript" src="starterkit/jquery.js"></script>

<form class="item" id="aaa">
  <input class="title"></input>
</form>
<form class="item" id="bbb">
  <input class="title"></input>
</form>

运行代码片段隐藏结果

当然，除了 vartest应该包含 id "aaa"，如果事件是从第一种形式触发的，并且"bbb"，如果事件是从第二种形式触发的。



解答

在 jQuery 中event.target总是引​​用触发事件的元素，event传递给函数的参数在哪里。http://api.jquery.com/category/events/event- object/

$(document).ready(function() {
    $("a").click(function(event) {
        alert(event.target.id);
    });
});

另请注意，它this也可以工作，但它不是 jQuery 对象，因此如果您希望在其上使用 jQuery 函数，则必须将其称为 ，例如$(this)：

$(document).ready(function() {
    $("a").click(function(event) {
        // this.append wouldn't work
        $(this).append(" Clicked");
    });
});

    var contentPanelId = jQuery(this).attr("id");
    alert(contentPanelId);

如何使 --no-ri --no-rdoc 成为 gem 安装的默认设置？

Arya

2023-02-17 06:10

我不使用我安装在我的机器或我处理的服务器中的 gem 的 RI 或 RDoc 输出（我使用其他文档方式）。

默认情况下，我安装的每个 gem 都会安装 RI 和 RDoc 文档，因为我忘记设置--no-ri --no-rdoc.

有没有办法让这两个标志成为默认标志？



解答

您只需将以下行添加到您的本地~/.gemrc文件（它在您的 主 文件夹中）：

gem: --no-document

经过

echo 'gem: --no-document' >> ~/.gemrc

或者您可以将此行添加到全局gemrc配置文件中。

这是找到它的方法（在 Linux 中）：

strace gem source 2>&1 | grep gemrc

该--no-document选项记录在RubyGems CLI Reference中。


更多建议

来自RVM 的文档：

只需将此行添加到您的~/.gemrcor /etc/gemrc：

gem: --no-document

注： 原答案是：

install: --no-rdoc --no-ri 
update: --no-rdoc --no-ri

这不再有效；RVM 文档已经更新，因此当前仅包含gem指令的答案是正确的。

如何在 Java 中生成 MD5 哈希？

Arya

2023-02-17 06:10

有什么方法可以在 Java 中生成字符串的 MD5 哈希值吗？



解答

你需要java.security.MessageDigest。

调用MessageDigest.getInstance("MD5")以获取您可以使用的 MD5 实例MessageDigest。

通过执行以下操作之一计算哈希：

  • 将整个输入作为 a 提供byte[]，并使用 计算一次操作中的哈希值md.digest(bytes)。
  • 通过调用一次喂一个MessageDigest块。添加完输入字节后，使用 计算哈希值。byte[]``md.update(bytes)``md.digest()

返回byte[]的md.digest()是 MD5 哈希。
