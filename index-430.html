我如何从 git stash 中提取单个文件（或对文件的更改）？

Arya

2023-02-17 06:45

是否可以在不弹出存储变更集的情况下从 git 存储中提取单个文件或文件的差异？



解答

在git stash联机帮助页上，您可以阅读（在“讨论”部分，就在“选项”描述之后）：

存储表示为一个提交，其树记录了工作目录的状态，其第一个父级是创建存储时位于 HEAD 的提交。

因此，您可以将存储（例如stash@{0}是第一个/最顶层的存储）视为合并提交，并使用：

$ git diff stash@{0}^1 stash@{0} -- <filename>

说明：stash@{0}^1表示给定存储的第一个父级，如上面的解释所述，它是存储更改的提交。我们使用这种形式的“git diff”（有两个提交），因为stash@{0}/refs/stash是一个合并提交，我们必须告诉 git 我们想要比较哪个父级。更神秘：

$ git diff stash@{0}^! -- <filename>

也应该有效（请参阅git rev-parse联机帮助页以了解语法说明rev^!，在“指定范围”部分中）。

同样，您可以使用git checkout从存储中检查单个文件：

$ git checkout stash@{0} -- <filename>

或者以另一个文件名保存它：

$ git show stash@{0}:<full filename>  >  <newfile>

或者

$ git show stash@{0}:./<relative filename> > <newfile>

（ 请注意 ，此处 是相对于项目顶级目录的文件的完整路径名（认为：相对于stash@{0}））。

您可能需要防止stash@{0}shell 扩展，即使用"stash@{0}"or 'stash@{0}'。


更多建议

如果您使用git stash apply而不是git stash pop，它会将存储应用到您的工作树，但仍保留存储。

完成后，您可以add/commit您想要的文件，然后重置剩余的更改。

如何在 HTML 中显示 Base64 图像

Arya

2023-02-17 06:44

我在显示内联 Base64 图像时遇到问题。

我该怎么做？

<!DOCTYPE html>
<html>
  <head>
    <title>Display Image</title>
  </head>
  <body>
    <img style='display:block; width:100px;height:100px;' id='base64image'
       src='data:image/jpeg;base64, LzlqLzRBQ... <!-- Base64 data -->' />
  </body>
</html>



解答

我的怀疑当然是实际的 Base64 数据。否则它对我来说很好看。请参阅此小提琴，其中有类似的方案正在运行。您可以尝试指定字符集。

<div>
  <p>Taken from wikpedia</p>
  <img src="data:image/png;base64, iVBORw0KGgoAAAANSUhEUgAAAAUA
    AAAFCAYAAACNbyblAAAAHElEQVQI12P4//8/w38GIAXDIBKE0DHxgljNBAAO
        9TXL0Y4OHwAAAABJRU5ErkJggg==" alt="Red dot" />
</div>

您可以试试这个 Base64 解码器，看看您的 Base64 数据是否正确。


更多建议

您需要指定正确的 Content-type 、 Content-encoding 和 charset 。

喜欢

 data:image/jpeg;charset=utf-8;base64,

根据数据 URI 方案的语法：

 data:[<media type>][;charset=<character set>][;base64],<data>

如何编写捕获所有异常的 `try`/`except` 块？

Arya

2023-02-17 06:44

我如何编写一个捕获所有异常的try/块？except



解答

你可以，但你可能不应该：

try:
    do_something()
except:
    print("Caught it!")

但是，这也会捕获异常KeyboardInterrupt，而您通常不希望这样，是吗？除非您立即重新引发异常 - 请参阅文档中的以下示例：

try:
    f = open('myfile.txt')
    s = f.readline()
    i = int(s.strip())
except IOError as (errno, strerror):
    print("I/O error({0}): {1}".format(errno, strerror))
except ValueError:
    print("Could not convert data to an integer.")
except:
    print("Unexpected error:", sys.exc_info()[0])
    raise
