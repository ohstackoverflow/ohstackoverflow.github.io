如何从 Git 存储库中删除文件？

Arya

2023-02-16 21:29

如何"file1.txt"从我的存储库中删除？



解答

使用git rm。

如果要从 Git 存储库 和文件系统 中删除文件，请使用：

git rm file1.txt
git commit -m "remove file1.txt"

但是如果你只想从 Git 存储库中删除文件而不是从文件系统中删除它，请使用：

git rm --cached file1.txt
git commit -m "remove file1.txt"

并将更改推送到远程仓库

git push origin branch_name


更多建议

git rm file.txt从 repo 中删除文件，但 也将其从本地文件系统中删除 。

要从存储库中删除文件而 不是 从本地文件系统中删除它，请使用：
git rm --cached file.txt

使 .gitignore 忽略除少数文件之外的所有内容

Arya

2023-02-16 21:28

我知道.gitignore文件会隐藏来自 Git 版本控制的指定文件。

我如何告诉.gitignore忽略除我正在使用 Git 跟踪的文件之外的所有内容？就像是：

# Ignore everything:
*

# Do not ignore these files:
script.pl
template.latex



解答

!否定模式的可选前缀；先前模式排除的任何匹配文件将再次包含在内。如果否定模式匹配，这将覆盖优先级较低的模式源。

# Ignore everything
*

# But not these files...
!.gitignore
!script.pl
!template.latex
# etc...

# ...even if they are in subdirectories
!*/

# if the files to be tracked are in subdirectories
!*/a/b/file1.txt
!*/a/b/c/*


更多建议

如果你想忽略一个目录中除了其中一个文件之外的全部内容，你可以为文件路径中的每个目录编写一对规则。例如.gitignore忽略pippo文件夹除了来自pippo/pluto/paperino.xml

pippo/*
!pippo/pluto
pippo/pluto/*
!pippo/pluto/paperino.xml

请注意，如果您只是在上面写了：

pippo/*
!pippo/pluto/paperino.xml

它不会工作，因为plutoGit 不存在中间文件夹，因此paperino.xml无法找到存在的位置。

如何使用 Javascript 切断/切片/修剪字符串中的最后一个字符？

Arya

2023-02-16 21:27

我有一个字符串12345.00，我希望它返回12345.0。

我看过trim，但看起来它只是在修剪空格，slice我看不出它是如何工作的。有什么建议么？



解答

您可以使用子字符串函数：

let str = "12345.00";
str = str.substring(0, str.length - 1);
console.log(str);

这是公认的答案，但根据下面的对话，切片语法更加清晰：

let str = "12345.00";
str = str.slice(0, -1); 
console.log(str);


更多建议

你可以使用slice！你只需要确保你知道如何使用它。正#s 是相对于开头的，负数是相对于结尾的。

js>"12345.00".slice(0,-1)
12345.0
