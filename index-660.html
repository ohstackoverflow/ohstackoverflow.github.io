获取（可能）关联数组中的第一个键？

Arya

2023-02-17 12:43

确定可能关联数组中第一个键的最佳方法是什么？我首先想到的只是 foreach 数组然后立即打破它，就像这样：

foreach ($an_array as $key => $val) break;

因此让 $key 包含第一个键，但这似乎效率低下。有没有人有更好的解决方案？



解答

2019更新

从PHP 7.3 开始，有一个名为的新内置函数array_key_first()，它将从给定数组中检索第一个键而不重置内部指针。查看文档以获取更多信息。

您可以使用reset和key：

reset($array);
$first_key = key($array);

它与您的初始代码本质上相同，但开销少了一点，并且更明显地发生了什么。

请记住调用reset，否则您可能会获得数组中的任何键。您也可以使用endinstead ofreset来获取最后一个密钥。

如果您希望键获得第一个值，reset实际上返回它：

$first_value = reset($array);

不过，有一个特殊情况需要注意（因此请先检查数组的长度）：

$arr1 = array(false);
$arr2 = array();
var_dump(reset($arr1) === reset($arr2)); // bool(true)


更多建议

array_keys返回键数组。取第一个条目。或者，您可以调用reset数组，然后调用key. 后一种方法可能稍微快一些（虽然我没有测试它），但它有重置内部指针的副作用。

复制 MySQL 表、索引和数据

Arya

2023-02-17 12:43

如何将 MySQL 表的数据、结构和索引复制或克隆或复制到新表？

这是我到目前为止所发现的。

这将复制数据和结构，但不复制索引：

create table {new_table} select * from {old_table};

这将复制结构和索引，但不复制数据：

create table {new_table} like {old_table};



解答

要使用索引和触发器进行复制，请执行以下 2 个查询：

CREATE TABLE new_table LIKE old_table; 
INSERT INTO new_table SELECT * FROM old_table;

要仅复制结构和数据，请使用以下方法：

CREATE TABLE new_table AS SELECT * FROM old_table;

我以前问过这个：

复制包含索引的 MySQL 表


更多建议

除了上述解决方案外，您还可以使用AS一行来制作它。

CREATE TABLE tbl_new AS SELECT * FROM tbl_old;

例如：

p + p {
  /* Some declarations */
}

我不知道这+意味着什么。p这和只为without定义样式有什么区别+ p？



解答

请参阅W3.org 上的相邻选择器。

在这种情况下，选择器意味着该样式仅适用于紧跟在另一个段落之后的段落。

普通p选择器会将样式应用于页面中的每个段落。

这仅适用于 IE7 或更高版本。在 IE6 中，样式不会应用于任何元素。>顺便说一句，这也适用于组合器。

另请参阅 Microsoft 对Internet Explorer 中的 CSS 兼容性的概述。


更多建议

它是相邻兄弟选择器。

来自Splash of Style 博客。

要定义 CSS 相邻选择器，请使用加号。

h1+p {color:blue;}

上面的 CSS 代码会将任何 h1 标题之后（不是内部）的第一段格式化为蓝色。
