如何检查数组是否包含 JavaScript 中的值？

Arya

2023-02-16 18:54

找出 JavaScript 数组是否包含值的最简洁有效的方法是什么？

这是我知道的唯一方法：

function contains(a, obj) {
    for (var i = 0; i < a.length; i++) {
        if (a[i] === obj) {
            return true;
        }
    }
    return false;
}

有没有更好更简洁的方法来完成这个？

这与 Stack Overflow 问题Best way to find an item in a JavaScript Array? 密切相关。它解决了使用 . 在数组中查找对象的问题indexOf。



解答

现代浏览器有Array#includes，它 确实 做到了这一点，并得到除 IE 之外的所有人的广泛支持：

console.log(['joe', 'jane', 'mary'].includes('jane')); //true

您也可以使用Array#indexOf，它不那么直接，但对于过时的浏览器不需要 polyfill。

console.log(['joe', 'jane', 'mary'].indexOf('jane') >= 0); //true

许多框架也提供了类似的方法：

  • 查询：$.inArray(value, array, [fromIndex])
  • Underscore.js：（_.contains(array, value)也别名为_.includeand _.includes）
  • 道场工具包：dojo.indexOf(array, value, [fromIndex, findLast])
  • 原型：array.indexOf(value)
  • 工具：array.indexOf(value)
  • 摩奇套件：findValue(array, value)
  • 阿贾克斯女士：array.indexOf(value)
  • 分机：Ext.Array.contains(array, value)
  • Lodash：（_.includes(array, value, [from])在_.contains4.0.0 之前）
  • 拉姆达：R.includes(value, array)

请注意，一些框架将其作为函数实现，而其他框架则将函数添加到数组原型中。


更多建议

2019 年更新：此答案来自 2008 年（11 岁！），与现代 JS 用法无关。 承诺的性能改进是基于当时在浏览器中完成的基准测试。它可能与现代 JS 执行上下文无关。如果您需要一个简单的解决方案，请寻找其他答案。如果您需要最佳性能，请在相关执行环境中对自己进行基准测试。

正如其他人所说，遍历数组可能是最好的方法，但已经证明递减while循环是 JavaScript 中最快的迭代方法。所以你可能想重写你的代码如下：

function contains(a, obj) {
    var i = a.length;
    while (i--) {
       if (a[i] === obj) {
           return true;
       }
    }
    return false;
}

当然，你也可以扩展 Array 原型：

Array.prototype.contains = function(obj) {
    var i = this.length;
    while (i--) {
        if (this[i] === obj) {
            return true;
        }
    }
    return false;
}

现在您可以简单地使用以下内容：

alert([1, 2, 3].contains(2)); // => true
alert([1, 2, 3].contains('2')); // => false

2023-02-16 18:52

在我升级到最新的稳定版后node，npm我尝试了npm install moment --save。它package.json使用脱字符号^前缀保存条目。以前，它是波浪号~前缀。

  1. 为什么要进行这些更改npm？
  2. ~tilde和 caret 和有什么不一样^？
  3. 相比别人有什么优势？



解答

请参阅NPM 文档和semver 文档：

  • ~version “大约相当于版本” ，将更新到所有未来的补丁版本，而不会增加次要版本。~1.2.3将使用从 1.2.3 到 <1.3.0 的版本。

  • ^version “Compatible with version” ，将更新到所有未来的次要/补丁版本，而不增加主要版本。^2.3.4将使用从 2.3.4 到 <3.0.0 的版本。

有关例外情况，请参阅下面的评论，特别是对于 pre-one 版本，例如 ^0.2.3

如何水平居中元素？

Arya

2023-02-16 18:52

如何使用 CSS<div>在另一个中水平居中<div>？

<div id="outer">
  <div id="inner">Foo foo</div>
</div>



解答

使用flexbox它可以很容易地设置 div 水平和垂直居中的样式。

#inner {  
  border: 0.05em solid black;
}

#outer {
  border: 0.05em solid red;
  width:100%;
  display: flex;
  justify-content: center;
}


<div id="outer">
  <div id="inner">Foo foo</div>
</div>

要使 div 垂直居中对齐，请使用属性align-items: center。

其他解决方案

您可以将此 CSS 应用于内部<div>：

#inner {
  width: 50%;
  margin: 0 auto;
}

当然，您不必将widthto设置为50%。任何小于 containing 的宽度都<div>可以。这margin: 0 auto就是实际居中的原因。

如果您的目标是Internet Explorer 8（及更高版本），最好使用以下代码：

#inner {
  display: table;
  margin: 0 auto;
}

它将使内部元素水平居中，并且无需设置特定的width.

这里的工作示例：

#inner {
  display: table;
  margin: 0 auto;
  border: 1px solid black;
}

#outer {
  border: 1px solid red;
  width:100%
}


<div id="outer">
  <div id="inner">Foo foo</div>
</div>


更多建议

如果你不想在内部设置固定宽度，div你可以这样做：

#outer {
  width: 100%;
  text-align: center;
}

#inner {
  display: inline-block;
}


<div id="outer">  
    <div id="inner">Foo foo</div>
</div>

这使得 innerdiv成为一个内联元素，可以用 居中text-align。
