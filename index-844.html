选择 <a> 以某个字符串结尾的 href

Arya

2023-02-17 17:39

是否可以使用jQuery选择<a>href 以“ABC”结尾的所有链接？

例如，如果我想找到这个链接<a href="http://server/page.aspx?id=ABC">



解答

   $('a[href$="ABC"]')...

选择器文档可以在http://docs.jquery.com/Selectors找到

对于属性：

= is exactly equal
!= is not equal
^= is starts with
$= is ends with
*= is contains
~= is contains word
|= is starts with prefix (i.e., |= "prefix" matches "prefix-...")


更多建议

$('a[href$="ABC"]:first').attr('title');

这将返回具有以“ABC”结尾的 URL 的第一个链接的标题。

如何查找不包含给定字符串模式的文件？

Arya

2023-02-17 17:39

如何找出当前目录中不 包含 单词（使用）的 文件 ？ ****foo``grep



解答

以下命令为我提供了所有不包含该模式的文件foo：

find .  -not  -ipath '.*svn*' -exec  grep  -H -E -o -c  "foo"  {} \; | grep 0


更多建议

您可以单独使用 grep 来完成（无需查找）。

grep -riL "foo" .

这是对使用的参数的解释grep

     -L, --files-without-match
             each file processed.
     -R, -r, --recursive
             Recursively search subdirectories listed.

     -i, --ignore-case
             Perform case insensitive matching.

如果你使用l（小写）你会得到相反的结果（匹配的文件）

     -l, --files-with-matches
             Only the names of files containing selected lines are written
