需要将 git 分支重置为原始版本

Arya

2023-02-17 19:30

我不小心在一个我不应该有一段时间的分支上工作，所以我从它分支出来并给它起了一个合适的名字。现在我想覆盖我不应该在原始版本（github）上的分支。是否有捷径可寻？我尝试删除分支，然后重新设置跟踪分支，但它只是给了我再次使用的版本。



解答

如果你还没有推送到原点，你可以将你的分支重置为上游分支：

git checkout mybranch
git reset --hard origin/mybranch

（确保您在单独的分支中引用您的最新提交，就像您在问题中提到的那样）

请注意，在重置之后，mybranch@{1}指的是重置之前的旧提交。

但如果您已经推送，请参阅“创建 git 分支，并将原始状态恢复为上游状态”以了解其他选项。

使用 Git 2.23（2019 年 8 月） ，这将是一个命令： git switch 。
即：git switch -C mybranch origin/mybranch

例子

C:\Users\vonc\git\git>git switch -C master origin/master
Reset branch 'master'
Branch 'master' set up to track remote branch 'master' from 'origin'.
Your branch is up to date with 'origin/master'.

这会像往常一样恢复索引和工作树git reset --hard。

正如Brad Herman评论的那样，areset --hard会 删除任何新文件或将修改后的文件重置为 HEAD 。

git clean -f -d实际上，为了确保您从“干净的石板”开始，重置后的a将确保工作树与您刚刚重置到的分支完全相同 。

这篇博文建议使用这些别名（master仅适用于分支，但您可以调整/扩展这些别名）：

[alias]
   resetorigin = !git fetch origin && git reset --hard origin/master &&

git clean -f -d resetupstream = !git fetch upstream && git reset --hard upstream/master && git clean -f -d

然后你可以输入：

git resetupstream

或者

git resetorigin


更多建议

有一种更简单的方法可以做到这一点：

git reset --hard @{u}

@{u}是无论您的跟踪分支是什么的快捷方式，因此如果您在master并且跟踪origin/master，@{u}则指向origin/master。

使用它的好处是您不必记住（或输入）您的跟踪分支的全名。你也可以做一个别名：

git-reset-origin="git reset --hard @{u}"

无论您当前所在的分支如何，它都会起作用。

将边框放在 div 内部而不是在其边缘

Arya

2023-02-17 19:29

我有一个<div>元素，我想在它上面加一个边框。我知道我可以写style="border: 1px solid black"，但这会在 div 的任一侧添加 2px，这不是我想要的。

我宁愿这个边框距离 div 的边缘为 -1px。div 本身是 100px x 100px，如果我添加边框，那么我必须做一些数学运算才能使边框出现。

有什么方法可以使边框出现，并确保框仍为 100px（包括边框）？



解答

将box-sizing属性设置为border-box：

div {
    box-sizing: border-box;
    -moz-box-sizing: border-box;
    -webkit-box-sizing: border-box;
    width: 100px;
    height: 100px;
    border: 20px solid #f00;
    background: #00f;
    margin: 10px;
}

div + div {
    border: 10px solid red;
}


<div>Hello!</div>
<div>Hello!</div>

它适用于IE8 及更高版本。


更多建议

你也可以像这样使用 box-shadow：

div{
    -webkit-box-shadow:inset 0px 0px 0px 10px #f00;
    -moz-box-shadow:inset 0px 0px 0px 10px #f00;
    box-shadow:inset 0px 0px 0px 10px #f00;
}

此处示例： http: //jsfiddle.net/nVyXS/（悬停以查看边框）

这仅适用于现代浏览器。例如：不支持 IE 8。有关详细信息，请参阅 caniuse.com（框阴影功能） 。

如何将 var_dump 的结果捕获到字符串中？

Arya

2023-02-17 19:29

我想将输出捕获var_dump到一个字符串。

PHP 文档说；

与任何直接将结果输出到浏览器的东西一样，输出控制函数可用于捕获此函数的输出，并将其保存在一个字符串中（例如）。

这可能如何运作的一个例子是什么？

print_r()不是一个有效的可能性，因为它不会给我我需要的信息。



解答

使用输出缓冲：

<?php
ob_start();
var_dump($someVar);
$result = ob_get_clean();
?>
