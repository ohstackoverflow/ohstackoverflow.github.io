将列表转换为数据框

Arya

2023-02-17 22:37

我有一个嵌套的数据列表。它的长度是132，每一项都是一个长度为20的列表。有没有一种 快速的 方法可以将这个结构转换成一个有132行20列数据的数据框？

以下是一些要使用的示例数据：

l <- replicate(
  132,
  as.list(sample(letters, 20)),
  simplify = FALSE
)



解答

2020 年 7 月更新：

参数的默认值stringsAsFactors现在是默认值default.stringsAsFactors()。FALSE

假设您的列表列表称为l：

df <- data.frame(matrix(unlist(l), nrow=length(l), byrow=TRUE))

上面会将所有字符列转换为因子，为避免这种情况，您可以向 data.frame() 调用添加一个参数：

df <- data.frame(matrix(unlist(l), nrow=132, byrow=TRUE),stringsAsFactors=FALSE)


更多建议

2020 年 7 月更新：

参数的默认值stringsAsFactors现在是默认值default.stringsAsFactors()。FALSE

假设您的列表列表称为l：

df <- data.frame(matrix(unlist(l), nrow=length(l), byrow=TRUE))

上面会将所有字符列转换为因子，为避免这种情况，您可以向 data.frame() 调用添加一个参数：

df <- data.frame(matrix(unlist(l), nrow=132, byrow=TRUE),stringsAsFactors=FALSE)

如何使用 new 在 C++ 中声明一个二维数组？

Arya

2023-02-17 22:37

我如何使用 new 声明一个二维数组？

就像，对于“普通”数组，我会：

int* ary = new int[Size]

但

int** ary = new int[sizeY][sizeX]

a) 不工作/编译 b) 不完成什么：

int ary[sizeY][sizeX]

做。



解答

如果你的行长度是一个编译时常量，C++11 允许

auto arr2d = new int [nrows][CONSTANT];

看到这个答案。像 gcc 这样的编译器允许可变长度数组作为 C++ 的扩展，可以new 像此处所示那样使用来获得完全运行时可变的数组维度功能，就像 C99 允许的那样，但是可移植的 ISO C++ 仅限于第一个维度是可变的。

另一个有效的选择是手动将 2d 索引到一个大的 1d 数组中，如另一个答案所示，允许与真正的 2D 数组相同的编译器优化（例如证明或检查数组不相互别名/重叠）。

否则，您可以使用 指向数组的指针数组 来允许像连续二维数组这样的二维语法，即使它不是一个有效的单一大型分配。您可以使用循环对其进行初始化，如下所示：

int** a = new int*[rowCount];
for(int i = 0; i < rowCount; ++i)
    a[i] = new int[colCount];

以上，对于colCount= 5和rowCount = 4，将产生以下结果：

delete在删除指针数组之前，不要忘记用循环分别对每一行进行处理。另一个答案中的示例。


更多建议

int** ary = new int[sizeY][sizeX]

应该：

int **ary = new int*[sizeY];
for(int i = 0; i < sizeY; ++i) {
    ary[i] = new int[sizeX];
}

然后清理将是：

for(int i = 0; i < sizeY; ++i) {
    delete [] ary[i];
}
delete [] ary;

编辑： 正如 Dietrich Epp 在评论中指出的那样，这并不是一个轻量级的解决方案。另一种方法是使用一个大内存块：

int *ary = new int[sizeX*sizeY];

// ary[i][j] is then rewritten as
ary[i*sizeY+j]

将多个值返回给方法调用者

Arya

2023-02-17 22:36

我阅读了这个问题的 C++ 版本，但并没有真正理解它。

有人可以清楚地解释一下是否可以在 C# 中完成，以及如何完成？



解答

在 C# 7 及更高版本中，请参阅此答案。

在以前的版本中，您可以使用.NET 4.0+ 的 Tuple：

例如：

public Tuple<int, int> GetMultipleValue()
{
     return Tuple.Create(1,2);
}

具有两个值的元组具有Item1和Item2作为属性。


更多建议

现在 C# 7 已经发布，您可以使用新包含的元组语法

(string, string, string) LookupName(long id) // tuple return type
{
    ... // retrieve first, middle and last from data storage
    return (first, middle, last); // tuple literal
}

然后可以这样使用：

var names = LookupName(id);
WriteLine($"found {names.Item1} {names.Item3}.");

您还可以为您的元素提供名称（因此它们不是“Item1”、“Item2”等）。您可以通过在签名或返回方法中添加名称来实现：

(string first, string middle, string last) LookupName(long id) // tuple elements have names

或者

return (first: first, middle: middle, last: last); // named tuple elements in a literal

它们也可以被解构，这是一个非常好的新特性：

(string first, string middle, string last) = LookupName(id1); // deconstructing declaration
