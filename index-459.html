如何为 MySQL 日期时间列设置默认值？

Arya

2023-02-17 07:27

如何为 MySQL 日期时间列设置默认值？

在 SQL Server 中它是getdate(). MySQL 的等价物是什么？如果这是一个因素，我正在使用 MySQL 5.x。



解答

重要编辑：从 MySQL 5.6.5 开始，现在可以使用 DATETIME 字段实现此目的，请查看下面的另一篇文章...

以前的版本不能用 DATETIME 做到这一点......

但是您可以使用 TIMESTAMP 来做到这一点：

mysql> create table test (str varchar(32), ts TIMESTAMP DEFAULT CURRENT_TIMESTAMP);
Query OK, 0 rows affected (0.00 sec)

mysql> desc test;
+-------+-------------+------+-----+-------------------+-------+
| Field | Type        | Null | Key | Default           | Extra |
+-------+-------------+------+-----+-------------------+-------+
| str   | varchar(32) | YES  |     | NULL              |       | 
| ts    | timestamp   | NO   |     | CURRENT_TIMESTAMP |       | 
+-------+-------------+------+-----+-------------------+-------+
2 rows in set (0.00 sec)

mysql> insert into test (str) values ("demo");
Query OK, 1 row affected (0.00 sec)

mysql> select * from test;
+------+---------------------+
| str  | ts                  |
+------+---------------------+
| demo | 2008-10-03 22:59:52 | 
+------+---------------------+
1 row in set (0.00 sec)

mysql>

警告： 如果您将 CURRENT_TIMESTAMP 定义为默认的列，您将需要始终为此列指定一个值，否则该值将在更新时自动重置为“now()”。 这意味着如果您不想更改值，您的 UPDATE 语句必须包含“[您的列名] = [您的列名]”（或其他一些值），否则该值将变为“now()”。很奇怪，但确实如此。我正在使用 5.5.56-MariaDB


更多建议

在 5.6.5 版本中，可以在日期时间列上设置默认值，甚至可以创建一个在行更新时更新的列。类型定义：

CREATE TABLE foo (
    `creation_time`     DATETIME DEFAULT CURRENT_TIMESTAMP,
    `modification_time` DATETIME ON UPDATE CURRENT_TIMESTAMP
)

参考：http://optimize-this.blogspot.com/2012/04/datetime-default-now-finally- available.html

如何在不使用库的情况下在 JavaScript 中的另一个元素之后插入一个元素？

Arya

2023-02-17 07:27

在 JavaScript 中有insertBefore()，但如何在不使用 jQuery 或其他库的情况下在另一个元素 之后 插入一个元素？



解答

referenceNode.parentNode.insertBefore(newNode, referenceNode.nextSibling);

referenceNode你想放在后面的节点在哪里newNode。如果referenceNode是其父元素中的最后一个子元素，那很好，因为referenceNode.nextSiblingwill benull并insertBefore通过添加到列表末尾来处理这种情况。

所以：

function insertAfter(referenceNode, newNode) {
    referenceNode.parentNode.insertBefore(newNode, referenceNode.nextSibling);
}

您可以使用以下代码段对其进行测试：

显示代码片段

function insertAfter(referenceNode, newNode) {
  referenceNode.parentNode.insertBefore(newNode, referenceNode.nextSibling);
}

var el = document.createElement("span");
el.innerHTML = "test";
var div = document.getElementById("foo");
insertAfter(div, el);


<div id="foo">Hello</div>


更多建议

让 Git 与代理服务器一起工作 - 失败并显示“请求超时”

Arya

2023-02-17 07:26

如何让 Git 使用代理服务器？

我需要从 Git 服务器检出代码，但每次都显示“请求超时”。我该如何解决这个问题？

或者，如何设置代理服务器？



解答

使用命令：

git config --global http.proxy http://proxyuser:proxypwd@proxy.server.com:8080
  • 更改proxyuser为您的代理用户
  • 更改proxypwd为您的代理密码
  • 更改proxy.server.com为您的代理服务器的 URL
  • 更改8080为代理服务器上配置的代理端口

请注意，这适用于 http 和 https 存储库。

如果您随时决定重置此代理并在没有代理的情况下工作：

使用命令：

git config --global --unset http.proxy

最后，检查当前设置的代理：

git config --global --get http.proxy


更多建议

这对我有用，在公司防火墙后面的 Windows XP 中。

除了来自http://code.google.com/p/msysgit/downloads/list?can=3的 git v1.771 之外，我不需要安装任何本地代理或任何其他软件

$ git config --global http.proxy http://proxyuser:proxypwd@proxy.server.com:8080
$ git config --system http.sslcainfo /bin/curl-ca-bundle.crt
$ git remote add origin https://mygithubuser:mygithubpwd@github.com/repoUser/repoName.git
$ git push origin master

proxyuser = 我由我们的 IT 部门分配的代理用户，在我的例子中，它是我用来登录我的 PC 的同一个 Windows 用户，即 Active Directory 用户

proxypwd = 我的代理用户的密码

proxy.server.com:8080 = 代理服务器名称和端口，我是从“控制面板”、“Internet 选项”、“连接”、“局域网设置”按钮、“代理服务器”部分中的“高级”按钮获得的，使用第一行 (http) 上的服务器名称和端口.

mygithubuser = 我用来登录 github.com 的用户

mygithubpwd = 我的 github.com 用户的密码

repoUser = repo 的用户所有者

repoName = 回购的名称
