您如何测试 Python 函数是否抛出异常？

Arya

2023-02-17 05:07

如何编写仅在函数未抛出预期异常时才失败的单元测试？



解答

使用unittest 模块中的TestCase.assertRaises(or TestCase.failUnlessRaises)，例如：

import mymod

class MyTestCase(unittest.TestCase):
    def test1(self):
        self.assertRaises(SomeCoolException, mymod.myfunc)


更多建议

从 Python 2.7 开始，您可以使用上下文管理器来获取抛出的实际异常对象：

import unittest

def broken_function():
    raise Exception('This is broken')

class MyTestCase(unittest.TestCase):
    def test(self):
        with self.assertRaises(Exception) as context:
            broken_function()

        self.assertTrue('This is broken' in context.exception)

if __name__ == '__main__':
    unittest.main()

断言引发

在 Python 3.5 中，你必须包装context.exception在 中str，否则你会得到一个TypeError

self.assertTrue('This is broken' in str(context.exception))

格式化数字以始终显示 2 位小数

Arya

2023-02-17 05:06

我想将我的数字格式化为始终显示 2 位小数，并在适用的情况下四舍五入。

例子：

number     display
------     -------
1          1.00
1.341      1.34
1.345      1.35

我一直在使用这个：

parseFloat(num).toFixed(2);

但它显示1为1, 而不是1.00.



解答

(Math.round(num * 100) / 100).toFixed(2);

现场演示

显示代码片段

var num1 = "1";
document.getElementById('num1').innerHTML = (Math.round(num1 * 100) / 100).toFixed(2);

var num2 = "1.341";
document.getElementById('num2').innerHTML = (Math.round(num2 * 100) / 100).toFixed(2);

var num3 = "1.345";
document.getElementById('num3').innerHTML = (Math.round(num3 * 100) / 100).toFixed(2);


span {
    border: 1px solid #000;
    margin: 5px;
    padding: 5px;
}


<span id="num1"></span>
<span id="num2"></span>
<span id="num3"></span>

请注意，它将 四舍五入 到小数点后两位，因此输入1.346将返回1.35.


更多建议

Number(1).toFixed(2);         // 1.00
Number(1.341).toFixed(2);     // 1.34
Number(1.345).toFixed(2);     // 1.34 NOTE: See andy's comment below.
Number(1.3450001).toFixed(2); // 1.35

显示代码片段

document.getElementById('line1').innerHTML = Number(1).toFixed(2);
document.getElementById('line2').innerHTML = Number(1.341).toFixed(2);
document.getElementById('line3').innerHTML = Number(1.345).toFixed(2);
document.getElementById('line4').innerHTML = Number(1.3450001).toFixed(2);

将数组项复制到另一个数组

Arya

2023-02-17 05:06

我有一个 JavaScript 数组dataArray，我想将其推入一个新数组newArray。除了我不想newArray[0]成为dataArray。我想将所有项目推入新数组：

var newArray = [];

newArray.pushValues(dataArray1);
newArray.pushValues(dataArray2);
// ...

甚至更好：

var newArray = new Array (
   dataArray1.values(),
   dataArray2.values(),
   // ... where values() (or something equivalent) would push the individual values into the array, rather than the array itself
);

所以现在新数组包含各个数据数组的所有值。是否有一些像pushValuesavailable 这样的速记，这样我就不必遍历每个人dataArray，一个一个地添加项目？



解答

使用concat函数，如下所示：

var arrayA = [1, 2];
var arrayB = [3, 4];
var newArray = arrayA.concat(arrayB);

的值newArray将是[1, 2, 3, 4]（arrayA并且arrayB保持不变；concat为结果创建并返回一个新数组）。


更多建议

在 ECMAScript 6 中，您可以使用Spread 语法：

let arr1 = [0, 1, 2];
let arr2 = [3, 4, 5];
arr1.push(...arr2);

console.log(arr1)

Spread 语法在所有主流浏览器（IE11 除外）中都可用。有关当前兼容性，请参阅此（不断更新的）兼容性表。

但是，有关性能的更多评论，请参见下面 Jack Giffin 的回复。它似乎concat仍然比传播运算符更好更快。
