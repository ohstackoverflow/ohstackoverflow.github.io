解答

@angular-devkit/build-angular作为开发依赖安装。这个包是 Angular 6.0 中新引入的

npm install --save-dev @angular-devkit/build-angular

或者，

yarn add @angular-devkit/build-angular --dev


更多建议

npm update

它就像一个魅力。
