带有泛型的 Typescript 箭头函数的语法是什么？

Arya

2023-02-17 23:10

打字稿手册目前没有关于箭头功能的内容。可以使用以下语法对普通函数进行通用类型化：示例：

function identity<T>(arg: T): T {
    return arg;
}

箭头函数的语法是什么？



解答

语言规范在 p.64f 上说

< T > ( ... ) => { ... } 形式的构造可以解析为带有类型参数的箭头函数表达式或应用于没有类型参数的箭头函数的类型断言。它被解析为前者[..]

例子：

// helper function needed because Backbone-couchdb's sync does not return a jqxhr
let fetched = <
           R extends Backbone.Collection<any> >(c:R) => {
               return new Promise(function (fulfill, reject) {
                   c.fetch({reset: true, success: fulfill, error: reject})
               });
           };


更多建议

如果你在一个.tsx文件中，你不能只写<T>，但这有效：

const foo = <T, >(x: T) => x;

与黑客攻击相反extends {}，这种黑客攻击至少保留了意图。
