如何检查对象是否具有属性？

Arya

2023-02-16 21:50

如何检查对象是否具有某些属性？例如：

>>> a = SomeClass()
>>> a.property
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AttributeError: SomeClass instance has no attribute 'property'

在使用它之前如何判断是否a具有该属性property？



解答

尝试hasattr()：

if hasattr(a, 'property'):
    a.property

请参阅下面的zweiterlinde 的回答，他提供了关于请求宽恕的好建议！一个非常pythonic的方法！

python 中的一般做法是，如果该属性可能大部分时间都存在，只需调用它并让异常传播，或者用 try/except 块捕获它。这可能会比hasattr. 如果该属性在大多数时间可能不存在，或者您不确定，则使用hasattr可能比反复陷入异常块更快。


更多建议

正如Jarret Hardie回答的那样，hasattr会成功的。不过，我想补充一点，Python 社区中的许多人推荐一种“请求宽恕比获得许可更容易”(EAFP) 的策略，而不是“三思而后行”(LBYL)。请参阅这些参考资料：

EAFP vs LBYL（回复：到目前为止有点失望）
EAFP vs. LBYL @Code Like a Pythonista: Idiomatic Python

IE：

try:
    doStuff(a.property)
except AttributeError:
    otherStuff()

... 优先于：

if hasattr(a, 'property'):
    doStuff(a.property)
else:
    otherStuff()
