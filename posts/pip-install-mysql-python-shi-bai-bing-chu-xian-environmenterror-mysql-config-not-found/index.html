我该怎么做才能解决这个问题？



解答

您的系统上似乎缺少 mysql_config 或安装程序找不到它。确保确实安装了 mysql_config。

例如在 Debian/Ubuntu 上你必须安装这个包：

sudo apt-get install libmysqlclient-dev

可能mysql_config不在你的路径下，你自己编译mysql套件的时候就会这样。

更新： 对于最新版本的 debian/ubuntu（截至 2018 年），它是

sudo apt install default-libmysqlclient-dev


更多建议

在 Mac OS 中，我只是在终端中运行它来修复：

export PATH=$PATH:/usr/local/mysql/bin

/etc/paths这是我找到的最快的修复程序——它将它添加到路径中，但我认为如果您计划在另一个环境中安装 MySQL- python，您最好永久添加它（即添加到 ） 。

（在 OSX Mountain Lion 中测试）
