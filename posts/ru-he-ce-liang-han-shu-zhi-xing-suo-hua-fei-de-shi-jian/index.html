如何测量函数执行所花费的时间

Arya

2023-02-17 00:51

我需要以毫秒为单位获取执行时间。

我最初在 2008 年问过这个问题。当时接受的答案是使用new Date().getTime()但是，我们现在都同意使用标准performance.now()API 更合适。因此，我正在更改对此的公认答案。



解答

使用 performance.now() ：

var startTime = performance.now()

doSomething()   // <---- measured code goes between startTime and endTime

var endTime = performance.now()

console.log(`Call to doSomething took ${endTime - startTime} milliseconds`)

其中Node.js需要导入performance类

导入性能

const { performance } = require('perf_hooks');

使用 console.time : (生活标准)

console.time('doSomething')

doSomething()   // <---- The function you're measuring time for

console.timeEnd('doSomething')

注意：
传递给time()和timeEnd()方法的字符串必须匹配
（ 以便计时器按预期完成 ）。

console.time()文档：

  • MDN 文档
  • Node.js 文档


更多建议

使用新的 Date().getTime()

getTime() 方法返回自 1970 年 1 月 1 日午夜以来的毫秒数。

前任。

var start = new Date().getTime();

for (i = 0; i < 50000; ++i) {
// do something
}

var end = new Date().getTime();
var time = end - start;
alert('Execution time: ' + time);
