如何使用 Javascript 处理每个文本字母？

Arya

2023-02-18 04:41

我想提醒字符串中的每个字母，但我不确定该怎么做。

所以，如果我有：

var str = 'This is my string';

我希望能够分别提醒T、h、i、s等。这只是我正在研究的想法的开始，但我需要知道如何分别处理每个字母。

我在想我可能需要在测试字符串的长度后使用 split 函数。

我怎样才能做到这一点？



解答

如果警报的顺序很重要，请使用：

for (var i = 0; i < str.length; i++) {
  alert(str.charAt(i));
}

或者这个：（另见这个答案）

 for (var i = 0; i < str.length; i++) {
   alert(str[i]);
 }

如果警报的顺序无关紧要，请使用：

var i = str.length;
while (i--) {
  alert(str.charAt(i));
}

或者这个：（另见这个答案）

 var i = str.length;
while (i--) {
  alert(str[i]);
}

显示代码片段

var str = 'This is my string';

function matters() {
  for (var i = 0; i < str.length; i++) {
    alert(str.charAt(i));
  }
}

function dontmatter() {
  var i = str.length;
  while (i--) {
    alert(str.charAt(i));
  }
}

<p>If the order of alerts doesn't matter, use <a href="#" onclick="dontmatter()">this</a>.</p>


更多建议

它可能不仅仅是解决了。只想贡献另一个简单的解决方案：

var text = 'uololooo';

// With ES6
[...text].forEach(c => console.log(c))

// With the `of` operator
for (const c of text) {
    console.log(c)
}

// With ES5
for (var x = 0, c=''; c = text.charAt(x); x++) { 
    console.log(c); 
}

// ES5 without the for loop:
text.split('').forEach(function(c) {
    console.log(c);
});
