在 Markdown 中更改图像大小

Arya

2023-02-17 01:22

我刚开始使用 Markdown。我喜欢它，但有一件事困扰着我：如何使用 Markdown 更改图像的大小？

该文档仅对图像提供以下建议：

![drawing](drawing.jpg)

如果可能的话，我希望图片也居中。我要求的是通用的 Markdown，而不仅仅是 GitHub 的做法。



解答

您可以在 Markdown 中使用一些 HTML：

<img src="drawing.jpg" alt="drawing" width="200"/>

或者通过style属性（ GitHub 不支持 ）

<img src="drawing.jpg" alt="drawing" style="width:200px;"/>

或者，您可以使用有关Markdown 和图像对齐的 答案中所述的自定义 CSS 文件 __

![drawing](drawing.jpg)

另一个文件中的 CSS：

img[alt=drawing] { width: 200px; }


更多建议

使用某些 Markdown 实现（包括Mou和Marked 2（仅限 macOS）），您可以=WIDTHxHEIGHT在图形文件的 URL 之后附加以调整图像大小。不要忘记 之前的空格=。

![](./pic/pic1_50.png =100x20)

你可以跳过高度

![](./pic/pic1s.png =250x)

和宽度

![](./pic/pic1s.png =x250)
