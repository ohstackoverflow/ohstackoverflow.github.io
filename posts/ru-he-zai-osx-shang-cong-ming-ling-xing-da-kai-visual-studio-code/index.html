如何在 OSX 上从命令行打开 Visual Studio Code？

Arya

2023-02-18 06:44

文档提到了一个名为 的可执行文件，但我不确定在哪里可以找到它，所以我可以将它放在我的路径上。code我从 VSCode 站点下载的 zip 不包含任何此类可执行文件。（我能够运行得.app很好。）

这是 Windows 独有的东西吗？



解答

从Visual Studio 代码设置页面：

提示 ：如果您想通过简单地键入“code”从终端运行 VS Code，VS Code 有一个命令，Shell 命令：在 PATH 中安装“code”命令，将“code”添加到您的 $PATH 变量列表。

安装后，启动 VS Code。现在打开命令面板（在 Mac 上为F1 或⇧+ ⌘+ ）并键入以查找命令。P``shell command``Shell Command: Install 'code' command in PATH

执行命令后，重启终端使新的$PATH值生效。您只需键入“代码”即可。在任何文件夹中开始编辑该文件夹中的文件。
