使用列表中的 max()/min() 获取返回的最大或最小项的索引

Arya

2023-02-17 18:58

我在列表上使用 Python 的max和函数来实现极小极大算法，我需要ormin返回的值的索引。换句话说，我需要知道哪个动作产生了最大值（在第一位玩家的回合）或最小值（第二位玩家）。max()``min()

for i in range(9):
    new_board = current_board.new_board_with_move([i / 3, i % 3], player)

    if new_board:
        temp = min_max(new_board, depth + 1, not is_min_level)  
        values.append(temp)

if is_min_level:
    return min(values)
else:
    return max(values)

我需要能够返回最小值或最大值的实际索引，而不仅仅是值。



解答

if is_min_level:
    return values.index(min(values))
else:
    return values.index(max(values))


更多建议

if is_min_level:
    return values.index(min(values))

    return values.index(max(values))
