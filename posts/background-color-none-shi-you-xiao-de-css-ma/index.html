background-color:none 是有效的 CSS 吗？

Arya

2023-02-18 04:49

谁能告诉我以下 CSS 是否有效？

.class {
    background-color:none;
}



解答

您可能想要transparentasnone不是有效值background-color。

CSS 2.1 规范为该background-color属性规定了以下内容：

Value: <color> | transparent | inherit

<color>可以是关键字或颜色的数字表示。有效color关键字是：

浅绿色、黑色、蓝色、紫红色、灰色、绿色、青柠色、栗色、海军蓝、橄榄色、橙色、紫色、红色、银色、蓝绿色、白色和黄色

transparent并且inherit本身是有效的关键字，但none不是。


更多建议

不，transparent改用none. 请参阅此示例中的工作示例，如果您将更改为transparent它将none不起作用

使用像.class { background-color:transparent; }

其中 .class 是您将命名透明类的名称。
