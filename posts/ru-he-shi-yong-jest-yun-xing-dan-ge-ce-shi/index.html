如何使用 Jest 运行单个测试？

Arya

2023-02-17 14:43

我在文件 fix-order-test.js 中有一个“与嵌套子项一起工作”的测试。

运行以下运行文件中的所有测试。

jest fix-order-test

如何只运行一个测试？以下不起作用，因为它搜索指定的正则表达式的文件。

jest 'works with nested children'



解答

在命令行中，使用--testNamePatternor-t标志：

jest -t 'fix-order-test'

这将只运行与您提供的测试名称模式相匹配的测试。它在Jest 文档中。

另一种方法是在监视模式下运行测试，jest --watch然后按通过键入测试文件名或运行单个测试名称P来过滤测试。T

如果你有it一个块的内部describe，你必须运行

jest -t '<describeString> <itString>'


更多建议

Jest 文档推荐如下：

如果测试失败，首先要检查的事情之一应该是当它是唯一运行的测试时测试是否失败。在 Jest 中，只运行一个测试很简单 - 只需暂时将该test命令更改为test.only

test.only('this will be the only test that runs', () => {
   expect(true).toBe(false);
});

或者

it.only('this will be the only test that runs', () => {
   expect(true).toBe(false);
});
