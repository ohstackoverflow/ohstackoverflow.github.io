如何修复错误：使用 NodeJS 时收听 EADDRINUSE？

Arya

2023-02-18 06:33

如果我使用端口 80 运行服务器，并尝试使用XMLHttpRequest，我会收到此错误：Error: listen EADDRINUSE

当我在端口 80 上运行服务器时，如果我想发出请求，为什么 NodeJS 会出现问题？对于网络浏览器，这不是问题：我可以在 Internet 上冲浪，而服务器正在运行。

服务器是：

  net.createServer(function (socket) {
    socket.name = socket.remoteAddress + ":" + socket.remotePort;
    console.log('connection request from: ' + socket.remoteAddress);
    socket.destroy();
  }).listen(options.port);

和要求：

var xhr = new XMLHttpRequest();

xhr.onreadystatechange = function() {
    sys.puts("State: " + this.readyState);

    if (this.readyState == 4) {
        sys.puts("Complete.\nBody length: " + this.responseText.length);
        sys.puts("Body:\n" + this.responseText);
    }
};

xhr.open("GET", "http://mywebsite.com");
xhr.send();



解答

EADDRINUSE表示listen()尝试绑定服务器的端口号已被使用。

因此，在您的情况下，必须已经在端口 80 上运行服务器。

如果你有另一个网络服务器在这个端口上运行，你必须把 node.js 放在那个服务器后面并通过它代理它。

你应该像这样检查listening事件，看看服务器是否真的在监听：

var http=require('http');

var server=http.createServer(function(req,res){
    res.end('test');
});

server.on('listening',function(){
    console.log('ok, server is running');
});

server.listen(80);
