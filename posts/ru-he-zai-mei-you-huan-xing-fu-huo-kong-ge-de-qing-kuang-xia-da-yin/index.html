如何在没有换行符或空格的情况下打印

Arya

2023-02-16 21:39

print考虑在 Python 中使用这些示例：

>>> for i in range(4): print('.')
.
.
.
.
>>> print('.', '.', '.', '.')
. . . .

在每个值之间添加换行符或空格。我怎样才能避免这种情况，以便输出呢....？换句话说，如何将字符串“附加”到标准输出流？



解答

在 Python 3 中，您可以使用函数的sep=和参数：end=``print

不在字符串末尾添加换行符：

print('.', end='')

要在要打印的所有函数参数之间不添加空格：

print('a', 'b', 'c', sep='')

您可以将任何字符串传递给任一参数，并且可以同时使用这两个参数。

如果您在缓冲方面遇到问题，可以通过添加flush=True关键字参数来刷新输出：

print('.', end='', flush=True)

Python 2.6 和 2.7

从 Python 2.6 开始，您可以使用模块print从 Python 3 导入函数：__future__

from __future__ import print_function

这允许您使用上面的 Python 3 解决方案。

但是请注意，该关键字在从Python 2导入的函数flush版本中不可用；它仅适用于 Python 3，更具体地说是 3.3 及更高版本。在早期版本中，您仍然需要通过调用手动刷新。您还必须重写执行此导入的文件中的所有其他打印语句。print``__future__``sys.stdout.flush()

或者你可以使用sys.stdout.write()

import sys
sys.stdout.write('.')

您可能还需要致电

sys.stdout.flush()

以确保stdout立即冲洗。

是否可以打印一些东西，但不会自动附加回车符？

是的，在要打印的最后一个参数之后附加一个逗号。例如，此循环在以空格分隔的一行上打印数字 0..9。请注意添加最终换行符的无参数“打印”：

>>> for i in range(10):
...     print i,
... else:
...     print
...

>>>
