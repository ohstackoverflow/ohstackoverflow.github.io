在 virtualenv 中使用不同的 Python 版本

Arya

2023-02-17 01:20

如何为指定版本的 Python 创建虚拟环境？



解答

注意： 对于 Python 3.3+ ，请参阅下面 Aelfinn 的回答。

在创建 virtualenv 实例时使用--python（或简称）选项来指定要使用的 Python 可执行文件，例如：-p

virtualenv --python="/usr/bin/python2.6" "/path/to/new/virtualenv/"


更多建议

从 Python 3 开始，文档建议使用以下方法创建虚拟环境：

python3 -m venv "my_env_name"

请注意，venv不允许使用其他版本的 Python 创建虚拟环境。为此，安装并使用该virtualenv包。

过时的信息

该pyvenv脚本可用于创建虚拟环境：

pyvenv "/path/to/new/virtual/environment"

自 Python 3.6 起已弃用。
