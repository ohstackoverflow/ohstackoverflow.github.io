document.getElementById 与 jQuery $()

Arya

2023-02-17 18:14

这是：

var contents = document.getElementById('contents');

与此相同：

var contents = $('#contents');

鉴于 jQuery 已加载？



解答

不完全是！！

document.getElementById('contents'); //returns a HTML DOM Object

var contents = $('#contents');  //returns a jQuery Object

在 jQuery 中，要获得与 相同的结果document.getElementById，您可以访问 jQuery 对象并获取对象中的第一个元素（记住 JavaScript 对象的行为类似于关联数组）。

var contents = $('#contents')[0]; //returns a HTML DOM Object


更多建议

不。

调用document.getElementById('id')将返回原始 DOM 对象。

调用$('#id')将返回一个 jQuery 对象，它包装了 DOM 对象并提供了 jQuery 方法。

因此，您只能调用 jQuery 方法 likecss()或animate()on the $()call。

您也可以编写$(document.getElementById('id'))，这将返回一个 jQuery 对象并且等效于$('#id').

您可以通过编写 .jQuery 对象从 jQuery 对象中获取底层 DOM 对象$('#id')[0]。
