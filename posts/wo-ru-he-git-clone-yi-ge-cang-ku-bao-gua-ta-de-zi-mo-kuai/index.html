我如何“git clone”一个仓库，包括它的子模块？

Arya

2023-02-16 21:11

我如何克隆一个 git 存储库以便它也克隆它的子模块？

运行git clone $REPO_URL只会创建空的子模块目录。



解答

在 Git 2.13 及更高版本中，--recurse-submodules可以用来代替--recursive：

git clone --recurse-submodules -j8 git://github.com/foo/bar.git
cd bar

编者注：-j8是一个可选的性能优化，在 2.8 版中可用，一次可以并行获取多达 8 个子模块 — 请参阅man git-clone.

从 Git 1.9 版本到 2.12 版本（-j标志仅在 2.8+ 版本中可用）：

git clone --recursive -j8 git://github.com/foo/bar.git
cd bar

使用 Git 1.6.5 及更高版本，您可以使用：

git clone --recursive git://github.com/foo/bar.git
cd bar

对于已经克隆的 repos 或较旧的 Git 版本，请使用：

git clone git://github.com/foo/bar.git
cd bar
git submodule update --init --recursive


更多建议

在填充子模块之前，您必须做两件事：

git submodule init 
git submodule update
