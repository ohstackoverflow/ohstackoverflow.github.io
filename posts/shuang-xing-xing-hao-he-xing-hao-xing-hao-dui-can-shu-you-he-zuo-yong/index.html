**（双星/星号）和 *（星号/星号）对参数有何作用？

Arya

2023-02-16 19:54

什么*args意思**kwargs？

def foo(x, y, *args):
def bar(x, y, **kwargs):



解答

and是一种常见的习惯用法，允许任意数量的函数参数，如*argsPython文档中有关定义函数的更多部分所述。**kwargs

将为*args您提供所有函数参数作为元组：

def foo(*args):
    for a in args:
        print(a)

foo(1)
# 1

foo(1,2,3)
# 1
# 2
# 3

将为**kwargs您提供所有 关键字参数 ，但与作为字典的形式参数相对应的那些除外。

def bar(**kwargs):
    for a in kwargs:
        print(a, kwargs[a])

bar(name='one', age=27)
# name one
# age 27

这两个习语都可以与普通参数混合使用，以允许一组固定参数和一些可变参数：

def foo(kind, *args, **kwargs):
   pass

也可以反过来使用它：

def foo(a, b, c):
    print(a, b, c)

obj = {'b':10, 'c':'lee'}

foo(100,**obj)
# 100 10 lee

该习语的另一种用法*l是在调用函数时 解压参数列表。

def foo(bar, lee):
    print(bar, lee)

l = [1,2]

foo(*l)
# 1 2

在 Python 3 中，可以*l在赋值的左侧使用（Extended Iterable Unpacking），尽管在这种情况下它给出的是列表而不是元组：

first, *rest = [1,2,3,4]
first, *l, last = [1,2,3,4]

Python 3 也增加了新的语义（参考PEP 3102）：

def func(arg1, arg2, arg3, *, kwarg1, kwarg2):
    pass

例如，以下内容适用于 python 3 但不适用于 python 2：

>>> x = [1, 2]
>>> [*x]
[1, 2]
>>> [*x, 3, 4]
[1, 2, 3, 4]

>>> x = {1:1, 2:2}
>>> x
{1: 1, 2: 2}
>>> {**x, 3:3, 4:4}
{1: 1, 2: 2, 3: 3, 4: 4}

这样的函数只接受 3 个位置参数，后面的所有内容都*只能作为关键字参数传递。

笔记：

  • 语义dict上用于关键字参数传递的 Python 是任意排序的。然而，在 Python 3.6 中，关键字参数保证记住插入顺序。
  • “现在元素的顺序**kwargs对应于关键字参数传递给函数的顺序。” - Python 3.6 中的新功能
  • 事实上，CPython 3.6 中的所有字典都会记住插入顺序作为实现细节，这在 Python 3.7 中成为标准。


更多建议

还值得注意的是，您也可以在调用函数时使用*and **。这是一个快捷方式，允许您使用列表/元组或字典将多个参数直接传递给函数。例如，如果您具有以下功能：

def foo(x,y,z):
    print("x=" + str(x))
    print("y=" + str(y))
    print("z=" + str(z))

您可以执行以下操作：

>>> mylist = [1,2,3]
>>> foo(*mylist)
x=1
y=2
z=3

>>> mydict = {'x':1,'y':2,'z':3}
>>> foo(**mydict)
x=1
y=2
z=3

>>> mytuple = (1, 2, 3)
>>> foo(*mytuple)
x=1
y=2
z=3

注意： 中的键名mydict必须与 function 的参数完全相同foo。否则它会抛出一个TypeError：

>>> mydict = {'x':1,'y':2,'z':3,'badnews':9}
>>> foo(**mydict)
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: foo() got an unexpected keyword argument 'badnews'
