如何使用 Git 将标签推送到远程存储库？

Arya

2023-02-16 20:07

我在我机器上的 master 分支上添加了一个标签：

git tag mytag master

如何将其推送到远程存储库？运行git push给出消息：

一切都是最新的

但是，远程存储库不包含我的标签。



解答

要推送特定的，一个标签执行以下操作git push origin tag_name


更多建议

git push --follow-tags

这是 Git 1.8.3 中引入的一个明智的选项：

git push --follow-tags

它同时推送提交和仅推送两者的标签：

  • 注释的
  • 从推送的提交中可达（祖先）

这是理智的，因为：

  • 您应该只将带注释的标签推送到远程，并为本地开发保留轻量级标签以避免标签冲突。另请参阅：带注释和未注释的标签有什么区别？
  • 它不会在不相关的分支上推送带注释的标签

出于这些原因，--tags应该避免。

Git 2.4添加了push.followTags默认打开该标志的选项，您可以通过以下方式设置：

git config --global push.followTags true

或者通过添加followTags = true到文件[push]的部分~/.gitconfig。
