switch 语句中的多个 case

Arya

2023-02-17 17:03

有没有办法在不case value:重复陈述的情况下通过多个案例陈述？

我知道这有效：

switch (value)
{
   case 1:
   case 2:
   case 3:
      // Do some stuff
      break;
   case 4:
   case 5:
   case 6:
      // Do some different stuff
      break;
   default:
       // Default stuff
      break;
}

但我想做这样的事情：

switch (value)
{
   case 1,2,3:
      // Do something
      break;
   case 4,5,6:
      // Do something
      break;
   default:
      // Do the Default
      break;
}

这种语法是我从另一种语言中想到的，还是我遗漏了什么？



解答

对于您提到的第二种方法，C++ 和 C# 中都没有语法。

你的第一种方法没有问题。但是，如果您的范围很大，只需使用一系列 if 语句。


更多建议

对于您提到的第二种方法，C++ 和 C# 中都没有语法。

你的第一种方法没有问题。但是，如果您的范围很大，只需使用一系列 if 语句。
