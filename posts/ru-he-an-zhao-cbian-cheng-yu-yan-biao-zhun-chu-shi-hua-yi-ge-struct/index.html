如何按照C编程语言标准初始化一个struct

Arya

2023-02-18 01:54

我想初始化一个结构元素，分为声明和初始化。这就是我所拥有的：

typedef struct MY_TYPE {
  bool flag;
  short int value;
  double stuff;
} MY_TYPE;

void function(void) {
  MY_TYPE a;
  ...
  a = { true, 15, 0.123 }
}

MY_TYPE这是按照C编程语言标准（C89、C90、C99、C11等）声明和初始化局部变量的方式吗？或者有什么更好的或至少工作？

更新 我最终得到了一个静态初始化元素，我根据需要在其中设置了每个子元素。



解答

在 (ANSI) C99 中，您可以使用 指定的初始化程序 来初始化结构：

MY_TYPE a = { .flag = true, .value = 123, .stuff = 0.456 };

其他成员初始化为零：“省略的字段成员被隐式初始化为与具有静态存储持续时间的对象相同。” ( https://gcc.gnu.org/onlinedocs/gcc/Designated-Inits.html )


更多建议

您可以使用复合文字来完成。根据该页面，它适用于 C99 （也算作ANSI C）。

MY_TYPE a;

a = (MY_TYPE) { .flag = true, .value = 123, .stuff = 0.456 };
...
a = (MY_TYPE) { .value = 234, .stuff = 1.234, .flag = false };

初始化器中的指定是可选的；你也可以写：

a = (MY_TYPE) { true,  123, 0.456 };
...
a = (MY_TYPE) { false, 234, 1.234 };
