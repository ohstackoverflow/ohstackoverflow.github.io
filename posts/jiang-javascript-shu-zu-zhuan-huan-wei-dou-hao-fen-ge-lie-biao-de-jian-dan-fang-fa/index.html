将 JavaScript 数组转换为逗号分隔列表的简单方法？

Arya

2023-02-18 07:23

我在 JavaScript 中有一个一维字符串数组，我想将其转换为以逗号分隔的列表。普通 JavaScript（或 jQuery）中是否有一种简单的方法可以将其转换为以逗号分隔的列表？（如果这是唯一的方法，我知道如何遍历数组并通过连接自己构建字符串。）



解答

Array.prototype.join ()方法：

var arr = ["Zero", "One", "Two"];

document.write(arr.join(", "));


更多建议

实际上，toString()实现默认使用逗号连接：

var arr = [ 42, 55 ];
var str1 = arr.toString(); // Gives you "42,55"
var str2 = String(arr); // Ditto

我不知道这是否是 JS 规范强制要求的，但这就是 最多 几乎所有浏览器似乎都在做。
