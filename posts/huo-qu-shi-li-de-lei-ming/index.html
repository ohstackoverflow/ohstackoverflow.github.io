获取实例的类名

Arya

2023-02-16 23:48

如何找出用于在 Python 中创建对象实例的类的名称？

我不确定是应该使用inspect模块还是解析__class__属性。



解答

您是否尝试过类的__name__属性？即type(x).__name__会给你班级的名字，我认为这是你想要的。

>>> import itertools
>>> x = itertools.count(0)
>>> type(x).__name__
'count'

如果您仍在使用 Python 2，请注意上述方法仅适用于新式类（在 Python 3+ 中，所有类都是“新式”类）。您的代码可能会使用一些旧式类。以下适用于两者：

x.__class__.__name__


更多建议

你想要类名作为字符串吗？

instance.__class__.__name__
