创建一个重复多次相同元素的数组

Arya

2023-02-18 03:23

在 Python 中，where[2]是一个列表，下面的代码给出了这个输出：

[2] * 5 # Outputs: [2,2,2,2,2]

有没有一种简单的方法可以用 JavaScript 中的数组来做到这一点？

我写了下面的函数来做到这一点，但是有更短或更好的东西吗？

var repeatelem = function(elem, n){
    // returns an array with element elem repeated n times.
    var arr = [];

    for (var i = 0; i <= n; i++) {
        arr = arr.concat(elem);
    };

    return arr;
};



解答

你可以这样做：

function fillArray(value, len) {
  if (len == 0) return [];
  var a = [value];
  while (a.length * 2 <= len) a = a.concat(a);
  if (a.length < len) a = a.concat(a.slice(0, len - a.length));
  return a;
}

它在每次迭代中将数组加倍，因此它可以用很少的迭代创建一个非常大的数组。

push注意：您还可以通过使用instead of来大大改进您的功能concat，因为concat每次迭代都会创建一个新数组。像这样（仅作为如何使用数组的示例显示）：

function fillArray(value, len) {
  var arr = [];
  for (var i = 0; i < len; i++) {
    arr.push(value);
  }
  return arr;
}
