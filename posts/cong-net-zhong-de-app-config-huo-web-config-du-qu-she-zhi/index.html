从 .NET 中的 app.config 或 web.config 读取设置

Arya

2023-02-17 09:00

我正在开发一个 C# 类库，它需要能够从web.config或app.config文件中读取设置（取决于 DLL 是从 ASP.NET Web 应用程序还是从 Windows 窗体应用程序引用的）。

我发现了

ConfigurationSettings.AppSettings.Get("MySetting")

有效，但该代码已被 Microsoft 标记为已弃用。

我读过我应该使用：

ConfigurationManager.AppSettings["MySetting"]

但是，该类System.Configuration.ConfigurationManager似乎无法从 C# 类库项目中获得。

做这个的最好方式是什么？



解答

您需要在项目的 引用文件夹 中添加 对 的引用 。System.Configuration ****

您绝对应该使用ConfigurationManager过时的ConfigurationSettings.
