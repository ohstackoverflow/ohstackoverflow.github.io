使用 pip 将 Python 包安装到不同的目录中？

Arya

2023-02-18 03:28

我知道显而易见的答案是使用 virtualenv 和 virtualenvwrapper，但出于各种原因我不能/不想这样做。

那么如何修改命令

pip install package_name

将pip软件包安装在默认位置以外的地方site-packages？



解答

使用：

pip install --install-option="--prefix=$PREFIX_PATH" package_name

您可能还想--ignore-installed使用此新前缀强制重新安装所有依赖项。您可以--install-option多次使用 to 添加您可以使用的任何选项python setup.py install（--prefix可能是您想要的，但您可以使用更多选项）。
