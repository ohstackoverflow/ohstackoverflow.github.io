如何使用 node.js 漂亮地打印 JSON？

Arya

2023-02-18 04:30

这似乎是一个已解决的问题，但我无法找到解决方案。

基本上，我读取一个 JSON 文件，更改一个密钥，然后将新的 JSON 写回同一个文件。一切正常，但我松开了 JSON 格式。所以，而不是：

{
  name:'test',
  version:'1.0'
}

我得到

{name:'test',version:'1.1'}

Node.js 中有没有一种方法可以将格式良好的 JSON 写入文件？



解答

JSON.stringify的第三个参数定义了漂亮打印的空白插入。它可以是字符串或数字（空格数）。Node 可以使用fs. 例子：

var fs = require('fs');

fs.writeFile('test.json', JSON.stringify({ a:1, b:2, c:3 }, null, 4));
/* test.json:
{
     "a": 1,
     "b": 2,
     "c": 3,
}
*/

请参阅MDN 上的 JSON.stringify() 文档，Node fs 文档

  name:'test',
  version:'1.0'

var outputFilename = '/tmp/my.json';

fs.writeFile(outputFilename, JSON.stringify(myData, null, 4), function(err) {
    if(err) {
      console.log(err);
    } else {
      console.log("JSON saved to " + outputFilename);
    }
});
