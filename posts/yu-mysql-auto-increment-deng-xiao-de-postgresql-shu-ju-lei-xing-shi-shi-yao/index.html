与 MySQL AUTO INCREMENT 等效的 PostgreSQL 数据类型是什么？

Arya

2023-02-17 18:01

我正在从 MySQL 切换到 PostgreSQL，我想知道如何才能拥有一个INT包含AUTO INCREMENT. 我在 PostgreSQL 文档中看到一个名为 的数据类型SERIAL，但在使用它时出现语法错误。



解答

是的，SERIAL 是等效的功能。

CREATE TABLE foo (
    id SERIAL,
    bar varchar
);

INSERT INTO foo (bar) VALUES ('blah');
INSERT INTO foo (bar) VALUES ('blah');

SELECT * FROM foo;

+----------+
| 1 | blah |
+----------+
| 2 | blah |
+----------+

SERIAL 只是围绕序列创建表时间宏。您不能将 SERIAL 更改为现有列。


更多建议

您可以使用任何其他整数数据类型，例如smallint.

例子 ：

CREATE SEQUENCE user_id_seq;
CREATE TABLE user (
    user_id smallint NOT NULL DEFAULT nextval('user_id_seq')
);
ALTER SEQUENCE user_id_seq OWNED BY user.user_id;

最好使用您自己的数据类型，而不是用户串行数据类型。
