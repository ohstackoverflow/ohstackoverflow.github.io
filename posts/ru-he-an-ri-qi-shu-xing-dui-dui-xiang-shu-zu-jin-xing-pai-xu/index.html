如何按日期属性对对象数组进行排序？

Arya

2023-02-17 04:52

假设我有一些对象的数组：

var array = [{id: 1, date: Mar 12 2012 10:00:00 AM}, {id: 2, date: Mar 8 2012 08:00:00 AM}];

如何从最接近当前日期和时间的日期开始按日期元素对该数组进行排序？请记住，数组可能有很多对象，但为了简单起见，我使用了 2 个。

我会使用排序功能和自定义比较器吗？



解答

最简单的答案

array.sort(function(a,b){
  // Turn your strings into dates, and then subtract them
  // to get a value that is either negative, positive, or zero.
  return new Date(b.date) - new Date(a.date);
});

更通用的答案

array.sort(function(o1,o2){
  if (sort_o1_before_o2)    return -1;
  else if(sort_o1_after_o2) return  1;
  else                      return  0;
});

或者更简洁：

array.sort(function(o1,o2){
  return sort_o1_before_o2 ? -1 : sort_o1_after_o2 ? 1 : 0;
});

通用的，强大的答案

在所有数组上sortBy使用Schwartzian 变换定义自定义不可枚举函数：

(function(){
  if (typeof Object.defineProperty === 'function'){
    try{Object.defineProperty(Array.prototype,'sortBy',{value:sb}); }catch(e){}
  }
  if (!Array.prototype.sortBy) Array.prototype.sortBy = sb;

  function sb(f){
    for (var i=this.length;i;){
      var o = this[--i];
      this[i] = [].concat(f.call(o,o,i),o);
    }
    this.sort(function(a,b){
      for (var i=0,len=a.length;i<len;++i){
        if (a[i]!=b[i]) return a[i]<b[i]?-1:1;
      }
      return 0;
    });
    for (var i=this.length;i;){
      this[--i]=this[i][this[i].length-1];
    }
    return this;
  }
})();

像这样使用它：

array.sortBy(function(o){ return o.date });

如果您的日期不能直接比较，请从中制作一个可比较的日期，例如

array.sortBy(function(o){ return new Date( o.date ) });

如果返回值数组，也可以使用它按多个条件排序：

// Sort by date, then score (reversed), then name
array.sortBy(function(o){ return [ o.date, -o.score, o.name ] };

有关详细信息， 请参阅http://phrogz.net/JS/Array.prototype.sortBy.js 。


更多建议

@Phrogz 的答案都很好，但这里有一个很棒的、更简洁的答案：

array.sort(function(a,b){return a.getTime() - b.getTime()});

使用箭头函数方式

array.sort((a,b)=>a.getTime()-b.getTime());
