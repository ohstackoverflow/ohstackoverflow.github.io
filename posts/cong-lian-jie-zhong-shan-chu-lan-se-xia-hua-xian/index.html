从链接中删除蓝色下划线

Arya

2023-02-17 17:24

我试图让一个链接显示为白色，没有下划线。文本颜色正确显示为白色，但蓝色下划线顽固地存在。我尝试text-decoration: none;在text- decoration: none !important;CSS 中删除链接下划线。都没有用。

.boxhead .otherPage {
  color: #FFFFFF;
  text-decoration: none;
}


<div class="boxhead">
  <h2>
    <span class="thisPage">Current Page</span>
    <a href="myLink"><span class="otherPage">Different Page</span></a>
  </h2>
</div>

运行代码片段隐藏结果

如何从链接中删除蓝色下划线？



解答

您不是应用于text-decoration: none;锚点 ( .boxhead a)，而是应用于跨度元素 ( .boxhead)。

尝试这个：

.boxhead a {
    color: #FFFFFF;
    text-decoration: none;
}


更多建议

锚标签（链接）还有visited、hover、link、active等伪类。确保您的样式适用于有问题的州，并且没有其他样式冲突。

例如：

a:hover, a:visited, a:link, a:active
{
    text-decoration: none;
}

有关用户操作伪类 :hover、:active 和 :focus 的 更多信息，请参见W3.org 。 ****
