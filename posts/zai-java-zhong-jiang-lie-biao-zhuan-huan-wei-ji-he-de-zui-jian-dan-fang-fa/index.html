在 Java 中将列表转换为集合的最简单方法

Arya

2023-02-17 15:29

在 Java 中将a 转换List为 a 的最简单方法是什么Set？



解答

Set<Foo> foo = new HashSet<Foo>(myList);


更多建议

我同意 sepp2k，但还有一些其他细节可能很重要：

new HashSet<Foo>(myList);

会给你一个没有重复的未排序的集合。在这种情况下，使用对象上的 .equals() 方法识别重复项。这是结合 .hashCode() 方法完成的。（有关平等的更多信息，请看这里）

给出排序集的替代方法是：

new TreeSet<Foo>(myList);

如果 Foo 实现了 Comparable，这就有效。如果没有，那么您可能需要使用比较器：

Set<Foo> lSet = new TreeSet<Foo>(someComparator);
lSet.addAll(myList);

这取决于 compareTo() （来自可比接口​​）或 compare() （来自比较器）以确保唯一性。因此，如果您只关心唯一性，请使用 HashSet。如果您在排序之后，请考虑 TreeSet。（记住：稍后优化！）如果时间效率很重要，请使用 HashSet 如果空间效率很重要，请查看 TreeSet。请注意，可以通过 Trove（和其他位置）获得更有效的 Set 和 Map 实现。
