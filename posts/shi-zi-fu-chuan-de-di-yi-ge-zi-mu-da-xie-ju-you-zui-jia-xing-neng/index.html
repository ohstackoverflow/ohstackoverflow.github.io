使字符串的第一个字母大写（具有最佳性能）

Arya

2023-02-18 00:36

我有一个DetailsViewwith aTextBox并且我希望 输入数据 始终 以 _ 大写的第一个字母_ 保存。

例子：

"red" --> "Red"
"red house" --> " Red house"

我怎样才能实现这种 最大化的性能 ？

注意 ：

根据答案和答案下的评论，很多人认为这是在询问字符串中 所有单词的大写。 例如=> Red House ，它不是，但如果那是您要寻找的TextInfo，请寻找使用的方法的答案之一ToTitleCase。（注意：对于实际提出的问题，这些答案是 不正确的。）请参阅 TextInfo.ToTitleCase 文档了解注意事项（不要触及全部大写的单词 - 它们被视为首字母缩略词；可以在“不应该”的单词中间使用小写字母降低，例如，“McDonald”→“Mcdonald”；不保证处理所有文化特定的微妙之处重新大写规则。）

注意 ：

关于第一个字母之后的字母是否应 强制 为 小写的问题是 模棱两可的 。接受的答案假定 只有第一个字母应该被改变 。如果要强制 字符串中除第一个字母以外的所有字母都 为小写，请查找包含ToTitleCase且不 包含 ToTitleCase 的 答案。 **** ToLower



解答

不同 C# 版本的解决方案

至少具有 .NET Core 3.0 或 .NET Standard 2.1 的 C# 8

public static class StringExtensions
{
    public static string FirstCharToUpper(this string input) =>
        input switch
        {
            null => throw new ArgumentNullException(nameof(input)),
            "" => throw new ArgumentException($"{nameof(input)} cannot be empty", nameof(input)),
            _ => string.Concat(input[0].ToString().ToUpper(), input.AsSpan(1))
        };
}

由于 .NET Core 3.0 / .NET Standard 2.1String.Concat()支持ReadonlySpan<char>如果我们使用.AsSpan(1)而不是 .NET可以节省一次分配.Substring(1)。

＃8

public static class StringExtensions
{
    public static string FirstCharToUpper(this string input) =>
        input switch
        {

C# 7

public static class StringExtensions
{
    public static string FirstCharToUpper(this string input)
    {
        switch (input)
        {
            case null: throw new ArgumentNullException(nameof(input));
            case "": throw new ArgumentException($"{nameof(input)} cannot be empty", nameof(input));
            default: return input[0].ToString().ToUpper() + input.Substring(1);
        }
    }
}

可能最快的解决方案是Darren 的（甚至有一个基准），尽管我会更改它的string.IsNullOrEmpty(s)验证以抛出异常，因为最初的要求期望第一个字母存在以便它可以大写。请注意，此代码适用于通用字符串，而不是特别适用于Textbox.


更多建议

public string FirstLetterToUpper(string str)
{
    if (str == null)
        return null;

    if (str.Length > 1)
        return char.ToUpper(str[0]) + str.Substring(1);

    return str.ToUpper();
}

旧答案：这使得每个首字母大写

public string ToTitleCase(string str)
{
    return CultureInfo.CurrentCulture.TextInfo.ToTitleCase(str.ToLower());
}
