获取 Flask 请求中接收到的数据

Arya

2023-02-17 04:45

我希望能够将数据发送到我的 Flask 应用程序。我试过访问request.data，但它是一个空字符串。您如何访问请求数据？

from flask import request

@app.route('/', methods=['GET', 'POST'])
def parse_request():
    data = request.data  # data is empty
    # need posted data here

这个问题的答案让我问Get raw POST body in Python Flask regardless of Content-Type header next，这是关于获取原始数据而不是解析数据。



解答

文档描述了请求期间对象 ( )上request可用的属性。from flask import request在大多数常见情况下request.data将为空，因为它用作后备：

request.data包含作为字符串传入的请求数据，以防它带有 Flask 无法处理的 mimetype。

  • request.args: URL 查询字符串中的键/值对
  • request.form：正文中的键/值对，来自 HTML post 表单或非 JSON 编码的 JavaScript 请求
  • request.files: 主体中的文件，Flask 与form. 必须使用 HTML 表单enctype=multipart/form-data，否则将不会上传文件。
  • request.values: 组合argsand form，args如果键重叠则更喜欢
  • request.json: 解析的 JSON 数据。请求必须有application/json内容类型，或者使用request.get_json(force=True)忽略内容类型。

所有这些都是MultiDict实例（除了json）。您可以使用以下方法访问值：

  • request.form['name']：如果您知道密钥存在，请使用索引
  • request.form.get('name'):get如果密钥可能不存在则使用
  • request.form.getlist('name')``getlist:如果密钥被多次发送并且您想要一个值列表，则使用。get只返回第一个值。


更多建议

对于 URL 查询参数，请使用request.args.

search = request.args.get("search")
page = request.args.get("page")

对于发布的表单输入，请使用request.form.

email = request.form.get('email')
password = request.form.get('password')

对于使用 content type 发布的 JSON application/json，请使用request.get_json().

data = request.get_json()
