如何访问“？”后的GET参数 在快递？

Arya

2023-02-17 19:44

我知道如何为这样的查询获取参数：

app.get('/sample/:id', routes.sample);

在这种情况下，我可以使用req.params.id获取参数（例如2in /sample/2）。

但是，对于 url like /sample/2?color=red，如何访问变量color？

我试过了req.params.color，但没用。



解答

因此，在检查了express reference之后，我发现它req.query.color会返回我正在寻找的值。

req.params 指的是 URL 中带有“:”的项目，req.query 指的是与“?”关联的项目

例子：

GET /something?color1=red&color2=blue

然后在快递中，处理程序：

app.get('/something', (req, res) => {
    req.query.color1 === 'red'  // true
    req.query.color2 === 'blue' // true
})


更多建议

使用 req.query，获取路由中查询字符串参数中的值。参考req.query。假设在路由中，http://localhost:3000/?name=satyam你想获取名称参数的值，那么你的“获取”路由处理程序将像这样：-

app.get('/', function(req, res){
    console.log(req.query.name);
    res.send('Response send to client::'+req.query.name);

});
