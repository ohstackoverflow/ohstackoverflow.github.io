如何在 Linux shell 脚本中提示输入是/否/取消？

Arya

2023-02-17 00:09

我想在 shell 脚本中暂停输入，并提示用户进行选择。
标准Yes、No或Cancel类型问题。
我如何在典型的 bash 提示符下完成此操作？



解答

在 shell 提示符下获取用户输入的最简单和最广泛使用的方法是read命令。说明其使用的最佳方式是一个简单的演示：

while true; do
    read -p "Do you wish to install this program? " yn
    case $yn in
        [Yy]* ) make install; break;;
        [Nn]* ) exit;;
        * ) echo "Please answer yes or no.";;
    esac
done

Steven Huwig指出的另一种方法是 Bash 的命令。这是使用 的相同示例：select``select

echo "Do you wish to install this program?"
select yn in "Yes" "No"; do
    case $yn in
        Yes ) make install; break;;
        No ) exit;;
    esac
done

你不需要select清理输入——它会显示可用的选择，然后你输入一个与你的选择相对应的数字。while true它还会自动循环，因此如果输入无效，则无需循环重试。

此外，Léa Gris在她的回答中演示了一种使请求语言不可知的方法。调整我的第一个示例以更好地服务于多种语言可能如下所示：

set -- $(locale LC_MESSAGES)
yesexpr="$1"; noexpr="$2"; yesword="$3"; noword="$4"

while true; do
    read -p "Install (${yesword} / ${noword})? " yn
    if [[ "$yn" =~ $yesexpr ]]; then make install; exit; fi
    if [[ "$yn" =~ $noexpr ]]; then exit; fi
    echo "Answer ${yesword} / ${noword}."
done

显然，其他通信字符串在这里仍未翻译（安装、回答），需要在更完整的翻译中解决，但在许多情况下，即使是部分翻译也会有所帮助。

最后，请查看F. Hauri的出色回答。

1. POSIX通用解决方案

您可以使用read命令，然后是if ... then ... else：

printf 'Is this a good question (y/n)? '
read answer

~~

 # if echo "$answer" | grep -iq "^y" ;then

~~

if [ "$answer" != "${answer#[Yy]}" ] ;then # this grammar (the #[] operator) means that the variable $answer where any Y or y in 1st position will be dropped if they exist.
    echo Yes
else
    echo No
fi

（感谢Adam Katz 的评论：将上面的测试替换为更便携且避免一次分叉的测试：）

POSIX，但单键功能

但是，如果您不希望用户必须点击Return，您可以这样写：

（ 编辑： 正如@JonathanLeffler正确建议的那样， 保存 stty 的配置可能比简单地强制它们保持 理智 更好。）

printf 'Is this a good question (y/n)? '
old_stty_cfg=$(stty -g)
stty raw -echo ; answer=$(head -c 1) ; stty $old_stty_cfg # Careful playing with stty
if [ "$answer" != "${answer#[Yy]}" ];then
    echo Yes
else
    echo No
fi

注意： 这是在下面测试的嘘,狂欢,ksh,短跑和busybox！

相同，但明确等待yor n：

#/bin/sh
printf 'Is this a good question (y/n)? '
old_stty_cfg=$(stty -g)
stty raw -echo
answer=$( while ! head -c 1 | grep -i '[ny]' ;do true ;done )
stty $old_stty_cfg
if [ "$answer" != "${answer#[Yy]}" ];then
    echo Yes
else
    echo No
fi

使用专用工具

有许多工具是使用libncurses、或其他图形库构建libgtk的。libqt例如，使用whiptail：

if whiptail --yesno "Is this a good question" 20 60 ;then
    echo Yes
else
    echo No
fi

根据您的系统，您可能需要更换whiptail为另一个类似的工具：

dialog --yesno "Is this a good question" 20 60 && echo Yes

kdialog --yesno "Is this a good question" 20 60 && echo Yes

其中20是对话框的行数高度，60是对话框的宽度。这些工具都具有 几乎相同的 语法。

DIALOG=whiptail
if [ -x /usr/bin/gdialog ] ;then DIALOG=gdialog ; fi
if [ -x /usr/bin/xdialog ] ;then DIALOG=xdialog ; fi
...
$DIALOG --yesno ...

2. Bash具体解决方案

基本 在线 方法

read -p "Is this a good question (y/n)? " answer
case ${answer:0:1} in
    y|Y )
        echo Yes
    ;;
    * )
        echo No
    ;;
esac

我更喜欢使用case这样我什至可以测试yes | ja | si | oui是否需要...

符合 单键 功能_

在 bash 下，我们可以为命令指定预期输入的长度read：

read -n 1 -p "Is this a good question (y/n)? " answer

在 bash 下，read命令接受 超时 参数，这可能很有用。

read -t 3 -n 1 -p "Is this a good question (Y/n)? " answer
[ -z "$answer" ] && answer="Yes"  # if 'yes' have to be default choice

倒计时 超时：

i=6 ;while ((i-->1)) &&
! read -sn 1 -t 1 -p $'\rIs this a good question (Y/n)? '$i$'..\e[3D' answer;do
  :;done ;[[ $answer == [nN] ]] && answer=No || answer=Yes ;echo "$answer "

#!/bin/sh
while true ;do
    [ -x "$(which ${DIALOG%% *})" ] || DIALOG=dialog

            whiptail       "dialog boxes from shell scripts" >/dev/tty \
            dialog         "dialog boxes from shell with ncurses" \
            gdialog        "dialog boxes from shell with Gtk" \
            kdialog        "dialog boxes from shell with Kde" ) || break
    clear;echo "Choosed: $DIALOG."
    for i in `seq 1 100`;do
        date +"`printf "XXX\n%d\n%%a %%b %%T progress: %d\nXXX\n" $i $i`"
        sleep .0125
      done | $DIALOG --gauge "Filling the tank" 20 60 0
    $DIALOG --infobox "This is a simple info box\n\nNo action required" 20 60
    sleep 3
    if $DIALOG --yesno  "Do you like this demo?" 20 60 ;then
        AnsYesNo=Yes; else AnsYesNo=No; fi
    AnsInput=$($DIALOG --inputbox "A text:" 20 60 "Text here..." 2>&1 >/dev/tty)
    AnsPass=$($DIALOG --passwordbox "A secret:" 20 60 "First..." 2>&1 >/dev/tty)

        Fun        "This demo is nice"        off \

    out="Your answers:\nLike: $AnsYesNo\nInput: $AnsInput\nSecret: $AnsPass"
    $DIALOG --msgbox "$out\nAttribs: $AnsCkLst\nNote: $AnsRadio" 20 60
  done

5.使用readline的历史

例子：

#!/bin/bash
set -i
HISTFILE=~/.myscript.history
history -c
history -r
myread() {
    read -e -p '> ' $1
    history -s ${!1}
}
trap 'history -a;exit' 0 1 2 3 6
while myread line;do
    case ${line%% *} in
        exit )  break ;;
        *    )  echo "Doing something with '$line'" ;;
      esac
  done

.myscript.history这将在您的目录中创建一个文件$HOME，而不是您可以使用 readline 的历史命令，如Up, Down, Ctrl+r等。
