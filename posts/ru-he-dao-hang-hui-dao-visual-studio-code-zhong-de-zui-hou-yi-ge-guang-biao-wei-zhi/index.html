如何导航回到 Visual Studio Code 中的最后一个光标位置？

Arya

2023-02-17 05:25

在 Visual Studio Code 中导航回最后一个光标位置的键盘快捷键是什么？



解答

键盘快捷键命令 是Go Forward 和 Go Back 。

在 Windows 上：

Alt+ ←... 向后导航

Alt+ →... 向前导航

在 Mac 上：

Ctrl+ -... 向后导航

Ctrl+ Shift+ -... 向前导航

在 Ubuntu Linux 上：

Ctrl+ Alt+ -... 向后导航

Ctrl+ Shift+ -... 向前导航


更多建议

我在 Mac OS X 上，所以我无法回答 Windows 用户：

我添加了一个自定义键盘映射条目并将其设置为Ctrl+ ←+ Ctrl+ →，而原始默认值是Ctrl+-和Ctrl+ Shift+ -（在我的德语键盘上转换为Ctrl+ß和Ctrl+ Shift+ ß）。

可以在用户键盘映射设置中简单地修改它：

{ "key": "ctrl+left",  "command": "workbench.action.navigateBack" },
{ "key": "ctrl+right", "command": "workbench.action.navigateForward" }
