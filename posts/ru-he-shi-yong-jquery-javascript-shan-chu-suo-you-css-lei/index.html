如何使用 jQuery/JavaScript 删除所有 CSS 类？

Arya

2023-02-17 11:11

不是$("#item").removeClass()单独调用元素可能具有的每个类，是否有一个可以调用的函数从给定元素中删除所有 CSS 类？

jQuery 和原始 JavaScript 都可以工作。



解答

$("#item").removeClass();

不带参数调用removeClass将删除项目的所有类。

也可以使用（但不一定推荐， 正确的 方法是上面那个）：

$("#item").removeAttr('class');
$("#item").attr('class', '');
$('#item')[0].className = '';

如果你没有 jQuery，那么这几乎是你唯一的选择：

document.getElementById('item').className = '';


更多建议

等一下，如果没有指定任何特定内容，removeClass() 不会默认删除所有类吗？所以

$("#item").removeClass();

会自己做...
