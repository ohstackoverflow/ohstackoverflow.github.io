如何在 AngularJS 中使用 ng-repeat 遍历键和值？

Arya

2023-02-17 17:22

在我的控制器中，我有如下数据：$scope.object = data

现在这个数据是带有键和值的字典json。

object.name我可以在模板中访问属性。有什么方法可以遍历键并将它们显示在表中

<tr><td> {{key}} </td> <td> data.key </td>

数据是这样的

{
    "id": 2,
    "project": "wewe2012",
    "date": "2013-02-26",
    "description": "ewew",
    "eet_no": "ewew",
}



解答

怎么样：

<table>
  <tr ng-repeat="(key, value) in data">
    <td> {{key}} </td> <td> {{ value }} </td>
  </tr>
</table>

此方法列在文档中：https://docs.angularjs.org/api/ng/directive/ngRepeat


更多建议

如果您想使用两种方式绑定编辑属性值：

<tr ng-repeat="(key, value) in data">
    <td>{{key}}<input type="text" ng-model="data[key]"></td>
</tr>
