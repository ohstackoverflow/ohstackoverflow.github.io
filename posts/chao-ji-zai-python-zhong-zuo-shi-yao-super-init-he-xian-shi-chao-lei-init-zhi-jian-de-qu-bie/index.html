“超级”在 Python 中做什么？- super().__init__() 和显式超类 __init__() 之间的区别

Arya

2023-02-17 16:14

有什么区别：

class Child(SomeBaseClass):
    def __init__(self):
        super(Child, self).__init__()

和：

class Child(SomeBaseClass):
    def __init__(self):
        SomeBaseClass.__init__(self)

我已经看到super在只有单一继承的类中使用了很多。我明白你为什么要在多重继承中使用它，但不清楚在这种情况下使用它有什么好处。



解答

单继承的好处super()很小——主要是，您不必将基类的名称硬编码到使用其父方法的每个方法中。

但是，如果没有 . ，使用多重继承几乎是不可能的super()。这包括常见的习语，如混合、接口、抽象类等。这扩展到以后扩展你的代码。如果以后有人想写一个扩展的类Child和一个 mixin，他们的代码将无法正常工作。
