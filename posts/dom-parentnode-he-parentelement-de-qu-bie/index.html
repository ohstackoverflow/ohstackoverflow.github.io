DOM parentNode 和 parentElement 的区别

Arya

2023-02-17 18:23

有人可以简单地解释一下，经典 DOM parentNode和 Firefox 9 中新引入的parentElement有什么区别



解答

parentElement是 Firefox 9 和 DOM4 的新功能，但它已经存在于所有其他主要浏览器中很长时间了。

在大多数情况下，它与parentNode. 唯一的区别是当节点parentNode不是元素时。如果是这样，parentElement是null。

举个例子：

document.body.parentNode; // the <html> element
document.body.parentElement; // the <html> element

document.documentElement.parentNode; // the document node
document.documentElement.parentElement; // null

(document.documentElement.parentNode === document);  // true
(document.documentElement.parentElement === document);  // false

由于<html>元素 ( document.documentElement) 没有作为元素的父元素，parentElement因此null. （还有其他不太可能parentElement发生的情况null，但您可能永远不会遇到它们。）


更多建议

在 Internet Explorer 中，parentElement对于 SVG 元素未定义，而已parentNode定义。
