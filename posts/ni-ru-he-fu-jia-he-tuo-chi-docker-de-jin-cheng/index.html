你如何附加和脱离 Docker 的进程？

Arya

2023-02-18 04:52

我可以附加到一个 docker 进程，但Ctrl+C无法从它分离。exit基本上停止了这个过程。

让进程运行、偶尔附加到它以进行一些更改然后分离的推荐工作流是什么？



解答

要在不退出 shell 的情况下分离 tty，请使用转义序列Ctrl+P后跟Ctrl+ Q。更多细节在这里。

此来源的其他信息：

  • ^P^Qdocker run -t -i → 可以用 docker attach分离和重新附加
  • docker run -i → 不能分离^P^Q; 会破坏标准输入
  • ^P^Qdocker run → 不能与;分离 可以SIGKILL客户端；可以用 docker attach 重新附加


更多建议

另请查看选项--sig-proxy：

docker attach --sig-proxy=false 304f5db405ec

然后使用Ctrl+C分离
