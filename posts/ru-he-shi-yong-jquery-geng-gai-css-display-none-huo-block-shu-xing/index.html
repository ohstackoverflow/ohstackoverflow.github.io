如何使用 jQuery 更改 CSS display none 或 block 属性？



解答

正确的做法是使用showand hide：

$('#id').hide();
$('#id').show();

另一种方法是使用 jQuery css方法：

$("#id").css("display", "none");
$("#id").css("display", "block");


更多建议

有几种方法可以实现这一点，每种方法都有自己的预期目的。

1 .) 使用 内联 ，同时简单地为一个元素分配一个要做的事情列表

$('#ele_id').css('display', 'block').animate(....
$('#ele_id').css('display', 'none').animate(....

2 .) 在设置多个 CSS 属性时使用

$('#ele_id').css({
    display: 'none'
    height: 100px,
    width: 100px
});
$('#ele_id').css({
    display: 'block'
    height: 100px,
    width: 100px
});

3 .) 动态调用命令

$('#ele_id').show();
$('#ele_id').hide();

4 .) 在块和无之间动态切换，如果它是一个 div

  • 一些元素显示为内联、内联块或表格，具体 取决于标签名称

$('#ele_id').toggle();
