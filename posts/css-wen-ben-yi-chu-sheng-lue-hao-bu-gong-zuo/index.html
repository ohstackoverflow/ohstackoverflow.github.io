CSS 文本溢出：省略号；不工作？

Arya

2023-02-17 23:45

我不知道为什么这个简单的 CSS 不起作用...

.app a {
  height: 18px;
  width: 140px;
  padding: 0;
  overflow: hidden;
  position: relative;
  margin: 0 5px 0 5px;
  text-align: center;
  text-decoration: none;
  text-overflow: ellipsis;
  white-space: nowrap;
  color: #000;
}


<div class="app">
  <a href="">Test Test Test Test Test Test</a>
</div>

解答

text-overflow:ellipsis;仅在满足以下条件时有效：

  • 元素的宽度必须限制在px（像素）内。宽度%（百分比）不起作用。
  • 该元素必须具有overflow:hidden并white-space:nowrap设置。

您在这里遇到问题的原因是width您的a元素不受约束。您确实有一个width设置，但是因为该元素设置为display:inline（即默认值），所以它忽略了它，并且没有其他任何东西限制它的宽度。

您可以通过执行以下操作之一来解决此问题：

  • 将元素设置为display:inline-block或display:block（可能是前者，但取决于您的布局需要）。
  • 将其中一个容器元素设置为display:block并为该元素指定一个固定的widthor max-width。
  • 将元素设置为float:leftor float:right（可能是前者，但同样，就省略号而言，两者都应该具有相同的效果）。

我建议display:inline- block，因为这对您的布局的附带影响最小；它非常像display:inline就布局而言，它的工作方式与当前使用的我已尝试提供尽可能多的信息，以帮助您了解这些事物如何相互作用；理解 CSS 的很大一部分是理解各种样式如何协同工作。

这是您的代码片段，添加了一个display:inline-block，以显示您有多接近。

.app a {
  height: 18px;
  width: 140px;
  padding: 0;
  overflow: hidden;
  position: relative;
  display: inline-block;
  margin: 0 5px 0 5px;
  text-align: center;
  text-decoration: none;
  text-overflow: ellipsis;
  white-space: nowrap;
  color: #000;
}

更多建议

接受的答案很棒。但是，您仍然可以使用%width 和 attain text-overflow: ellipsis。解决方法很简单：

display: inline-block; /* for inline elements e.g. span, strong, em etc */
text-overflow: ellipsis;
width: calc(80%); /* The trick is here! */

似乎无论何时使用calc，最终值都以绝对像素呈现，因此会转换为类似于-width 容器的80%东西。因此，不使用，而是使用。800px``1000px``width: [YOUR PERCENT]%``width: calc([YOUR PERCENT]%)
