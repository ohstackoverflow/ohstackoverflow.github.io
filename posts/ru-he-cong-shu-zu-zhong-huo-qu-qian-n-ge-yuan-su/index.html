如何从数组中获取前 N 个元素

Arya

2023-02-17 07:18

我正在使用 Javascript(ES6) /FaceBook 做出反应并尝试获取大小不同的数组的前 3 个元素。我想做相当于 Linq take(n) 的事情。

在我的 Jsx 文件中，我有以下内容：

var items = list.map(i => {
  return (
    <myview item={i} key={i.id} />
  );
});

然后得到我试过的前 3 个项目

  var map = new Map(list);
    map.size = 3;
    var items = map(i => {
      return (<SpotlightLandingGlobalInboxItem item={i} key={i.id} />);
    });

这不起作用，因为地图没有设置功能。

你能帮忙吗？



解答

我相信你要找的是：

// ...inside the render() function

var size = 3;
var items = list.slice(0, size).map(i => {
    return <myview item={i} key={i.id} />
});                       
return (
  <div>
    {items}
  </div>   
)
