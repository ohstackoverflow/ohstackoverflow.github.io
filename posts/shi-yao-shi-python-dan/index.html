什么是 Python 蛋？

Arya

2023-02-17 18:55

我试图了解 Python 包的工作原理。大概eggs是某种打包机制，但如何快速概述它们所扮演的角色，以及关于它们为何有用以及如何创建它们的一些信息？



解答

注意： Egg 包装已被 Wheel 包装取代。

与 Java 中文件的概念相同.jar，它是.zip一些元数据文件重命名的文件.egg，用于将代码作为包分发。

具体来说：Python Eggs 的内部结构

“Python 蛋”是一种逻辑结构，体现了 Python 项目特定版本的发布，包括其代码、资源和元数据。有多种格式可用于对 Python egg 进行物理编码，并且可以开发其他格式。然而，Python eggs 的一个关键原则是它们应该是可发现和可导入的。也就是说，Python 应用程序应该可以轻松高效地找出系统中存在的鸡蛋，并确保所需鸡蛋的内容是可导入的。

该.egg格式非常适合代码的分发和轻松卸载或升级，因为该项目本质上是独立于单个目录或文件中的，没有与任何其他项目的代码或资源混合。它还使得同时安装一个项目的多个版本成为可能，这样各个程序就可以选择他们希望使用的版本。


更多建议

该.egg文件是 Python 包的分发格式。它只是源代码分发或 Windows exe. 但请注意，对于 pure Python，该.egg文件是完全跨平台的。

文件.egg本身本质上是一个.zip文件。如果将扩展名更改为“ zip”，您可以看到它在存档中有文件夹。

另外，如果你有一个.egg文件，你可以使用easy_install

示例：要为一个目录创建一个.egg文件，mymath比如它本身可能有多个 python 脚本，请执行以下步骤：

# setup.py
from setuptools import setup, find_packages
setup(
    name = "mymath",
    version = "0.1",
    packages = find_packages()
    )

然后，从终端执行：

 $ python setup.py bdist_egg

这将生成大量输出，但是当它完成时，您会看到您拥有 三个 新文件夹： build 、 dist 和 mymath.egg-info 。我们唯一关心的文件夹是 dist 文件夹，您可以在其中找到您的.egg文件， mymath-0.1-py3.5.egg 以及您的默认 python（安装）版本号（我在这里：3.5）
