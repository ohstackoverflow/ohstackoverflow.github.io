如何禁用 Python 警告？

Arya

2023-02-17 13:12

我正在处理使用库抛出很多（目前对我而言）无用警告的代码warnings。阅读（/扫描）文档我只找到了一种方法来禁用单个功能的警告。但我不想更改那么多代码。

有没有旗子之类的python -no-warning foo.py？

你会推荐什么？



解答

有-W选择权。

python -W ignore foo.py


更多建议

有-W选择权。

python -W ignore foo.py
