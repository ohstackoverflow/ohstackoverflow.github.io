getColor(int id) 在 Android 6.0 Marshmallow (API 23) 上弃用

Arya

2023-02-17 13:53

该Resources.getColor(int id)方法已被弃用。

@ColorInt
@Deprecated
public int getColor(@ColorRes int id) throws NotFoundException {
    return getColor(id, null);
}

我应该怎么办？



解答

从 Android 支持库 23 开始，
新的getColor()方法已添加到ContextCompat.

它来自官方 JavaDoc 的描述：

返回与特定资源 ID 关联的颜色

从 M 开始，返回的颜色将根据指定的上下文主题设置样式。

所以， 只需致电 ：

ContextCompat.getColor(context, R.color.your_color);

ContextCompat.getColor() 你可以在 GitHub 上查看源代码。


更多建议

长话短说：

ContextCompat.getColor(context, R.color.my_color)

解释：

您将需要使用ContextCompat.getColor()，它是支持 V4 库的一部分（它将适用于所有以前的 API）。

ContextCompat.getColor(context, R.color.my_color)

如果您还没有使用支持库，则需要将以下行添加到dependencies应用程序中的数组中build.gradle（注意： 如果您已经使用了 appcompat (V7) 库，则它是可选的 ）：

compile 'com.android.support:support-v4:23.0.0' # or any version above

如果您关心主题，文档会指定：

从 M 开始，返回的颜色将根据指定的上下文主题设置样式
