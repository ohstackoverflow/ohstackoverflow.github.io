__init__ 和 __call__ 有什么区别？

Arya

2023-02-17 15:33

__init__我想知道和方法之间的区别__call__。

例如：

class test:

  def __init__(self):
    self.a = 10

  def __call__(self): 
    b = 20



解答

第一个用于初始化新创建的对象，并接收用于执行此操作的参数：

class Foo:
    def __init__(self, a, b, c):
        # ...

x = Foo(1, 2, 3) # __init__

第二个实现函数调用运算符。

class Foo:
    def __call__(self, a, b, c):
        # ...

x = Foo()
x(1, 2, 3) # __call__


更多建议

定义自定义__call__()方法允许类的实例作为函数调用，而不是总是修改实例本身。

In [1]: class A:
   ...:     def __init__(self):
   ...:         print "init"
   ...:         
   ...:     def __call__(self):
   ...:         print "call"
   ...:         
   ...:

In [2]: a = A()
init

In [3]: a()
call
