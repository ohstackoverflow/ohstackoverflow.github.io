将 JavaScript 日期初始化为午夜的最佳方法是什么？

Arya

2023-02-18 00:30

获取实例new Date()但将时间设置为午夜的最简单方法是什么？



解答

该setHours方法可以采用可选的minutes,seconds和ms参数，例如：

var d = new Date();
d.setHours(0,0,0,0);

这会将时间设置为00:00:00.000您 当前的时区 ，如果您想以 UTC 时间工作，可以使用该setUTCHours方法。


更多建议

只是想澄清一下，接受的答案中的片段给出了过去 最近的午夜 ：

var d = new Date();
d.setHours(0,0,0,0); // last midnight

如果你想在未来 获得最近的午夜，请使用以下代码：

var d = new Date();
d.setHours(24,0,0,0); // next midnight
