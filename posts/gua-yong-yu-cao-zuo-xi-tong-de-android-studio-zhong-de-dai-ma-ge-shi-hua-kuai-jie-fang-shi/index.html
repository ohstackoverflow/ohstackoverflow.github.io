适用于操作系统的 Android Studio 中的代码格式化快捷方式

Arya

2023-02-17 10:52

我已经开始使用Android Studio进行开发。在 Eclipse 中我使用的是Ctrl++ ，但在 Android StudioShift中F它不起作用。这将是不同的。

如何跳转到.java文件中的任何方法？我在 Eclipse 中使用Ctrl+ 。O

是可用的么？代码格式化的快捷方式是什么？



解答

窗口：Ctrl+ Alt+L

Linux：Ctrl+ Shift+ Alt+L

macOS：Option+ Command+L

参考：Key Commands和此处是适用于Windows/Linux 用户和Mac 用户的所有命令。

由于 Rohit 在 Ubuntu 中遇到格式代码快捷方式的问题，这是由于在 Ubuntu 中使用Ctrl+ Alt+L键锁定屏幕。

我发现 Ubuntu 首先处理这个键盘快捷键。所以你应该将Ctrl+ Alt+L键盘快捷键绑定到其他东西，这样它就不会与 Ubuntu 冲突。

脚步

  1. 转到 系统工具 → 系统设置 → 键盘 → 快捷方式 选项卡 → 系统 → 锁定屏幕 。

  2. 选择New Accelerator... 行，然后按带有 Alpha 键的任意特殊键（例如Shift+ L）。您应该已经成功更改了键盘快捷键。

  3. 检查键盘快捷键现在是否可以在 Android Studio 中使用。

替代方法

Ctrl+ Alt+ Shift+ L（显示对话框）

您还可以使用 Eclipse 快捷方式。

  • Windows 和 Linux：转到菜单 文件 → 设置 → 键盘映射
  • macOS：转到菜单 Preferences → Keymap 并从下拉菜单中选择 Eclipse。

使用菜单栏

  • 选择要重新格式化的代码

  • 单击菜单栏中的 代码，然后选择 重新格式化代码 ，如下图所示：

在Android studio 3.5中，重新排列xml代码有问题，它也会重新排列视图，所以需要先使用这些设置。

设置 -> 编辑器 -> 代码样式 -> XML -> 设置自 -> 预定义样式 > Android。

适用于 Android Studio 4+ 的更新

从 Android Studio 4+ 开始，您可能会在 Mac 上遇到快捷方式问题。在 Mac 中更新到 Android studio 4 后快捷方式不起作用？有一个答案。


更多建议

您还可以使用 Eclipse 的键盘快捷键：只需转至菜单 Preferences → keymap 并从下拉菜单中选择 Eclipse 。

实际路径为：菜单 File → Settings → Keymap (IDE settings下)
