GitHub“致命：远程源已经存在”

Arya

2023-02-17 23:06

我正在尝试按照Michael Hartl 的 Rails 教程进行操作，但遇到了一个错误。

我在 GitHub 上注册，发布了一个新的 SSH 密钥，并创建了一个新的存储库。但是当我在终端中输入下一行时，出现以下错误：

Parkers-MacBook-Pro:.ssh ppreyer$ git remote add origin git@github.com:ppreyer/first_app.git
fatal: remote origin already exists.

只是想知道是否还有其他人遇到过这个问题？



解答

简洁版本：

您应该只更新现有的遥控器：

$ git remote set-url origin git@github.com:ppreyer/first_app.git

长版：

如错误消息所示，已经有一个配置了相同名称的远程。因此，您可以添加具有不同名称的新遥控器，或者在不需要时更新现有遥控器。

要添加一个新的遥控器，例如调用github而不是origin（显然已经存在于您的系统中），请执行以下操作：

$ git remote add github git@github.com:ppreyer/first_app.git

不过请记住，在本教程的任何地方，您都可以看到 “origin” ，您应该将其替换为 “github” 。例如$ git push origin master现在应该是$ git push github master。

但是，如果您想查看已经存在的origin遥控器是什么，您可以执行$ git remote -v. 如果您认为这是错误的，您可以像这样更新它：

$ git remote set-url origin git@github.com:ppreyer/first_app.git


更多建议

这有效：

git remote rm origin
git remote add origin git@github.com:username/myapp.git
