如何访问 Javascript 对象的第一个属性？

Arya

2023-02-17 11:06

有没有一种优雅的方法来访问对象的第一个属性......

  1. 你不知道你的房产名称的地方
  2. 不使用类似循环for .. in或 jQuery 的$.each

例如，我需要foo1在不知道 foo1 名称的情况下访问对象：

var example = {
    foo1: { /* stuff1 */},
    foo2: { /* stuff2 */},
    foo3: { /* stuff3 */}
};



解答

var obj = { first: 'someVal' };
obj[Object.keys(obj)[0]]; //returns 'someVal'

Object.values(obj)[0]; // returns 'someVal'

使用它，您还可以通过索引访问其他属性。请注意！Object.keys或Object.values根据 ECMAScript 不保证退货顺序，但所有主要浏览器实现都非正式地保证，请阅读https://stackoverflow.com/a/23202095了解详细信息。


更多建议

尝试for … in循环并在第一次迭代后中断：

for (var prop in object) {
    // object[prop]
    break;
}
