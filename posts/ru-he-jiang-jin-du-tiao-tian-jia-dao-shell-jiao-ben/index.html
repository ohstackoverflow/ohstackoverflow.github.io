如何将进度条添加到 shell 脚本？

Arya

2023-02-18 07:16

当在 bash 或 *NIX 中的任何其他 shell 中编写脚本时，在运行将花费几秒钟以上的命令时，需要一个进度条。

例如，复制一个大文件，打开一个大的 tar 文件。

你推荐用什么方法给 shell 脚本添加进度条？



解答

您可以通过覆盖一行来实现。用于\r返回行首而不写入\n终端。

\n当你完成前进时写信。

用于echo -ne：

  1. 不打印\n和
  2. 识别转义序列，如\r.

这是一个演示：

echo -ne '#####                     (33%)\r'
sleep 1
echo -ne '#############             (66%)\r'
sleep 1
echo -ne '#######################   (100%)\r'
echo -ne '\n'

在下面的评论中，puk 提到如果您从一长行开始然后想写一个短行会“失败”：在这种情况下，您需要覆盖长行的长度（例如，使用空格）。


更多建议

您可能还对如何制作微调器感兴趣：

我可以在 Bash 中做一个微调器吗？

当然！

i=1
sp="/-\|"
echo -n ' '
while true
do
    printf "\b${sp:i++%${#sp}:1}"
done

每次循环迭代时，它都会显示 sp 字符串中的下一个字符，并在到达末尾时回绕。（i 是当前要显示的字符的位置，${#sp} 是 sp 字符串的长度）。

\b 字符串被“退格”字符替换。或者，您可以使用 \r 返回到该行的开头。

如果你想让它慢下来，在循环中放置一个睡眠命令（在 printf 之后）。

一个 POSIX 等价物是：

sp='/-\|'
printf ' '
while true; do
    printf '\b%.1s' "$sp"
    sp=${sp#?}${sp%???}
done

如果您已经有一个执行大量工作的循环，则可以在每次迭代开始时调用以下函数来更新微调器：

sp="/-\|"
sc=0
spin() {
   printf "\b${sp:sc++:1}"
   ((sc==${#sp})) && sc=0
}
endspin() {
   printf "\r%s\n" "$@"
}

until work_done; do
   spin
   some_work ...
done
endspin
