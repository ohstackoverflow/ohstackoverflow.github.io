何时使用 IMG 与 CSS 背景图片？

Arya

2023-02-17 14:13

IMG在什么情况下使用 HTML标签显示图像比使用 CSS更合适background-image，反之亦然？

因素可能包括可访问性、浏览器支持、动态内容或任何类型的技术限制或可用性原则。



解答

正确使用 IMG

  1. IMG如果您打算让人们打印您的页面并且您希望默认包含图像，请使用。—杰蒂
  2. 当图像具有重要的语义意义时使用IMG(with text)，例如警告图标。这确保图像的含义可以在所有用户代理中传达，包括屏幕阅读器。alt

IMG 的实用用途

  1. IMG如果图像是内容的一部分，例如徽标或图表或人物（真人，而不是照片人物），请使用plus alt 属性。— sanchothefat
  2. IMG如果您依赖浏览器缩放比例来呈现与文本大小成比例的图像，请使用。
  3. 用于IE6 中IMG的多个叠加图像。

  4. IMG 与 a 一起使用z-index以拉伸背景图像以填充其整个窗口。
    注意，CSS3 background-size 不再适用；请参阅下面的#6。

  5. 使用imginstead ofbackground-image可以显着提高背景动画的性能。

何时使用 CSS 背景图片

  1. 如果图像不是内容的一部分，请使用 CSS 背景图像。— sanchothefat
  2. 在进行文本图像替换时使用 CSS 背景图像，例如。段落/标题。— sanchothefat
  3. background-image如果您打算让人们打印您的页面并且您不希望默认包含图像，请使用。—杰蒂
  4. background-image如果您需要缩短下载时间，请使用CSS sprites。
  5. background-image如果您只需要图像的一部分可见，请使用 CSS 精灵。
  6. 使用background-imagewithbackground-size:cover以拉伸背景图像以填充其整个窗口。


更多建议

对我来说这是一个非黑即白的决定。如果图像是内容的一部分，例如徽标或图表或人物（真人，而不是照片人物），则使用标签<img />加 alt 属性。对于其他所有内容，都有 CSS 背景图像。

另一个使用 CSS 背景图像的时间是在对文本进行图像替换时，例如。段落/标题。
