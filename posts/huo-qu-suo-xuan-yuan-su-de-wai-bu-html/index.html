获取所选元素的外部 HTML

Arya

2023-02-17 12:47

我正在尝试使用 jQuery 获取选定对象的 HTML。我知道这个.html()功能；问题是我需要包含所选对象的 HTML（在本例中为表格行，其中.html()仅返回行内的单元格）。

我四处搜索，发现了一些非常“hackish”类型的克隆对象的方法，将其添加到新创建的 div 等，但这看起来真的很脏。有没有更好的方法，或者新版本的 jQuery (1.4.2) 是否提供任何类型的outerHtml功能？



解答

2014 编辑：问题和回复来自 2010 年。当时，没有更好的解决方案被广泛使用。 现在，许多其他回复更好：例如 Eric Hu 或 Re Capcha 的回复。

这个站点似乎有适合您的解决方案：jQuery: outerHTML | 黄豆腐

jQuery.fn.outerHTML = function(s) {
    return s
        ? this.before(s).remove()
        : jQuery("<p>").append(this.eq(0).clone()).html();
};


更多建议

无需为其生成函数。就这样做：

$('a').each(function(){
    var s = $(this).clone().wrap('<p>').parent().html();
    console.log(s);
});

（顺便说一句，您的浏览器控制台会显示记录的内容。自 2009 年左右以来的大多数最新浏览器都具有此功能。）

最后的神奇之处在于：

.clone().wrap('<p>').parent().html();

克隆意味着您实际上并没有扰乱 DOM。在没有它的情况下运行它，您会看到p在所有超链接之前/之后插入的标签（在本例中），这是不希望的。所以，是的，使用.clone().

它的工作方式是获取每个a标签，在 RAM 中复制它，用标签包装p，获取它的父级（即标签p），然后获取innerHTML它的属性。

编辑 ：接受建议并将div标签更改为p标签，因为它的输入更少并且工作原理相同。
