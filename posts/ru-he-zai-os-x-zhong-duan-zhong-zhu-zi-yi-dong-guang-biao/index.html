如何在 OS X 终端中逐字移动光标

Arya

2023-02-17 10:14

我知道组合Ctrl+A跳转到当前命令的开头，Ctrl+E跳转到结尾。

但是有什么方法可以逐字跳转，就像Cocoa 应用程序中的Alt+ ←/→那样吗？



解答

开箱即用，您可以使用非常奇怪的Esc+F移动到下一个单词的开头，并使用Esc+B移动到当前单词的开头。


更多建议

在 macOS（所有版本）上，以下键盘快捷键默认工作。

  • ALT+ 向前F跳一个词。 ****
  • ALT+向后B跳 B 一个词。

请注意，您必须将 Option 键设置为类似于 Meta 键。您可以在终端中通过访问首选项 (CMD+,) 并选择配置文件 -> 键盘来执行此操作。在 iTerm2 Pselect rofiles -> Keys -> General 中选择“Option key as Esc+”。

此外，一些用于简单文本导航的 Emacs 风格的键绑定似乎也适用于 bash shell。您可以使用：

  • CTRL+F向前移动一个字符
  • CTRL+B向后移动一个字符
  • CTRL+A跳转到行首
  • CTRL+E跳转到行尾
  • CTRL+K杀死从光标位置开始的行
  • ALT+D删除从当前光标位置开始的单词
  • CTRL+W从光标位置向后删除单词
  • CTRL+Y粘贴终止缓冲区中的文本
  • CTRL+R从您的历史记录中反向搜索您过去输入的命令。
  • CTRL+S转发搜索（对我来说在 ZSH 中有效，但在 bash 中无效）
