在背景中拉伸和缩放 CSS 图像 - 仅使用 CSS

Arya

2023-02-17 09:19

我希望我的背景图像根据浏览器视口大小进行拉伸和缩放。

我在 Stack Overflow 上看到了一些可以完成这项工作的问题，例如 Stretch and scale CSS background 。background它运行良好，但我想使用而不是标签放置图像img。

在那个标签中img放置了一个标签，然后我们使用 CSS 向img标签致敬。

width:100%; height:100%;

它有效，但这个问题有点陈旧，并指出在 CSS 3 中调整背景图像的大小会很好地工作。我已经在第一个示例中尝试过这个示例，但它对我来说没有用。

有没有一个很好的方法来做声明background-image？



解答

CSS3 有一个不错的小属性，叫做background-size:cover.

这会缩放图像，以便背景区域完全被背景图像覆盖，同时保持纵横比。整个区域都将被覆盖。但是，如果调整后的图像的宽度/高度太大，则部分图像可能不可见。


更多建议

您可以使用 CSS3 属性来很好地完成它。它会调整比例，因此不会出现图像失真（尽管它会放大小图像）。请注意，它尚未在所有浏览器中实现。

background-size: 100%;
