如何“取消还原”还原的 Git 提交？

Arya

2023-02-17 14:43

鉴于已使用 提交commit，然后使用 还原的更改revert，然后撤消该还原的最佳方法是什么？

理想情况下，这应该通过新提交来完成，以免重写历史记录。



解答

如果你还没有推动那个改变，git reset --hard HEAD^

否则，还原还原是完全可以的。

另一种方法是git checkout HEAD^^ -- .然后git add -A && git commit。
