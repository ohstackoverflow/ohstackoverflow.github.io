如何检查数组是否包含 JavaScript 中的值？

Arya

2023-02-16 18:54

找出 JavaScript 数组是否包含值的最简洁有效的方法是什么？

这是我知道的唯一方法：

function contains(a, obj) {
    for (var i = 0; i < a.length; i++) {
        if (a[i] === obj) {
            return true;
        }
    }
    return false;
}

有没有更好更简洁的方法来完成这个？

这与 Stack Overflow 问题Best way to find an item in a JavaScript Array? 密切相关。它解决了使用 . 在数组中查找对象的问题indexOf。



解答

现代浏览器有Array#includes，它 确实 做到了这一点，并得到除 IE 之外的所有人的广泛支持：

console.log(['joe', 'jane', 'mary'].includes('jane')); //true

您也可以使用Array#indexOf，它不那么直接，但对于过时的浏览器不需要 polyfill。

console.log(['joe', 'jane', 'mary'].indexOf('jane') >= 0); //true

许多框架也提供了类似的方法：

  • 查询：$.inArray(value, array, [fromIndex])
  • Underscore.js：（_.contains(array, value)也别名为_.includeand _.includes）
  • 道场工具包：dojo.indexOf(array, value, [fromIndex, findLast])
  • 原型：array.indexOf(value)
  • 工具：array.indexOf(value)
  • 摩奇套件：findValue(array, value)
  • 阿贾克斯女士：array.indexOf(value)
  • 分机：Ext.Array.contains(array, value)
  • Lodash：（_.includes(array, value, [from])在_.contains4.0.0 之前）
  • 拉姆达：R.includes(value, array)

请注意，一些框架将其作为函数实现，而其他框架则将函数添加到数组原型中。

正如其他人所说，遍历数组可能是最好的方法，但已经证明递减while循环是 JavaScript 中最快的迭代方法。所以你可能想重写你的代码如下：

function contains(a, obj) {
    var i = a.length;
    while (i--) {
       if (a[i] === obj) {
           return true;
       }
    }
    return false;
}

当然，你也可以扩展 Array 原型：

Array.prototype.contains = function(obj) {
    var i = this.length;
    while (i--) {
        if (this[i] === obj) {
            return true;
        }
    }
    return false;
}

现在您可以简单地使用以下内容：

alert([1, 2, 3].contains(2)); // => true
alert([1, 2, 3].contains('2')); // => false
