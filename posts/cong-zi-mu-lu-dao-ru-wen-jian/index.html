从子目录导入文件？

Arya

2023-02-17 23:58

我有一个名为 的文件tester.py，位于/project.

/project有一个名为 的子目录lib，其中有一个名为 的文件BoxTime.py：

/project/tester.py
/project/lib/BoxTime.py

我想BoxTime从tester. 我试过这个：

import lib.BoxTime

结果是：

Traceback (most recent call last):
  File "./tester.py", line 3, in <module>
    import lib.BoxTime
ImportError: No module named lib.BoxTime

任何想法如何BoxTime从子目录导入？

编辑

这__init__.py就是问题所在，但不要忘记引用BoxTimeaslib.BoxTime或使用：

import lib.BoxTime as BT
...
BT.bt_function()



解答

查看 包文档（第 6.4 节） 。

简而言之，您需要放置一个名为

__init__.py

在lib目录中。


更多建议

  • 创建一个名为lib.
  • 创建一个名为lib\__init__.py.

  • 在 中，编写如下lib\BoxTime.py函数：foo()

    def foo():
print "foo!"

  • 在上面目录中的客户端代码中lib，写入：

    from lib import BoxTime

    BoxTime.foo()

  • 运行您的客户端代码。你会得到：

    foo!

很久以后——在 Linux 中，它看起来像这样：

% cd ~/tmp
% mkdir lib
% touch lib/__init__.py
% cat > lib/BoxTime.py << EOF
heredoc> def foo():
heredoc>     print "foo!"
heredoc> EOF
% tree lib
lib
├── BoxTime.py
└── __init__.py

0 directories, 2 files
% python

[GCC 4.8.2] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>> from lib import BoxTime
>>> BoxTime.foo()
foo!
