jQuery - 通过其文本描述设置选择控件的选定值

Arya

2023-02-18 05:14

我有一个选择控件，在一个 javascript 变量中我有一个文本字符串。

使用 jQuery，我想将选择控件的选定元素设置为具有我拥有的文本描述的项目（与我没有的值相反）。

我知道按值设置它非常简单。例如

$("#my-select").val(myVal);

但是我有点难以通过文本描述来做到这一点。我想一定有办法从文本描述中获取价值，但我的大脑在星期五下午太忙了，无法计算出来。



解答

按描述选择 jQuery v1.6+

var text1 = 'Two';
$("select option").filter(function() {
  //may want to use $.trim in here
  return $(this).text() == text1;
}).prop('selected', true);


<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>

<select>
  <option value="0">One</option>
  <option value="1">Two</option>
</select>

低于 1.6 且高于或等于 1.4 的 jQuery 版本

var text1 = 'Two';
$("select option").filter(function() {
  //may want to use $.trim in here
  return $(this).text() == text1;
}).attr('selected', true);


<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/1.4.0/jquery.min.js"></script>

<select>
  <option value="0">One</option>
  <option value="1">Two</option>
</select>

请注意，虽然此方法适用于 1.6 以上但低于 1.9 的版本，但自 1.6 起已弃用。它 不适用于 jQuery 1.9+。

之前的版本

val()应该处理这两种情况。

$('select').val('1'); // selects "Two"
$('select').val('Two'); // also selects "Two"


<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/1.2.3/jquery.min.js"></script>

<select>
  <option value="0">One</option>
  <option value="1">Two</option>
</select>


更多建议

我没有测试过这个，但这可能对你有用。

$("select#my-select option")
   .each(function() { this.selected = (this.text == myVal); });
