如何以编程方式确定 Java 中的操作系统？

Arya

2023-02-17 23:43

我想确定我的 Java 程序正在以编程方式运行的主机的操作系统（例如：我希望能够根据我是在 Windows 还是 Unix 平台上加载不同的属性）。以 100% 的可靠性做到这一点的最安全方法是什么？



解答

您可以使用：

System.getProperty("os.name")

PS 你可能会发现这段代码很有用：

class ShowProperties {
    public static void main(String[] args) {
        System.getProperties().list(System.out);
    }
}

它所做的只是打印出您的 Java 实现提供的所有属性。它会让您了解可以通过属性找到有关 Java 环境的哪些信息。:-)


更多建议

如其他答案所示，System.getProperty 提供原始数据。但是，Apache Commons Lang 组件为 java.lang.System提供了一个包装器，它具有方便的属性SystemUtils.IS_OS_WINDOWS，例如 ，非常类似于前面提到的 Swingx OS 实用程序。
