如何查找给定键是否存在于 C++ std::map 中

Arya

2023-02-18 01:07

我正在尝试检查给定的键是否在地图中，但有点做不到：

typedef map<string,string>::iterator mi;
map<string, string> m;
m.insert(make_pair("f","++--"));
pair<mi,mi> p = m.equal_range("f");//I'm not sure if equal_range does what I want
cout << p.first;//I'm getting error here

那么如何打印 p 中的内容呢？



解答

使用map::find和map::end：

if (m.find("f") == m.end()) {
  // not found
} else {
  // found
}


更多建议

要检查映射中的特定键是否存在，请count按以下方式之一使用成员函数：

m.count(key) > 0
m.count(key) == 1
m.count(key) != 0

的文档说：map::find“另一个成员函数，map::count可用于检查特定键是否存在。”

的文档说map::count：“因为地图容器中的所有元素都是唯一的，所以该函数只能返回 1（如果找到该元素）或零（否则）。”

要通过您知道存在的键从地图中检索值，请使用map::at：

value = m.at(key)

与map::operator[]不同，map::at如果指定键不存在，则不会在地图中创建新键。
