如何强制除法为浮点数？除法一直向下舍入为 0？

Arya

2023-02-17 14:01

我有两个整数值a和b，但我需要它们的浮点比。我知道并且a < b我想计算a / b，所以如果我使用整数除法，我总是会得到 0 的余数a。

在下面的 Python 2 中，如何强制c成为浮点数？

c = a / b

在 3.x 中，行为相反；请参阅为什么整数除法会产生浮点数而不是另一个整数？相反，3.x 特定的问题。



解答

在 Python 2 中，两个整数相除产生一个整数。在 Python 3 中，它产生一个浮点数。我们可以通过从 导入来获得新的行为__future__。

>>> from __future__ import division
>>> a = 4
>>> b = 6
>>> c = a / b
>>> c
0.66666666666666663


更多建议

你可以通过做来转换为浮动c = a / float(b)。如果分子或分母是浮点数，那么结果也将是。

b警告：正如评论者所指出的，如果不是整数或浮点数（或代表一个的字符串），这将不起作用。如果您可能正在处理其他类型（例如复数），则需要检查这些类型或使用不同的方法。
