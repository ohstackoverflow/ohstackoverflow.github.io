Flex-box：将最后一行与网格对齐

Arya

2023-02-18 05:45

我有一个简单的 flex-box 布局，容器如下：

.grid {
  display: flex;
  flex-flow: row wrap;
  justify-content: space-between;
}

现在我希望最后一行中的项目与其他项目对齐。justify-content: space-between;应该使用，因为可以调整网格的宽度和高度。

解答

添加一个::after自动填充空间的。无需污染您的 HTML。这是一个显示它的codepen：http://codepen.io/DanAndreasson/pen/ZQXLXj

.grid {
  display: flex;
  flex-flow: row wrap;
  justify-content: space-between;
}

.grid::after {
  content: "";
  flex: auto;
}


更多建议

正如其他海报所提到的那样——没有一种干净的方法可以将最后一行与 flexbox 左对齐（至少按照当前规范）

然而，就其价值而言：使用CSS 网格布局模块，这非常容易生成：

基本上相关的代码归结为：

ul {
  display: grid; /* 1 */
  grid-template-columns: repeat(auto-fill, 100px); /* 2 */
  grid-gap: 1rem; /* 3 */
  justify-content: space-between; /* 4 */
}

1）使容器元素成为网格容器

2) 将网格设置为宽度为 100px 的自动列。（请注意自动填充的使用（与auto-fit- 它（对于 1 行布局）将空轨道折叠为 0 - 导致项目扩展以占用剩余空间。这将导致合理的'space-between ' 当网格只有一行时的布局，在我们的例子中这不是我们想要的。（查看此演示以了解它们之间的区别））。

3) 为网格行和列设置间隙/间距 - 在这里，因为想要一个“间隔”布局 - 间隙实际上是最小间隙，因为它会根据需要增长。

4) 类似于flexbox。

显示代码片段

ul {
  display: grid;
  grid-template-columns: repeat(auto-fill, 100px);
  grid-gap: 1rem;
  justify-content: space-between;

  /* boring properties */
  list-style: none;
  background: wheat;
  padding: 2rem;
  width: 80vw;
  margin: 0 auto;
}

li {
  height: 50px;
  border: 1px solid green;
}


<ul>
  <li></li>
  <li></li>
  <li></li>
  <li></li>
  <li></li>
  <li></li>
  <li></li>
</ul>
