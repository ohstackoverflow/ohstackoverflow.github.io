echo 输出到 stderr

Arya

2023-02-17 01:46

是否有一个标准的 Bash 工具，它的作用类似于 echo 但输出到 stderr 而不是 stdout？

我知道我能做到echo foo 1>&2，但它有点难看，而且我怀疑容易出错（例如，当事情发生变化时更有可能被编辑错误）。



解答

你可以这样做，这有助于阅读：

>&2 echo "error"

>&2将文件描述符 #2 复制到文件描述符 #1。 因此，在执行此重定向后，两个文件描述符将引用同一个文件：最初 引用的文件描述符#2 。有关详细信息，请参阅Bash Hackers Illustrated Redirection Tutorial。


更多建议

你可以定义一个函数：

echoerr() { echo "$@" 1>&2; }
echoerr hello world

这将比脚本更快并且没有依赖性。

Camilo Martin 的 bash 具体建议使用“here string”并将打印您传递给它的任何内容，包括 echo 通常会吞下的参数 (-n)：

echoerr() { cat <<< "$@" 1>&2; }

Glenn Jackman 的解决方案也避免了参数吞噬问题：

echoerr() { printf "%s\n" "$*" >&2; }
