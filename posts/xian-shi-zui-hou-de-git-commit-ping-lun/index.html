显示最后的 git commit 评论

Arya

2023-02-18 05:51

通常在提交 ( $ git -commit -m "") 期间，我希望阅读我的最后评论以记住我取得的进展。有没有一种简单的方法可以通过命令行直接访问最后的提交消息？（我正在使用 Windows。）



解答

git show

是最快的输入方式，但也会向您显示差异。

git log -1

快速简单。

git log -1 --pretty=%B

如果您只需要提交消息而别无其他。


更多建议

一般来说：

git log -n

将向您显示最后的n提交消息

更优雅——如果你想快速概览你的提交

git log --oneline -n

这将仅显示最后n提交消息的第一行。

您可以使用更短的命令将其保存为 git 别名或 shell 别名。例如，我在我的 shell 中得到了它glog，我可以看到我最后 10 条提交消息glog -10。
