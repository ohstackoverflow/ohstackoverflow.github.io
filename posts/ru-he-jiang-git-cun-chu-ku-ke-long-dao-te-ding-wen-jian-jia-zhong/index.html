如何将 Git 存储库克隆到特定文件夹中？

Arya

2023-02-16 20:25

该命令git clone git@github.com:whatever创建一个名为的目录whatever，其中包含一个 Git 存储库：

./
    whatever/
        .git

我想将 Git 存储库的内容克隆到我当前的目录./中：

./
    .git



解答

选项A：

git clone git@github.com:whatever folder-name

因此，right here使用：

git clone git@github.com:whatever .

选项B：

.git也移动文件夹。请注意，该.git文件夹在大多数图形文件资源管理器中都是隐藏的，因此请务必显示隐藏文件。

mv /where/it/is/right/now/* /where/I/want/it/
mv /where/it/is/right/now/.* /where/I/want/it/

第一行抓取所有普通文件，第二行抓取点文件。shopt -s dotglob也可以通过启用 dotglob（即）在一行中完成此操作，但如果您要问这个答案回答的问题，那可能是一个不好的解决方案。

更好的是：

将您的工作副本保存在其他地方，并创建一个符号链接。像这样：

ln -s /where/it/is/right/now /the/path/I/want/to/use

对于您的情况，这将类似于：

ln -sfn /opt/projectA/prod/public /httpdocs/public

如果您需要，可以轻松更改以测试它，即：

ln -sfn /opt/projectA/test/public /httpdocs/public

无需移动文件。添加-fn以防有人复制这些行（-f强制，-n避免与已经存在和不存在的链接进行一些通常不需要的交互）。

如果您只是想让它工作，请使用选项 A，如果其他人要查看您所做的，请使用选项 C。


更多建议

我想很多人问这个问题的例子就是这个。如果您位于要将 git 存储库的内容转储到的目录 中，请运行：

git clone git@github.com:whatever .

这 ”。” 最后将当前文件夹指定为结帐文件夹。
