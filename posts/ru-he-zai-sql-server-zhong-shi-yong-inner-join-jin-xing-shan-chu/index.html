如何在 SQL Server 中使用 INNER JOIN 进行删除？

Arya

2023-02-17 00:50

我想 删除INNER JOIN在 SQL Server 2008 中使用。

但我收到此错误：

消息 156，级别 15，状态 1，第 15行 关键字 “INNER”附近的语法
不正确。 ****

我的代码：

DELETE 
FROM WorkRecord2 
INNER JOIN Employee 
        ON EmployeeRun=EmployeeNo
WHERE Company = '1' 
    AND Date = '2013-05-06'



解答

您需要指定要从中删除的表。这是一个带有别名的版本：

DELETE w
FROM WorkRecord2 w
INNER JOIN Employee e
  ON EmployeeRun=EmployeeNo
WHERE Company = '1' AND Date = '2013-05-06'


更多建议

只需在要删除记录的位置之间添加表名即可，因为我们必须指定要删除的DELETE表。FROM还要删除该ORDER BY子句，因为删除记录时没有任何命令。

所以你的最终查询应该是这样的：

    DELETE WorkRecord2 
      FROM WorkRecord2 
INNER JOIN Employee 
        ON EmployeeRun=EmployeeNo
     WHERE Company = '1' 
       AND Date = '2013-05-06';
