插入多行而不重复语句的“INSERT INTO ...”部分？

Arya

2023-02-18 01:31

我知道我几年前就这样做过，但我不记得语法了，而且由于调出了大量关于“批量导入”的帮助文档和文章，我在任何地方都找不到它。

这是我想做的，但语法不完全正确......请以前做过的人帮帮我:)

INSERT INTO dbo.MyTable (ID, Name)
VALUES (123, 'Timmy'),
    (124, 'Jonny'),
    (125, 'Sally')

我知道这 接近于 正确的语法。我可能需要“BULK”这个词，或者其他我不记得的东西。任何想法？

我需要这个用于 SQL Server 2005 数据库。我试过这段代码，但无济于事：

DECLARE @blah TABLE
(
    ID INT NOT NULL PRIMARY KEY,
    Name VARCHAR(100) NOT NULL
)

INSERT INTO @blah (ID, Name)
    VALUES (123, 'Timmy')
    VALUES (124, 'Jonny')
    VALUES (125, 'Sally')

SELECT * FROM @blah

我越来越Incorrect syntax near the keyword 'VALUES'.



解答

INSERT INTO dbo.MyTable (ID, Name)
SELECT 123, 'Timmy'
UNION ALL
SELECT 124, 'Jonny'
UNION ALL
SELECT 125, 'Sally'

对于 SQL Server 2008，可以完全 按照您问题中的语句在一个 VALUES 子句中完成（您只需要添加一个逗号来分隔每个值语句）...


更多建议

INSERT INTO dbo.MyTable (ID, Name)
SELECT 123, 'Timmy'
UNION ALL
SELECT 124, 'Jonny'
