转到特定修订

Arya

2023-02-17 13:07

我克隆了某个项目的 git 存储库。我能否将文件恢复到初始状态，然后在查看文件时转到修订版 2、3、4 ……最新的？我想大致了解一下该项目的进展情况。



解答

在执行 此命令之前，请记住它会使您处于 分离头状态

用于git checkout <sha1>签出特定的提交。

<sha1>您可以获得的提交唯一编号在哪里git log

以下是处于分离头状态后的一些选项：

  • 复制文件或对 git 文件夹外的文件夹进行所需的更改，检查分支是否需要它们git checkout <existingBranch>并替换文件
  • 创建一个新的本地分支git checkout -b <new_branch_name> <sha1>


更多建议

要转到特定版本/提交，请运行以下命令。哈希码你可以从git log --oneline -n 10

git reset --hard HASH-CODE

注意git pull --rebase- 在重置为特定版本/提交后，如果您想恢复所有被丢弃的提交，您可以运行。
