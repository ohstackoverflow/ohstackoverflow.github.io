将列表转换为数据框

Arya

2023-02-17 22:37

我有一个嵌套的数据列表。它的长度是132，每一项都是一个长度为20的列表。有没有一种 快速的 方法可以将这个结构转换成一个有132行20列数据的数据框？

以下是一些要使用的示例数据：

l <- replicate(
  132,
  as.list(sample(letters, 20)),
  simplify = FALSE
)



解答

2020 年 7 月更新：

参数的默认值stringsAsFactors现在是默认值default.stringsAsFactors()。FALSE

假设您的列表列表称为l：

df <- data.frame(matrix(unlist(l), nrow=length(l), byrow=TRUE))

上面会将所有字符列转换为因子，为避免这种情况，您可以向 data.frame() 调用添加一个参数：

df <- data.frame(matrix(unlist(l), nrow=132, byrow=TRUE),stringsAsFactors=FALSE)
