Visual Studio Code 中的垂直标尺

Arya

2023-02-17 02:34

在 VS Code 中渲染多个标尺

VS Code 的标尺默认配置如下所示。

  "editor.ruler": 80

我在使用默认 VS 代码配置（如上所示） 时遇到的问题是它只呈现一个标尺。在 Sublime Text Editor 中，我可以使用以下 Sublime 配置呈现任意数量的标尺。

  "rulers": [72, 80, 100, 120]

是否可以在 VS Code 中呈现多个标尺。 如果可能的话， VS Code 中的多标尺配置是什么样的？



解答

Visual Studio Code 0.10.10 引入了此功能。要配置它，请转到菜单 文件 → 首选项 → 设置 并将其添加到您的用户或工作区设置中：

"editor.rulers": [80,120]

标尺的颜色可以这样定制：

"workbench.colorCustomizations": {
    "editorRuler.foreground": "#ff4081"
}


更多建议

除了全局"editor.rulers"设置之外，还可以在每种语言级别上进行设置。

例如，Python 项目的风格指南通常指定 79 或 120 个字符，而 Git 提交消息不应超过 50 个字符。

所以在你的settings.json，你会把：

"[git-commit]": {"editor.rulers": [50]},
"[python]": {
    "editor.rulers": [
        79,
        120
    ]
}
