通配符 * 在 CSS 中用于类

Arya

2023-02-17 13:20

我有这些用于设计样式的 div .tocolor，但我还需要唯一标识符 1、2、3、4 等，所以我将其添加为另一个类tocolor-1。

<div class="tocolor tocolor-1">   tocolor 1   </div>
<div class="tocolor tocolor-2">   tocolor 2   </div>
<div class="tocolor tocolor-3">   tocolor 3   </div>
<div class="tocolor tocolor-4">   tocolor 4   </div>

.tocolor{
  background: red;
}

有没有办法只有一节课tocolor-*。我尝试像在这个 css 中一样使用通配符*，但它没有用。

.tocolor-*{
  background: red;
}



解答

你需要的是所谓的属性选择器。使用您的 html 结构的示例如下：

div[class^="tocolor-"], div[class*=" tocolor-"] {
    color:red 
}

在 的位置，div您可以添加任何元素或完全删除它，在 的位置，class您可以添加指定元素的任何属性。

[class^="tocolor-"]— 以“tocolor-”开头。
[class*=" tocolor-"]— 包含直接出现在空格字符之后的子字符串“tocolor-”。

演示： http: //jsfiddle.net/K3693/1/

是的，你可以做到这一点。

*[id^='term-']{
    [css here]
}

这将选择所有以开头的 ID 'term-'。

至于不这样做的原因，我看到选择这种方式更可取的地方; 至于风格，我不会自己做，但这是可能的。
