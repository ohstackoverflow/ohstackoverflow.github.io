如何在 Objective-C 中测试字符串是否为空？

Arya

2023-02-17 18:08

在 Objective-C 中如何测试 an 是否NSString为空？



解答

你可以检查是否[string length] == 0。这将检查它是否是一个有效但空的字符串 (@"") 以及它是否为 nil，因为调用lengthnil 也会返回 0。


更多建议

马克的回答是正确的。但我会借此机会包括一个指向 Wil Shipley 的 generalized 的指针isEmpty，他在他的博客上分享了它：

static inline BOOL IsEmpty(id thing) {
return thing == nil
|| ([thing respondsToSelector:@selector(length)]
&& [(NSData *)thing length] == 0)
|| ([thing respondsToSelector:@selector(count)]
&& [(NSArray *)thing count] == 0);
}
