Visual Studio Code 中的多行编辑

Arya

2023-02-17 08:49

是否可以像Sublime Text那样启用多行编辑？

例如，按下Ctrl以放置额外的光标插入符号，并能够一次在文档的多个位置写入/删除。



解答

在 Windows 上，按住Ctrl+键Alt的同时按向上↑或向下↓箭头键可添加光标。

麦克：⌥ Opt+ ⌘ Cmd+ ↑/↓

Linux：Shift+ Alt+ ↑/↓

请注意，第三方软件可能会干扰这些快捷方式，使它们无法按预期工作（特别是 Windows 上的英特尔高清显卡软件；有关详细信息，请参阅评论）。

如果遇到此问题，您可以禁用 Intel/其他软件热键，或修改 VS Code 快捷方式（如下所述）。

按Esc重置为单个光标。

或者，正如 Isidor Nikolic 指出的那样，您可以按住Alt并左键单击以任意放置光标。

您可以通过以下方式查看和编辑键盘快捷键：

文件 → 首选项 → 键盘快捷键

文档：

https://code.visualstudio.com/docs/customization/keybindings

两步解决！

  1. Ctrl+F

  2. Alt+Enter

它等于 Sublime Text 的Alt+ F3。

注意：对于 macOS，⌘+F后跟⌥+Enter等同于 Sublime Text 的⌃+ ⌘+ G。
