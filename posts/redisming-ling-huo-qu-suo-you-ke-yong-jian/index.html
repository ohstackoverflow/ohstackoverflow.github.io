Redis命令获取所有可用键？

Arya

2023-02-17 20:34

是否有用于获取数据库中所有键的 Redis 命令？我已经看到一些 python-redis 库在获取它们。但是想知道是否可以从 redis-client 获得。



解答

尝试查看KEYS命令。KEYS *将列出存储在 redis 中的所有键。

编辑： 请注意文档页面顶部的警告KEYS：

时间复杂度： O(N)，其中 N 是数据库中键的数量，假设数据库中的键名称和给定模式的长度有限。

UPDATE（V2.8 或更高版本）： SCAN是 的高级替代品KEYS，因为它不会阻塞服务器，也不会消耗大量资源。更喜欢使用它。


更多建议

针对 Redis 2.8 及更高版本更新

正如之前对该问题的回答的评论中所指出的，KEYS这是一个潜在的危险命令，因为您的 Redis 服务器在为它提供服务时将无法执行其他操作。另一个风险KEYS是它会消耗（取决于密钥空间的大小）大量 RAM 来准备响应缓冲区，从而可能耗尽服务器的内存。

Redis 2.8 版引入了SCAN系列命令，这些命令更加礼貌，可用于相同目的。

CLI 还提供了一种使用它的好方法：

$ redis-cli --scan --pattern '*'
