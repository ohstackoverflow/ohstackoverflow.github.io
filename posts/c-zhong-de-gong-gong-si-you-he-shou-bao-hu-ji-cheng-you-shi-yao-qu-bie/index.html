C++ 中的公共、私有和受保护继承有什么区别？

Arya

2023-02-17 04:34

public, private, 和 之间有什么区别protectedC++ 中的

我在 SO 上发现的所有问题都涉及特定案例。



解答

为了回答这个问题，我想先用我自己的话来描述成员的访问器。如果您已经知道这一点，请跳至标题“下一步：”。

我知道三种访问器：public,protected和private.

让：

class Base {
    public:
        int publicMember;
    protected:
        int protectedMember;
    private:
        int privateMember;
};
  • 意识到的一切Base也意识到Base包含publicMember。
  • 只有孩子（和他们的孩子）知道Base包含protectedMember.
  • 无人Base不知privateMember。

“意识到”是指“承认存在并因此能够访问”。

下一个：

公有、私有和受保护的继承也是如此。让我们考虑一个类Base和一个Child继承自Base.

  • 如果继承是public，所有知道的Base和Child也知道的都Child继承自Base。
  • 如果继承是protected，则只有Child及其子级知道他们继承自Base。
  • 如果继承是private，则除此之外没有人Child知道继承。
