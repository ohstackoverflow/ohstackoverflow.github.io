如何计算列表项的出现次数？

Arya

2023-02-16 22:45

给定一个项目，我如何在 Python 中计算它在列表中的出现次数？

一个相关但不同的问题是计算集合中 每个不同元素的出现次数，获取字典或列表作为直方图结果而不是单个整数。 对于该问题，请参阅使用字典对列表中的项目进行计数。



解答

如果您只想要单个项目的计数，请使用以下count方法：

>>> [1, 2, 3, 4, 1, 4, 1].count(1)
3
重要提示：如果您计算 多个 不同的项目，这会非常慢

每次count调用都会遍历整个元素列表n。count循环调用n意味着n * n全面检查，这对性能来说可能是灾难性的。

如果要计算多个项目，请使用Counter，它只会进行n总计检查。


更多建议

Counter如果您使用的是 Python 2.7 或 3.x，并且您想要每个元素出现的次数，请使用：

>>> from collections import Counter
>>> z = ['blue', 'red', 'blue', 'yellow', 'blue', 'red']
>>> Counter(z)
Counter({'blue': 3, 'red': 2, 'yellow': 1})
