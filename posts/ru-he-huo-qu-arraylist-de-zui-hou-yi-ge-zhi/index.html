如何获取 ArrayList 的最后一个值

Arya

2023-02-17 15:00

如何获取 ArrayList 的最后一个值？



解答

以下是List接口的一部分（ArrayList 实现）：

E e = list.get(list.size() - 1);

E是元素类型。如果列表为空，get则抛出一个IndexOutOfBoundsException. 您可以在此处找到完整的 API 文档。


更多建议

普通 Java 中没有优雅的方式。

谷歌番石榴

Google Guava图书馆很棒 - 查看他们的Iterables课程。NoSuchElementException如果列表为空，此方法将抛出 a ，而不是 an IndexOutOfBoundsException，就像典型的size()-1方法一样 - 我发现NoSuchElementException更好，或者能够指定默认值：

lastElement = Iterables.getLast(iterableList);

如果列表为空，您还可以提供默认值，而不是异常：

lastElement = Iterables.getLast(iterableList, null);

或者，如果您使用的是选项：

lastElementRaw = Iterables.getLast(iterableList, null);
lastElement = (lastElementRaw == null) ? Option.none() : Option.some(lastElementRaw);
