如何在 TypeScript 中将字符串转换为数字？

Arya

2023-02-17 02:07

给定数字的字符串表示形式，如何将其转换为numberTypeScript 类型？

var numberString: string = "1234";
var numberValue: number = /* what should I do with `numberString`? */;



解答

与在 JavaScript 中完全一样，您可以使用parseIntorparseFloat函数，或者简单地使用一元+运算符：

var x = "32";
var y: number = +x;

所有提到的技术都将具有正确的类型，并将正确解析简单的十进制整数字符串，如"123"，但对于其他各种可能预期的情况（如"123.45"）和极端情况（如null），其行为会有所不同。

更多建议

执行此操作的 Typescript 方法是：

Number('1234') // 1234
Number('9BX9') // NaN
