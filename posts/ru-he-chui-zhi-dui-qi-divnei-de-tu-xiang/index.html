解答

据我所知，唯一（也是最好的跨浏览器）方法是在两个元素上使用inline-block帮助程序。height: 100%``vertical-align: middle

于是有了解决方案：http://jsfiddle.net/kizu/4RPFa/4570/

或者，如果您不想在现代浏览器中使用额外的元素并且不介意使用 Internet Explorer 表达式，您可以使用伪元素并使用方便的表达式将其添加到 Internet Explorer，每个元素只运行一次，所以不会有任何性能问题：

:before适用于 Internet Explorer 的解决方案expression()：http://jsfiddle.net/kizu/4RPFa/4571/

显示代码片段

.frame {
    height: 25px;      /* Equals maximum image height */
    width: 160px;
    border: 1px solid red;
    white-space: nowrap;

    text-align: center;
    margin: 1em 0;
}

.frame:before,
.frame_before {
    content: "";
    display: inline-block;
    height: 100%;
    vertical-align: middle;
}

img {
    background: #3A6F9A;
    vertical-align: middle;
    max-height: 25px;
    max-width: 160px;
}

/* Move this to conditional comments */
.frame {
    list-style:none;
    behavior: expression(
        function(t){
            t.insertAdjacentHTML('afterBegin','<span class="frame_before"></span>');
            t.runtimeStyle.behavior = 'none';
        }(this)
    );
}

怎么运行的：

  1. 当你有两个inline-block彼此靠近的元素时，你可以将彼此对齐到对方的一边，所以vertical-align: middle你会得到这样的东西：

  1. 当你有一个固定高度的块（inpx或em其他绝对单位）时，你可以设置内部块的高度 in %。

  2. 因此，在具有固定高度的块中添加一个inline-blockwithheight: 100%将使inline-block其中的另一个元素（<img/>在您的情况下）垂直对齐它。
