从 GitHub 中删除分叉的回购

Arya

2023-02-17 14:31

我从 git 和 GitHub 开始，我正在关注 GitHub 上的一个项目。我无意中点击了它。现在它对我来说是一个新项目。

我对此有一些疑问：

  • 我知道如果对我的分叉回购提交或做其他事情，它会被更新，但更新后的代码只有在项目作者请求拉取后才会生效。正确的？
  • 如果我转到 GitHub 上的管理面板，会有一个删除选项。如果按上面的选项删除，对原来的有影响吗？

我想删除它。到目前为止，我只是在研究代码，并不真的需要叉子。



解答

删除它不会对原始项目有任何影响。编辑它只会在你的回购页面上编辑你的叉子。


更多建议
