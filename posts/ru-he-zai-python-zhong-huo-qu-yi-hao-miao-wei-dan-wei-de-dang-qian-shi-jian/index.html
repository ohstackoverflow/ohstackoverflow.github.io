如何在 Python 中获取以毫秒为单位的当前时间？



解答

使用time.time()：

import time

def current_milli_time():
    return round(time.time() * 1000)

然后：

>>> current_milli_time()
1378761833768


更多建议

对于 Python 3.7+，time.time_ns()给出自纪元以来经过的时间（以纳秒为单位）。

这给出了以毫秒为单位的整数时间：

import time

ms = time.time_ns() // 1_000_000
