如何在调试模式的 Android Studio 中获取 SHA-1 指纹证书？

Arya

2023-02-17 18:16

我正在尝试在 Windows 上的 Android Studio 中制作地图应用程序。如何找到 SHA-1 指纹证书编号？

当我使用 Eclipse 时，它​​就在 Windows - > Preferences -> Android -> Build下。但是在 Android Studio 中我找不到类似的选项。

我在文档中读到

当您从 IDE 运行或调试项目时，Android Studio 会自动在调试模式下为您的应用签名。

所以我尝试设置我的 Java bin 路径并运行以下命令，从这里获取：

keytool -list -v -keystore "%USERPROFILE%\.android\debug.keystore" -alias androiddebugkey -storepass android -keypass android

但是，我收到一条错误消息，说这是一个非法选项。

一步一步， 我如何在 Android Studio 中获取这些信息 ？

解决方法：我之前使用 Eclipse 生成的 SHA-1 指纹注册了我的应用程序。我可以使用相同的 API 密钥在 Android Studio 中处理我的项目吗？



解答

有史以来最简单的方法：

在最后一步中为 Android Studio V 2.2 添加了更新

有两种方法可以做到这一点。

1. 更快的方式 ：

  1. 打开安卓工作室
  2. 打开你的项目
  3. 单击 Gradle（在右侧面板中，您将看到 Gradle Bar ）
  4. 点击 Refresh（点击 Refresh from Gradle Bar ，你会看到 List Gradle scripts of your Project）
  5. 单击您的项目（您的项目名称表单 列表 （根））
  6. 点击任务
  7. 点击安卓
  8. 双击 signingReport（您将在 运行栏中获得 SHA1 和 MD5 （有时它会在 Gradle 控制台中）） ****
  9. 从 模块选择下拉列表 中选择 应用程序 模块以运行或调试您的应用程序 ****

2. 使用谷歌地图活动 ：

  1. 打开安卓工作室
  2. 打开你的项目
  3. 点击 文件 菜单 -> 选择 新建 -> 点击 谷歌 -> 选择 谷歌地图活动
  4. 将出现一个对话框 -> 单击 “完成”
  5. Android Studio 会自动生成一个 名为 的 XML 文件 ****google_maps_api.xml
  6. 您将在此处获得 调试 SHA1 密钥（在XML 文件的 第 10 行）

更多建议

我的问题与我也将我的Google Maps项目从 Eclipse 转移到 Android Studio 一样。我已经通过以下方式解决了我的问题：

通过以下命令转到您的 Java bin 目录：

C:\Program Files\Java\jdk1.7.0_71\bin>

现在在命令窗口 (CMD.EXE) 中输入以下命令：

keytool -list -v -keystore c:\users\your_user_name\.android\debug.keystore -alias androiddebugkey -storepass android -keypass android

例子：

keytool -list -v -keystore c:\users\James\.android\debug.keystore -alias androiddebugkey -storepass android -keypass android

或者如果你不知道用户名，你可以在 cmd 中写这个：

keytool -list -v -keystore "%USERPROFILE%\.android\debug.keystore" -alias androiddebugkey -storepass android -keypass android

你会得到 SHA1。

然后我从https://code.google.com/apis/console创建了一个新密钥，因为我的包名称已更改，然后在我的应用程序中使用此密钥。它工作正常。

确保你在jdkX.X.X_XX 的 BIN 文件夹中 （我的文件夹名称是 jdk1.7.0_71） ，否则如果你不知道你有什么文件夹名称然后通过打开 Java 文件夹找到它，你会看到文件夹名称是什么，但您必须在 BIN 文件夹中，然后运行此命令。今天，我通过上述过程在 cmd 命令窗口中获得了我的 SHA1 值。
