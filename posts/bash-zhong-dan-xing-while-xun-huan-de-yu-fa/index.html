Bash 中单行 while 循环的语法

Arya

2023-02-17 12:30

我无法想出分号和/或大括号的正确组合。我想这样做，但作为命令行的单行代码：

while [ 1 ]
do
    foo
    sleep 2
done



解答

while true; do foo; sleep 2; done

顺便说一句，如果您在命令提示符下将其作为多行输入（如您所显示的那样），然后使用向上箭头调用历史记录，您将在一行中获得它，并正确标点符号。

$ while true
> do
>    echo "hello"
>    sleep 2
> done
hello
hello
hello
^C
$ <arrow up> while true; do    echo "hello";    sleep 2; done


更多建议

也可以在 while 的条件下使用 sleep 命令。使单线看起来更干净恕我直言。

while sleep 2; do echo thinking; done
