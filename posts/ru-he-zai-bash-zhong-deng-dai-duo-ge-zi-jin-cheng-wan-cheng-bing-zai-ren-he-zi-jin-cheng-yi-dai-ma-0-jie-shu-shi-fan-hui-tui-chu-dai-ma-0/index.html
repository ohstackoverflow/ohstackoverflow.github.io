如何在 bash 中等待多个子进程完成，并在任何子进程以代码 !=0 结束时返回退出代码 !=0？

Arya

2023-02-17 16:28

如何在 bash 脚本中等待从该脚本生成的多个子进程完成，然后在!=0任何子进程以代码结束时返回退出代码!=0？

简单脚本：

#!/bin/bash
for i in `seq 0 9`; do
  doCalculations $i &
done
wait

上面的脚本将等待所有 10 个派生的子进程，但它总是会给出退出状态0（参见 参考资料help wait）。我如何修改这个脚本，以便它会发现衍生子进程的退出状态，并1在任何子进程以代码结束时返回退出代码!=0？

有没有比收集子进程的 PID、按顺序等待它们并总结退出状态更好的解决方案？



解答

wait还（可选）获取要PID等待的进程的权限，并$!获得PID在后台启动的最后一个命令的权限。修改循环以将PID每个派生的子进程存储到一个数组中，然后再次循环等待每个PID。

# run processes and store pids in array
for i in $n_procs; do
    ./procs[${i}] &
    pids[${i}]=$!
done

# wait for all pids
for pid in ${pids[*]}; do
    wait $pid
done

#!/bin/bash
FAIL=0
echo "starting"
./sleeper 2 0 &
./sleeper 2 1 &
./sleeper 3 0 &
./sleeper 2 0 &
for job in `jobs -p`
do
echo $job
    wait $job || let "FAIL+=1"
done
echo $FAIL
if [ "$FAIL" == "0" ];
then
echo "YAY!"
else
echo "FAIL! ($FAIL)"
fi
