如何在 Java 中将 Map 转换为 List？

Arya

2023-02-17 13:18

如何将 a 转换Map<key,value>为 a List<value>？我应该遍历所有地图值并将它们插入到列表中吗？



解答

List<Value> list = new ArrayList<Value>(map.values());

假设：

Map<Key,Value> map;


更多建议

这里的问题是它Map有两个值（一个键和一个值），而一个List只有一个值（一个元素）。

因此，最好的办法是获取List键或值。（除非我们制作一个包装器来保存键/值对）。

假设我们有一个Map：

Map<String, String> m = new HashMap<String, String>();
m.put("Hello", "World");
m.put("Apple", "3.14");
m.put("Another", "Element");

作为 a 的键可以通过从方法返回的a 中List创建一个新的来获得：ArrayList``Set``Map.keySet

List<String> list = new ArrayList<String>(m.keySet());

虽然可以从方法返回的a中List创建一个新值来获取值：ArrayList``Collection``Map.values

List<String> list = new ArrayList<String>(m.values());
