如何将键/值对添加到 JavaScript 对象？

Arya

2023-02-16 23:34

这是我的对象文字：

var obj = {key1: value1, key2: value2};

如何向对象添加key3字段value3？



解答

有两种方法可以向对象添加新 属性：

var obj = {
    key1: value1,
    key2: value2
};

使用点符号：

obj.key3 = "value3";

使用方括号表示法：

obj["key3"] = "value3";

当您知道属性的名称时，将使用第一种形式。当动态确定属性名称时使用第二种形式。就像在这个例子中：

var getProperty = function (propertyName) {
    return obj[propertyName];
};

getProperty("key1");
getProperty("key2");
getProperty("key3");

可以使用以下任一方法构造 真正的 JavaScript数组：

数组文字符号：

var arr = [];

数组构造函数符号：

var arr = new Array();


更多建议

2017年答案：Object.assign()

Object.assign(dest, src1, src2, ...)合并对象。

它dest用（无论多少）源对象的属性和值覆盖，然后返回dest。

该Object.assign()方法用于将所有可枚举自身属性的值从一个或多个源对象复制到目标对象。它将返回目标对象。

实例

var obj = {key1: "value1", key2: "value2"};
Object.assign(obj, {key3: "value3"});

document.body.innerHTML = JSON.stringify(obj);

2018年答案：对象传播运算符{...} 

obj = {...obj, ...pair, scalar};

来自MDN：

它将自己的可枚举属性从提供的对象复制到新对象上。

现在可以使用比Object.assign().

请注意，Object.assign()触发设置器而传播语法则不会。

实例

它适用于当前的 Chrome 和当前的 Firefox。他们说它在当前的 Edge 中不起作用。

var obj = {key1: "value1", key2: "value2"};
var pair = {key3: "value3"};
var scalar = "value4"
obj = {...obj, ...pair, scalar};

document.body.innerHTML = JSON.stringify(obj);
