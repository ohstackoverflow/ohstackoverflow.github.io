Java 中的 native 关键字有什么用？

Arya

2023-02-18 06:04

在玩这个谜题（这是一个 Java 关键字问答游戏）时，我遇到了native关键字。

Java中的native关键字是干什么用的？



解答

关键字native应用于方法以指示该方法是使用 JNI（Java 本机接口）在本机代码中实现的。


更多建议

它标志着一种方法，它将以其他语言而不是 Java 实现。它与 JNI（Java 本机接口）一起工作。

过去使用本机方法来编写性能关键部分，但随着 Java 变得越来越快，这种方法现在不那么常见了。当前需要使用本地方法

  • 您需要从用其他语言编写的 Java 中调用一个库。

  • 您需要访问只能从其他语言（通常是 C）访问的系统或硬件资源。实际上，许多与真实计算机交互的系统功能（例如磁盘和网络 IO）只能这样做，因为它们调用了本机代码。
