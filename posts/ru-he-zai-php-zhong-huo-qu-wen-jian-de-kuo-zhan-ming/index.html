如何在 PHP 中获取文件的扩展名？

Arya

2023-02-17 11:43

这是一个您可以在网络上随处阅读的问题以及各种答案：

$ext = end(explode('.', $filename));
$ext = substr(strrchr($filename, '.'), 1);
$ext = substr($filename, strrpos($filename, '.') + 1);
$ext = preg_replace('/^.*\.([^.]+)$/D', '$1', $filename);

$exts = split("[/\\.]", $filename);
$n    = count($exts)-1;
$ext  = $exts[$n];

ETC。

然而，总是有“最好的方法”，它应该在 Stack Overflow 上。



解答

来自其他脚本语言的人总是认为他们的更好，因为他们有一个内置的函数来做到这一点，而不是 PHP（我现在正在看 Pythonistas :-)）。

其实它确实存在，只是很少有人知道。见面pathinfo()：

$ext = pathinfo($filename, PATHINFO_EXTENSION);

这是快速且内置的。pathinfo()可以为您提供其他信息，例如规范路径，具体取决于您传递给它的常量。

请记住，如果您希望能够处理非 ASCII 字符，则需要先设置语言环境。例如：

setlocale(LC_ALL,'en_US.UTF-8');

另外，请注意，这并没有考虑文件内容或 MIME 类型，您只会得到扩展名。但这是你要求的。

最后，请注意，这仅适用于文件路径，不适用于使用 PARSE_URL 覆盖的 URL 资源路径。
