我可以强制 pip 重新安装当前版本吗？

Arya

2023-02-17 14:58

我遇到过当前版本的软件包似乎无法正常工作并且需要重新安装的情况。但pip install -U不会触及已经是最新的包。我看到了如何通过先卸载（使用 ）然后安装来强制重新pip uninstall安装，但是有没有一种方法可以一步简单地强制“更新”到名义上的当前版本？



解答

pip install --upgrade --force-reinstall <package>

升级时，重新安装所有软件包，即使它们已经是最新的。

pip install -I <package>
pip install --ignore-installed <package>

忽略已安装的软件包（改为重新安装）。


更多建议

您可能希望拥有所有三个选项：--upgrade并--force-reinstall确保重新安装，同时--no-deps避免重新安装依赖项。

$ sudo pip install --upgrade --no-deps --force-reinstall <packagename>

否则你可能会遇到 pip 开始重新编译 Numpy 或其他大型包的问题。
