JavaScript 中有常量吗？

Arya

2023-02-17 04:55

有没有办法在JavaScript 中使用常量？

如果不是，指定用作常量的变量的常见做法是什么？



解答

从ES2015开始，JavaScript 有一个概念const：

const MY_CONSTANT = "some-value";

这几乎适用于除 IE 8、9 和 10 之外的所有浏览器。有些可能还需要启用严格模式。

如果您需要支持较旧的浏览器或正在使用旧代码，您可以使用varALL_CAPS 等约定来表明不应修改某些值：

var MY_CONSTANT = "some-value";


更多建议

您是否试图保护变量不被修改？如果是这样，那么您可以使用模块模式：

var CONFIG = (function() {
     var private = {
         'MY_CONST': '1',
         'ANOTHER_CONST': '2'
     };

     return {
        get: function(name) { return private[name]; }
    };
})();

alert('MY_CONST: ' + CONFIG.get('MY_CONST'));  // 1

CONFIG.MY_CONST = '2';
alert('MY_CONST: ' + CONFIG.get('MY_CONST'));  // 1

CONFIG.private.MY_CONST = '2';                 // error
alert('MY_CONST: ' + CONFIG.get('MY_CONST'));  // 1

使用这种方法，无法修改这些值。但是，您必须在 CONFIG 上使用 get() 方法 :(。

如果您不需要严格保护变量值，那么只需按照建议进行操作并使用全部大写的约定即可。
