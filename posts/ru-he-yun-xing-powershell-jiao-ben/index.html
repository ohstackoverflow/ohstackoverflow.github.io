如何运行 PowerShell 脚本

Arya

2023-02-17 08:57

如何运行 PowerShell 脚本？

  • 我有一个名为 myscript.ps1 的脚本
  • 我已经安装了所有必要的框架
  • 我设置了那个执行策略的东西
  • 我已按照此 MSDN 帮助页面上的说明进行操作，并尝试像这样运行它：（powershell.exe 'C:\my_path\yada_yada\run_import_script.ps1'有或没有--noexit）

除了输出文件名外，它什么都不返回。

没有错误，没有消息，什么都没有。哦，当我添加时-noexit，同样的事情发生了，但我仍然在 PowerShell 中并且必须手动退出。

.ps1 文件应该运行一个程序并根据该程序的输出返回错误级别。但我很确定我什至还没有到达那里。

我究竟做错了什么？



解答

先决条件：

  • 您需要能够以管理员身份运行 PowerShell
  • 您需要将 PowerShell 执行策略设置为允许值或能够绕过它

脚步：

  1. 以管理员身份启动 Windows PowerShell，并等待PS>提示出现

  2. 在 PowerShell 中导航到脚本所在的目录：

    PS> cd C:\my_path\yada_yada\ (enter)

  3. 执行脚本：

    PS> .\run_import_script.ps1 (enter)

或者：您可以从命令提示符 ( cmd.exe) 运行 PowerShell 脚本，如下所示：

powershell -noexit "& ""C:\my_path\yada_yada\run_import_script.ps1""" (enter)

根据Kirk Munro 从 cmd.exe（或开始 | 运行）调用 PowerShell 脚本。

或者您甚至可以从您的 C# 应用程序异步运行您的 PowerShell 脚本。


更多建议

如果您使用的是 PowerShell 2.0，请使用 PowerShell.exe 的-File参数从另一个环境调用脚本，例如 cmd.exe。例如：

Powershell.exe -File C:\my_path\yada_yada\run_import_script.ps1
