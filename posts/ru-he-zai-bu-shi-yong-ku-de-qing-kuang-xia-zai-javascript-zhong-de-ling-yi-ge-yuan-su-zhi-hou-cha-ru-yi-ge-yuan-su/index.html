如何在不使用库的情况下在 JavaScript 中的另一个元素之后插入一个元素？

Arya

2023-02-17 07:27

在 JavaScript 中有insertBefore()，但如何在不使用 jQuery 或其他库的情况下在另一个元素 之后 插入一个元素？



解答

referenceNode.parentNode.insertBefore(newNode, referenceNode.nextSibling);

referenceNode你想放在后面的节点在哪里newNode。如果referenceNode是其父元素中的最后一个子元素，那很好，因为referenceNode.nextSiblingwill benull并insertBefore通过添加到列表末尾来处理这种情况。

所以：

function insertAfter(referenceNode, newNode) {
    referenceNode.parentNode.insertBefore(newNode, referenceNode.nextSibling);
}

您可以使用以下代码段对其进行测试：

显示代码片段

function insertAfter(referenceNode, newNode) {
  referenceNode.parentNode.insertBefore(newNode, referenceNode.nextSibling);
}

var el = document.createElement("span");
el.innerHTML = "test";
var div = document.getElementById("foo");
insertAfter(div, el);


<div id="foo">Hello</div>


更多建议

简单的 JavaScript 如下所示：

附加在前面：

element.parentNode.insertBefore(newElement, element);

追加在：

element.parentNode.insertBefore(newElement, element.nextSibling);

但为了便于使用，将一些 原型 放入其中：

通过构建以下原型，您将能够直接从新创建的元素中调用这些函数。

  • newElement.appendBefore(element);

  • newElement.appendAfter(element);

.appendBefore(元素) 原型

Element.prototype.appendBefore = function (element) {
  element.parentNode.insertBefore(this, element);
},false;

.appendAfter(element) 原型

Element.prototype.appendAfter = function (element) {
  element.parentNode.insertBefore(this, element.nextSibling);
},false;

显示代码片段

/* Adds Element BEFORE NeighborElement */
Element.prototype.appendBefore = function(element) {
  element.parentNode.insertBefore(this, element);
}, false;

/* Adds Element AFTER NeighborElement */
Element.prototype.appendAfter = function(element) {
  element.parentNode.insertBefore(this, element.nextSibling);
}, false;

NewElement.appendAfter(document.getElementById('Neighbor2'));
