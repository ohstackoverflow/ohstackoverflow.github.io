Git 不断询问我的 ssh 密钥密码

Arya

2023-02-17 14:05

我按照 github 教程中的说明创建了密钥，将它们注册到 github，并尝试显式使用 ssh-agent — 但每次我尝试执行拉取或推送时，git 都会继续询问我的密码。

可能是什么原因？



解答

使用以下命令启动 SSH 代理后：

eval $(ssh-agent)

执行以下任一操作：

  1. 要将您的私钥添加到其中：
     ssh-add

这只会询问您一次密码，然后您应该被允许推送，前提是您将公钥上传到 Github。

  1. 要在macOS 上永久添加和保存您的密钥：
     ssh-add -K

这将在您关闭并通过将其存储在用户的钥匙串中重新打开后保留它。

如果您看到有关标志的警告deprecated，请尝试新变体：

     ssh-add --apple-use-keychain
  1. 要在Ubuntu （或等效的）上永久添加和保存您的密钥：
      ssh-add ~/.ssh/id_rsa


更多建议

自从从 OS X El Capitan (10.11) 升级到 macOS Sierra (10.12) 后，这在我身上一直发生。该ssh- add解决方案暂时有效，但不会在另一次重启后持续存在。

永久解决方案是编辑（或创建）~/.ssh/config并启用该UseKeychain选项。

Host *
    UseKeychain yes
