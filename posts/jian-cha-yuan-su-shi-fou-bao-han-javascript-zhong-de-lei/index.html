检查元素是否包含 JavaScript 中的类？

Arya

2023-02-17 10:08

使用纯 JavaScript（不是 jQuery），有什么方法可以检查元素是否 包含 类？

目前，我正在这样做：

var test = document.getElementById("test");
var testClass = test.className;

switch (testClass) {
  case "class1":
    test.innerHTML = "I have class1";
    break;
  case "class2":
    test.innerHTML = "I have class2";
    break;
  case "class3":
    test.innerHTML = "I have class3";
    break;
  case "class4":
    test.innerHTML = "I have class4";
    break;
  default:
    test.innerHTML = "";
}


<div id="test" class="class1"></div>

运行代码片段隐藏结果

问题是，如果我将 HTML 更改为此...

<div id="test" class="class1 class5"></div>

...不再有完全匹配，所以我得到默认输出 nothing ( "")。但我仍然希望输出是I have class1因为<div>仍然 包含 类.class1。



解答

使用方法：element.classList .contains

element.classList.contains(class);

这适用于所有当前的浏览器，并且有 polyfills 也支持旧的浏览器。

或者 ，如果您使用较旧的浏览器并且不想使用 polyfill 来修复它们，使用indexOf是正确的，但您必须稍微调整一下：

function hasClass(element, className) {
    return (' ' + element.className + ' ').indexOf(' ' + className+ ' ') > -1;
}

true否则，如果您要查找的类是另一个类名的一部分，您也会得到。

演示

jQuery 使用类似（如果不相同）的方法。

应用于示例：

由于这不能与 switch 语句一起使用，您可以使用以下代码实现相同的效果：

var test = document.getElementById("test"),
    classes = ['class1', 'class2', 'class3', 'class4'];

test.innerHTML = "";

for(var i = 0, j = classes.length; i < j; i++) {
    if(hasClass(test, classes[i])) {
        test.innerHTML = "I have " + classes[i];
        break;
    }
}

它也不那么多余；）


更多建议

简单有效的解决方案是尝试 .contains 方法。

test.classList.contains(testClass);
