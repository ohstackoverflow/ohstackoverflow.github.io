在 Git 中获取从 master 到分支的更改

Arya

2023-02-17 09:20

aq在我的存储库中，我有一个正在处理的分支。

然后我在master.

将这些提交提交到分支的最佳方法是什么aq？从中创建另一个新分支master并将其与aq?



解答

检查aq分支，并从master.

git checkout aq
git rebase master


更多建议

git merge origin/master当你在你的 aq 分支上时，你应该能够。

git checkout aq
git merge origin/master
