在 Java 中将 InputStream 转换为字节数组

Arya

2023-02-17 08:08

如何将整个读InputStream入字节数组？



解答

您可以使用 Apache Commons IO来处理这个任务和类似的任务。

该IOUtils类型有一个静态方法来读取 anInputStream并返回 a byte[]。

InputStream is;
byte[] bytes = IOUtils.toByteArray(is);

这在内部创建一个ByteArrayOutputStream并将字节复制到输出，然后调用toByteArray(). 它通过复制 4KiB 块中的字节来处理大文件。


更多建议

您需要从您的文件中读取每个字节InputStream并将其写入ByteArrayOutputStream.

然后，您可以通过调用检索底层字节数组toByteArray()：

InputStream is = ...
ByteArrayOutputStream buffer = new ByteArrayOutputStream();

int nRead;
byte[] data = new byte[16384];

while ((nRead = is.read(data, 0, data.length)) != -1) {
  buffer.write(data, 0, nRead);
}

return buffer.toByteArray();
