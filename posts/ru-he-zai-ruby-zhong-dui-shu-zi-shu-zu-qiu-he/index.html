如何在 Ruby 中对数字数组求和？

Arya

2023-02-17 23:21

我有一个整数数组。

例如：

array = [123,321,12389]

有什么好的方法可以得到它们的总和吗？

我知道

sum = 0
array.each { |a| sum+=a }

会工作。



解答

尝试这个：

array.inject(0){ |sum, x| sum + x }

请参阅 Ruby 的可枚举文档

（注意：0需要基本情况​​，以便0返回一个空数组而不是nil）
