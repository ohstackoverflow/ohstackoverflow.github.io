如何在 python 中“测试”NoneType？

Arya

2023-02-17 22:19

我有一个有时会返回 NoneType 值的方法。那么我如何质疑一个 NoneType 变量呢？我需要使用 if 方法，例如

if not new:
    new = '#'

我知道这是错误的方式，我希望你明白我的意思。



解答

那么我如何质疑一个 NoneType 变量呢？

使用is运算符，像这样

if variable is None:

为什么这有效？

由于None是 Python 中唯一的单例对象NoneType，我们可以使用is运算符来检查变量None是否包含在其中。

引用is文档，

对象标识的运算符is和is not测试：x is y当且仅当x和y是同一对象时才为真。x is not y产生逆真值。

由于 只能有一个实例None，is因此将是检查 的首选方法None。

从马口中听到

引用Python's Coding Style Guidelines - PEP-008（由 Guido 自己共同定义），

与像这样的单例的比较 None 应该始终使用isoris not来完成， 永远不要使用相等运算符 。


更多建议

if variable is None:
   ...

if variable is not None:
   ...
