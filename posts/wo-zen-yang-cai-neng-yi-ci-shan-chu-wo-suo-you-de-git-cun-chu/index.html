我怎样才能一次删除我所有的Git存储？

具体来说，我的意思是，输入一个命令。



解答

以下命令删除您所有的藏匿处：

git stash clear

来自git 文档：

clear

删除所有隐藏状态。

重要警告： 这些状态将被修剪，并且可能无法恢复（...）。


更多建议

此命令使您能够查看所有隐藏的更改：

git stash list

以下是使用它清除所有隐藏更改的命令：

git stash clear

现在，如果你想从隐藏区域中删除一个隐藏的更改：

git stash drop stash@{index} # Index will be shown after getting stash list

注意：git stash list使您能够从 git 的存储区获取索引。
