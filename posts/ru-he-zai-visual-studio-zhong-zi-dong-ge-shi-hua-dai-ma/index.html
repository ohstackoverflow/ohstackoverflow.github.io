如何在 Visual Studio 中自动格式化代码？

Arya

2023-02-17 05:44

我知道 Visual Studio 可以自动格式化以使我的方法和循环正确缩进，但我找不到设置。



解答

格式化选择：Ctrl+ K, Ctrl+F

格式化文档：Ctrl+ K, Ctrl+D

请参阅预定义的键盘快捷键。（这两个是Edit.FormatSelection和Edit.FormatDocument。）

macOS 注意事项

在 macOS 上，使用CMD ⌘key 而不是Ctrl:

  • 格式化选择：CMD ⌘+ K, CMD ⌘+F
  • 格式化文档：CMD ⌘+ K, CMD ⌘+D


更多建议

对于 Visual Studio 2010/2013/2015/2017/2019

  • 格式化文档 (Ctrl+K, Ctrl+D)，即按住Ctrl，按下并释放K然后点击，D因为它是一个序列
  • 格式选择 (Ctrl+K, Ctrl+F)

工具栏 Edit - > Advanced（如果看不到 Advanced ，请在解决方案资源管理器中选择一个代码文件并重试）

您的快捷方式可能与我的显示不同，因为我为 C# 编码设置，但通过工具栏导航将带您进入您的快捷方式。

如果它不起作用，请查找代码中的错误，例如缺少括号会阻止自动格式工作
