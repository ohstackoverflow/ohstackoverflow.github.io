安全地将 JSON 字符串转换为对象

Arya

2023-02-17 02:02

给定一串 JSON 数据，我如何才能将该字符串安全地转换为 JavaScript 对象？

显然我可以用类似的东西不安全地做到这一点：

var obj = eval("(" + json + ')');

但这让我容易受到包含其他代码的 JSON 字符串的攻击，简单地评估它似乎非常危险。



解答

JSON.parse(jsonString)是一种纯 JavaScript 方法，只要您可以保证一个相当现代的浏览器。


更多建议

jQuery 方法现已弃用。请改用此方法：

let jsonObject = JSON.parse(jsonString);

使用已弃用的 jQuery 功能的原始答案 ：

如果您使用的是 jQuery，只需使用：

jQuery.parseJSON( jsonString );

这正是您要查找的内容（请参阅 jQuery文档）。
