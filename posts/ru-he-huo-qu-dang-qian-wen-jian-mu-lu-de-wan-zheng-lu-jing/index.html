如何获取当前文件目录的完整路径？

Arya

2023-02-17 03:03

如何获取当前文件的目录路径？我试过：

>>> os.path.abspath(__file__)
'C:\\python27\\test.py'

但我想要：

'C:\\python27\\'



解答

特殊变量__file__包含当前文件的路径。pathlib从那里我们可以使用任一模块获取目录os.path。

蟒蛇3

对于正在运行的脚本的目录：

import pathlib
pathlib.Path(__file__).parent.resolve()

对于当前工作目录：

import pathlib
pathlib.Path().resolve()

Python 2 和 3

对于正在运行的脚本的目录：

import os
os.path.dirname(os.path.abspath(__file__))

如果您指的是当前工作目录：

import os
os.path.abspath(os.getcwd())

注意before和afterfile是两个下划线，而不是一个。

另请注意，如果您以交互方式运行或从文件以外的其他内容（例如：数据库或在线资源）加载代码，则__file__可能无法设置，因为没有“当前文件”的概念。上面的答案假设了运行文件中的 python 脚本的最常见场景。

更多建议

从 Python 3 开始，推荐使用Pathfrom ：pathlib

from pathlib import Path
print("File      Path:", Path(__file__).absolute())
print("Directory Path:", Path().absolute()) # Directory of current working directory, not __file__

注意： 如果使用 Jupyter Notebook，则__file__不会返回预期值，因此Path().absolute()必须使用。
