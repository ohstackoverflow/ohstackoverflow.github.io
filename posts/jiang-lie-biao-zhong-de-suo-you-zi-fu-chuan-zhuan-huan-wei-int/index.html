将列表中的所有字符串转换为 int

Arya

2023-02-17 13:47

如何将列表中的所有字符串转换为整数？

['1', '2', '3']  ⟶  [1, 2, 3]



解答

鉴于：

xs = ['1', '2', '3']

然后使用mapthenlist获取整数列表：

list(map(int, xs))

在 Python 2 中，list没有必要，因为map它返回了一个列表：

map(int, xs)


更多建议

在列表上使用列表理解xs：

[int(x) for x in xs]

例如

>>> xs = ["1", "2", "3"]
>>> [int(x) for x in xs]
[1, 2, 3]
