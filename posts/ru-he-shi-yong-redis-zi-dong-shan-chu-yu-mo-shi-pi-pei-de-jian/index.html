如何使用 Redis 自动删除与模式匹配的键

Arya

2023-02-17 16:23

在我的 Redis 数据库中，我有一些prefix:<numeric_id>哈希值。

有时我想原子地清除它们。如果不使用某些分布式锁定机制，我该怎么做？



解答

从 redis 2.6.0 开始，您可以运行原子执行的 lua 脚本。我从来没有写过，但我认为它看起来像这样

EVAL "return redis.call('del', unpack(redis.call('keys', ARGV[1])))" 0 prefix:[YOUR_PREFIX e.g delete_me_*]

警告 ：正如Redis 文档所说，由于性能问题，keys命令不应该用于生产中的常规操作，该命令用于调试和特殊操作。阅读更多

请参阅EVAL 文档。
