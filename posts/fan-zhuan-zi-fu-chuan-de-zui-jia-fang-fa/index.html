反转字符串的最佳方法

Arya

2023-02-18 04:30

我不得不在 C# 2.0 中编写一个字符串反转函数（即 LINQ 不可用）并想出了这个：

public string Reverse(string text)
{
    char[] cArray = text.ToCharArray();
    string reverse = String.Empty;
    for (int i = cArray.Length - 1; i > -1; i--)
    {
        reverse += cArray[i];
    }
    return reverse;
}

就个人而言，我对这个功能并不着迷，并且相信有更好的方法来实现它。有没有？



解答

public static string Reverse( string s )
{
    char[] charArray = s.ToCharArray();
    Array.Reverse(charArray);
    return new string(charArray);
}


更多建议

这里有一个解决方案可以正确地将字符串反转"Les Mise\u0301rables"为"selbare\u0301siM seL". 这应该呈现为selbarésiM seL, 而不是selbaŕesiM seL（注意重音的位置），就像大多数基于代码单元（Array.Reverse等）甚至代码点（特别注意代理对的反转）的实现的结果一样。

using System;
using System.Collections.Generic;
using System.Globalization;
using System.Linq;

public static class Test
{
    private static IEnumerable<string> GraphemeClusters(this string s) {
        var enumerator = StringInfo.GetTextElementEnumerator(s);
        while(enumerator.MoveNext()) {
            yield return (string)enumerator.Current;
        }
    }
    private static string ReverseGraphemeClusters(this string s) {
        return string.Join("", s.GraphemeClusters().Reverse().ToArray());
    }

    public static void Main()
    {
        var s = "Les Mise\u0301rables";
        var r = s.ReverseGraphemeClusters();
        Console.WriteLine(r);
    }
}
