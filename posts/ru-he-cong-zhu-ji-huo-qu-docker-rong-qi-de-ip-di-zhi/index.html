如何从主机获取 Docker 容器的 IP 地址

Arya

2023-02-16 23:00

创建新容器后，是否可以运行命令从主机获取容器的 IP 地址？

基本上，一旦 Docker 创建了容器，我就想滚动我自己的代码部署和容器配置脚本。



解答

--format的选项来inspect拯救。

现代 Docker 客户端语法是：

docker inspect -f '{{range.NetworkSettings.Networks}}{{.IPAddress}}{{end}}' container_name_or_id

旧的 Docker 客户端语法是：

docker inspect --format '{{ .NetworkSettings.IPAddress }}' container_name_or_id

这些命令将返回 Docker 容器的 IP 地址。

如评论中所述：如果您使用的是 Windows，请在花括号周围使用双引号"而不是单引号。'


更多建议

你可以使用docker inspect <container id>。

例如：

CID=$(docker run -d -p 4321 base nc -lk 4321);
docker inspect $CID
