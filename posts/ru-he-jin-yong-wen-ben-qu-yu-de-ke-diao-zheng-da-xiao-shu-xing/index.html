如何禁用文本区域的可调整大小属性？

Arya

2023-02-16 19:58

我想禁用textarea.

目前，我可以textarea通过单击右下角textarea并拖动鼠标来调整 a 的大小。我怎样才能禁用它？



解答

以下 CSS 规则禁用textarea元素的调整大小行为：

textarea {
  resize: none;
}

要为某些（但不是全部）禁用它textarea，有几个选项。

class您可以在标签 ( ) 中使用属性<textarea class="textarea1">：

.textarea1 {
  resize: none;
}

要禁用属性设置为（即， ）textarea的特定对象：name``foo``<textarea name="foo"></textarea>

textarea[name=foo] {
  resize: none;
}

或者，使用id属性（即<textarea id="foo"></textarea>）：

#foo {
  resize: none;
}

W3C页面列出了调整大小限制的可能值：none、both、horizo​​ntal、vertical 和 inherit：

textarea {
  resize: vertical; /* user can resize vertically, but width is fixed */
}

查看一个体面的兼容性页面以查看当前支持此功能的浏览器。正如 Jon Hulka 评论的那样，可以使用 max-width、max-height、min- width 和 min-height 在 CSS 中进一步限制尺寸。

超级重要的是要知道：

除非 overflow 属性不是可见的，否则此属性不执行任何操作，这是大多数元素的默认设置。所以通常要使用它，你必须设置类似 overflow: scroll; 的东西。

引用 Sara Cope，http://css-tricks.com/almanac/properties/r/resize/
