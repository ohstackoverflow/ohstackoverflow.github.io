如何在签出之前的提交后返回到最新的提交？

Arya

2023-02-17 18:43

我有时会检查一些以前版本的代码来检查或测试。如果我想修改以前的提交，我已经看到了如何操作的说明——但假设我不做任何更改。在我完成 eg 之后git checkout HEAD^，我如何回到分支的顶端？..git log不再显示最新提交的 SHA。



解答

如果您知道要返回的提交是某个分支的负责人，或者被标记了，那么您可以

git checkout branchname

您还可以使用它git reflog来查看您的 HEAD（或任何其他 ref）过去指向的其他提交。

编辑添加：

在较新版本的 Git 中，如果您只运行git checkout或执行其他操作来移动HEAD一次，您也可以这样做

git checkout -

切换回上次结帐前的位置。这是由于类似于 shell 惯用语cd -返回到之前所在的任何工作目录。


更多建议

git结帐大师

master 是提示，或最后一次提交。 gitk 只会显示你当时在树中的位置。 git reflog 将显示所有提交，但在这种情况下，您只需要提示，所以 git checkout master。
