如何将新的本地分支推送到远程 Git 存储库并跟踪它？

Arya

2023-02-16 18:40

我如何能：

  1. 从另一个分支创建本地分支（通过git branch或git checkout -b）。

  2. 将本地分支推送到远程存储库（即发布），但使其可跟踪，以便git pull和git push将起作用。



解答

在 Git 1.7.0 及更高版本中，您可以检出一个新分支：

git checkout -b <branch>

编辑文件，添加并提交。然后使用-u（简称--set-upstream）选项推送：

git push -u origin <branch>

Git 将在推送期间设置跟踪信息。


更多建议

如果您不与其他人共享您的存储库，这对于将您的 所有 分支推送到远程并--set-upstream为您正确跟踪很有用：

git push --all -u

（不完全是 OP 的要求，但这种单线很受欢迎）

如果您与其他人共享您的存储库，这不是很好的形式，因为您会用所有狡猾的实验分支堵塞存储库。
