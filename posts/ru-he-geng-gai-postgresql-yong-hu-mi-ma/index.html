如何更改 PostgreSQL 用户的密码？



解答

不使用密码登录：

sudo -u user_name psql db_name

如果您忘记了密码，请重设密码：

ALTER USER user_name WITH PASSWORD 'new_password';


更多建议

要更改 PostgreSQL 用户的密码，请执行以下步骤：

  1. 登录到 psql 控制台：

    sudo -u postgres psql

  2. 然后在 psql 控制台，修改密码并退出：

    postgres=# \password postgres

    Enter new password: postgres=# \q

或者使用查询：

ALTER USER postgres PASSWORD '<new-password>';

或者在一行中

sudo -u postgres psql -c "ALTER USER postgres PASSWORD '<new-password>';"
笔记：

如果这不起作用，请通过编辑/etc/postgresql/9.1/main/pg_hba.conf（路径会有所不同）重新配置身份验证并更改：

local     all         all             peer # change this to md5

到

local     all         all             md5 # like this

然后重启服务器：

sudo service postgresql restart
