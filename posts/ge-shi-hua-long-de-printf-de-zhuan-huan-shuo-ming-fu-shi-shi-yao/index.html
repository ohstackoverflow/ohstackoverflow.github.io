格式化 long 的 printf 的转换说明符是什么？

Arya

2023-02-18 01:24

该printf函数采用参数类型，例如%dor %ifor a signed int。但是，我看不到任何有价值的东西long。



解答

l直接在说明符之前放置一个（小写字母 L）。

unsigned long n;
long m;

printf("%lu %ld", n, m);


更多建议

我想你的意思是：

unsigned long n;
printf("%lu", n);   // unsigned long

或者

long n;
printf("%ld", n);   // signed long
