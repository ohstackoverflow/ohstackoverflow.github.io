如何从十六进制颜色代码（例如 ）中获取颜色#FFDFD991？

我正在读取一个文件并得到一个十六进制颜色代码。我需要System.Windows.Media.Color为十六进制颜色代码创建相应的实例。框架中是否有内置方法来执行此操作？



解答

我假设那是一个 ARGB 代码......你指的是System.Drawing.Coloror 吗System.Windows.Media.Color？例如，后者用于 WPF。我还没有看到有人提到它，以防万一你正在寻找它：

using System.Windows.Media;

Color color = (Color)ColorConverter.ConvertFromString("#FFDFD991");


更多建议

假设您指的是 HTML 类型的 RGB 代码（称为十六进制代码，例如#FFCC66），请使用ColorTranslator类：

System.Drawing.Color col = System.Drawing.ColorTranslator.FromHtml("#FFCC66");

但是，如果您使用的是 ARGB 十六进制代码，则可以使用System.Windows.Media 命名空间中的ColorConverter类：

Color col = ColorConverter.ConvertFromString("#FFDFD991") as Color;
//or      = (Color) ColorConverter.ConvertFromString("#FFCC66") ;
