如何默认以管理员身份运行 Visual Studio？

Arya

2023-02-18 04:24

我最近发现，即使以管理员身份登录我的个人笔记本电脑，Visual Studio 也不会以管理员模式运行，您需要明确使用 Run As Administrator 。

有没有办法让它默认以管理员身份运行，而不是创建快捷方式等？



解答

从此处复制并粘贴“使用高级属性”部分。这将允许您在打开程序时始终以管理员身份运行该程序。

  1. 右键单击程序的快捷方式，然后单击属性。
  2. 单击程序快捷方式的快捷方式选项卡，然后单击高级按钮。
  3. 选中“以管理员身份运行”框，然后单击“确定”。
  4. 单击确定。
  5. 打开程序。
  6. 如果 UAC 提示，则单击“是”以应用权限以允许程序以管理员身份以完全权限运行。

注意：如果您是在以标准用户而非管理员身份登录时执行此操作，则您需要提供管理员密码才能以管理员身份运行程序。


更多建议

在 Windows 10 上尝试以下步骤：

  • 在“开始”窗口中搜索 Visual Studio 并选择“打开文件位置”：

  • 选择“解决兼容性问题”：

  • 选择“疑难解答程序”：

* 提升权限：

  • 选择“是，为此程序保存这些设置”

  • 选择“关闭”

完成后，Visual Studio 应该以管理员身份运行。
