生成 0 到 9 之间的随机整数

Arya

2023-02-17 00:04

如何在 Python 中生成 0 到 9（含）之间的随机整数？

例如0，,,,,,,,,,,, 1 2 3 4 5 6 7 89



解答

尝试random.randrange：

from random import randrange
print(randrange(10))


更多建议

尝试random.randint：

import random
print(random.randint(0, 9))

文档状态：

random.randint(a, b)

返回一个随机整数 N，使得 a <= N <= b。
