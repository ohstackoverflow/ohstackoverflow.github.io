'Microsoft.ACE.OLEDB.12.0' 提供程序未在本地计算机上注册

Arya

2023-02-18 04:31

我试图在按钮单击事件中从 Excel 文件中获取数据。我的连接字符串是：

 string connString = "Provider=Microsoft.ACE.OLEDB.12.0;Data Source=C:\\source\\SiteCore65\\Individual-Data.xls;Extended Properties=Excel 8.0;";

当我点击按钮时，出现以下错误：

“Microsoft.ACE.OLEDB.12.0”提供程序未在本地计算机上注册。

我不知道如何解决这个问题。我的操作系统是 Windows 7。



解答

那么，你需要安装它。您正在寻找：

  • 2007 Office System 驱动程序：数据连接组件。


更多建议

此处提供了 64 位版本的“Microsoft Access Database Engine 2010 Redistributable”，可让您使用“Microsoft.ACE.OLEDB.12.0”提供程序：
http://www.microsoft.com/en-us/下载/details.aspx?id=13255

如果您使用已接受答案的下载，您将需要为 x86 构建，如 @backtestbroker.com 所指出的。
