“+”（加号）CSS 选择器是什么意思？

Arya

2023-02-17 12:42

这个问题的答案是 社区的努力。编辑现有答案以改进这篇文章。它目前不接受新的答案或互动。

例如：

p + p {
  /* Some declarations */
}

我不知道这+意味着什么。p这和只为without定义样式有什么区别+ p？



解答

请参阅W3.org 上的相邻选择器。

在这种情况下，选择器意味着该样式仅适用于紧跟在另一个段落之后的段落。

普通p选择器会将样式应用于页面中的每个段落。

这仅适用于 IE7 或更高版本。在 IE6 中，样式不会应用于任何元素。>顺便说一句，这也适用于组合器。

另请参阅 Microsoft 对Internet Explorer 中的 CSS 兼容性的概述。


更多建议

它是相邻兄弟选择器。

来自Splash of Style 博客。

要定义 CSS 相邻选择器，请使用加号。

h1+p {color:blue;}

上面的 CSS 代码会将任何 h1 标题之后（不是内部）的第一段格式化为蓝色。

h1>p``p选择作为元素的直接（第一代）子元素（内部）的任何元素h1。

  • h1>p火柴<h1> <p></p> </h1>（<p>里面<h1>）

h1+p将选择作为元素的第一个p兄弟元素（在 dom 的同一级别）h1。

  • h1+p匹配<h1></h1> <p><p/>（<p>旁边/之后<h1>）
