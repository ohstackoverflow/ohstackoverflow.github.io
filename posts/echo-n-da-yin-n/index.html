“echo -n” 打印“-n”

Arya

2023-02-17 21:58

echo我的脚本有问题：

echo -n "Some string..."

印刷

-n Some string...

并移动到下一行。在控制台中，它在没有换行符的情况下正常工作：

Some string...



解答

该命令有多个版本echo，具有不同的行为。显然，用于脚本的 shell 使用了无法识别-n.

该printf命令具有更加一致的行为。echo适用于像 之类的简单事物echo hello，但我建议使用它printf来处理更复杂的事物。

你在什么系统上，你的脚本使用什么 shell？


更多建议

bash有一个名为“echo”的“内置”命令：

$ type echo
echo is a shell builtin

此外，还有一个“echo”命令是一个正确的可执行文件（即 shell forks 和 execs /bin/echo，而不是解释echo和执行它）：

$ ls -l /bin/echo
-rwxr-xr-x 1 root root 22856 Jul 21  2011 /bin/echo

echoeither的关于\cand的行为各-n不相同。你最好的选择是使用printf，它在我看过的四种不同的 *NIX 风格上可用：

$ printf "a line without trailing linefeed"
$ printf "a line with trailing linefeed\n"
