LINQ 相当于 IEnumerable<T> 的 foreach

Arya

2023-02-17 11:30

我想在 LINQ 中执行与以下等效的操作，但我不知道该怎么做：

IEnumerable<Item> items = GetItems();
items.ForEach(i => i.DoStuff());

真正的语法是什么？



解答

没有 ForEach 扩展IEnumerable；只为List<T>. 所以你可以做

items.ToList().ForEach(i => i.DoStuff());

或者，编写您自己的 ForEach 扩展方法：

public static void ForEach<T>(this IEnumerable<T> enumeration, Action<T> action)
{
    foreach(T item in enumeration)
    {
        action(item);
    }
}


更多建议

Fredrik 提供了修复程序，但可能值得考虑为什么这不在框架中。我相信这个想法是 LINQ 查询运算符应该没有副作用，适合以合理的功能方式看待世界。显然 ForEach 恰恰相反——一个 纯粹 基于副作用的构造。

这并不是说这是一件坏事——只是想一想这个决定背后的哲学原因。
