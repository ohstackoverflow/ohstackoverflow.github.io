fig.add_subplot(111) 中的参数是什么意思？

Arya

2023-02-18 03:18

有时我会遇到这样的代码：

import matplotlib.pyplot as plt
x = [1, 2, 3, 4, 5]
y = [1, 4, 9, 16, 25]
fig = plt.figure()
fig.add_subplot(111)
plt.scatter(x, y)
plt.show()

哪个产生：

我一直在疯狂地阅读文档，但找不到对111. 有时我看到一个212。

是什么意思fig.add_subplot()？



解答

这些是编码为单个整数的子图网格参数。例如，“111”表示“1x1 网格，第一个子图”，“234”表示“2x3 网格，第 4 个子图”。

的替代形式add_subplot(111)是add_subplot(1, 1, 1)。
