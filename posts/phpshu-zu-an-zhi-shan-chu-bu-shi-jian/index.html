PHP数组按值删除（不是键）

Arya

2023-02-17 05:32

我有一个 PHP 数组，如下所示：

$messages = [312, 401, 1599, 3, ...];

我想删除包含该值的元素$del_val（例如，$del_val=401），但我不知道它的键。这可能会有所帮助： 每个值只能存在一次 。

我正在寻找执行此任务的最简单的功能。



解答

使用array_search()和unset，尝试以下操作：

if (($key = array_search($del_val, $messages)) !== false) {
    unset($messages[$key]);
}

array_search()返回它找到的元素的键，可用于使用从原始数组中删除该元素unset()。它会在失败时返回FALSE，但它可以在成功时返回 false-y 值（0例如，您的键可能是），这就是!==使用严格比较运算符的原因。

该if()语句将检查是否array_search()返回了一个值，如果返回了则只执行一个操作。


更多建议

好吧，从数组中删除一个元素基本上只是与一个元素设置差异。

array_diff( [312, 401, 15, 401, 3], [401] ) // removing 401 returns [312, 15, 3]

它概括得很好，如果你愿意，你可以同时删除任意数量的元素。

免责声明： 请注意，我的解决方案生成数组的新副本，同时保持旧副本完好无损，这与接受的变异答案形成对比。选择你需要的那个。
