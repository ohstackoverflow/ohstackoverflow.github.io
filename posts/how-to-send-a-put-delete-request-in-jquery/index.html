How to send a PUT/DELETE request in jQuery?

Arya

2023-02-18 00:13

GET:$.get(..)

POST:$.post()..

What about PUT/DELETE?



Answer

You could use the ajax method:

$.ajax({
    url: '/script.cgi',
    type: 'DELETE',
    success: function(result) {
        // Do something with the result
    }
});


Suggest

$.ajax will work.

$.ajax({
   url: 'script.php',
   type: 'PUT',
   success: function(response) {
     //...
   }
});
