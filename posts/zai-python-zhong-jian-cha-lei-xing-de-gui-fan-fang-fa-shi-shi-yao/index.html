解答

用于isinstance检查是否o是的实例str或任何子类str：

if isinstance(o, str):

要检查的类型是否o完全是str， 不包括子类str：

if type(o) is str:

有关相关信息，请参阅Python 库参考中的内置函数。

在 Python 2 中检查字符串

对于 Python 2，这是检查是否o为字符串的更好方法：

if isinstance(o, basestring):

因为这也会捕获 Unicode 字符串。unicode不是str;的子类 和str都是unicode的子类basestring。在 Python 3 中，basestring不再存在，因为字符串 ( ) 和二进制数据 ( )严格分开。str``bytes

或者，isinstance接受类的元组。True如果o是以下任何子类的实例，这将返回(str, unicode)：

if isinstance(o, (str, unicode)):


更多建议

检查对象类型的 最 Pythonic方法是……不检查它。

由于 Python 鼓励Duck Typing，您应该按照try...except您想要的方式使用对象的方法。所以如果你的函数正在寻找一个可写的文件对象， 不要 检查它是否是 的子类file，只需尝试使用它的.write()方法即可！

当然，有时这些漂亮的抽象会分解，而isinstance(obj, cls)这正是您所需要的。但要谨慎使用。
