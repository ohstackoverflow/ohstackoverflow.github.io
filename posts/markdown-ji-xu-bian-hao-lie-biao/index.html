Markdown：继续编号列表

Arya

2023-02-17 17:20

在下面的降价代码中，我想item 3从列表编号 3 开始。但是由于降价之间的代码块将此列表项作为新列表启动。有什么办法可以防止这种行为吗？

期望的输出：

1. item 1
2. item 2

```
Code block
```

3. item 3

产生的输出：

  1. 第 1 项
  2. 第 2 项

Code block

  1. 第 3 项



解答

在项目符号点之间使用四个空格缩进内容

1. item 1
2. item 2

    ```
    Code block
    ```
3. item 3

生产：

  1. 第 1 项
  2. 第 2 项

Code block

  1. 第 3 项


更多建议

作为现有答案的扩展。对于那些试图在代码块以外的东西之后继续编号列表的人。例如第二段。只需将第二段缩进至少 1 个空格。

降价：

1. one
2. two

 three
3. four

输出：

  1. 一

  2. 二

三

  1. 四
