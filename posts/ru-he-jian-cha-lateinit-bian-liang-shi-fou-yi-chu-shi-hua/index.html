如何检查“lateinit”变量是否已初始化？

Arya

2023-02-17 13:19

我想知道是否有办法检查lateinit变量是否已初始化。例如：

class Foo() {

    private lateinit var myFile: File

    fun bar(path: String?) {
        path?.let { myFile = File(it) }
    }

    fun bar2() {
        myFile.whateverMethod()
        // May crash since I don't know whether myFile has been initialized
    }
}



解答

Kotlin 1.2 中有一个lateinit改进，允许lateinit直接检查变量的初始化状态：

lateinit var file: File

if (this::file.isInitialized) { ... }

请参阅JetBrains 博客上的公告或KEEP 提案。

更新： Kotlin 1.2 已经发布。您可以在此处找到lateinit增强功能：

  • 检查 lateinit var 是否已初始化
  • Lateinit 顶级属性和局部变量


更多建议

使用.isInitialized属性可以检查 lateinit 变量的初始化状态。

if (::file.isInitialized) {
    // File is initialized
} else {
    // File is not initialized
}
