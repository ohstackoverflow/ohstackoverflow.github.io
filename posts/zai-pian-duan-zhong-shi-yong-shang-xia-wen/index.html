如何获取片段中的上下文？

我需要使用我的数据库，其构造函数接受上下文，但getApplicationContext()不FragmentClass.this工作，我该怎么办？

数据库构造器

public Database(Context ctx)
{
    this.context = ctx;
    DBHelper = new DatabaseHelper(context);
}



解答

您可以使用getActivity()，它会返回与 a 关联的活动fragment。
该活动是一个context （自Activityextends Context）。


更多建议

按照上面的答案，你可以覆盖onAttach片段的方法：

public static class DummySectionFragment extends Fragment{
...
    @Override
    public void onAttach(Activity activity) {
        super.onAttach(activity);
        DBHelper = new DatabaseHelper(activity);
    }
}
