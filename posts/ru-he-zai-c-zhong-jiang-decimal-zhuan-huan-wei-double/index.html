如何在 C# 中将 Decimal 转换为 Double？

Arya

2023-02-17 13:32

我想将十进制变量“trans”分配给双变量“this.Opacity”。

decimal trans = trackBar1.Value / 5000;
this.Opacity = trans;

当我构建应用程序时，出现以下错误：

无法将类型 decimal 隐式转换为 double



解答

double不需要像这样的显式转换：

double trans = (double) trackBar1.Value / 5000.0;

将常量标识为5000.0（或5000d）就足够了：

double trans = trackBar1.Value / 5000.0;
double trans = trackBar1.Value / 5000d;


更多建议

通用问题“Decimal vs Double？”的更通用答案：

Decimal 用于货币计算以保持精度。 Double 用于不受微小差异影响的科学计算。由于 Double 是 CPU 的原生类型（内部表示存储在 基数 2 中），因此使用 Double 进行的计算比 Decimal 执行得更好（在内部以 基数 10 表示）。
