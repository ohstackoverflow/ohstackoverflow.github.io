单击锚链接时平滑滚动

Arya

2023-02-17 23:31

我的页面上有几个超链接。用户访问我的帮助部分时会阅读的常见问题解答。

使用锚链接，我可以使页面滚动到锚点并引导用户到那里。

有没有办法使滚动平滑？

但请注意，他使用的是自定义 JavaScript 库。也许 jQuery 提供了类似这样的东西？



解答

不需要任何 js 只需使用 scroll-behavior: smooth at html tag 就是这样

html{
scroll-behavior: smooth;
}


更多建议

CSS3 中的新热点。这比此页面上列出的每个方法都容易得多，并且不需要 Javascript。只需将下面的代码输入到您的 CSS 中，突然之间指向您自己页面内的位置的链接就会有一个流畅的滚动动画。

html{scroll-behavior:smooth}

之后，任何指向 div 的链接都会平滑地滑到这些部分。

<a href="#section">Section1</a>

编辑：对于那些对上面的标签感到困惑的人。基本上它是一个可点击的链接。然后你可以在你的网页的某处有另一个 div 标签，比如

<div id="section">content</div>

在这方面，a 链接将是可点击的，并将转到#section 所在的任何位置，在这种情况下，它是我们称为 section 的 div。

顺便说一句，我花了几个小时试图让它发挥作用。在一些晦涩的评论部分找到了解决方案。它有问题，在某些标签中不起作用。对身体没有作用。当我把它放在 CSS 文件中的 html{} 中时，它终于起作用了。
