是否有撤消 git init 的命令？

Arya

2023-02-17 09:16

我只是用错误的用户 Git 初始化了一个回购协议，然后想撤消它。有什么命令吗？我真的必须进入并编辑 .git 目录吗？



解答

您可以删除 .git。通常：

rm -rf .git

然后，重新创建为正确的用户。


更多建议
