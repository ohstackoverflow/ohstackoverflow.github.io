Eclipse：设置自动格式化的最大行长？

Arya

2023-02-18 03:26

我正在使用 Java。如果我在 Eclipse Helios 中点击Ctrl+ Shift+ ，它会自动格式化我的代码。F在某一点，它换行。我想增加最大行长度。我怎样才能做到这一点？



解答

在首选项 Java - > Code Style -> Formatter中，编辑配置文件。在 Line Wrapping 选项卡下是线宽的主要选项（最大线宽：）。在 Comments 选项卡中，您有一个单独的选项 Maximum line width for comments: ，它也需要更改以影响评论换行。

如果您使用其中一个，您将需要创建自己的配置文件以进行这些更改[Built-in]。只需单击格式化程序首选项页面上的“新建...”。
