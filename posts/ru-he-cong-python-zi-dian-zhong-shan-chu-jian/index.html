如何从 Python 字典中删除键？

Arya

2023-02-16 20:40

如果存在，我想从字典中删除一个键。我目前使用此代码：

if 'key' in my_dict:
    del my_dict['key']

如果没有该语句，如果密钥 不if存在，代码将引发。我怎样才能更简单地处理这个问题？KeyError __

有关从字典中删除键（包括生成修改副本的键）问题的更一般方法，请参阅从字典中删除元素。



解答

要删除一个键，而不管它是否在字典中，请使用双参数形式dict.pop()：

my_dict.pop('key', None)

my_dict[key]如果key字典中存在，这将返回，None否则。如果第二个参数未指定（即my_dict.pop('key')）且不key存在，则KeyError引发 a 。

要删除保证存在的密钥，您还可以使用

del my_dict['key']

KeyError如果键不在字典中，这将引发一个。


更多建议

专门回答“有没有一种方法可以做到这一点？”

if 'key' in my_dict: del my_dict['key']

...好吧，你 问了 ;-)

不过，您应该考虑，这种从 a 中删除对象的方法dict不是原子的——它可能在语句期间'key'存在，但可能在执行之前被删除，在这种情况下将失败并返回 a 。鉴于此，最安全的方法是使用或类似的东西my_dict``if``del``del``KeyError``dict.pop

try:
    del my_dict['key']
except KeyError:
    pass

当然，这绝对 不是 单行的。
