如何在 Notepad++ 中重新格式化 JSON？

Arya

2023-02-17 03:56

我需要 Notepad++ 从中获取一个 json 字符串

{"menu": {"id": "file","value": "File","popup": {"menuitem": [{"value": "New", "onclick": "CreateNewDoc()"},{"value": "Open", "onclick": "OpenDoc()"},{"value": "Close", "onclick": "CloseDoc()"}]}}}

对这个...

{"menu": {
  "id": "file",
  "value": "File",
  "popup": {
    "menuitem": [
      {"value": "New", "onclick": "CreateNewDoc()"},
      {"value": "Open", "onclick": "OpenDoc()"},
      {"value": "Close", "onclick": "CloseDoc()"}
    ]
  }
}}

我查看了所有 TextFX 选项，但找不到任何有效的选项。



解答

更新：

从 Notepad++ v7.6 开始，使用 Plugin Admin按照 这个答案安装 JSTool

安装

从http://sourceforge.net/projects/jsminnpp/下载，将JSMinNpp.dll复制到Notepad++的插件目录下。或者您可以从 Notepad++ 的插件管理器中安装“JSTool”。

新的 Notepad++ 安装以及 PluginManager 去了哪里？请参见如何在 Notepad++ 中查看插件管理器

{
  "menu" : {
    "id" : "file",
    "value" : "File",
    "popup" : {
      "menuitem" : [{
      "value" : "New",
          "onclick" : "CreateNewDoc()"
        }, {
          "value" : "Open",
          "onclick" : "OpenDoc()"
        }, {
          "value" : "Close",
          "onclick" : "CloseDoc()"
        }
      ]
    }
  }
}

提示：选择要重新格式化的代码，然后选择插件 | JS工具| JS格式。


更多建议

对于 Notepad++ v.7.6 及更高版本， 插件管理... 可用。

  1. 打开菜单插件 > 插件管理...

  2. 搜索 JSON 查看器

  3. 检查列表中的 JSON 查看器

  4. 点击 安装 按钮

  5. 重启记事本++

  6. 选择 JSON 文本

  7. 转到插件 > JSON 查看器 > 格式化 JSON (Ctrl+ Alt+ Shift+ M)

我们可以使用 Plugins Admin 安装任何 Notepad++ 支持的插件...
