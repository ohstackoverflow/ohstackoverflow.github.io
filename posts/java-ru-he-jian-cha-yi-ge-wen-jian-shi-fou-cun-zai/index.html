Java 如何检查一个文件是否存在？

Arya

2023-02-17 08:33

在用Java （相当于 Perl 的 ）打开文件进行阅读之前，如何检查文件是否存在-e $filename？

SO 上唯一类似的问题涉及写入文件，因此使用FileWriter此处显然不适用的方式进行了回答。

如果可能的话，我更喜欢真正的 API 调用返回 true/false 而不是一些“调用 API 以打开文件并在它抛出异常时捕获你检查文本中的‘无文件’”，但我可以忍受后者。



解答

使用java.io.File：

File f = new File(filePathString);
if(f.exists() && !f.isDirectory()) { 
    // do something
}


更多建议

我建议使用isFile()而不是exists(). 大多数时候，您不仅要检查路径是否指向文件，还要检查它是否存在。请记住，exists()如果您的路径指向一个目录，它将返回 true。

new File("path/to/file.txt").isFile();

new File("C:/").exists()将返回 true 但不允许您将其作为文件打开和读取。
