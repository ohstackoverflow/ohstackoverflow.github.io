如何从 Java 中的字符串值获取枚举值

Arya

2023-02-16 21:47

假设我有一个枚举，它只是

public enum Blah {
    A, B, C, D
}

我想找到一个字符串的枚举值，"A"例如Blah.A. 怎么可能做到这一点？

这是Enum.valueOf()我需要的方法吗？如果是这样，我将如何使用它？



解答

是的，Blah.valueOf("A")会给你Blah.A。

请注意，名称必须 完全 匹配，包括大小写：Blah.valueOf("a")并且Blah.valueOf("A ")都抛出一个IllegalArgumentException.

静态方法valueOf()和values()是在编译时创建的，不会出现在源代码中。不过，它们确实出现在 Javadoc 中；例如，Dialog.ModalityType显示两种方法。


更多建议

如果文本与枚举值不同，另一种解决方案：

public enum Blah {
    A("text1"),
    B("text2"),
    C("text3"),
    D("text4");

    private String text;

    Blah(String text) {
        this.text = text;
    }

    public String getText() {
        return this.text;
    }

    public static Blah fromString(String text) {
        for (Blah b : Blah.values()) {
            if (b.text.equalsIgnoreCase(text)) {
                return b;
            }
        }
        return null;
    }
}
