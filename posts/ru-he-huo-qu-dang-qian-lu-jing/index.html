如何获取当前路径

Arya

2023-02-17 16:36

当前的文档只讨论获取路由参数，而不是实际的路由段。

例如，如果我想找到当前路线的父级，那怎么可能呢？



解答

新的 V3 路由器有一个 url 属性。

this.router.url === '/login'


更多建议

角度 RC4：

您可以Router从导入@angular/router

然后注入它：

constructor(private router: Router ) {

}

然后调用它的 URL 参数：

console.log(this.router.url); //  /routename
