如何播放音频？

Arya

2023-02-17 08:49

我正在用 HTML5 和 JavaScript 制作游戏。

我如何通过 JavaScript 播放游戏音频？



解答

如果你不想弄乱 HTML 元素：

var audio = new Audio('audio_file.mp3');
audio.play();

显示代码片段

function play() {
  var audio = new Audio('https://interactive-examples.mdn.mozilla.net/media/cc0-audio/t-rex-roar.mp3');
  audio.play();
}


<button onclick="play()">Play Audio</button>

这使用了接口，它以与元素HTMLAudioElement相同的方式播放音频。<audio>

如果您需要更多功能，我使用了howler.js库，发现它简单实用。

显示代码片段

<script src="https://cdnjs.cloudflare.com/ajax/libs/howler/2.1.1/howler.min.js"></script>
<script>
    var sound = new Howl({
      src: ['https://interactive-examples.mdn.mozilla.net/media/cc0-audio/t-rex-roar.mp3'],
      volume: 0.5,
      onend: function () {
        alert('Finished!');
      }
    });
    sound.play()
</script>


更多建议

这很简单，只需获取您的audio元素并调用该play()方法即可：

document.getElementById('yourAudioTag').play();

看看这个例子： http: //www.storiesinflight.com/html5/audio.html

该站点揭示了您可以做的一些其他很酷的事情，例如load()、pause()和元素的一些其他属性audio。
