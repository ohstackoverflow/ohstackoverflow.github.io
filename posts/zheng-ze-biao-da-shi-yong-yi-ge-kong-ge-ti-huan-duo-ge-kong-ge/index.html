正则表达式用一个空格替换多个空格

Arya

2023-02-17 15:45

给定一个字符串，例如：

“狗有一条长尾巴，而且是红色的！”

什么样的 jQuery 或 JavaScript 魔法可以用来将空间保持为最大一个空间？

目标：

“狗有一条长尾巴，而且是红色的！”



解答

鉴于您还想覆盖制表符、换行符等，只需替换\s\s+为' '：

string = string.replace(/\s\s+/g, ' ');

如果你真的只想覆盖空格（而不是制表符、换行符等），请这样做：

string = string.replace(/  +/g, ' ');


更多建议

由于您似乎对性能感兴趣，因此我使用萤火虫对这些进行了分析。这是我得到的结果：

str.replace( /  +/g, ' ' )       ->  380ms
str.replace( /\s\s+/g, ' ' )     ->  390ms
str.replace( / {2,}/g, ' ' )     ->  470ms
str.replace( / +/g, ' ' )        ->  790ms
str.replace( / +(?= )/g, ' ')    -> 3250ms

这是在 Firefox 上运行 100k 字符串替换。

如果您认为性能是个问题，我鼓励您使用 firebug 进行自己的分析测试。众所周知，人类不善于预测程序中的瓶颈所在。

（另请注意，IE 8 的开发人员工具栏还内置了一个分析器——可能值得检查一下 IE 中的性能。）
