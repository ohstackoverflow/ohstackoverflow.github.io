PHP 中的 startsWith() 和 endsWith() 函数

Arya

2023-02-17 00:48

如果它以指定的字符/字符串开头或以它结尾，我如何编写两个函数来获取字符串并返回？

例如：

$str = '|apples}';

echo startsWith($str, '|'); //Returns true
echo endsWith($str, '}'); //Returns true



解答

PHP 8.0 及更高版本

从 PHP 8.0 开始，您可以使用

str_starts_with 手动和

str_ends_with 手动的

例子

echo str_starts_with($str, '|');

PHP 8.0 之前

function startsWith( $haystack, $needle ) {
     $length = strlen( $needle );
     return substr( $haystack, 0, $length ) === $needle;
}



function endsWith( $haystack, $needle ) {
    $length = strlen( $needle );
    if( !$length ) {
        return true;
    }
    return substr( $haystack, -$length ) === $needle;
}


更多建议

您可以使用substr_compare函数来检查开始和结束：

function startsWith($haystack, $needle) {
    return substr_compare($haystack, $needle, 0, strlen($needle)) === 0;
}
function endsWith($haystack, $needle) {
    return substr_compare($haystack, $needle, -strlen($needle)) === 0;
}

这应该是 PHP 7 上最快的解决方案之一（基准脚本）。针对 8KB 的干草堆、各种长度的针以及完整、部分和不匹配的情况进行了测试。strncmpstarts-with 稍微快一点，但它不能检查 ends-with。
