设置 flexbox 项目之间距离的更好方法

Arya

2023-02-17 03:08

设置我在容器margin: 0 5px上.item和margin: 0 -5px容器上使用的 flexbox 项目之间的最小距离。对我来说，这似乎是一种 hack，但我找不到更好的方法来做到这一点。

#box {
  display: flex;
  width: 100px;
  margin: 0 -5px;
}

.item {
  background: gray;
  width: 50px;
  height: 50px;
  margin: 0 5px;
}


<div id='box'>
  <div class='item'></div>
  <div class='item'></div>
  <div class='item'></div>
  <div class='item'></div>
</div>



解答

  • Flexbox 没有折叠边距。
  • Flexbox 没有任何类似于border-spacing表格的东西（编辑： CSS 属性gap在较新的浏览器中扮演这个角色，我可以使用吗）

因此，实现您的要求有点困难。

根据我的经验，不使用:first-child/:last-child并且无需任何修改即可工作的“最干净”方式是在容器和子级上flex- wrap:wrap设置。这将在每个孩子之间以及每个孩子与他们的父母之间产生差距。padding:5px``margin:5px``10px

演示

.upper {
  margin: 30px;
  display: flex;
  flex-direction: row;
  width: 300px;
  height: 80px;
  border: 1px red solid;

  padding: 5px; /* this */
}

.upper > div {
  flex: 1 1 auto;
  border: 1px red solid;
  text-align: center;

  margin: 5px;  /* and that, will result in a 10px gap */
}

.upper.mc /* multicol test */ {
  flex-direction: column;
  flex-wrap: wrap;
  width: 200px;
  height: 200px;
}


<div class="upper">
  <div>aaa<br/>aaa</div>
  <div>aaa</div>
  <div>aaa<br/>aaa</div>
  <div>aaa<br/>aaa<br/>aaa</div>
  <div>aaa</div>
  <div>aaa</div>
</div>

<div class="upper mc">
  <div>aaa<br/>aaa</div>
  <div>aaa</div>
  <div>aaa<br/>aaa</div>
  <div>aaa<br/>aaa<br/>aaa</div>
  <div>aaa</div>
  <div>aaa</div>
</div>
