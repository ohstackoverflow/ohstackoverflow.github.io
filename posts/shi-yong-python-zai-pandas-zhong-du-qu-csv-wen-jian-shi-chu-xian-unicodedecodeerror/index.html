解答

read_csv可以选择encoding处理不同格式的文件。我主要使用read_csv('file', encoding = "ISO-8859-1"), 或者用于encoding = "utf-8"阅读，通常utf-8用于to_csv.

您还可以使用多个alias选项之一，例如'latin'或'cp1252'(Windows) 而不是'ISO-8859-1'（请参阅python 文档，以及您可能遇到的许多其他编码）。

请参阅相关的 Pandas 文档、csv 文件上的 python 文档示例，以及此处关于 SO 的大量相关问题。What every developer should know about unicode and character sets是一个很好的背景资源。

要检测编码（假设文件包含非 ascii 字符），您可以使用enca（参见手册页）或file -i（linux）或file -I（osx）（参见手册页）。


更多建议

所有解决方案中最简单的：

import pandas as pd
df = pd.read_csv('file_name.csv', engine='python')

替代解决方案：

崇高的文字：

  • 在Sublime 文本编辑器 或 VS Code 中打开 csv 文件。
  • 以 utf-8 格式保存文件。
  • 在 sublime 中，点击 File -> Save with encoding -> UTF-8

VS代码：

在 VSCode 的底部栏中，您会看到标签 UTF-8。点击它。弹出窗口打开。单击使用编码保存。您现在可以为该文件选择一个新的编码。

然后，您可以像往常一样阅读您的文件：

import pandas as pd
data = pd.read_csv('file_name.csv', encoding='utf-8')

其他不同的编码类型是：

encoding = "cp1252"
encoding = "ISO-8859-1"
