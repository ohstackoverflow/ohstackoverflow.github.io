制作一个带有参数的 Bash 别名？

Arya

2023-02-17 00:35

我曾经使用 CShell (csh)，它可以让你创建一个带有参数的别名。符号是这样的

alias junk="mv \\!* ~/.Trash"

在 Bash 中，这似乎不起作用。鉴于 Bash 有许多有用的特性，我认为这个已经实现了，但我想知道如何实现。



解答

Bash 别名不直接接受参数。您将必须创建一个函数。

alias不接受参数，但可以像别名一样调用函数。例如：

myfunction() {
    #do things with parameters like $1 such as
    mv "$1" "$1.bak"
    cp "$2" "$1"
}


myfunction old.conf new.conf #calls `myfunction`

顺便说一下，在您的.bashrc文件和其他文件中定义的 Bash 函数在您的 shell 中可以作为命令使用。因此，例如，您可以像这样调用较早的函数

$ myfunction original.conf my.conf


更多建议

细化上面的答案，您可以获得像别名一样的 1 行语法，这对于 shell 或 .bashrc 文件中的临时定义更方便：

bash$ myfunction() { mv "$1" "$1.bak" && cp -i "$2" "$1"; }

bash$ myfunction original.conf my.conf

不要忘记右括号前的分号。同样，对于实际问题：

csh% alias junk="mv \\!* ~/.Trash"

bash$ junk() { mv "$@" ~/.Trash/; }

或者：

bash$ junk() { for item in "$@" ; do echo "Trashing: $item" ; mv "$item" ~/.Trash/; done; }
