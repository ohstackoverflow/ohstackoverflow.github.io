JavaScript 窗口调整大小事件

Arya

2023-02-17 16:56

如何挂接到浏览器窗口调整大小事件？

有一种 jQuery 方法可以侦听调整大小事件，但我不希望仅出于这一要求将其引入我的项目中。



解答

最佳做法是添加到调整大小事件，而不是替换它：

window.addEventListener('resize', function(event) {
    ...
}, true);

另一种方法是为 DOM 事件创建一个处理程序（但只能有一个），例如。

window.onresize = function(event) {
    ...
};

jQuery 可能会 做一些工作来确保在所有浏览器中一致地触发调整大小事件，但我不确定是否有任何浏览器不同，但我鼓励您在 Firefox、Safari 和 IE 中进行测试。


更多建议

首先，我知道addEventListener上面的评论中提到了该方法，但我没有看到任何代码。由于这是首选方法，因此它是：

window.addEventListener('resize', function(event){
  // do stuff here
});

这是一个工作示例。
