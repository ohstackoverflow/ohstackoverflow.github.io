如何从 Git 存储库中删除远程源

Arya

2023-02-17 03:01

我只是git init将我的文件夹初始化为 Git 存储库，然后使用git remote add origin URL. 现在我想删除它git remote add origin并添加一个新的存储库git remote add origin new-URL。我该怎么做？



解答

您可以这样做，而不是删除和重新添加：

git remote set-url origin git://new.url.here

看到这个问题：How to change the URI (URL) for a remote Git repository?

要删除远程使用此：

git remote remove origin


更多建议

如果你坚持删除它：

git remote remove origin

或者，如果你的 Git 版本为 1.7.10 或更早

git remote rm origin

但是kahowell的回答更好。
