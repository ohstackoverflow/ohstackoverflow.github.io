解答

注意 ：不久前，PyPI 中有适用于 Windows 的二进制轮子，因此这对于 Windows 用户来说应该不再是个问题。以下是针对 Linux、Mac 用户的解决方案，因为他们中的很多人都是通过网络搜索找到这篇文章的。

选项1

安装psycopg2-binaryPyPI 包，它有适用于 Linux 和 Mac OS 的 Python 轮子。

pip install psycopg2-binary

选项 2

psycopg2安装从源代码构建包的先决条件：

Debian/Ubuntu

蟒蛇3

sudo apt install libpq-dev python3-dev

python3.8-dev对于 Python 3.8，您可能需要安装或类似的。

蟒蛇2

sudo apt install libpq-dev python-dev

如果这还不够，请尝试

sudo apt install build-essential

或者

sudo apt install postgresql-server-dev-all

以及再次安装 psycopg2 之前。

中央操作系统 6

查看Banjer 的回答

苹果系统

查看nichochar 的回答


更多建议

在 CentOS 上，您需要 postgres 开发包：

sudo yum install python-devel postgresql-devel

这至少是 CentOS 6 上的解决方案。
