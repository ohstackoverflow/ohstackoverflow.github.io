如何检查列表是否为空？

Arya

2023-02-16 19:54

这个问题的答案是 社区的努力。编辑现有答案以改进这篇文章。它目前不接受新的答案或互动。

例如，如果通过以下内容：

a = []

如何检查是否a为空？



解答

if not a:
    print("List is empty")

使用空的隐式布尔值list非常 Pythonic。


更多建议

实现它的 Pythonic 方法来自PEP 8 风格指南。

对于序列（字符串、列表、元组），使用空序列为假的事实：

# Correct:
if not seq:
if seq:

# Wrong:
if len(seq):
if not len(seq):
