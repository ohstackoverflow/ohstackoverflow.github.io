如何在 Android XML 可绘制文件中定义圆形？

Arya

2023-02-17 12:35

我在查找 Android XML 中形状定义的文档时遇到了一些问题。我想在 XML 文件中定义一个用纯色填充的简单圆圈，以将其包含到我的布局文件中。

遗憾的是，android.com 上的文档没有涵盖 Shape 类的 XML 属性。我想我应该使用ArcShape来绘制一个圆，但是没有关于如何设置大小、颜色或从圆弧中制作圆所需的角度的说明。



解答

这是一个简单的圆形，作为 Android 中的可绘制对象。

<?xml version="1.0" encoding="utf-8"?>
<shape
    xmlns:android="http://schemas.android.com/apk/res/android"
    android:shape="oval">

   <solid 
       android:color="#666666"/>

   <size 
       android:width="120dp"
        android:height="120dp"/>
</shape>


更多建议

将此设置为您的视图背景

<shape xmlns:android="http://schemas.android.com/apk/res/android"
    android:shape="oval">
    <stroke
        android:width="1dp"
        android:color="#78d9ff"/>
</shape>

对于实心圆使用：

<shape xmlns:android="http://schemas.android.com/apk/res/android"
    android:shape="oval">
    <solid
        android:color="#48b3ff"/>
</shape>

实心描边：

<shape xmlns:android="http://schemas.android.com/apk/res/android"
    android:shape="oval">
    <solid android:color="#199fff"/>
    <stroke
        android:width="2dp"
        android:color="#444444"/>
</shape>

注意 ：要使oval形状显示为圆形，在这些示例中，您使用此形状作为其背景的视图应该是正方形，或者您必须将形状标签的height和属性设置为相等的值。width
