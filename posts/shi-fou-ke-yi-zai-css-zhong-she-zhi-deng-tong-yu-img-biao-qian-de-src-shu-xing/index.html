是否可以在 CSS 中设置等同于 img 标签的 src 属性？

Arya

2023-02-17 19:34

是否可以在 CSS 中设置src属性值？
大多数情况下，我们是这样使用的：

<img src="pathTo/myImage.jpg" />

我希望它是这样的

<img class="myClass" />



.myClass {
    some-src-property: url("pathTo/myImage.jpg");
}

我想知道是否有一种方法可以 不 使用CSS 中的background或background-image属性。



解答

使用content:url("image.jpg")。

完整的工作解决方案（Live Demo）：

<!doctype html>

<style>
.MyClass123{
    content:url("http://imgur.com/SZ8Cm.jpg");
}
</style>

<img class="MyClass123"/>

测试和工作：

  • 铬 14.0.835.163
  • 野生动物园 4.0.5
  • 歌剧 10.6
  • Firefox 100 及更新版本

测试和 不 工作：

  • FireFox 40.0.2（观察Developer Network Tools，可以看到URL加载了，但是图片不显示）
  • Internet Explorer 11.0.9600.17905（从不加载 URL）


更多建议

我今天找到了一个解决方案（适用于 IE6+、FF、Opera、Chrome）：

<img src='willbehidden.png' 
 style="width:0px; height:0px; padding: 8px; background: url(newimage.png);">

怎么运行的：

  • 图像缩小，直到宽度和高度不再可见。
  • 然后，您需要使用填充“重置”图像大小。这个给出了一个 16x16 的图像。当然你可以使用 padding-left / padding-top 来制作矩形图像。
  • 最后，使用背景将新图像放在那里。
  • 如果新的背景图像太大或太小，我建议使用background-size例如：background-size:cover;它适合您的图像到分配的空间。

它也适用于提交输入图像，它们保持可点击。

查看现场演示：http://www.audenaerde.org/csstricks.html#imagereplacecss
