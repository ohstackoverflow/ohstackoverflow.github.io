如何在 Visual Studio Code 中搜索文件？

Arya

2023-02-17 06:28

我习惯了 Resharper，我可以在其中搜索文件，而不是内容，而是文件名，这样可以快速打开新文件。

此功能是否在 Visual Studio Code 中实现，是否有快捷方式？



解答

使用 转到 菜单下的 转到文件... 或使用键盘快捷键： ****

  • 在 Windows Ctrl+p或Ctrl+e
  • 在 macOS Cmd ⌘+上p
  • 在 Linux Ctrl+p或Ctrl+e

然后键入文件名。

还请务必检查您是否可以设置自己的键绑定，以及是否有适用于Windows、macOS和Linux的备忘单。


更多建议

自2022 年 7 月发布 1.70.0以来，您可以在查找资源管理器等树视图中查找和筛选。您可以在树内按Ctrl+F弹出 Find 控件。

在 1.70.0 发布之前

当您单击资源管理器树中的任意位置并开始在键盘上键入内容时，搜索关键字会出现在屏幕的右上角：（“ module.ts ”）

当您将鼠标光标悬停在关键字上时，您可以单击“ 启用类型过滤器 ”以使用您的搜索过滤树！
