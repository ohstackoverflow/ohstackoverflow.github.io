如何检查元素是否隐藏在 jQuery 中？

Arya

2023-02-16 18:18

如何使用.hide()、.show()或切换元素的可见性.toggle()？

我如何测试一个元素是visibleor hidden？



解答

由于问题涉及单个元素，因此这段代码可能更合适：

// Checks CSS content for display:[none|block], ignores visibility:[true|false]
$(element).is(":visible");

// The same works with hidden
$(element).is(":hidden");

它与twernt 的建议相同，但应用于单个元素；它与jQuery FAQ 中推荐的算法相匹配。

我们使用 jQuery 的is()来检查所选元素与另一个元素、选择器或任何 jQuery 对象。此方法遍历 DOM 元素以找到满足传递参数的匹配项。如果匹配则返回 true，否则返回 false。


更多建议

您可以使用hidden选择器：

// Matches all elements that are hidden
$('element:hidden')

和visible选择器：

// Matches all elements that are visible
$('element:visible')
