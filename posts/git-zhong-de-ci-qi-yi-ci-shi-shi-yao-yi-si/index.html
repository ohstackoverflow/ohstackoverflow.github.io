Git 中的“瓷器”一词是什么意思？

Arya

2023-02-18 03:37

“瓷器”一词偶尔会出现在 Git 文档中。这是什么意思？



解答

“瓷器”是通常用来制造马桶（有时还有洗脸盆等其他固定装置）的材料。这与“管道”（实际的管道和排水管）不同，瓷器为管道提供了更加用户友好的界面。

Git 类比地使用这个术语，将用户通常不需要直接使用的低级命令（“管道”）与更用户友好的高级命令（“瓷器”）分开。


更多建议

更重要的是，术语“瓷器”适用于高级命令，输出：

  • 意味着人类可读
  • 不打算被解析
  • 容易受到变化/演变的影响

这是关键：如果您编写脚本，则应尽可能使用具有稳定输出的 管道 命令。不是陶瓷命令。

但是， 您可以使用 porcelain 命令的输出，该命令--porcelain在脚本中有一个选项（见下文），例如：

git status --porcelain
git push --porcelain
git blame --porcelain

虽然 git 包含了它自己的 porcelain 层 ，但是它的底层命令足以支持替代 porcelain 的开发。 这些低级命令的
接口（输入、输出、选项集和语义）比 Porcelain 级命令要稳定得多，因为 这些命令主要用于脚本使用 。另一方面，为了改善最终用户体验，Porcelain 命令的界面可能会发生变化。 * *

请参阅“如何以编程方式确定是否有未提交的更改？ ”作为使用管道命令而不是陶瓷命令的示例。

注意：porcelain 命令可以有一个--porcelain选项。
例如：git status --porcelain，它指定 要解析的输出 。

--porcelain

以易于解析的格式为脚本提供输出。这类似于短输出，但无论用户配置如何，跨 git 版本都将保持稳定。详情见下文。
