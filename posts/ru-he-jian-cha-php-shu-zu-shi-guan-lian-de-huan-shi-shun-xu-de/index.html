如何检查 PHP 数组是关联的还是顺序的？

Arya

2023-02-17 10:26

PHP 将所有数组视为关联数组，因此没有任何内置函数。谁能推荐一种相当有效的方法来检查数组是否 “是列表” （仅包含从 0 开始的数字键）？

基本上，我希望能够区分这一点：

$sequentialArray = [
    'apple', 'orange', 'tomato', 'carrot'
];

和这个：

$assocArray = [
    'fruit1' => 'apple',
    'fruit2' => 'orange',
    'veg1' => 'tomato',
    'veg2' => 'carrot'
];



解答

您问了两个不完全相同的问题：

  • 一、如何判断数组是否只有数字键
  • 其次，如何判断一个数组是否有 连续的 数字键，从0开始

考虑您实际需要哪些行为。（可能两者都适合您的目的。）

Captain kurO 很好地回答了第一个问题（简单地检查所有键是否为数字）。

对于第二个问题（检查数组是否为零索引和顺序），可以使用以下函数：

function isAssoc(array $arr)
{
    if (array() === $arr) return false;
    return array_keys($arr) !== range(0, count($arr) - 1);
}

var_dump(isAssoc(['a', 'b', 'c'])); // false
var_dump(isAssoc(["0" => 'a', "1" => 'b', "2" => 'c'])); // false
var_dump(isAssoc(["1" => 'a', "0" => 'b', "2" => 'c'])); // true
var_dump(isAssoc(["a" => 'a', "b" => 'b', "c" => 'c'])); // true


更多建议

仅检查数组是否具有非整数键（而不是数组是顺序索引还是零索引）：

function has_string_keys(array $array) {
  return count(array_filter(array_keys($array), 'is_string')) > 0;
}

如果至少有一个字符串键，$array将被视为一个关联数组。
