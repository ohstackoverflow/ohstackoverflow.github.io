如何强制maven更新？

Arya

2023-02-17 10:28

我在另一台计算机上导入了我已经在工作的项目，它开始下载依赖项。

显然我的互联网连接崩溃了，现在我得到以下信息：

    >Build errors for comics; org.apache.maven.lifecycle.LifecycleExecutionException:

    >Original error: Could not transfer artifact org.springframework:spring-context:jar:3.0.5.RELEASE from central (http://repo1.maven.org/maven2): No response received after 60000

我如何强制maven更新？



解答

mvn clean install -U

-U表示强制更新 快照 依赖项。

如果以前从未成功下载过发布依赖项，它们将以这种方式更新。参考：https://stackoverflow.com/a/29020990/32453


更多建议

-U如果您的本地存储库因发布 jar 而不是快照（并且仅更新快照）以某种方式被破坏--update- snapshots，您可以使用以下方法清除本地存储库：

 mvn dependency:purge-local-repository

然后您可能想再次清理并安装：

 mvn dependency:purge-local-repository clean install

https://maven.apache.org/plugins/maven-dependency-plugin/examples/purging- local-repository.html提供更多信息
