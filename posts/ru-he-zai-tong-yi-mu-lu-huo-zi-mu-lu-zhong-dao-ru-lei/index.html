如何在同一目录或子目录中导入类？

Arya

2023-02-17 10:20

我有一个存储所有 .py 文件的目录。

bin/
   main.py
   user.py # where class User resides
   dir.py # where class Dir resides

我想在 main.py 中使用 user.py 和 dir.py 中的类。如何将这些 Python 类导入 main.py ？此外，如果 user.py 在子目录中，我该如何导入类？ ****

User ****

 bin/
    dir.py
    main.py
    usr/
        user.py



解答

蟒蛇2

__init__.py在与文件相同的目录中创建一个名为的空文件。这将向 Python 表明“可以从该目录导入”。

然后就做...

from user import User
from dir import Dir

如果文件位于子目录中，同样如此 -__init__.py也将 an 放在子目录中，然后使用带点符号的常规导入语句。对于每一级目录，都需要添加到导入路径中。

bin/
    main.py
    classes/
        user.py
        dir.py

所以如果目录被命名为“classes”，那么你会这样做：

from classes.user import User
from classes.dir import Dir

蟒蛇3

与之前相同，但在模块名称前加上.if not using a subdirectory：

from .user import User
from .dir import Dir


更多建议

我刚刚了解到（感谢martineau 的评论），为了从同一目录中的文件导入类，您现在可以用 Python 3 编写：

from .user import User
from .dir import Dir
