如何检查加载了哪个版本的 jQuery？

Arya

2023-02-18 05:15

如何检查客户端计算机上加载了哪个版本的 jQuery？客户端可能加载了 jQuery，但我不知道如何检查它。如果他们加载了它，我该如何检查版本和前缀，例如：

$('.class')
JQuery('.class')



解答

if (typeof jQuery != 'undefined') {  
    // jQuery is loaded => print the version
    alert(jQuery.fn.jquery);
}


更多建议

我的 goto 是判断版本的意思：

$.fn.jquery

另一个类似的选择：

$().jQuery

如果担心可能有多种实现$— 使$.歧义 — 那么请jQuery改用：

jQuery.fn.jquery

最近我在一些网站上使用$.fn.jquery和 时遇到问题，所以我想记下第三个简单的命令来拉取 jQuery 版本。$().jQuery

如果你得到一个版本号——通常是一个字符串——然后 jQuery 被加载，这就是你正在使用的版本。如果没有加载，那么你应该回来undefined或者甚至是一个错误。
