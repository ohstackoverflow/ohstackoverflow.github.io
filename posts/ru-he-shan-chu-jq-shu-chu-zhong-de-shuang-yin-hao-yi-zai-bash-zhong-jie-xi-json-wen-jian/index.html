如何删除 jq 输出中的双引号以在 bash 中解析 json 文件？

Arya

2023-02-18 03:32

我正在使用 jq 来解析 JSON 文件，如此处所示。但是，字符串值的结果包含预期的“双引号”，如下所示：

$ cat json.txt | jq '.name'
"Google"

我如何将其通过管道传输到另一个命令中以删除“”？所以我得到

$ cat json.txt | jq '.name' | some_other_command
Google

some_other_command我能用什么？



解答

使用-r(or --raw-output) 选项发出原始字符串作为输出：

jq -r '.name' <json.txt


更多建议

{"name": "Google"}所以对于一个包含yes 的文件

sample='{"name":"Google"}'
echo $sample| jq '.name'

"Google"

使用--raw-input帮助

echo $sample| jq --raw-output '.name'

Google

--raw-output但是我偶然发现了这个问题，因为我在这样的 json 数组上使用

sample='[{"name":"Yahoo"},{"name":"Google"}]'
echo $sample | jq --raw-output 'map(.name)'

[
  "Yahoo",
  "Google"
]

而且我不明白为什么引号仍然存在。我看到了这篇文章，现在我知道添加| .[]就可以了！

echo $sample | jq --raw-output 'map(.name)| .[]'

Yahoo
Google
