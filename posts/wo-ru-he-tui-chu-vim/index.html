我如何退出 Vim？

Arya

2023-02-16 18:50

我被困住了，无法逃脱。它说：

type :quit<Enter> to quit VIM

但是当我输入它时，它只会出现在对象主体中。



解答

按下Esc键进入“普通模式”。然后你可以键入:进入“命令行模式”。冒号 ( :) 将出现在屏幕底部，您可以输入以下命令之一。要执行命令，请按Enter键。

  • :q退出（简称:quit）
  • :q!退出而不保存（简称:quit!）
  • :wq写作和退出
  • :wq!写入并退出，如果文件缺少写入权限则尝试强制写入
  • :x写作和退出；喜欢:wq但仅在修改时写入（ 的缩写:exit）
  • :qa退出所有（简称:quitall）
  • :cq退出，不保存，使用非零退出代码表示失败（简称:cquit）

您也可以直接从“正常模式”退出 Vim，方法是键入ZZ保存并退出（与 相同:x）或ZQ直接退出（与 相同:q!）。（请注意，这里的大小写很重要。ZZ并不是zz同一件事。）

Vim 有广泛的帮助——你可以用:help命令访问——在那里你可以找到所有问题的答案和初学者教程。

我每个学期都把这个画给我的学生，他们似乎在之后掌握了 vi。

Vi 是一个具有三个主要状态的有限状态机。

它以 COMMAND 模式启动，您可以在其中 盲目地 使用非常短的击键序列执行编辑器功能。你知道你在做什么；这不适合业余爱好者。

当你想真正编辑文本时，你应该通过一些按键进入 INSERT 模式；常见的包括：

  • i: 在光标 前 插入
  • I：移至行首并 (i)nsert
  • a: 追加在光标 之后
  • A：移至行尾并 (a)append
  • o：在当前行 下方 新开一行
  • O：在当前行上方 新 开一行
  • R：进入REPLACE模式（类似INSERT模式）

现在，回答这个问题：退出。

您可以从 EX 模式退出 vi：

  • q - 如果您没有进行任何修改或事先保存
  • 问！- 忽略任何修改并 退出
  • wq - 保存并退出
  • x - 这类似于 wq

w并x接受一个文件名参数。如果 vi 已经知道要使用的文件名（例如，它以 开头vi file），则无需在此处再次提供。

最后，最重要的是：如何才能达到 EX 模式？

EX 模式适用于长命令，您可以在屏幕的最后一行看到这些命令。在 COMMAND 模式下，您按冒号，:，冒号将出现在底行，您可以在其中键入上述命令。

从 INSERT 模式，您需要按下 ESC，即 Escape 按钮，进入 COMMAND 模式，然后： 进入 EX 模式。

如果您不确定，请按 ESC 键，这将带您进入命令模式。

可靠的方法是 ESC-:-x-Enter ，它会保存您的文件并退出。
