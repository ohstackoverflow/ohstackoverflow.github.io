如何检查 Pandas 中是否存在列

Arya

2023-02-18 04:25

如何检查 Pandas DataFrame 中是否存在列df？

   A   B    C
0  3  40  100
1  6  30  200

我将如何检查上面的 DataFrame 中是否存在该列"A"，以便我可以计算：

df['sum'] = df['A'] + df['C']

如果"A"不存在：

df['sum'] = df['B'] + df['C']



解答

这将起作用：

if 'A' in df:

但为了清楚起见，我可能会把它写成：

if 'A' in df.columns:


更多建议

要检查一列 或多 列是否全部存在，您可以使用set.issubset, 如：

if set(['A','C']).issubset(df.columns):
   df['sum'] = df['A'] + df['C']

正如@brianpck 在评论中指出的那样，set([])也可以用花括号构造，

if {'A', 'C'}.issubset(df.columns):

有关花括号语法的讨论，请参阅此问题。

或者，您可以使用生成器理解，如：

if all(item in df.columns for item in ['A','C']):
