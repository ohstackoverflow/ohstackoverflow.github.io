使用 javascript（或 jQuery）选择和操作 CSS 伪元素，例如 ::before 和 ::after

Arya

2023-02-17 05:45

有什么方法可以使用 jQuery 选择/操作 CSS 伪元素，例如::beforeand ::after（以及带有一个分号的旧版本）？

例如，我的样式表具有以下规则：

.span::after{ content:'foo' }

如何使用普通 JS 或 jQuery 将“foo”更改为“bar”？



解答

您还可以将内容传递给具有数据属性的伪元素，然后使用 jQuery 对其进行操作：

在 HTML 中：

<span>foo</span>

在 jQuery 中：

$('span').hover(function(){
    $(this).attr('data-content','bar');
});

在 CSS 中：

span:after {
    content: attr(data-content) ' any other text you may want';
}

如果你想防止“其他文本”出现，你可以将它与 seucolega 的解决方案结合起来，如下所示：

在 HTML 中：

<span>foo</span>

在 jQuery 中：

$('span').hover(function(){
    $(this).addClass('change').attr('data-content','bar');
});

在 CSS 中：

span.change:after {
    content: attr(data-content) ' any other text you may want';
}


更多建议

你会认为这是一个简单的问题，jQuery 可以做的所有其他事情。不幸的是，问题归结为技术问题： css :after 和 :before 规则不是 DOM 的一部分， 因此不能使用 jQuery 的 DOM 方法进行更改。

有 一些 方法可以使用 JavaScript 和/或 CSS 变通方法来操纵这些元素；您使用哪一个取决于您的具体要求。

我将从被广泛认为是“最佳”的方法开始：

1) 添加/删除预定类

在这种方法中，您已经在 CSS 中创建了一个具有不同:after或:before样式的类。稍后将这个“新”类放在样式表中以确保它覆盖：

p:before {
    content: "foo";
}
p.special:before {
    content: "bar";
}

然后，您可以使用 jQuery（或原生 JavaScript）轻松添加或删除此类：

$('p').on('click', function() {
    $(this).toggleClass('special');
});

    });


p:before {
  content: "foo";
  color: red;
  cursor: pointer;
}
p.special:before {
  content: "bar";
}

  • 优点： 易于使用 jQuery 实现；一次快速改变多种风格；强制关注点分离（将 CSS 和 JS 与 HTML 隔离）
  • 缺点： CSS 必须预先写好，所以:beforeor的内容:after不是完全动态的

2) 将新样式直接添加到文档的样式表中

可以使用 JavaScript 将样式直接添加到文档样式表，包括:after和:before样式。jQuery 没有提供方便的快捷方式，但幸运的是 JS 并没有那么复杂：

var str = "bar";
document.styleSheets[0].addRule('p.special:before','content: "'+str+'";');

3) 改变不同的 DOM 属性

您还可以在 CSS 中使用attr()它来读取特定的 DOM 属性。（如果浏览器支持:before，它attr()也支持。）通过将其与content:一些精心准备的 CSS 相结合，我们可以动态更改和的内容（但 不能更改其他属性， 如边距或颜色） ：:before``:after

p:before {
    content: attr(data-before);
    color: red;
    cursor: pointer;
}
