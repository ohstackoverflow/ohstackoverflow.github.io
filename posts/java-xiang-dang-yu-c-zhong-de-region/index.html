Java 相当于 C# 中的#region

Arya

2023-02-18 05:45

我想在Eclipse 中使用区域进行代码折叠；这怎么能在 Java 中完成呢？

C#中的示例用法：

#region name
//code
#endregion



解答

没有这样的 标准 等价物。某些 IDE（例如 Intellij 或 Eclipse）可以根据所涉及的代码类型（构造函数、导入等）进行折叠，但没有什么比#region.
