从命令行运行 R 脚本

Arya

2023-02-17 22:34

我有一个文件，名为a.r，它的值为chmod755，

sayHello <- function(){
   print('hello')
}

sayHello()

我如何通过命令行运行它？



解答

如果您希望输出打印到终端，最好使用 Rscript

Rscript a.R

请注意，当使用R CMD BATCH a.R它而不是将输出重定向到标准输出并在终端上显示时，将创建一个名为 a.Rout 的新文件。

R CMD BATCH a.R
# Check the output
cat a.Rout

关于使用 Rscript 的另一件事是它不会methods默认加载包，这可能会导致混淆。因此，如果您依赖方法提供的任何内容，您将希望在脚本中显式加载它。

如果你真的想使用./a.R调用脚本的方式，你可以#!在脚本的顶部添加一个适当的

#!/usr/bin/env Rscript
sayHello <- function(){
   print('hello')
}
sayHello()

我还会注意到，如果您在 *unix 系统上运行，那么有一个有用的littler包，它提供了简单的命令行管道到 R。可能需要使用 littler 通过脚本运行闪亮的应用程序？可以在这个问题中找到更多详细信息。


更多建议

这并没有直接回答问题。但是有人可能会在这里结束，因为他们想从终端运行 R 的 oneliner。例如，如果你只是想安装一些丢失的包然后退出，这个 oneliner 可以很方便。我经常用它，突然发现少了一些包，想安装到我想安装的地方。

  • 要安装到默认位置：

    R -e 'install.packages(c("package1", "package2"))'

  • 要安装到需要权限的位置root：

    R -e 'install.packages(c("package1", "package2"), lib="/usr/local/lib/R/site-library")'
