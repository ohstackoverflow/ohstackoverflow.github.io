在 Git 中可视化分支拓扑

Arya

2023-02-17 06:41

我在自己的机器上单独使用 Git，我发现很难维护我所有分支和提交的心智模型。我知道我可以git log从我所在的地方查看提交历史记录，但是有没有办法查看整个分支地形图，比如这些 ASCII 映射似乎到处都在用来解释分支？

      .-A---M---N---O---P
     /     /   /   /   /
    I     B   C   D   E
     \   /   /   /   /
      `-------------'

感觉就像有人走过来并试图获取我的存储库一样，很难弄清楚到底发生了什么。

我想我受到了 AccuRev 的流浏览器的影响......



解答

使用git log --graph或gitk。（两者都接受--all，这将显示所有分支而不仅仅是当前分支。）

对于分支名称和紧凑视图，请尝试：

git log --graph --decorate --oneline


更多建议

我有 3 个别名 （和 4 个用于快速使用的别名）， 我通常会在我的~/.gitconfig文件中使用这些别名：

[alias]
    lg = lg1
    lg1 = lg1-specific --all
    lg2 = lg2-specific --all
    lg3 = lg3-specific --all

    lg1-specific = log --graph --abbrev-commit --decorate --format=format:'%C(bold blue)%h%C(reset) - %C(bold green)(%ar)%C(reset) %C(white)%s%C(reset) %C(dim white)- %an%C(reset)%C(auto)%d%C(reset)'
    lg2-specific = log --graph --abbrev-commit --decorate --format=format:'%C(bold blue)%h%C(reset) - %C(bold cyan)%aD%C(reset) %C(bold green)(%ar)%C(reset)%C(auto)%d%C(reset)%n''          %C(white)%s%C(reset) %C(dim white)- %an%C(reset)'
    lg3-specific = log --graph --abbrev-commit --decorate --format=format:'%C(bold blue)%h%C(reset) - %C(bold cyan)%aD%C(reset) %C(bold green)(%ar)%C(reset) %C(bold cyan)(committed: %cD)%C(reset) %C(auto)%d%C(reset)%n''          %C(white)%s%C(reset)%n''          %C(dim white)- %an <%ae> %C(reset) %C(dim white)(committer: %cn <%ce>)%C(reset)'
