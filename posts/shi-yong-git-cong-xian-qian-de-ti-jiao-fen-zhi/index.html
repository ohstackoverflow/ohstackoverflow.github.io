使用 Git 从先前的提交分支

Arya

2023-02-16 21:25

如果我有N提交，我如何从N-3提交分支？



解答

使用提交哈希创建分支：

git branch branch_name <commit-hash>

或者通过使用符号引用：

git branch branch_name HEAD~3

要在创建分支时检出分支，请使用：

git checkout -b branch_name <commit-hash or HEAD~3>


更多建议

要在 github.com 上执行此操作：

  1. 转到您的项目。
  2. 单击“提交”。
  3. 单击要从中分支的提交上的 < >（“浏览历史记录中此时的存储库”）。
  4. 单击左上角的“树：xxxxxx”。在语言统计栏下方，您将获得“查找或创建分支”的选项（只需在此处输入新的分支名称）
