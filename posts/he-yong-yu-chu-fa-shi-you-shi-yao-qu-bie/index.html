'/' 和 '//' 用于除法时有什么区别？

Arya

2023-02-18 04:11

使用一个比另一个有好处吗？在 Python 2 中，它们似乎都返回相同的结果：

>>> 6/3
2
>>> 6//3
2



解答

在 Python 3.x 中，5 / 2将返回2.5并将5 // 2返回2. 前者是浮点除法，后者是 _ 底除法 ，有时也叫 _ 整数除法 。

在 Python 2.2 或更高版本的 2.x 行中，整数没有区别，除非您执行 a from __future__ import division，这会导致 Python 2.x 采用 3.x 行为。

不管以后的导入，5.0 // 2都会返回2.0，因为那是地板分割运算的结果。

您可以在PEP 238：更改除法运算符 中找到详细说明。
