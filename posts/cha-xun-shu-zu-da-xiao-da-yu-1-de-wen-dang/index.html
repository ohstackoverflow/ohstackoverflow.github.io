查询数组大小大于 1 的文档

Arya

2023-02-17 09:58

我有一个 MongoDB 集合，其中包含以下格式的文档：

{
  "_id" : ObjectId("4e8ae86d08101908e1000001"),
  "name" : ["Name"],
  "zipcode" : ["2223"]
}
{
  "_id" : ObjectId("4e8ae86d08101908e1000002"),
  "name" : ["Another ", "Name"],
  "zipcode" : ["2224"]
}

我目前可以获得与特定数组大小匹配的文档：

db.accommodations.find({ name : { $size : 2 }})

这会正确返回数组中包含 2 个元素的文档name。但是，我无法执行命令来返回字段数组大小大于 2 的$gt所有文档：name

db.accommodations.find({ name : { $size: { $gt : 1 } }})

如何选择数组name大小大于 1 的所有文档（最好不必修改当前数据结构）？



解答

更新：

对于 mongodb 2.2+版本， @JohnnyHK 在另一个答案中描述了更有效的方法。

  1. 使用$where

db.accommodations.find({ $where: "this.name.length > 1" } );

但...

Javascript 的执行速度比本页列出的本机运算符慢，但非常灵活。有关详细信息，请参阅服务器端处理页面。

  1. 创建 额外的 字段NamesArrayLength，用名称数组长度更新它，然后在查询中使用：

db.accommodations.find({"NamesArrayLength": {$gt: 1} });

这将是更好的解决方案，并且工作速度更快（您可以在其上创建索引）。
