如何在 Vim 中复制到剪贴板？

Arya

2023-02-17 07:25

是否可以直接从 Vim 复制到剪贴板？yy只将内容复制到 Vim 的内部缓冲区。我想复制到操作系统的剪贴板。Vim 中是否有这样的命令，或者您只能在 Vim 中提取内容？



解答

寄存器*将执行此操作。在 Windows 中，+和*是等价的。+在 unix 中，和之间存在细微差别*：

在 Windows 下， 和 + 寄存器是等效的。但是，对于 X11 系统，它们是不同的。对于 X11 系统， 是选择，+ 是剪切缓冲区（如剪贴板）。http://vim.wikia.com/wiki/Accessing_the_system_clipboard

*可能是您大多数时候想要的，所以我使用*它是因为它在两种环境中都像我期望的那样工作。

在 Linux 发行版中，您必须先安装vim-gtk（又名gvim）才能获得剪贴板功能。这是因为非 gtk vim 通常在没有 X11 支持的情况下编译。这是为了允许它在仅控制台机器（通常是服务器）上运行。

对于那些对在 yanking 或 put 时如何使用寄存器感到困惑的人，您只需写下"寄存器的名称。因此，要将某些内容复制到剪贴板，请注册您键入的"*y内容，然后再输入内容"*p（来源：Kyle Mathews）


更多建议

在 Mac 操作系统上

  • 复制所选部分：目视选择文本（键入v或V在正常模式下）并键入:w !pbcopy

  • 复制整个文件:%w !pbcopy

  • 从剪贴板粘贴:r !pbpaste

在 大多数 Linux 发行版上，您可以替代：

  • pbcopy以上与xclip -i -sel c或xsel -i -b
  • pbpaste使用xclip -o -sel -c或xsel -o -b
    —— 注意： 如果这些工具（xsel和xclip）都没有预装在你的发行版上，你可能可以在 repos 中找到它们
