HTTPS 标头是否加密？

Arya

2023-02-17 15:21

通过 HTTPS 发送数据时，我知道内容已加密，但我听到关于标头是否已加密或标头有多少已加密的混合答案。

有多少 HTTPS 标头 是 加密的？

包括GET/POST请求URL、Cookies等。



解答

所有 HTTP 标头都经过加密†。这就是为什么虚拟主机上的 SSL 不能很好地工作——你需要一个专用的 IP 地址，因为主机头是加密的。

†服务器名称标识 (SNI) 标准意味着如果您使用 TLS，主机名可能不会被加密。此外，无论您是否使用 SNI，TCP 和 IP 标头都不会加密。（如果是，您的数据包将无法路由。）


更多建议

标题是完全加密的。通过网络“明文”传输的唯一信息与 SSL 设置和 D/H 密钥交换有关。这种交换经过精心设计，不会向窃听者提供任何有用的信息，一旦发生，所有数据都会被加密。
