在 JavaScript 中获取客户端的时区（和偏移量）

Arya

2023-02-17 09:20

如何收集访客的时区信息？

我需要两者：

  1. 时区（例如，欧洲/伦敦）
  2. 以及与 UTC 或 GMT 的偏移量（例如 UTC+01）



解答

使用getTimezoneOffset()

您可以像这样以分钟为单位获得时区 偏移量：

var offset = new Date().getTimezoneOffset();
console.log(offset);
// if offset equals -60 then the time zone offset is UTC+01

运行代码片段隐藏结果

时区偏移量是 UTC 和本地时间之间的差异（以分钟为单位）。请注意，这意味着如果本地时区落后于 UTC，则偏移量为正，如果领先于 UTC，则偏移量为负。例如，如果您的时区是 UTC+10（澳大利亚东部标准时间），则将返回 -600。即使对于给定的语言环境，夏令时也会阻止此值成为常量

  • Mozilla 日期对象参考

请注意，并非所有时区都按整小时偏移：例如，纽芬兰是 UTC 负 3 小时 30 米（不考虑夏令时）。

另请注意，这只会为您提供时区偏移量（例如：UTC+01），不会为您提供时区（例如：欧洲/伦敦）。
