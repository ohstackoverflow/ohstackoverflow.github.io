如何将 String 转换为 Int？

Arya

2023-02-17 13:26

我有一个TextBoxD1.Text，我想将它转换为一个int以将其存储在数据库中。

我怎样才能做到这一点？



解答

尝试这个：

int x = Int32.Parse(TextBoxD1.Text);

或者更好：

int x = 0;

Int32.TryParse(TextBoxD1.Text, out x);

此外，由于Int32.TryParse返回 abool你可以使用它的返回值来决定解析尝试的结果：

int x = 0;

if (Int32.TryParse(TextBoxD1.Text, out x))
{
    // you know that the parsing attempt
    // was successful
}

Parse如果你很好奇，最好将和之间的区别TryParse总结如下：

TryParse 方法类似于 Parse 方法，只是 TryParse 方法在转换失败时不会抛出异常。它消除了在 s 无效且无法成功解析的情况下使用异常处理来测试 FormatException 的需要。-MSDN _


更多建议

Convert.ToInt32( TextBoxD1.Text );

如果您确信文本框的内容是有效的，请使用此选项int。一个更安全的选择是

int val = 0;
Int32.TryParse( TextBoxD1.Text, out val );

这将为您提供一些您可以使用的默认值。Int32.TryParse还返回一个布尔值，指示它是否能够解析，因此您甚至可以将其用作语句的条件if。

if( Int32.TryParse( TextBoxD1.Text, out val ){
  DoSomething(..);
} else {
  HandleBadInput(..);
}
