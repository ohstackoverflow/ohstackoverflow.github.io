使用 Git，显示一个分支中的所有提交，但不显示其他分支中的所有提交

Arya

2023-02-18 03:18

我有一个旧分支，我想删除它。但是，在这样做之前，我想检查对该分支所做的所有提交是否在某个时候合并到其他分支中。因此，我希望看到对我当前分支所做的所有提交尚未应用于任何其他分支[或者，如果没有一些脚本就不可能做到这一点，那么如何查看一个分支中尚未应用的所有提交到另一个给定的分支？]。



解答

你可能只是想要

git branch --contains branch-to-delete

这将列出所有包含来自“branch-to-delete”的提交的分支。如果它报告的不仅仅是“branch-to-delete”，则该分支已被合并。

您的替代方案实际上只是 rev-list 语法。例如git log one-branch..another-branch显示一切one- branch需要拥有的一切another-branch。

git show-branch您可能还对查看位置的方式感兴趣。
