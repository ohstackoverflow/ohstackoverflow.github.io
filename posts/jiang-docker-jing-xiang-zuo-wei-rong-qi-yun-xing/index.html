将 Docker 镜像作为容器运行

Arya

2023-02-17 17:01

从 a 构建 Docker 镜像后dockerfile，我看到镜像已成功构建，但我该如何处理它呢？我不应该能够将它作为容器运行吗？



解答

运行它的具体方式取决于你是否给图像一个标签/名称。

$ docker images
REPOSITORY          TAG                 ID                  CREATED             SIZE
ubuntu              12.04               8dbd9e392a96        4 months ago        131.5 MB (virtual 131.5 MB)

使用名称（让我们使用 Ubuntu ）：

$ docker run -i -t ubuntu:12.04 /bin/bash

没有名称，仅使用 ID：

$ docker run -i -t 8dbd9e392a96 /bin/bash

有关详细信息，请参阅 Docker 运行参考。


更多建议

执行以下步骤：

  1. $ docker images

您将获得带有指定标签的所有本地 Docker 映像的列表。

  1. $ docker run image_name:tag_name

如果您没有指定，tag_name它将自动运行带有“最新”标签的图像。

除了image_name，您还可以指定图像 ID（无 tag_name）。
