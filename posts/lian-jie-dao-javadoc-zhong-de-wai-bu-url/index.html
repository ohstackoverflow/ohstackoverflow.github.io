链接到 Javadoc 中的外部 URL？

Arya

2023-02-17 10:27

就像是：

/**
 * See {@linktourl http://google.com}
 */



解答

这将创建一个包含链接的“另见”标题，即：

/**
 * @see <a href="http://google.com">http://google.com</a>
 */

将呈现为：

另请参阅： http:
//google.com

而这个：

/**
 * See <a href="http://google.com">http://google.com</a>
 */

将创建一个内嵌链接：

请参阅http://google.com

取自javadoc 规范

@see <a href="URL#value">label</a>: 添加由 定义的链接URL#value。是URL#value相对或绝对 URL。Javadoc 工具通过查找小于号 ( <) 作为第一个字符来将这种情况与其他情况区分开来。

例如 ：@see <a href="http://www.google.com">Google</a>
