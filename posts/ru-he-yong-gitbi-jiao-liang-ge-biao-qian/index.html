如何用git比较两个标签？

Arya

2023-02-18 04:55

我想在两个标签之间做一个差异，并在这两个标签之间提交更改。你能告诉我命令吗？



解答

$ git diff tag1 tag2

或显示它们之间的日志：

$ git log tag1..tag2

有时只查看已更改的文件列表可能会很方便：

$ git diff tag1 tag2 --stat

然后查看某些特定文件的差异：

$ git diff tag1 tag2 -- some/file/name

标签只是对“该标签上”的最新提交的引用，因此您正在对它们之间的提交进行比较。

（一定要先做git pull --tags）

另外，一个很好的参考：https://git-scm.com/docs/git-diff


更多建议

如果源代码在 Github 上，您可以使用他们的比较工具： https: //help.github.com/articles/comparing- commits-across-time/
