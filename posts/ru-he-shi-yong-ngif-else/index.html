如何使用“*ngIf else”？

Arya

2023-02-17 09:12

*ngIf else我正在使用 Angular，我想在此示例中使用（自版本 4 起可用）：

<div *ngIf="isValid">
  content here ...
</div>

<div *ngIf="!isValid">
 other content here...
</div>

我怎样才能实现相同的行为ngIf else？



解答

角度 4 和 5 ：

使用else：

<div *ngIf="isValid;else other_content">
    content here ...
</div>

<ng-template #other_content>other content here...</ng-template>

您还可以使用then else：

<div *ngIf="isValid;then content else other_content">here is ignored</div>
<ng-template #content>content here...</ng-template>
<ng-template #other_content>other content here...</ng-template>

或者then单独：

<div *ngIf="isValid;then content"></div>
<ng-template #content>content here...</ng-template>

细节：

<ng-template>: 是 Angular 自己根据 MDN<template>实现的标签：

HTML<template>元素是一种用于保存客户端内容的机制，这些内容不会在页面加载时呈现，但随后可能会在运行时使用 JavaScript 实例化。


更多建议

在 Angular 4.xx 中

您可以通过四种方式使用ngIf来实现简单的 if - else 过程：

  1. 只需使用 If

    <div *ngIf="isValid">
If isValid is true

  2. 将If 与 Else 一起 使用（请注意 templateName ）

    <div *ngIf="isValid; else templateName">
If isValid is true

    If isValid is false

  3. 使用 If with Then （请注意 templateName ）

    <div *ngIf="isValid; then templateName">
Here is never showing

    If isValid is true

  4. 将If 与 Then 和 Else 一起 使用 ****

    <div *ngIf="isValid; then thenTemplateName else elseTemplateName">
Here is never showing

    If isValid is true

    If isValid is false

提示： ngIf 计算 表达式 ，然后在表达式分别为 truthy 或 falsy 时渲染 then 或 else模板。

通常是：

  • 那么template 是 ngIf 的内联模板，除非绑定到不同的值。
  • 否则 模板是空白的，除非它被绑定。
