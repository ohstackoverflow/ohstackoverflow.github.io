Java中如何将数组转换为集合

Arya

2023-02-17 12:54

我想用 Java 将数组转换为 Set。有一些明显的方法可以做到这一点（即使用循环），但我想要一些更简洁的方法，例如：

java.util.Arrays.asList(Object[] a);

有任何想法吗？



解答

像这样：

Set<T> mySet = new HashSet<>(Arrays.asList(someArray));

在 Java 9+ 中，如果 unmodifiable set 没问题：

Set<T> mySet = Set.of(someArray);

在 Java 10+ 中，泛型类型参数可以从数组组件类型中推断出来：

var mySet = Set.of(someArray);

当心

Set.of 抛出 IllegalArgumentException - 如果 someArray 中有任何重复元素。查看更多详细信息：https://docs.oracle.com/en/java/javase/11/docs/api/java.base/java/util/Set.html#of(E...)


更多建议

Set<T> mySet = new HashSet<T>();
Collections.addAll(mySet, myArray);

这是JDK 6 中的Collections.addAll(java.util.Collection, T...) 。

另外： 如果我们的数组充满了原语怎么办？

对于 JDK < 8，我只会编写一个明显的for循环来一次完成包装和添加到集合。

对于 JDK >= 8，一个有吸引力的选择是这样的：

Arrays.stream(intArray).boxed().collect(Collectors.toSet());
