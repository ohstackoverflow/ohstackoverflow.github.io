在 JavaScript 中将数字转换为字符串的最佳方法是什么？

Arya

2023-02-17 16:17

将数字转换为字符串的“最佳”方法是什么（在速度优势、清晰度优势、内存优势等方面）？

一些例子：

  1. String(n)

  2. n.toString()

  3. ""+n

  4. n+""



解答

像这样：

var foo = 45;
var bar = '' + foo;

实际上，尽管我通常为了简单的方便而这样做，但超过 1,000 次的迭代 对于原始速度来说似乎有一个优势.toString()

请在此处查看性能测试（不是我自己做的，而是我自己写的时候发现的）：http://jsben.ch/#/ghQYR

最快基于上面的 JSPerf 测试：str = num.toString();

应该注意的是，当您认为它可以 在 0.1 秒内以任何方式进行 100 万次 转换时，速度差异并不过分。

更新： 速度似乎因浏览器而异。基于此测试，在 Chrome 中num + ''似乎是最快的http://jsben.ch/#/ghQYR

更新 2： 再次基于我上面的测试，应该注意到 Firefox 20.0.1 的执行速度.toString()比'' + num示例慢大约 100 倍。


更多建议

在我看来，n.toString()它因其清晰度而获奖，而且我认为它不会带来任何额外的开销。
