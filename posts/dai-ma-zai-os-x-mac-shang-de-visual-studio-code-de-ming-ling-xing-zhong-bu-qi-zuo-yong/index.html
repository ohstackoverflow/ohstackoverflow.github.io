“代码 。” 在 OS X/Mac 上的 Visual Studio Code 的命令行中不起作用

Arya

2023-02-17 06:47

该命令在本手册code .中不起作用。

之前的所有其他步骤都有效。如何在 OS X 终端中调用 Visual Studio Code？

pwd

/用户/mona/nodejs/myExpressApp

code .

-bash：代码：找不到命令

我最终通过更改工作区从 Visual Code Studio 中打开它，但为什么该命令code .部分不起作用？



解答

1. 确保将 Visual Studio Code 应用程序拖到 应用程序 文件夹中

否则（如评论中所述），您必须在重新启动后再次执行此过程。

2.接下来，打开Visual Studio Code

通过打开 命令面板⌘``⇧``P并键入shell command以查找 Shell 命令：

在“在 PATH 中安装‘代码’命令”命令之前，使用 PATH 命令中的卸载‘代码’命令。

执行命令后，重启终端使新的$PATH值生效。您只需键入“代码”即可。在任何文件夹中开始编辑该文件夹中的文件。这 ”。” 简单的意思是“当前目录”

（来源：Visual Studio Code 文档）

注意：如果您正在运行基于 OSS存储库的构建，则需要运行code-oss .（Dzeimsas Zvirblis 的评论）


更多建议

如果你想永久添加它：

如果您运行的是macOS v10.15 (Catalina) 或更高版本，请将其添加到您的~/.bash_profile, 或添加到。~/.zshrc

export PATH="$PATH:/Applications/Visual Studio Code.app/Contents/Resources/app/bin"
