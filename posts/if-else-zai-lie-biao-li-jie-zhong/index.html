if/else 在列表理解中

Arya

2023-02-17 01:32

xs我有一个包含字符串和值混合的列表None。我如何使用列表推导式在每个字符串上调用函数，但将值转换None为''（而不是将它们传递给函数）？

我试过：

[f(x) for x in xs if x is not None else '']

但它给出了一个SyntaxError。正确的语法是什么？

如果您尝试创建一个基于条件省略值 的 列表推导，请参阅带条件的列表推导。 __

如果您需要考虑两个以上的条件结果，请注意 Python 的条件表达式不支持elif. 相反，有必要嵌套if/else条件。有关详细信息，请参阅列表理解条件中的 elif。



解答

你完全可以做到。这只是一个订购问题：

[f(x) if x is not None else '' for x in xs]

一般来说，

[f(x) if condition else g(x) for x in sequence]

并且，对于if仅具有条件的列表理解，

[f(x) for x in sequence if condition]

请注意，这实际上使用了不同的语言构造，即条件表达式，它本身不是理解语法的一部分，而if后面的for…in是列表理解的一部分，用于从源可迭代中 过滤元素。

条件表达式可用于您希望根据某些条件在两个表达式值之间进行选择的各种情况。这与存在于其他语言中的三元运算符的?:作用相同。例如：

value = 123
print(value, 'is', 'even' if value % 2 == 0 else 'odd')

基础知识

if/else列表理解中的语句涉及两件事：

  • 列表理解
  • 条件表达式（三元运算符）

1. 列表理解

它们提供了一种创建列表的简洁方法。

它的结构包括：“ 括号包含一个表达式，后跟一个 for 子句，然后是零个或多个 for 或 if 子句 ”。

情况1

这里我们没有条件。iterable 中的每个项目都添加到new_list.

new_list = [expression for item in iterable]
new_list = [x for x in range(1, 10)]
> [1, 2, 3, 4, 5, 6, 7, 8, 9]

案例二

这里我们有一个条件。

示例 1

条件：只有 偶数 会被添加到new_list.

new_list = [expression for item in iterable if condition == True]
new_list = [x for x in range(1, 10) if x % 2 == 0]
> [2, 4, 6, 8]

示例 2

条件：只有3 的倍数的 偶数 才会被加到new_list.

new_list = [expression for item in iterable if condition == True]
new_list = [x for x in range(1, 10) if x % 2 == 0 if x % 3 == 0]
> [6]

但是如果我们使用两个ifin怎么会有一个条件new_list呢？

先验表达式可以写成：

new_list = [x for x in range(1, 10) if x % 2 and x % 3 == 0]
> [6]

我们只使用一个if语句。

这就像做：

new_list = []
for x in range(1, 10):
    if x % 2 == 0 and x % 3 == 0:
        new_list.append(x)
> [6]

示例 3

只是为了争论，你也可以使用or.

条件：将 偶数 或3的倍数加到new_list。

new_list = [x for x in range(1, 10) if x % 2 == 0 or x % 3 == 0]
> [2, 3, 4, 6, 8, 9]

案例三

不止一种情况：

这里我们需要条件表达式（三元运算符）的帮助。

2.条件表达式

什么是条件表达式？顾名思义：具有某些条件的 Python 表达式。

<Exp1> if condition else <Exp2>

首先condition评估。如果condition是True，则<Exp1>评估并返回。如果condition是False，则<Exp2>评估并返回。

具有多个条件的条件表达式：

<Exp1> if condition else <Exp2> if condition else <Exp3>...

来自Real Python的示例：

age = 12
s = 'minor' if age < 21 else 'adult'
> minor

的价值s以age价值为条件。

3.List Comprehensions with Conditionals

我们像这样将列表理解和条件放在一起。

new_list = [<Conditional Expression> for <item> in <iterable>]

new_list = [<Exp1> if condition else <Exp2> if condition else <Exp3> for <item> in <iterable>]

条件： 偶数 加为'even'，数字三加为'number three'，其余加为'odd'。

new_list = ['even' if x % 2 == 0 else 'number three' if x == 3 else 'odd' 
             for x in range(1, 10)]
> ['odd', 'even', 'number three', 'even', 'odd', 'even', 'odd', 'even', 'odd']

问题的答案

[f(x) for x in xs if x is not None else '']

这里我们遇到了列表结构的问题：for x in xs应该在表达式的末尾。

正确方法：

[f(x) if x is not None else '' for x in xs]
