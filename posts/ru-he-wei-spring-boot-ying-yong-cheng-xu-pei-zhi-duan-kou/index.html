如何为 Spring Boot 应用程序配置端口

Arya

2023-02-17 07:29

如何配置 Spring Boot 应用程序侦听的 TCP/IP 端口，使其不使用默认端口 8080。



解答

正如文档中所说，要么使用 jvm 的命令行选项设置server.port为系统属性，-Dserver.port=8090要么application.properties添加/src/main/resources/

server.port=8090

对于随机端口使用：

server.port=0

同样application.yml添加/src/main/resources/：

server:
  port: 8090


更多建议

在 Spring Boot 应用程序中更改嵌入式 Tomcat 中的端口有两种主要方法。

修改application.properties

首先你可以试试 /resources 文件夹下的 application.properties 文件：

server.port = 8090

修改虚拟机选项

第二种方式，如果你想避免修改任何文件并签入你只在本地需要的东西，你可以使用 vm arg：

转到运行 - > 编辑配置 -> VM 选项

-Dserver.port=8090

此外，如果您需要更多信息，可以在此处查看以下博客文章：Changing the port on a Spring Boot Application
