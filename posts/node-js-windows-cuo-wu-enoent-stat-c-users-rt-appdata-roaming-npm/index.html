Node.js/Windows 错误：ENOENT，stat 'C:\Users\RT\AppData\Roaming\npm'

Arya

2023-02-17 07:29

我有 Windows 7 32 位。我安装了最新的Node.js 32 位。
当我尝试运行命令时npm install jquery，我收到错误：

错误：ENOENT，stat 'C:\Users\RT\AppData\Roaming\npm

如何解决？



解答

在显示的路径中手动创建一个名为“npm”的文件夹解决了这个问题。

可以在故障排除页面上找到更多信息


更多建议

我在通过 npm 安装包时遇到了同样的问题。

npm在该特定错误中手动创建文件夹后C:\Users\UserName\AppData\Roaming\消失了，但它给出了类似的多个错误，因为它试图在文件夹中创建其他目录npm但失败了。 以管理员身份运行命令提示符 后问题得到解决。
