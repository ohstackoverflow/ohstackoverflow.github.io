在避免使用循环的同时用另一个字典扩展字典的最佳方法是什么for？例如：

>>> a = { "a" : 1, "b" : 2 }
>>> b = { "c" : 3, "d" : 4 }
>>> a
{'a': 1, 'b': 2}
>>> b
{'c': 3, 'd': 4}

结果：

{ "a" : 1, "b" : 2, "c" : 3, "d" : 4 }

就像是：

a.extend(b)  # This does not work



解答

a.update(b)

更多建议

这个封闭问题中的一颗美丽的宝石：

“oneliner way”，既不改变输入口述，是

basket = dict(basket_one, **basket_two)

在此处了解**basket_two(the **) 的含义。

如果发生冲突，来自的项目basket_two将覆盖来自的项目basket_one。随着一行行的发展，这是非常可读和透明的，并且我不会在任何时候使用它，因为它是两个其他混合的字典派上用场（任何难以理解它的读者实际上会得到很好的服务这促使他或她学习dict和**形式的方式;-)。因此，例如，使用如下：

x = mungesomedict(dict(adict, **anotherdict))

在我的代码中经常出现。

最初由Alex Martelli提交

注意： 在 Python 3 中，只有当 basket_two 中的每个键都是string.
