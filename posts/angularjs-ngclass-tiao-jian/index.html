AngularJS ngClass 条件

Arya

2023-02-18 06:37

有没有什么办法可以表达像ng-class条件一样的东西？

例如，我尝试了以下操作：

<span ng-class="{test: 'obj.value1 == \'someothervalue\''}">test</span>

这段代码的问题在于，无论是什么obj.value1，类测试总是应用于元素。这样做：

<span ng-class="{test: obj.value2}">test</span>

只要obj.value2不等于真值，就不会应用该类。现在我可以通过这样做来解决第一个示例中的问题：

<span ng-class="{test: checkValue1()}">test</span>

函数checkValue1看起来像这样：

$scope.checkValue1 = function() {
  return $scope.obj.value === 'somevalue';
}

我只是想知道这是否应该ng- class工作。我还在构建一个自定义指令，我想在其中执行与此类似的操作。但是，我找不到观看表达式的方法（也许这是不可能的，也是它这样工作的原因）。

这是一个plnkr来说明我的意思。



解答

您的第一次尝试几乎是正确的，它应该在没有引号的情况下工作。

{test: obj.value1 == 'someothervalue'}

这是一个plnkr。

ngClass 指令适用于任何求值为 truthy 或 falsey 的表达式，有点类似于 Javascript 表达式，但有一些区别，您可以在此处阅读相关信息。如果您的条件太复杂，那么您可以使用返回 truthy 或 falsey 的函数，就像您在第三次尝试中所做的那样。

补充一下：您还可以使用逻辑运算符来形成逻辑表达式，例如

ng-class="{'test': obj.value1 == 'someothervalue' || obj.value2 == 'somethingelse'}"


更多建议

在 ng-repeat 中使用 ng-class

<table>
    <tbody>
            <tr ng-repeat="task in todos"
                ng-class="{'warning': task.status == 'Hold' , 'success': task.status == 'Completed',
              'active': task.status == 'Started', 'danger': task.status == 'Pending' } ">
                <td>{{$index + 1}}</td>
                <td>{{task.name}}</td>
                <td>{{task.date|date:'yyyy-MM-dd'}}</td>
                <td>{{task.status}}</td>
            </tr>
    </tbody>
</table>

对于 task.status 中的每个状态，该行使用不同的类。
