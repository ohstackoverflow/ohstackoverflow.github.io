如何分析 Python 脚本？

Arya

2023-02-17 01:05

Project Euler和其他编码竞赛通常有最长运行时间，或者人们吹嘘他们的特定解决方案运行速度有多快。使用 Python，有时这些方法有些笨拙——即，将计时代码添加到__main__.

分析 Python 程序运行时间的好方法是什么？



解答

Python 包含一个名为cProfile的分析器。它不仅给出了总运行时间，而且还分别对每个函数进行了计时，并告诉您每个函数被调用了多少次，从而很容易确定您应该在哪里进行优化。

您可以从您的代码中或从解释器中调用它，如下所示：

import cProfile
cProfile.run('foo()')

更有用的是，您可以在运行脚本时调用 cProfile：

python -m cProfile myscript.py

为了让它更容易，我制作了一个名为“profile.bat”的小批处理文件：

python -m cProfile %1

所以我所要做的就是运行：

profile euler048.py

更多建议

不久前我做了pycallgraph一个从你的 Python 代码生成可视化的东西。 编辑： 我已经更新示例以使用 3.3，这是撰写本文时的最新版本。

在pip install pycallgraph安装GraphViz之后，您可以从命令行运行它：

pycallgraph graphviz -- ./mypythonscript.py

或者，您可以分析代码的特定部分：

from pycallgraph import PyCallGraph
from pycallgraph.output import GraphvizOutput

with PyCallGraph(output=GraphvizOutput()):
    code_to_profile()
