如何在 Node.js 中卸载 npm 模块？

Arya

2023-02-17 02:46

众所周知，任何 npm 模块都可以通过运行一个简单的命令来安装：npm install <module_name>：

我已经安装了一些我不再使用的模块，我只想将它们关闭。我对此有几个问题：

  • 我们是否有任何命令或进程从根目录中卸载模块（如npm uninstall <module_name>），或者只是删除模块文件？

  • 如果保留未使用的模块，对我们有何影响？



解答

命令很简单npm uninstall <name>

Node.js 文档https://npmjs.org/doc/包含您需要了解的使用 npm 的所有命令。

本地安装将在node_modules/您的应用程序目录中。如果模块保留在那里而没有对它的引用，这不会影响应用程序。

但是，如果您要删除一个全局包，任何引用它的应用程序都会崩溃。

这里有不同的选择：

npm uninstall <name>从中删除模块node_modules但不更新package.json

npm uninstall <name> --save也将其从中dependencies删除package.json

npm uninstall <name> --save-dev也将其从中devDependencies删除package.json

npm uninstall -g <name> --save也将其全局删除


更多建议

如果它不起作用，npm uninstall <module_name>请通过键入全局尝试-g。

也许您只需要以超级用户/管理员的身份使用sudo npm uninstall <module_name>.
