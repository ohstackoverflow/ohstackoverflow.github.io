对象字面量/初始值设定项中的自引用

Arya

2023-02-17 10:09

有什么方法可以让类似下面的东西在 JavaScript 中工作吗？

var foo = {
    a: 5,
    b: 6,
    c: this.a + this.b  // Doesn't work
};

在当前形式中，这段代码显然会引发引用错误，因为它this没有引用foo. 但是 有 什么方法可以让对象字面量的属性值依赖于之前声明的其他属性吗？



解答

好吧，我唯一能告诉你的就是吸气剂：

var foo = {
  a: 5,
  b: 6,
  get c() {
    return this.a + this.b;
  }
}

console.log(foo.c) // 11

这是 ECMAScript 第 5 版规范引入的语法扩展，大多数现代浏览器（包括 IE9）都支持该语法。


更多建议

你可以这样做：

var foo = {
   a: 5,
   b: 6,
   init: function() {
       this.c = this.a + this.b;
       return this;
   }
}.init();

这将是对象的某种一次性初始化。

init()请注意，您实际上是将的返回值分配给foo，因此您必须return this。
