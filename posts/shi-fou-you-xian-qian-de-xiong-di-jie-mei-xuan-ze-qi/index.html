是否有“先前的兄弟姐妹”选择器？

Arya

2023-02-17 00:25

加号选择器 ( +) 用于选择下一个相邻的兄弟。

是否有前一个兄弟姐妹的等价物？



解答

不，没有“前一个兄弟”选择器。

在相关说明中，~是用于一般后继兄弟（意味着该元素出现在该元素之后，但不一定紧随其后）并且是一个 CSS3 选择器。+是给下一个兄弟姐妹的，是CSS2.1。

请参阅Selectors Level 3中的Adjacent sibling combinator和5.7 Adjacent sibling selectors from Cascading Style Sheets Level 2 Revision 1 (CSS 2.1) Specification。


更多建议

我找到了一种方法来设置所有以前的兄弟姐妹（与 相反~）的样式，这些兄弟姐妹可能会根据您的需要工作。

假设您有一个链接列表，当鼠标悬停在一个链接上时，所有之前的链接都应该变成红色。你可以这样做：

/* default link color is blue */
.parent a {
  color: blue;
}

/* prev siblings should be red */
.parent:hover a {
  color: red;
}
.parent a:hover,
.parent a:hover ~ a {
  color: blue;
}


<div class="parent">
  <a href="#">link</a>
  <a href="#">link</a>
  <a href="#">link</a>
  <a href="#">link</a>
  <a href="#">link</a>
</div>
