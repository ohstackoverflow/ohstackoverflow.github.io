如何删除使用 .css() 函数添加的样式？

Arya

2023-02-17 08:53

我正在用 jQuery 更改 CSS ，我希望删除我根据输入值添加的样式： ****

 if(color != '000000') $("body").css("background-color", color); else // remove style ?

我怎样才能做到这一点？
请注意，只要使用颜色选择器选择颜色（即当鼠标在色轮上移动时），上面的行就会运行。

第二个注意事项：我不能这样做，因为它会从 CSScss("background-color", "none")文件中删除默认样式。我只想删除 jQuery 添加的内联样式。 *
background-color *



解答

将属性更改为空字符串似乎可以完成这项工作：

$.css("background-color", "");


更多建议

接受的答案有效，但style在我的测试中在 DOM 上留下了一个空属性。没什么大不了的，但这一切都消除了：

removeAttr( 'style' );

这假定您要删除所有动态样式并返回到样式表样式。
