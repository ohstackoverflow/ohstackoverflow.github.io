在 CSS 中按属性选择元素

Arya

2023-02-18 05:43

是否可以通过 HTML5 数据属性（例如data-role）来选择 CSS 中的元素？



解答

如果你的意思是使用attribute selector，当然，为什么不呢：

[data-role="page"] {
    /* Styles */
}

有多种属性选择器可用于各种场景，这些都包含在我链接到的文档中。请注意，尽管自定义数据属性是“新的 HTML5 功能”，

  • 浏览器通常支持非标准属性没有任何问题，因此您应该能够使用属性选择器过滤它们；和

  • 您也不必担心 CSS 验证，因为 CSS 不关心非命名空间属性名称，只要它们不破坏选择器语法即可。


更多建议

在现代浏览器中，也可以选择不考虑其内容的属性

和：

[data-my-attribute] {
   /* Styles */
}

[anything] {
   /* Styles */
}

例如： http: //codepen.io/jasonm23/pen/fADnu

适用于非常大比例的浏览器。

请注意，这也可以用在 JQuery 选择器中，或者使用document.querySelector
