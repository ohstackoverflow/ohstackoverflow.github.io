在 .NET 中以换行符拆分字符串

Arya

2023-02-17 08:57

我需要在 .NET 中将字符串拆分为换行符，我知道拆分字符串的唯一方法是使用Split方法。但是，这不允许我（轻松地）在换行符上拆分，那么最好的方法是什么？



解答

要拆分字符串，您需要使用采用字符串数组的重载：

string[] lines = theText.Split(
    new string[] { Environment.NewLine },
    StringSplitOptions.None
);

编辑：
如果你想处理文本中不同类型的换行符，你可以使用匹配多个字符串的能力。这将在任何一种类型的换行符上正确拆分，并保留文本中的空行和间距：

string[] lines = theText.Split(
    new string[] { "\r\n", "\r", "\n" },
    StringSplitOptions.None
);

使用一个怎么样StringReader？

using (System.IO.StringReader reader = new System.IO.StringReader(input)) {
    string line = reader.ReadLine();
