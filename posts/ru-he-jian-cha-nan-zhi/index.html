如何检查 NaN 值？

Arya

2023-02-17 01:44

float('nan')代表 NaN（不是数字）。但是我该如何检查呢？



解答

使用math.isnan：

>>> import math
>>> x = float('nan')
>>> math.isnan(x)
True


更多建议

测试 NaN 的通常方法是看它是否等于它自己：

def isNaN(num):
    return num != num
