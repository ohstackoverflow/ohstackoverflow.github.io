自动或使用快捷方式删除尾随空格

Arya

2023-02-17 09:37

有没有办法删除 Visual Studio Code 中的尾随空格（自动或使用快捷方式）？

我一直在命令面板和编辑器设置中寻找它，但找不到我要找的东西。



解答

您可以在文件保存时 从设置中启用空格修剪：

  1. 打开 Visual Studio Code 用户设置（菜单 文件 → 首选项 → 设置 → 用户设置选项 卡）。
  2. 单击窗口右上角的图标。这将打开一个文档。
  3. 将新"files.trimTrailingWhitespace": true设置添加到用户设置文档（如果尚不存在）。这样您就不会直接编辑默认设置，而是添加它。
  4. 保存用户设置文件。

我们还添加了一个新命令来手动触发此操作（来自命令面板的 Trim Trailing Whitespace ）。


更多建议

菜单 代码 → 首选项 → 设置 →

检查 “修剪尾随空格” 选项 - “启用后，将在保存文件时修剪尾随空格” 。
