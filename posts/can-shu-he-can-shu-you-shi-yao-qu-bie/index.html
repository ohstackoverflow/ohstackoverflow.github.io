参数和参数有什么区别？

Arya

2023-02-17 08:12

当口头谈论方法时，我永远不确定是使用 参数 还是 参数 或其他词。无论哪种方式，其他人都知道我的意思，但什么是正确的，术语的历史是什么？

我是一名 C# 程序员，但我也想知道人们是否在不同的语言中使用不同的术语。

作为记录，我是在没有计算机科学背景的情况下自学成才的。（请不要告诉我阅读Code Complete，因为我问这个是为了其他还没有Steve McConnell的精彩书籍副本的人的利益。）

概括

普遍的共识似乎是在团队环境中可以互换使用这些术语。除非您正在定义精确的术语；那么你也可以使用“ 形式 参数/参数”和“ 实际 参数/参数”来消除歧义。



解答

参数是方法定义中的变量。调用方法时，参数是您传递给方法参数的数据。

public void MyMethod(string myParam) { }

...

string myArg1 = "this is my argument";
myClass.MyMethod(myArg1);


更多建议

参数 是函数声明中的变量。

参数 是传递给函数的此变量的实际值。
