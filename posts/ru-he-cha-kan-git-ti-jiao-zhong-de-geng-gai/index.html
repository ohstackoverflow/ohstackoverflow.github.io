如何查看 Git 提交中的更改？

Arya

2023-02-16 22:28

当我这样做时，git diff COMMIT我会看到该提交和 HEAD 之间的更改（据我所知），但我希望看到该单个提交所做的更改。

我还没有在diff/上找到任何明显的选项log可以给我那个输出。



解答

要查看特定哈希的差异COMMIT，COMMIT提交的哈希在哪里：

git diff COMMIT~ COMMIT``COMMIT将向您展示 that 's ancestor 和 the之间的区别COMMIT。请参阅git diff的手册页以了解有关命令的详细信息以及有关表示法及其朋友的gitrevisions 。~

或者，git show COMMIT将做一些非常相似的事情。（提交的数据，包括它的差异 - 但不用于合并提交。）请参阅git show 联机帮助页。

（还将向您展示它与头部git diff COMMIT之间的区别。）COMMIT


更多建议

正如“ git commit 与其父级差异的简写？ ”中提到的，您还可以使用git diffwith：

git diff COMMIT^!

或者

git diff-tree -p COMMIT

使用 git show，你需要（为了单独关注 diff）做：

git show --color --pretty=format:%b COMMIT

该COMMIT参数是一个 commit-ish ：

提交对象或可以递归取消引用到提交对象的对象。以下都是提交对象：一个提交对象，一个指向提交对象的标记对象，一个指向一个标记对象的标记对象，一个标记对象指向一个提交对象，等等。

请参阅gitrevision“SPECIFYING REVISIONS”以引用 commit-ish。
另请参阅“ Git 中的树状结构是什么意思？ ”。
