抛出异常时如何获取 JavaScript 堆栈跟踪？

Arya

2023-02-17 18:36

如果我自己抛出 JavaScript 异常（例如，throw "AArrggg"），我如何获取堆栈跟踪（在 Firebug 中或其他方式中）？现在我刚收到消息。

编辑 ：正如下面许多人发布的那样，可以获取 JavaScript 异常的堆栈跟踪，但 我 想获取异常的堆栈跟踪。例如：

function foo() {
    bar(2);
}
function bar(n) {
    if (n < 2)
        throw "Oh no! 'n' is too small!"
    bar(n-1);
}

调用时，我想获得一个堆栈跟踪，其中包括对, ,foo的调用。foo``bar``bar



解答

编辑 2（2017 年）：

在所有现代浏览器中，您可以简单地调用：console.trace(); （MDN 参考）

编辑 1（2013 年）：

正如对原始问题的评论中指出的那样，更好（更简单）的解决方案是使用对象stack的属性，Error如下所示：

function stackTrace() {
    var err = new Error();
    return err.stack;
}

这将生成如下输出：

DBX.Utils.stackTrace@http://localhost:49573/assets/js/scripts.js:44
DBX.Console.Debug@http://localhost:49573/assets/js/scripts.js:9
.success@http://localhost:49573/:462
x.Callbacks/c@http://localhost:49573/assets/js/jquery-1.10.2.min.js:4
x.Callbacks/p.fireWith@http://localhost:49573/assets/js/jquery-1.10.2.min.js:4
k@http://localhost:49573/assets/js/jquery-1.10.2.min.js:6
.send/r@http://localhost:49573/assets/js/jquery-1.10.2.min.js:6

提供调用函数的名称以及 URL、它的调用函数等。

原版（2009 年）：

此代码段的修改版本可能会有所帮助：

function stacktrace() { 
  function st2(f) {
    return !f ? [] : 
        st2(f.caller).concat([f.toString().split('(')[0].substring(9) + '(' + f.arguments.join(',') + ')']);
  }
  return st2(arguments.callee.caller);
}


更多建议

Chrome/Chromium 和其他使用 V8 的浏览器，以及 Firefox，有一个方便的接口来通过对象stack的属性获取堆栈跟踪Error：

    try {
        // Code throwing an exception
        throw new Error();
    } catch(e) {
        console.log(e.stack);
    }

请参阅V8 文档中的详细信息
