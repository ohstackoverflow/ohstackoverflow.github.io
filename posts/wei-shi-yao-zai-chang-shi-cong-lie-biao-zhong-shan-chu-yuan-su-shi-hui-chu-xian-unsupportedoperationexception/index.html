为什么在尝试从列表中删除元素时会出现 UnsupportedOperationException？

Arya

2023-02-18 01:00

我有这段代码：

public static String SelectRandomFromTemplate(String template,int count) {
   String[] split = template.split("|");
   List<String> list=Arrays.asList(split);
   Random r = new Random();
   while( list.size() > count ) {
      list.remove(r.nextInt(list.size()));
   }
   return StringUtils.join(list, ", ");
}

我明白了：

06-03 15:05:29.614: ERROR/AndroidRuntime(7737): java.lang.UnsupportedOperationException
06-03 15:05:29.614: ERROR/AndroidRuntime(7737):     at java.util.AbstractList.remove(AbstractList.java:645)

这怎么是正确的方法？Java.15



解答

您的代码有很多问题：

返回Arrays.asList固定大小的列表

从 API：

Arrays.asList：返回由指定数组支持的 固定大小列表。

你做不到add；你不能remove从它。您不能在结构上修改List.

使固定

创建一个LinkedList，它支持更快remove。

List<String> list = new LinkedList<String>(Arrays.asList(split));

关于split正则表达式

从 API：

String.split(String regex): 围绕给定正则表达式的匹配拆分此字符串。

|是正则表达式元字符；如果要拆分文字|，则必须将其转义\|为 Java 字符串文字"\\|"。

使固定：

template.split("\\|")

关于更好的算法

与其remove使用随机索引一次调用一个，不如在范围内生成足够多的随机数，然后使用Lista 遍历一次listIterator()，remove()在适当的索引处调用。stackoverflow 上有关于如何在给定范围内生成随机但不同的数字的问题。

有了这个，你的算法将是O(N)。


更多建议

这个已经烧了我很多次了。Arrays.asList创建一个不可修改的列表。来自 Javadoc：返回由指定数组支持的 固定大小列表。

创建一个具有相同内容的新列表：

newList.addAll(Arrays.asList(newArray));

这会产生一些额外的垃圾，但您可以对其进行变异。
