在文件中查找重复行并计算每行重复了多少次？

Arya

2023-02-17 18:50

假设我有一个类似如下的文件：

123 
123 
234 
234 
123 
345

我想知道“123”被重复了多少次，“234”被重复了多少次等等。理想情况下，输出应该是这样的：

123  3 
234  2 
345  1



解答

假设每行一个数字：

sort <file> | uniq -c

--count您也可以在 GNU 版本中使用更详细的标志，例如，在 Linux 上：

sort <file> | uniq --count


更多建议

这将 仅打印重复行 ，计数：

sort FILE | uniq -cd

或者，使用 GNU 长选项（在 Linux 上）：

sort FILE | uniq --count --repeated

在BSD 和 OSX 上你必须使用 grep来过滤掉独特的行：

sort FILE | uniq -c | grep -v '^ *1 '

对于给定的示例，结果将是：

  3 123
  2 234

如果要 打印所有行的计数， 包括只出现一次的行：

sort FILE | uniq -c

或者，使用 GNU 长选项（在 Linux 上）：

sort FILE | uniq --count

对于给定的输入，输出是：

  3 123
  2 234
  1 345

为了将 出现频率最高的行排 在最前面，您可以执行以下操作（以获取所有结果）：

sort FILE | uniq -c | sort -nr

或者，要仅获取重复的行，首先是最常见的行：

sort FILE | uniq -cd | sort -nr

在 OSX 和 BSD 上，最后一个变为：

sort FILE | uniq -c | grep -v '^ *1 ' | sort -nr
