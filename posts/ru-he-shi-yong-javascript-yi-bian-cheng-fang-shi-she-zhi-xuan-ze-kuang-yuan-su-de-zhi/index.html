如何使用 JavaScript 以编程方式设置选择框元素的值？

Arya

2023-02-18 02:14

我有以下 HTML<select>元素：

<select id="leaveCode" name="leaveCode">
  <option value="10">Annual Leave</option>
  <option value="11">Medical Leave</option>
  <option value="14">Long Service</option>
  <option value="17">Leave Without Pay</option>
</select>

使用将数字作为参数的 JavaScript 函数leaveCode，如何在列表中选择适当的选项？



解答

您可以使用此功能：

function selectElement(id, valueToSelect) {    
    let element = document.getElementById(id);
    element.value = valueToSelect;
}

selectElement('leaveCode', '11');


<select id="leaveCode" name="leaveCode">
  <option value="10">Annual Leave</option>
  <option value="11">Medical Leave</option>
  <option value="14">Long Service</option>
  <option value="17">Leave Without Pay</option>
</select>

或者，如果您还想触发 onchange 事件，您可以使用：

element.dispatchEvent(new Event('change'))


更多建议

如果你使用 jQuery，你也可以这样做：

$('#leaveCode').val('14');

这将选择<option>值为 14 的 。

Document使用纯 Javascript，这也可以通过两种方法实现：

  • 使用document.querySelector，您可以根据 CSS 选择器选择元素：

    document.querySelector('#leaveCode').value = '14'

  • 将更成熟的方法与 一起使用document.getElementById()，正如函数名称所暗示的那样，您可以根据其选择一个元素id：

    document.getElementById('leaveCode').value = '14'
