如何使用子图更改图形大小？

Arya

2023-02-18 01:09

我如何增加这个数字的数字大小？这什么都不做：

fig.figsize(15, 15)



解答

在图形对象上使用它：

fig.set_figheight(15)
fig.set_figwidth(15)

或者，在使用.subplots()创建新图形时，指定figsize=：

fig, axs = plt.subplots(2, 2, figsize=(15, 15))


更多建议

除了前面的答案之外，这里还有一个选项可以通过以下方式分别设置图形的大小和图形中子图的大小gridspec_kw：

import matplotlib.pyplot as plt
import numpy as np
import pandas as pd

#generate random data
x,y=range(100), range(10)
z=np.random.random((len(x),len(y)))
Y=[z[i].sum() for i in range(len(x))]
z=pd.DataFrame(z).unstack().reset_index()

#Plot data
fig, axs = plt.subplots(2,1,figsize=(16,9), gridspec_kw={'height_ratios': [1, 2]})
axs[0].plot(Y)
axs[1].scatter(z['level_1'], z['level_0'],c=z[0])
