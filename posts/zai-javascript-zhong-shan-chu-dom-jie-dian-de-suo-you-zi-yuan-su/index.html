在 JavaScript 中删除 DOM 节点的所有子元素

Arya

2023-02-17 03:51

我将如何着手删除 JavaScript 中 DOM 节点的所有子元素？

假设我有以下（丑陋的）HTML：

<p id="foo">
    <span>hello</span>
    <div>world</div>
</p>

然后我像这样抓住我想要的节点：

var myNode = document.getElementById("foo");

foo我怎么能删除这样的孩子<p id="foo"></p>呢？

我可以这样做吗：

myNode.childNodes = new Array();

或者我应该使用一些组合removeElement？

我希望答案是直截了当的 DOM；如果您还提供了 jQuery 中的答案以及仅 DOM 的答案，则可以加分。



解答

2022+，使用replaceChildren()API​​！

现在可以使用（支持跨浏览器的）replaceChildrenAPI替换所有子项：

container.replaceChildren(...arrayOfNewChildren);

这将同时做：

  • 删除所有现有的孩子，和
  • 在一次操作中追加所有给定的新孩子。

您还可以使用相同的 API 只删除现有的子项，而不替换它们：

container.replaceChildren();

这在 Chrome/Edge 86+、Firefox 78+ 和 Safari 14+ 中得到完全支持。 这是完全指定的行为。这可能比这里提出的任何其他方法 都快 ，因为移除旧孩子和添加新孩子不需要innerHTML, 并且一步而不是多步完成。
