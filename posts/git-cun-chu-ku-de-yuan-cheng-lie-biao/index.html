Git 存储库的远程列表？

Arya

2023-02-17 15:53

我有一个 Git 存储库。这个存储库有多个远程存储库（我认为）。如何获取属于该存储库的远程存储库列表？

像git list --remotes或类似的东西？



解答

您可以使用命令获取任何配置的远程 URL 的列表git remote -v。

这将为您提供如下内容：

base    /home/***/htdocs/base (fetch)
base    /home/***/htdocs/base (push)
origin  git@bitbucket.org:*** (fetch)
origin  git@bitbucket.org:*** (push)


更多建议

如果您只需要远程存储库的名称（而不是任何其他数据），那么简单git remote就足够了。

$ git remote
iqandreas
octopress
origin
