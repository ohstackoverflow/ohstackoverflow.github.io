如何根据号码检测信用卡类型？

Arya

2023-02-18 02:11

我想弄清楚如何仅根据信用卡号来检测信用卡的类型。有谁知道找到这个的明确可靠的方法？



解答

信用卡/借记卡号称为 PAN 或 主帐号 。PAN 的前六位数字取自属于发行银行的 IIN 或 发行人标识号 （IIN 以前称为 BIN — 银行标识号 — 因此您可能会在某些文档中看到对该术语的引用）。这六位数字受国际标准ISO/IEC 7812的约束，可用于根据号码确定卡的类型。

不幸的是，实际的 ISO/IEC 7812 数据库不是公开的，但是，有非官方的列表，包括商业的和免费的，包括在维基百科上。

无论如何，要从数字中检测类型，您可以使用如下所示的正则表达式：Credit for original expressions

Visa： ^4[0-9]{6,}$ Visa 卡号以 4 开头。

MasterCard： ^5[1-5][0-9]{5,}|222[1-9][0-9]{3,}|22[3-9][0-9]{4,}|2[3-6][0-9]{5,}|27[01][0-9]{4,}|2720[0-9]{3,}$ 2016 年之前，MasterCard 编号以数字 51 到 55 开头， 但这只会检测 MasterCard 信用卡 ；还有其他使用万事达卡系统发行的卡不属于这个 IIN 范围。2016 年，他们将在 (222100-272099) 范围内添加数字。

美国运通： ^3[47][0-9]{5,}$美国运通卡号以 34 或 37 开头。

Diners Club： ^3(?:0[0-5]|[68][0-9])[0-9]{4,}$ Diners Club 卡号以 300 到 305、36 或 38 开头。有些 Diners Club 卡以 5 开头，有 16 位数字。这些是 Diners Club 和 MasterCard 的合资企业，应该像 MasterCard 一样处理。

Discover： ^6(?:011|5[0-9]{2})[0-9]{3,}$ Discover 卡号以 6011 或 65 开头。

JCB： ^(?:2131|1800|35[0-9]{3})[0-9]{3,}$ JCB 卡以 2131、1800 或 35 开头。

遗憾的是，有许多由 MasterCard 系统处理的卡片类型不在 MasterCard 的 IIN 范围内（编号从 51...55 开始）；最重要的案例是 Maestro 卡，其中许多是从其他银行的 IIN 范围发行的，因此遍布整个号码空间。因此， 最好假定您接受的不是其他类型的任何卡一定是 MasterCard 。

重要提示 ：卡号的长度确实有所不同；例如，Visa 过去曾发行过带有 13 位 PAN 和 16 位 PAN 的卡。Visa 的文档目前表明它可能会或可能已经发布了 12 到 19 位数字的号码。 因此，您不应该检查卡号的长度，除非验证它至少有 7 位数字（完整的 IIN 加上一个校验位，应该与 Luhn 算法预测的值匹配）。

一个进一步的提示： 在处理持卡人 PAN 之前，从输入中去除所有空格和标点符号 。为什么？因为按组输入数字通常要容易 得多 ，类似于它们在实际信用卡正面的显示方式，即

4444 4444 4444 4444

比正确输入更容易

4444444444444444

惩罚用户确实没有任何好处，因为他们在这里输入了您不希望看到的字符。

这也意味着确保您的输入字段 至少 有24 个字符的空间，否则输入空格的用户将用完空间。我建议您将字段设置得足够宽以显示 32 个字符，最多允许 64 个字符；这为扩展提供了充足的空间。

更多建议

在javascript中：

function detectCardType(number) {
    var re = {
        electron: /^(4026|417500|4405|4508|4844|4913|4917)\d+$/,
        maestro: /^(5018|5020|5038|5612|5893|6304|6759|6761|6762|6763|0604|6390)\d+$/,
        dankort: /^(5019)\d+$/,
        interpayment: /^(636)\d+$/,
        unionpay: /^(62|88)\d+$/,
        visa: /^4[0-9]{12}(?:[0-9]{3})?$/,
        mastercard: /^5[1-5][0-9]{14}$/,
        amex: /^3[47][0-9]{13}$/,
        diners: /^3(?:0[0-5]|[68][0-9])[0-9]{11}$/,
        discover: /^6(?:011|5[0-9]{2})[0-9]{12}$/,
        jcb: /^(?:2131|1800|35\d{3})\d{11}$/
    }

    for(var key in re) {
        if(re[key].test(number)) {
            return key
        }
    }
}
