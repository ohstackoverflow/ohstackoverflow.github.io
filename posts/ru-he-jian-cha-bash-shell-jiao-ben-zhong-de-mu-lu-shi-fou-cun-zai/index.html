如何检查 Bash shell 脚本中的目录是否存在？

Arya

2023-02-16 19:04

什么命令检查 Bash shell 脚本中目录是否存在？



解答

检查目录是否存在：

if [ -d "$DIRECTORY" ]; then
  echo "$DIRECTORY does exist."
fi

检查目录是否不存在：

if [ ! -d "$DIRECTORY" ]; then
  echo "$DIRECTORY does not exist."
fi

然而，正如Jon Ericson指出的那样，如果您不考虑到目录的符号链接也将通过此检查，则后续命令可能无法按预期工作。例如运行这个：

ln -s "$ACTUAL_DIR" "$SYMLINK"
if [ -d "$SYMLINK" ]; then 
  rmdir "$SYMLINK" 
fi

将产生错误信息：

rmdir: failed to remove `symlink': Not a directory

因此，如果后续命令需要目录，则可能必须区别对待符号链接：

if [ -d "$LINK_OR_DIR" ]; then 
  if [ -L "$LINK_OR_DIR" ]; then
    # It is a symlink!
    # Symbolic link specific commands go here.
    rm "$LINK_OR_DIR"
  else
    # It's a directory!
    # Directory command goes here.
    rmdir "$LINK_OR_DIR"
  fi
fi

请特别注意用于包装变量的双引号。8jean在另一个回答中解释了这样做的原因。

如果变量包含空格或其他不寻常的字符，则可能会导致脚本失败。


更多建议

在 Bash 脚本中引用变量时，始终将变量用双引号引起来。

if [ -d "$DIRECTORY" ]; then
    # Will enter here if $DIRECTORY exists, even if it contains spaces
fi

现在的孩子们在他们的目录名称中加入了空格和许多其他有趣的字符。（空间！回到我的时代，我们没有没有花哨的空间！）有一天，那些孩子中的一个将运行您的脚本设置为$DIRECTORY并且"My M0viez"您的脚本会爆炸。你不想要那个。所以使用双引号。
