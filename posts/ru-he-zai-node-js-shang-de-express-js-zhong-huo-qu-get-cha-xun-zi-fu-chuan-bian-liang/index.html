如何在 Node.js 上的 Express.js 中获取 GET（查询字符串）变量？

Arya

2023-02-17 01:45

我们可以像在 PHP 中获取变量一样在 Node.js 中获取查询字符串中的变量吗$_GET？

我知道在 Node.js 中我们可以获取请求中的 URL。是否有获取查询字符串参数的方法？



解答

在 Express 中，它已经为你完成了，你可以简单地使用req.query来实现：

var id = req.query.id; // $_GET["id"]

否则，在 NodeJS 中，您可以访问req.url和内置url模块以手动进行url.parse：

var url = require('url');
var url_parts = url.parse(request.url, true);
var query = url_parts.query;
