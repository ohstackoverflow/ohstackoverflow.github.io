C# 中的多行字符串文字

Arya

2023-02-17 03:03

有没有一种简单的方法可以在 C# 中创建多行字符串文字？

这是我现在拥有的：

string query = "SELECT foo, bar"
+ " FROM table"
+ " WHERE id = 42";

我知道 PHP 有

<<<BLOCK

BLOCK;

C#有类似的东西吗？



解答

您可以使用@a 前面的符号string来形成逐字字符串文字：

string query = @"SELECT foo, bar
FROM table
WHERE id = 42";

使用此方法时，您也不必转义特殊字符，但 Jon Skeet 的回答中所示的双引号除外。


更多建议

它在 C# 中称为 逐字字符串文字 ，只需将 @ 放在文字之前即可。这不仅允许多行，而且还关闭了转义。所以例如你可以这样做：

string query = @"SELECT foo, bar
FROM table
WHERE name = 'a\b'";

但是，这包括字符串中的换行符（使用您的源中的任何换行符）。对于 SQL，这不仅无害，而且可能会 提高 您看到字符串的任何地方的可读性 - 但在其他地方可能不需要它，在这种情况下，您要么不需要使用多行逐字字符串文字，或从结果字符串中删除它们。

唯一的转义是，如果你想要一个双引号，你必须添加一个额外的双引号：

string quote = @"Jon said, ""This will work,"" - and it did!";
