在 Notepad++ 中将制表符转换为空格

Arya

2023-02-17 04:18

如何在 Notepad++ 中将制表符转换为空格？

我发现一个网页表明这是可能的，但我找不到任何关于如何做到这一点的信息。

我希望能够做到这一点，因为某些网络表单不尊重其中包含选项卡的代码。



解答

要将现有制表符转换为空格，请按Edit->Blank Operations->TAB to Space。

如果以后你想在按 tab 键时输入空格而不是 tab：

  1. 转到Settings->Preferences...->Language（从 7.1 版开始）或Settings->Preferences...->Tab Settings（以前的版本）
  2. 查看Replace by space
  3. （ 可选 ）您可以通过更改字段来设置用于代替制表符的空格数Tab size。


更多建议

已过时：此答案仅适用于较旧版本的 Notepad++。 制表符/空格之间的转换现在内置于 Notepad++ 中，TextFX 插件在插件管理器对话框中不再可用。

  • 首先设置中的 “用空格替换”Preferences -> Language Menu/Tab Settings设置。
  • 接下来，打开您希望用来替换制表符的文档。
  • 突出显示所有文本 ( CTRL+ A)。
  • 然后选择TextFX -> TextFX Edit -> Leading spaces to tabs or tabs to spaces。

注意： 确保已安装 TextFX CharactersPlugins -> Plugin manager -> Show plugin manager插件（ ，Installed选项卡）。否则，将没有 TextFX 菜单。
