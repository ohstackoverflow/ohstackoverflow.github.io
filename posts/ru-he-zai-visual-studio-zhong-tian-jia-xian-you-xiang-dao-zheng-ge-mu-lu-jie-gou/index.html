如何在 Visual Studio 中“添加现有项”到整个目录结构？

Arya

2023-02-17 13:41

我有一组完全不隶属于任何 C# 项目的独立文件，这些文件位于一个复杂的嵌套目录结构中。

我想将它们以那种格式添加到我正在处理的 ASP.NET Web 应用程序中的不同目录中；同时保留相同的结构。因此，我将该文件夹复制到项目的目标位置，并尝试“添加现有项目”，结果却丢失了之前的文件夹层次结构。

通常我会手动重新创建目录，一对一复制，然后添加现有项目。在这种情况下，目录/项目太多了。

那么如何在 Visual Studio 2008 中添加现有的目录和文件呢？



解答

将文件/文件夹从 Windows 资源管理器拖到解决方案资源管理器中。它将全部添加。请注意，如果 Visual Studio 处于管理员模式，这将不起作用，因为 Windows 资源管理器是一个用户模式进程。


更多建议

为特定项目启用 “显示所有文件” （您可能需要点击“刷新”才能看到它们）**。

不属于您的项目的文件夹/文件在项目树中 显得稍微“轻” 。

右键单击 要添加的文件夹/文件，然后单击 “包含在项目中” 。它将递归地将文件夹/文件添加到项目中。

** 这些按钮位于迷你解决方案资源管理器工具栏上。

** 确保您未处于调试模式。
