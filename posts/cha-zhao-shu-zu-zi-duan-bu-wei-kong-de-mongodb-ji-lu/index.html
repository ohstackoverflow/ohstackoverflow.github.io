查找数组字段不为空的 MongoDB 记录

Arya

2023-02-17 16:56

我所有的记录都有一个名为“图片”的字段。该字段是一个字符串数组。

我现在想要这个数组不为空的最新 10 条记录。

我四处搜索，但奇怪的是我没有找到太多关于这个的信息。我已经阅读了 $where 选项，但我想知道它对本机函数有多慢，以及是否有更好的解决方案。

即便如此，这也行不通：

ME.find({$where: 'this.pictures.length > 0'}).sort('-created').limit(10).execFind()

什么都不返回。没有长度位的离开this.pictures确实有效，但当然它也会返回空记录。



解答

如果您还有没有密钥的文档，您可以使用：

ME.find({ pictures: { $exists: true, $not: {$size: 0} } })

$size如果涉及到MongoDB 不使用索引，那么这里有一个更好的解决方案：

ME.find({ pictures: { $exists: true, $ne: [] } })

如果您的财产可能具有无效值（例如或其他），那么您可以按照此答案中的建议null boolean添加额外的检查：$types

使用 mongo >= 3.2：

ME.find({ pictures: { $exists: true, $type: 'array', $ne: [] } })

使用 mongo < 3.2：

ME.find({ pictures: { $exists: true, $type: 4, $ne: [] } })

自 MongoDB 2.6 版本开始，您可以与 operator 进行比较$gt，但这可能会导致意想不到的结果（您可以在这个答案中找到详细解释）：

ME.find({ pictures: { $gt: [] } })


更多建议

经过更多的查看，尤其是在 mongodb 文档中，并将一些令人费解的部分放在一起，这就是答案：

ME.find({pictures: {$exists: true, $not: {$size: 0}}})
