如何使用 将文件夹从远程主机复制到本地主机scp？

我用来ssh登录我的服务器。
然后，我想将远程文件夹复制foo到本地/home/user/Desktop。

我如何实现这一目标？



解答

scp -r user@your.server.example.com:/path/to/foo /home/user/Desktop/

通过不包括 foo 末尾的尾随 '/'，您将复制目录本身（包括内容），而不仅仅是目录的内容。

来自man scp（参见在线手册）

-r 递归复制整个目录


更多建议

要使用scp 的全部功能，您需要完成以下步骤：

  1. 公钥授权
  2. 创建 SSH 别名

然后，例如，如果你有这个 ~/.ssh/config ：

Host test
    User testuser
    HostName test-site.example
    Port 22022

Host prod
    User produser
    HostName production-site.example
    Port 22022

您将免于输入密码并简化 scp 语法，如下所示：

scp -r prod:/path/foo /home/user/Desktop   # copy to local
scp -r prod:/path/foo test:/tmp            # copy from remote prod to remote test

此外，您将能够使用远程路径补全：

scp test:/var/log/  # press tab twice
Display all 151 possibilities? (y or n)

要启用远程 bash-completion，您需要在主机<source>和<target>主机上都有 bash-shell，并且 bash- completion 可以正常工作。有关更多信息，请参阅相关问题：

使用 scp 时如何为远程路径启用自动完成？
