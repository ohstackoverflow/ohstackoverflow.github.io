如何使用 sed 将每个换行符 (\n) 替换为空格？

Arya

2023-02-17 00:53

如何使用命令将换行符 (" \n") 替换为空格 (" `")sed`？

我尝试失败：

sed 's#\n# #g' file
sed 's#^$# #g' file

我如何解决它？



解答

将此解决方案与 GNU 一起使用sed：

sed ':a;N;$!ba;s/\n/ /g' file

这将循环读取整个文件 ( ':a;N;$!ba)，然后用空格 ( ) 替换换行符s/\n/ /g。如果需要，可以简单地附加其他替换。

解释：

  1. sed首先将不包括换行符的第一行读入模式空间。
  2. 通过创建标签:a。
  3. 通过 将换行符和下一行附加到模式空间N。
  4. 如果我们在最后一行之前，则分支到创建的标签$!ba（$!意味着不要在最后一行执行此操作。这是避免N再次执行所必需的，如果没有更多输入，脚本将终止！）。
  5. 最后，替换用模式空间（整个文件）上的空格替换每个换行符。

这是适用于 BSD 和 OS X 的跨平台兼容语法sed（根据@Benjie 评论）：

sed -e ':a' -e 'N' -e '$!ba' -e 's/\n/ /g' file

如您所见，使用sedfor 这个原本简单的问题是有问题的。有关更简单和适当的解决方案，请参阅此答案。
