写入 Visual Studio 的输出窗口

Arya

2023-02-17 16:12

我正在尝试向输出窗口写入一条消息以进行调试。我搜索了像 Java 的system.out.println(""). 我试过Debug.Write，，Console.Write和Trace.Write。它不会报错，但也不会打印任何内容。

选中“定义 DEBUG 常量”和“定义 TRACE 常量”选项。

菜单 工具 → 选项 → 调试 → “将所有输出窗口文本重定向到立即窗口” 选项未选中。

配置：活动（调试）

注意：如果相关，我使用向导创建了一个项目作为“Windows 窗体应用程序”。我不知道去哪里看。



解答

添加System.Diagnostics命名空间，然后就可以使用它Debug.WriteLine()来快速打印消息到IDE的输出窗口。更多详情，请参考这些：

  • 如何在 Visual C# 中进行跟踪和调试
  • 关于使用调试和跟踪类的论文，包括异常处理


更多建议

这将写入调试输出窗口：

using System.Diagnostics;

Debug.WriteLine("Send to debug output.");
