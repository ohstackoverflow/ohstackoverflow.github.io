使用 jQuery 转义键的键码

如何检测 IE、Firefox 和 Chrome 中的转义键按下？下面的代码在 IE 和警报中有效27，但在 Firefox 中它会发出警报0

$('body').keypress(function(e){
    alert(e.which);
    if(e.which == 27){
        // Close my modal window
    }
});



解答

注意：keyCode即将弃用，请key改用。

function keyPress (e) {
    if(e.key === "Escape") {
        // write your logic here.
    }
}

该keydown事件将适用于 Escape 并具有允许您在所有浏览器中使用的好处keyCode。此外，您需要将听众附加到document而不是身体。

keyCode现在正处于被弃用的过程中，大多数现代浏览器key现在都提供该属性，尽管您现在仍然需要后备以获得良好的浏览器支持（在撰写本文时，当前版本的 Chrome 和 Safari 不支持它）。

*evt.key所有现代浏览器现在都支持 *2018 年 9 月更新。

document.onkeydown = function(evt) {
    evt = evt || window.event;
    var isEscape = false;
    if ("key" in evt) {
        isEscape = (evt.key === "Escape" || evt.key === "Esc");
    } else {
        isEscape = (evt.keyCode === 27);
    }
    if (isEscape) {
        alert("Escape");
    }
};
