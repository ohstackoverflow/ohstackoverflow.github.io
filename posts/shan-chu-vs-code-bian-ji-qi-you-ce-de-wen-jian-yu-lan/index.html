删除 VS Code 编辑器右侧的文件预览

Arya

2023-02-18 06:48

这个美女是最近一次更新后出现的，让我很分心。

我搜索了菜单，但没有找到任何设置来删除它。有什么办法可以从编辑器中消除它吗？



解答

这称为minimap，并且如该链接中所述，

如果您想禁用迷你地图，您可以"editor.minimap.enabled": false在您的用户或工作区设置中进行设置。

保存设置文件后，小地图就会消失。


更多建议

您还可以从命令面板切换小地图。您可以转到“查看”并选择“命令面板”或只是Ctrl+ Shift+P并输入“迷你地图”一词，您将立即获得切换迷你地图的选项。
