使用值列表从 Pandas 数据框中选择行

Arya

2023-02-17 04:16

假设我有以下 Pandas 数据框：

df = DataFrame({'A' : [5,6,3,4], 'B' : [1,2,3, 5]})
df

     A   B
0    5   1
1    6   2
2    3   3
3    4   5

我可以根据特定值进行子集化：

x = df[df['A'] == 3]
x

     A   B
2    3   3

但是我如何根据值列表进行子集化？- 是这样的：

list_of_values = [3,6]

y = df[df['A'] in list_of_values]

要得到：

     A    B
1    6    2
2    3    3



解答

您可以使用以下isin方法：

In [1]: df = pd.DataFrame({'A': [5,6,3,4], 'B': [1,2,3,5]})

In [2]: df
Out[2]:
   A  B
0  5  1
1  6  2
2  3  3
3  4  5

In [3]: df[df['A'].isin([3, 6])]
Out[3]:
   A  B
1  6  2
2  3  3

并获得相反的用途~：

In [4]: df[~df['A'].isin([3, 6])]
Out[4]:
   A  B
0  5  1
3  4  5


更多建议

您可以使用方法查询：

df.query('A in [6, 3]')
# df.query('A == [6, 3]')

或者

lst = [6, 3]
df.query('A in @lst')
# df.query('A == @lst')
