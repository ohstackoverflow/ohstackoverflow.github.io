如何查看哪些 Git 分支正在跟踪哪个远程/上游分支？

Arya

2023-02-17 07:18

我知道我能做到git branch --all，这向我展示了本地和远程分支机构，但它在向我展示它们之间的关系方面没有多大用处。

如何以显示哪个本地分支正在跟踪哪个远程分支的方式列出分支？



解答

非常瓷器命令，如果你想用它来编写脚本就不好了：

git branch -vv   # doubly verbose!

请注意，对于 git 1.8.3，上游分支显示为 蓝色 （请参阅“什么是 git 中的分支跟踪（如果有的话）？ ”）

如果您想要干净的输出，请参阅Carl Suster 的回答- 它使用了一个瓷器命令，我认为在我最初写这个答案时不存在，所以它更简洁一些并且适用于为变基配置的分支，而不仅仅是合并。


更多建议

git remote show origin

将“来源”替换为您的遥控器的任何名称。
