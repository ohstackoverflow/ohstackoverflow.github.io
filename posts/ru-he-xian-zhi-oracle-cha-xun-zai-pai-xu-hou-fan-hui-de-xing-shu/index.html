如何限制 Oracle 查询在排序后返回的行数？

Arya

2023-02-17 03:50

有没有办法让Oracle查询表现得像包含 MySQLlimit子句一样？

在 MySQL 中，我可以这样做：

select * 
from sometable
order by name
limit 20,10

获得第 21 到第 30 行（跳过前 20 行，给出接下来的 10 行）。行是在 之后选择的order by，所以它实际上是按字母顺序从第 20 个名字开始的。

在 Oracle 中，人们唯一提到的是rownum伪列，但它是 在之前 order by评估的，这意味着：

select * 
from sometable
where rownum <= 10
order by name

将返回一组按名称排序的随机十行，这通常不是我想要的。它还不允许指定偏移量。



解答

您可以为此使用子查询

select *
from  
( select * 
  from emp 
  order by sal desc ) 
where ROWNUM <= 5;

另请查看Oracle/AskTom 上的主题On ROWNUM 和限制结果以获取更多信息。

更新 ：为了限制下限和上限的结果，事情变得更加臃肿

select * from 
( select a.*, ROWNUM rnum from 
  ( <your_query_goes_here, with order by> ) a 
  where ROWNUM <= :MAX_ROW_TO_FETCH )
where rnum  >= :MIN_ROW_TO_FETCH;

（从指定的 AskTom 文章中复制）

更新 2 ：从 Oracle 12c (12.1) 开始，有一种语法可用于限制行或从偏移量开始。

SELECT * 
FROM   sometable
ORDER BY name
OFFSET 20 ROWS FETCH NEXT 10 ROWS ONLY;

有关更多示例，请参见此答案。感谢 Krumia 的提示。


更多建议

从 Oracle 12c R1 (12.1) 开始，有 一个 row limiting clause。它不使用熟悉的LIMIT语法，但可以通过更多选项更好地完成工作。您可以在此处找到完整的语法。（另请阅读此答案中有关 Oracle 内部工作原理的更多信息）。

要回答原始问题，这里是查询：

SELECT * 
FROM   sometable
ORDER BY name
OFFSET 20 ROWS FETCH NEXT 10 ROWS ONLY;

（更早的Oracle版本，请参考本题其他答案）

  val  NUMBER

  INTO rownum_order_test
SELECT level
FROM   dual
CONNECT BY level <= 10;

20 rows selected.

获取第一N行

SELECT val
FROM   rownum_order_test
ORDER BY val DESC
FETCH FIRST 5 ROWS ONLY;

       VAL
----------
        10
        10
         9
         9
         8

5 rows selected.

获取第一N行，如果N第行有联系，则获取所有联系的行

SELECT val
FROM   rownum_order_test
ORDER BY val DESC
FETCH FIRST 5 ROWS WITH TIES;

       VAL
----------
        10
        10
         9
         9
         8
         8

6 rows selected.

前x% 的行

SELECT val
FROM   rownum_order_test
ORDER BY val
FETCH FIRST 20 PERCENT ROWS ONLY;

       VAL
----------
         1
         1
         2
         2

4 rows selected.

使用偏移量，对分页非常有用

SELECT val
FROM   rownum_order_test
ORDER BY val
OFFSET 4 ROWS FETCH NEXT 4 ROWS ONLY;

       VAL
----------
         3
         3
         4
         4

4 rows selected.

您可以将偏移量与百分比结合使用

SELECT val
FROM   rownum_order_test
ORDER BY val
OFFSET 4 ROWS FETCH NEXT 20 PERCENT ROWS ONLY;

       VAL
----------
         3
         3
         4
         4

4 rows selected.
