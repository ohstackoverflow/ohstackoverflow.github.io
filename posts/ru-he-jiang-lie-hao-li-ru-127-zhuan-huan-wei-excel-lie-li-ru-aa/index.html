如何将列号（例如 127）转换为 Excel 列（例如 AA）

Arya

2023-02-18 05:25

如何在不使用自动化直接从 Excel 获取值的情况下，将数字转换为 C# 中的 Excel 列名称。

Excel 2007 的可能范围是 1 到 16384，这是它支持的列数。结果值应采用 excel 列名称的形式，例如 A、AA、AAA 等。



解答

我是这样做的：

private string GetExcelColumnName(int columnNumber)
{
    string columnName = "";

    while (columnNumber > 0)
    {
        int modulo = (columnNumber - 1) % 26;
        columnName = Convert.ToChar('A' + modulo) + columnName;
        columnNumber = (columnNumber - modulo) / 26;
    }

    return columnName;
}


更多建议

如果有人需要在没有 VBA 的情况下在 Excel 中执行此操作，可以使用以下方法：

=SUBSTITUTE(ADDRESS(1;colNum;4);"1";"")

其中 colNum 是列号

在 VBA 中：

Function GetColumnName(colNum As Integer) As String
    Dim d As Integer
    Dim m As Integer
    Dim name As String
    d = colNum
    name = ""
    Do While (d > 0)
        m = (d - 1) Mod 26
        name = Chr(65 + m) + name
        d = Int((d - m) / 26)
    Loop
    GetColumnName = name
End Function
