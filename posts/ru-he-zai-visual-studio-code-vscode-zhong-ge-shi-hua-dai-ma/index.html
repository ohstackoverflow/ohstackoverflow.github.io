解答

通过以下快捷方式，可以在 Visual Studio Code 中使用代码格式：

  • 在 Windows Shift+ Alt+F
  • 在 Mac Shift+ Option+F
  • 在 Linux Ctrl+ Shift+I

或者，您可以通过子菜单找到快捷方式以及其他快捷方式，View / Command Palette编辑器中也提供了Ctrl+ Shift+ P（或Mac 上的Command+ Shift+ P），然后搜索 _ 格式文档 。_

对于未保存的片段

  1. 打开命令面板（Win:F1或Ctrl+ Shift+ P）

  2. 找到“更改语言模式”

  3. 选择语言，例如json。现在应该突出显示语法。

  4. 格式化文档（例如打开命令面板 -> “ 格式化文档 ”）

取消格式化

  1. 选择文本
  2. 命令面板 -> 连接线

更多建议

代码格式化快捷方式：

Windows 上的 Visual Studio Code - Shift+ Alt+F

MacOS 上的 Visual Studio Code - Shift+ Option+F

Ubuntu 上的 Visual Studio Code - Ctrl+ Shift+I

如果需要，您还可以使用首选项设置自定义此快捷方式。

保存文件时的代码格式：

Visual Studio Code 允许用户自定义默认设置。

如果您想在保存时自动格式化您的内容，请在 Visual Studio Code 的工作空间设置中添加以下代码片段。

菜单 文件 → 首选项 → 工作区设置

{
    // Controls if the editor should automatically format the line after typing
    "beautify.onSave": true,

    "editor.formatOnSave": true,

    // You can auto format any files based on the file extensions type.
    "beautify.JSfiles": [
        "js",
        "json",
        "jsbeautifyrc",
        "jshintrc",
        "ts"
    ]
}

注意：现在您可以自动格式化 TypeScript 文件。检查我的更新。
