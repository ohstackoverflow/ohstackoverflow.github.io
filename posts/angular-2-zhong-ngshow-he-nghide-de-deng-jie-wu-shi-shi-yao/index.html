Angular 2+ 中 ngShow 和 ngHide 的等价物是什么？

Arya

2023-02-17 16:25

我有许多元素希望在特定条件下可见。

在 AngularJS 中我会写

<div ng-show="myVar">stuff</div>

我如何在 Angular 2+ 中执行此操作？



解答

该hidden属性可用于此

[hidden]="!myVar"

也可以看看

  • https://developer.mozilla.org/en-US/docs/Web/HTML/Global_attributes/hidden

问题

hidden但是有一些问题，因为它可能与display属性的 CSS 冲突。

看看 Plunker 示例some是如何不被隐藏的，因为它有一个样式 ****

:host {display: block;}

放。（这在其他浏览器中可能表现不同——我用 Chrome 50 测试过）

解决方法

您可以通过添加来修复它

[hidden] { display: none !important;}

在index.html.

另一个陷阱

hidden="false"
hidden="{{false}}"
hidden="{{isHidden}}" // isHidden = false;

是一样的

hidden="true"

并且不会显示该元素。

hidden="false"``"false"将分配被认为是真实的字符串。
只有值false或删除属性才会真正使元素可见。

使用{{}}还会将表达式转换为字符串，并且不会按预期工作。

只有绑定 with[]才能按预期工作，因为 thisfalse被指定为 asfalse而不是"false".

*ngIf 对比[hidden]

*ngIf有效地从 DOM 中删除它的内容，同时[hidden]修改display属性，只指示浏览器不显示内容，但 DOM 仍然包含它。


更多建议

使用[hidden]属性：

[hidden]="!myVar"

或者你可以使用*ngIf

*ngIf="myVar"

这是显示/隐藏元素的两种方式。唯一的区别是：*ngIf将从 DOM 中删除元素，同时[hidden]通过将元素保留在 DOM 中来告诉浏览器使用 CSSdisplay属性显示/隐藏元素。
