“python -m SimpleHTTPServer”的 Python 3 等价物是什么

Arya

2023-02-17 01:18

Python 3 等价于什么python -m SimpleHTTPServer？



解答

从文档：

该SimpleHTTPServer模块已合并到http.serverPython 3.0 中。将源代码转换为 3.0 时，2to3 工具会自动调整导入。

因此，您的命令是python -m http.server，或者根据您的安装，它可以是：

python3 -m http.server


更多建议

等价于：

python3 -m http.server
