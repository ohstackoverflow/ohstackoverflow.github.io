在 Java 中将“ArrayList<String>”转换为“String[]”

Arya

2023-02-17 04:11

我如何将ArrayList<String>对象转换为String[]Java 中的数组？



解答

List<String> list = ..;
String[] array = list.toArray(new String[0]);

例如：

List<String> list = new ArrayList<String>();
//add some stuff
list.add("android");
list.add("apple");
String[] stringArray = list.toArray(new String[0]);

toArray()不传递任何参数的方法返回Object[]。所以你必须传递一个数组作为参数，它将用列表中的数据填充并返回。您也可以传递一个空数组，但您也可以传递一个具有所需大小的数组。

重要更新 ：最初使用上面的代码new String[list.size()]。然而，这篇博文表明，由于 JVM 优化，new String[0]现在使用更好。


更多建议

Java 8 中的替代方案：

String[] strings = list.stream().toArray(String[]::new);

Java 11+：

String[] strings = list.toArray(String[]::new);
