将数组项复制到另一个数组

Arya

2023-02-17 05:06

我有一个 JavaScript 数组dataArray，我想将其推入一个新数组newArray。除了我不想newArray[0]成为dataArray。我想将所有项目推入新数组：

var newArray = [];

newArray.pushValues(dataArray1);
newArray.pushValues(dataArray2);
// ...

甚至更好：

var newArray = new Array (
   dataArray1.values(),
   dataArray2.values(),
   // ... where values() (or something equivalent) would push the individual values into the array, rather than the array itself
);

所以现在新数组包含各个数据数组的所有值。是否有一些像pushValuesavailable 这样的速记，这样我就不必遍历每个人dataArray，一个一个地添加项目？



解答

使用concat函数，如下所示：

var arrayA = [1, 2];
var arrayB = [3, 4];
var newArray = arrayA.concat(arrayB);

的值newArray将是[1, 2, 3, 4]（arrayA并且arrayB保持不变；concat为结果创建并返回一个新数组）。


更多建议

在 ECMAScript 6 中，您可以使用Spread 语法：

let arr1 = [0, 1, 2];
let arr2 = [3, 4, 5];
arr1.push(...arr2);

console.log(arr1)

Spread 语法在所有主流浏览器（IE11 除外）中都可用。有关当前兼容性，请参阅此（不断更新的）兼容性表。

但是，有关性能的更多评论，请参见下面 Jack Giffin 的回复。它似乎concat仍然比传播运算符更好更快。
