'SELECT' 语句中的 'IF' - 根据列值选择输出值

Arya

2023-02-17 15:24 

SELECT id, amount FROM report

我需要amount是amountifreport.type='P'和-amountif report.type='N'。如何将其添加到上述查询中？



解答

SELECT id, 
       IF(type = 'P', amount, amount * -1) as amount
FROM report

请参阅http://dev.mysql.com/doc/refman/5.0/en/control-flow-functions.html。

此外，您可以处理条件为空的情况。在金额为空的情况下：

SELECT id, 
       IF(type = 'P', IFNULL(amount,0), IFNULL(amount,0) * -1) as amount
FROM report

该部分IFNULL(amount,0)表示 当 amount 不为 null 时返回 amount 否则返回 0 。


更多建议

使用case语句：

select id,
    case report.type
        when 'P' then amount
        when 'N' then -amount
    end as amount
from
    `report`
