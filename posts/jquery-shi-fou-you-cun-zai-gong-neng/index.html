jQuery 是否有“存在”功能？

Arya

2023-02-16 20:09

如何检查 jQuery 中的元素是否存在？

我目前的代码是这样的：

if ($(selector).length > 0) {
    // Do something
}

有没有更优雅的方法来解决这个问题？也许是插件或功能？



解答

在 JavaScript 中，一切都是“真实的”或“虚假的”，对于数字0意味着false，一切true。所以你可以写：

if ($(selector).length)

你不需要那>0部分。


更多建议

是的！

jQuery.fn.exists = function(){ return this.length > 0; }

if ($(selector).exists()) {
    // Do something
}
