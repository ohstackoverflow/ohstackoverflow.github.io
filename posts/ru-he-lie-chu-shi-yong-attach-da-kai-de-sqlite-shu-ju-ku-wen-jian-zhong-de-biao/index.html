如何列出使用 ATTACH 打开的 SQLite 数据库文件中的表？

Arya

2023-02-17 03:10

什么 SQL 可用于列出表以及 SQLite 数据库文件中这些表中的行 - 一旦我将其附加到ATTACHSQLite 3 命令行工具上的命令？



解答

,.tables和.schema“helper”函数不查看 ATTACHed 数据库：它们只是查询SQLITE_MASTER“主”数据库的表。因此，如果您使用

ATTACH some_file.db AS my_db;

那么你需要做

SELECT name FROM my_db.sqlite_master WHERE type='table';

请注意，临时表不会显示.tables：您必须sqlite_temp_master为此列出：

SELECT name FROM sqlite_temp_master WHERE type='table';
