有没有一种简单的方法可以按值删除列表元素？

Arya

2023-02-17 05:04

我想从列表中删除一个值，如果它存在于列表中（它可能不存在）。

a = [1, 2, 3, 4]
b = a.index(6)

del a[b]
print(a)

以上给出了错误：

ValueError: list.index(x): x not in list

所以我必须这样做：

a = [1, 2, 3, 4]

try:
    b = a.index(6)
    del a[b]
except:
    pass

print(a)

但是有没有更简单的方法来做到这一点？



解答

要删除元素的 第一次出现list.remove，请使用：

>>> xs = ['a', 'b', 'c', 'd']
>>> xs.remove('b')
>>> print(xs)
['a', 'c', 'd']

要删除 所有出现 的元素，请使用列表理解：

>>> xs = ['a', 'b', 'c', 'd', 'b', 'b', 'b', 'b']
>>> xs = [x for x in xs if x != 'b']
>>> print(xs)
['a', 'c', 'd']


更多建议

通常，如果你告诉 Python 做一些它不能做的事情，Python 会抛出一个异常，所以你必须做以下任一事情：

if c in a:
    a.remove(c)

或者：

try:
    a.remove(c)
except ValueError:
    pass

Exception 不一定是坏事，只要它是您期望并正确处理的异常。
