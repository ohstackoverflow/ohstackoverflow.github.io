Bash 中的 Echo 换行符打印文字 \n

Arya

2023-02-16 19:49

如何打印换行符？这只是打印\n：

$ echo -e "Hello,\nWorld!"
Hello,\nWorld!



解答

改用printf：

printf "hello\nworld\n"

printf在不同环境中表现比echo.


更多建议

确保你在 Bash 中。

$ echo $0
bash

所有这四种方式都对我有用：

echo -e "Hello\nworld"
echo -e 'Hello\nworld'
echo Hello$'\n'world
echo Hello ; echo world
