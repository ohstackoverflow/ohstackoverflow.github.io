在 JavaScript 中返回多个值？

Arya

2023-02-17 05:29

我试图在 JavaScript 中返回两个值。这可能吗？

var newCodes = function() {  
    var dCodes = fg.codecsCodes.rs;
    var dCodes2 = fg.codecsCodes2.rs;
    return dCodes, dCodes2;
};



解答

不，但您可以返回一个包含您的值的数组：

function getValues() {
    return [getFirstValue(), getSecondValue()]
}

然后你可以像这样访问它们：

const [first, second] = getValues()

这称为解构赋值，每个主要的 JS 环境都支持它。它等效于以下内容：

const values = getValues()
const first = values[0]
const second = values[1]

如果你想为每个值分配一个名称，你也可以返回一个对象：

function getValues() {
    return {
        first: getFirstValue(),
        second: getSecondValue(),
    }
}

并访问它们：

const {first, second} = getValues()

这与：

const values = getValues()
const first = values.first
const second = values.second

强烈建议 返回对象而不是数组，除非值作为简单元组有意义，例如坐标对[x, y]。使用数组，很容易忘记哪个值是哪个，以后添加更多值就更难了，使用 TypeScript 或 JSDoc 正确输入也稍微困难一些。


更多建议

从 ECMAScript 6 开始，您可以使用数组和“解构赋值”来做到这一点。请注意，这些在旧的 Javascript 版本中不可用（意思是 - ECMAScript 第 3 版和第 5 版均不可用）。

它允许您同时分配给 1+ 个变量：

var [x, y] = [1, 2];
x; // 1
y; // 2

// or

[x, y] = (function(){ return [3, 4]; })();
x; // 3
y; // 4

您还可以使用对象解构结合属性值速记来命名对象中的返回值并挑选出您想要的：

let {baz, foo} = (function(){ return {foo: 3, bar: 500, baz: 40} })();
baz; // 40
foo; // 3

顺便说一下，不要被 ECMAScript 允许您这样做的事实所迷惑return 1, 2, ...。那里真正发生的事情并不像看起来那样。return 语句中的表达式 — 1, 2, 3— 只不过是按顺序应用于数字文字 ( 1, 2, 和3) 的逗号运算符，它最终计算为其最后一个表达式 — 的值3。这就是为什么return 1, 2, 3is 在功能上只等同于return 3.

return 1, 2, 3;
// becomes
return 2, 3;
// becomes
return 3;
