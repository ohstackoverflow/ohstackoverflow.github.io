Python 和 pip，列出可用包的所有版本？

Arya

2023-02-17 14:46

给定可以使用pip安装的 Python 包的名称，是否有任何方法可以找出 pip 可以安装的所有可能版本的列表？现在是反复试验。

我正在尝试为第三方库安装一个版本，但最新版本太新，进行了向后不兼容的更改。所以我想以某种方式列出 pip 知道的所有版本，以便我可以测试它们。



解答

（更新：截至 2020 年 3 月，许多人报告说通过安装的 yolkpip install yolk3k仅返回最新版本。Chris的回答似乎获得了最多的赞成票并且对我有用）

pastebin 上的脚本确实有效。但是，如果您正在使用多个环境/主机，这不是很方便，因为您每次都必须复制/创建它。

更好的全方位解决方案是使用yolk3k，它可以通过 pip 安装。例如，查看可用的 Django 版本：

$ pip install yolk3k
$ yolk -V django
