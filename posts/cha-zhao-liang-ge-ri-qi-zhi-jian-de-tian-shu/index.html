查找两个日期之间的天数

Arya

2023-02-17 14:39

如何使用 PHP 查找两个日期之间的天数？



解答

$now = time(); // or your date as well
$your_date = strtotime("2010-01-31");
$datediff = $now - $your_date;

echo round($datediff / (60 * 60 * 24));


更多建议

如果您使用PHP 5.3 >，这是迄今为止计算绝对差值最准确的方法：

$earlier = new DateTime("2010-07-06");
$later = new DateTime("2010-07-09");

$abs_diff = $later->diff($earlier)->format("%a"); //3

如果您需要相对（有符号）天数，请改用：

$earlier = new DateTime("2010-07-06");
$later = new DateTime("2010-07-09");

$pos_diff = $earlier->diff($later)->format("%r%a"); //3
$neg_diff = $later->diff($earlier)->format("%r%a"); //-3

更多关于 phpDateInterval格式的信息可以在这里找到： https: //www.php.net/manual/en/dateinterval.format.php
