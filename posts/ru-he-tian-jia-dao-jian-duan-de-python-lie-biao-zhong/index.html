如何添加到简短的 python 列表中？

Arya

2023-02-17 16:09

list.append()附加到列表的末尾。这解释了由于list.prepend()大型列表的性能问题不存在。对于一个简短的列表，我如何在前面加上一个值？



解答

表格s.insert(0, x)是最常见的。

每当你看到它时，可能是时候考虑使用collections.deque而不是列表了。前置到双端队列运行在恒定时间内。在列表前添加以线性时间运行。


更多建议

这将创建一个x带有前缀的新列表，而不是修改现有列表：

new_list = [x] + old_list
