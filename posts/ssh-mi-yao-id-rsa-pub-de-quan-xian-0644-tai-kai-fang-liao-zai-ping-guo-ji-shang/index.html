SSH 密钥：“'id_rsa.pub' 的权限 0644 太开放了。” 在苹果机上

Arya

2023-02-18 03:12

密钥文件权限：

解答

debug1: identity file /Users/tudouya/.ssh/vm/vm_id_rsa.pub type 1

您似乎正在尝试使用错误的密钥文件。扩展名为“.pub”的文件包含密钥的 公共部分。 没有“.pub”扩展名的相应文件包含密钥的 私有部分。 当您运行 ssh 客户端连接到远程服务器时，您必须向 ssh 客户端提供私钥文件。

.ssh/config您的文件（或）中可能有一行/etc/ssh_config如下所示：

IdentityFile .../.ssh/vm/vm_id_rsa.pub

您需要从文件名中删除“.pub”扩展名：

IdentityFile .../.ssh/vm/vm_id_rsa
