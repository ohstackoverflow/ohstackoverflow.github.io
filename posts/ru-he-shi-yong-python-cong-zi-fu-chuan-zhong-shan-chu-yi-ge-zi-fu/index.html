如何使用 Python 从字符串中删除一个字符

Arya

2023-02-18 04:16

例如，有一个字符串。EXAMPLE.

如何删除中间字符，即M从中删除？我不需要代码。我想知道：

  • Python 中的字符串是否以任何特殊字符结尾？
  • 哪个更好-从中间字符开始将所有内容从右向左移动或创建新字符串而不复制中间字符？



解答

在 Python 中，字符串是不可变的，因此您必须创建一个新字符串。您有几个关于如何创建新字符串的选项。如果您想删除出现的“M”：

newstr = oldstr.replace("M", "")

如果要删除中心字符：

midlen = len(oldstr) // 2
newstr = oldstr[:midlen] + oldstr[midlen+1:]

您询问字符串是否以特殊字符结尾。不，您正在像 C 程序员一样思考。在 Python 中，字符串与其长度一起存储，因此任何字节值（包括\0）都可以出现在字符串中。


更多建议

要替换特定位置：

s = s[:pos] + s[(pos+1):]

要替换特定字符：

s = s.replace('M','')
