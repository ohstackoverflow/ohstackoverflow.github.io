MySQL select 10 random rows from 600K rows fast

Arya

2023-02-18 06:01

How can I best write a query that selects 10 rows randomly from a total of 600k?



Answer

A great post handling several cases, from simple, to gaps, to non-uniform with gaps.

http://jan.kneschke.de/projects/mysql/order-by-rand/

For most general case, here is how you do it:

SELECT name
  FROM random AS r1 JOIN
       (SELECT CEIL(RAND() *
                     (SELECT MAX(id)
                        FROM random)) AS id)
        AS r2
 WHERE r1.id >= r2.id
 ORDER BY r1.id ASC
 LIMIT 1

This supposes that the distribution of ids is equal, and that there can be gaps in the id list. See the article for more advanced examples


Suggest

SELECT column FROM table
ORDER BY RAND()
LIMIT 10

Not the efficient solution but works

How do I find and replace all occurrences (in all files) in Visual Studio Code?

Arya

2023-02-18 06:01

I can't figure out how to find and replace all occurrences of a word in different files using Visual Studio Code version 1.0.

I get the impression this should be possible since doing Ctrl + Shift + F allows me to simply search a folder, but i am clueless to how to proceed from here.

I looked in various key combination https://code.visualstudio.com/Docs/customization/keybindings

But I only found how to replace all occurrences in the open file (Ctrl + h)

All help gratefully appreciated, this is driving me crazy.



Answer

Since version 1.3 of vscode this is possible

  1. Navigate to the search, click icon to the left or:
    • (mac) cmd + shift + h
    • (PC) ctrl + shift + h
  2. expand replace
  3. enter search term and replace term
  4. confirm!

How to concatenate strings in twig

Arya

2023-02-18 06:00

Anyone knows how to concatenate strings in twig? I want to do something like:

{{ concat('http://', app.request.host) }}



Answer

This should work fine:

{{ 'http://' ~ app.request.host }}

To add a filter - like 'trans' - in the same tag use

{{ ('http://' ~ app.request.host) | trans }}

As Adam Elsodaney points out, you can also use string interpolation, this does require double quoted strings:

{{ "http://#{app.request.host}" }}


Suggest

Also a little known feature in Twig is string interpolation:

{{ "http://#{app.request.host}" }}
