返回传递给方法的值

Arya

2023-02-18 06:40

我在接口上有一个方法：

string DoSomething(string whatever);

我想用最小起订量模拟它，以便它返回传入的任何内容 - 类似于：

_mock.Setup( theObject => theObject.DoSomething( It.IsAny<string>( ) ) )
   .Returns( [the parameter that was passed] ) ;

有任何想法吗？



解答

您可以使用带输入参数的 lambda，如下所示：

.Returns((string myval) => { return myval; });

或者稍微更具可读性：

.Returns<string>(x => x);


更多建议

更有用的是，如果你有多个参数，你可以通过以下方式访问任何/所有参数：

_mock.Setup(x => x.DoSomething(It.IsAny<string>(),It.IsAny<string>(),It.IsAny<string>())
     .Returns((string a, string b, string c) => string.Concat(a,b,c));

您始终需要引用所有参数，以匹配方法的签名，即使您只打算使用其中一个。

在 Java 中解码 Base64 数据

Arya

2023-02-18 06:40

我有一个 Base64 编码的图像。在 Java 中解码它的最佳方法是什么？希望只使用 Sun Java 6 中包含的库。



解答

从 v6 开始，Java SE 附带了 JAXB。javax.xml.bind.DatatypeConverter有使这很容易的静态方法。见parseBase64Binary()和printBase64Binary()。

更新 ：Java 不再附带 JAXB（自 Java 11 起）。如果您的项目需要 JAXB，则需要通过依赖管理系统配置相关库，例如 Maven。如果您需要编译器 ( xjc.exe)，您还需要单独下载它。

如何切换布尔值？

Arya

2023-02-18 06:39

有没有真正简单的方法来切换javascript 中的布尔值？

到目前为止，我在编写自定义函数之外得到的最好的是三元：

bool = bool ? false : true;



解答

bool = !bool;

这适用于大多数语言。


更多建议

如果您不介意将布尔值转换为数字（即 0 或 1），则可以使用按位异或赋值运算符。像这样：

bool ^= true;   //- toggle value.

如果您使用长的、描述性的布尔名称，这尤其有用，例如：

let inDynamicEditMode   = true;     // Value is: true (boolean)
inDynamicEditMode      ^= true;     // Value is: 0 (number)
inDynamicEditMode      ^= true;     // Value is: 1 (number)
inDynamicEditMode      ^= true;     // Value is: 0 (number)

这比在每一行中重复变量更容易扫描。

此方法适用于所有（主要）浏览器（和大多数编程语言）。
