反转字符串的最佳方法

Arya

2023-02-18 04:30

我不得不在 C# 2.0 中编写一个字符串反转函数（即 LINQ 不可用）并想出了这个：

public string Reverse(string text)
{
    char[] cArray = text.ToCharArray();
    string reverse = String.Empty;
    for (int i = cArray.Length - 1; i > -1; i--)
    {
        reverse += cArray[i];
    }
    return reverse;
}

就个人而言，我对这个功能并不着迷，并且相信有更好的方法来实现它。有没有？



解答

public static string Reverse( string s )
{
    char[] charArray = s.ToCharArray();
    Array.Reverse(charArray);
    return new string(charArray);
}


更多建议

这里有一个解决方案可以正确地将字符串反转"Les Mise\u0301rables"为"selbare\u0301siM seL". 这应该呈现为selbarésiM seL, 而不是selbaŕesiM seL（注意重音的位置），就像大多数基于代码单元（Array.Reverse等）甚至代码点（特别注意代理对的反转）的实现的结果一样。

using System;
using System.Collections.Generic;
using System.Globalization;
using System.Linq;

public static class Test
{
    private static IEnumerable<string> GraphemeClusters(this string s) {
        var enumerator = StringInfo.GetTextElementEnumerator(s);
        while(enumerator.MoveNext()) {
            yield return (string)enumerator.Current;
        }
    }
    private static string ReverseGraphemeClusters(this string s) {
        return string.Join("", s.GraphemeClusters().Reverse().ToArray());
    }

    public static void Main()
    {
        var s = "Les Mise\u0301rables";
        var r = s.ReverseGraphemeClusters();
        Console.WriteLine(r);
    }
}

如何使用 node.js 漂亮地打印 JSON？

Arya

2023-02-18 04:30

这似乎是一个已解决的问题，但我无法找到解决方案。

基本上，我读取一个 JSON 文件，更改一个密钥，然后将新的 JSON 写回同一个文件。一切正常，但我松开了 JSON 格式。所以，而不是：

{
  name:'test',
  version:'1.0'
}

我得到

{name:'test',version:'1.1'}

Node.js 中有没有一种方法可以将格式良好的 JSON 写入文件？



解答

JSON.stringify的第三个参数定义了漂亮打印的空白插入。它可以是字符串或数字（空格数）。Node 可以使用fs. 例子：

var fs = require('fs');

fs.writeFile('test.json', JSON.stringify({ a:1, b:2, c:3 }, null, 4));
/* test.json:
{
     "a": 1,
     "b": 2,
     "c": 3,
}
*/

请参阅MDN 上的 JSON.stringify() 文档，Node fs 文档

鉴于此功能，我想用随机颜色生成器替换 颜色 。

document.overlay = GPolyline.fromEncoded({
    color: "#0000FF",
    weight: 10,
    points: encoded_points,
    zoomFactor: 32,
    levels: encoded_levels,
    numLevels: 4
});

我该怎么做？



解答

使用getRandomColor()代替"#0000FF"：

function getRandomColor() {
  var letters = '0123456789ABCDEF';
  var color = '#';
  for (var i = 0; i < 6; i++) {
    color += letters[Math.floor(Math.random() * 16)];
  }
  return color;
}



function setRandomColor() {
  $("#colorpad").css("background-color", getRandomColor());
}
