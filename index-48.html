如何列出提交中的所有文件？

Arya

2023-02-16 19:38

如何打印属于给定提交的所有文件的简单列表？

虽然下面列出了文件，但它还包含每个文件不需要的差异信息：

git show a303aa90779efdd2f6b9d90693e2cbbbe4613c1d



解答

首选方式 （因为它是 管道 命令；意味着编程）：

$ git diff-tree --no-commit-id --name-only bd61ad98 -r
index.html
javascript/application.js
javascript/ie6.js

另一种方式 （不太适合脚本，因为它是 瓷器 命令；意味着面向用户）

$ git show --pretty="" --name-only bd61ad98    
index.html
javascript/application.js
javascript/ie6.js
  • 抑制--no-commit-id提交 ID 输出。
  • 该--pretty参数指定一个空的格式字符串，以避免在开头出现麻烦。
  • 该--name-only参数仅显示受影响的文件名（感谢 Hank）。--name-status如果您想查看每个文件发生了什么（ 删除 、 修改 、 添加 ），请改用
  • 参数-r是递归到子树


更多建议

如果要获取已更改文件的列表：

git diff-tree --no-commit-id --name-only -r <commit-ish>

如果要获取提交中所有文件的列表，可以使用

git ls-tree --name-only -r <commit-ish>

在 Git 中仅提交文件的部分更改

Arya

2023-02-16 19:38

当我在 Git 中对文件进行更改时，如何才能只提交部分更改？

例如，我如何才能提交文件中已更改的 30 行中的 15 行？



解答

您可以使用：

git add --patch <filename>

或简称：

git add -p <filename>

Git 会将您的文件分解成它认为合理的“块”（文件的一部分）。然后它会提示你这个问题：

Stage this hunk [y,n,q,a,d,/,j,J,g,s,e,?]?

以下是每个选项的说明：

  • y为下一次提交暂存这个大块头
  • n不要为下一次提交准备这个大块头
  • q辞职; 不要上演这个大块头或任何剩余的大块头
  • a在文件中暂存这个大块头和所有后来的大块头
  • d不要在文件中暂存这个大块或任何后来的大块
  • g选择一个大块头去
  • /搜索匹配给定正则表达式的大块头
  • j让这个大块头犹豫不决，看看下一个犹豫不决的大块头
  • J让这个大块头犹豫不决，看看下一个大块头
  • k让这个大块头犹豫不决，看看以前的未定大块头
  • K让这个大块头犹豫不决，看看以前的大块头
  • s将当前的大块分割成更小的大块
  • e手动编辑当前的大块头
    • +然后，您可以通过将/-替换为#（感谢veksen）来手动编辑大块头
  • ?打印大佬帮助

如果该文件还不在存储库中，您可以先执行git add -N <filename>. 之后你可以继续git add -p <filename>。

之后，您可以使用：

  • git diff --staged检查您是否进行了正确的更改
  • git reset -p解除错误添加的帅哥
  • git commit -v在编辑提交消息时查看您的提交。

请注意，这与命令有很大不同git format-patch，后者的目的是将提交数据解析为.patch文件。

未来参考：Git 工具 - 交互式暂存


更多建议

您可以使用git add --interactiveor ，然后( not ); 请参阅git-add联机帮助页中的 交互模式 ，或简单地按照说明进行操作。git add -p < _file_ >``git commit ****git commit -a __

现代 Git 也有git commit --interactive（和git commit --patch，这是交互式提交中补丁选项的快捷方式）。

如果您更喜欢从 GUI 进行操作，可以使用git-gui。您可以简单地标记要包含在提交中的块，方法是右键单击它们并选择：

提交的舞台大块头

如果你只想添加大块头的某些行，你也可以选择你想要的行，右键单击它们并：

提交阶段行

我个人觉得它比使用git add -i. 其他 git GUI，如 QGit 或 GitX，也可能具有此功能。

如何在 Java 中将 String 转换为 int？

Arya

2023-02-16 19:36

如何将 a 转换String为 an int？

"1234"  →  1234



解答

String myString = "1234";
int foo = Integer.parseInt(myString);

如果您查看Java 文档，您会注意到“捕获”是该函数可以抛出一个NumberFormatException，您可以处理它：

int foo;
try {
   foo = Integer.parseInt(myString);
}
catch (NumberFormatException e) {
   foo = 0;
}

（此处理将格式错误的数字默认为0，但您可以根据需要做其他事情。）

或者，您可以使用IntsGuava 库中的一种方法，该方法与 Java 8 相结合Optional，提供了一种将字符串转换为 int 的强大而简洁的方法：

import com.google.common.primitives.Ints;

int foo = Optional.ofNullable(myString)
 .map(Ints::tryParse)
 .orElse(0)


更多建议

例如，这里有两种方式：

Integer x = Integer.valueOf(str);
// or
int y = Integer.parseInt(str);

这些方法之间存在细微差别：

  • valueOf返回一个新的或缓存的实例java.lang.Integer
  • parseInt返回原始的int。

所有情况都相同：Short.valueOf/ parseShort、Long.valueOf/parseLong等。
