使用已删除的远程分支从原点获取？

Arya

2023-02-18 01:17

当我这样做git fetch origin并且 origin 有一个已删除的分支时，它似乎没有在我的存储库中更新它。当我这样做时git branch -r它仍然显示origin/DELETED_BRANCH.

我怎样才能解决这个问题？



解答

您需要执行以下操作

git fetch -p

or-p参数--prune将更新远程分支的本地数据库。


更多建议

来自http://www.gitguys.com/topics/adding-and-removing-remote-branches/

当有人从远程仓库中删除一个分支后，当用户执行 git pull 或 git fetch 时，git 不会自动删除本地仓库分支。但是，如果用户希望从其本地存储库中删除已在远程存储库中删除的所有跟踪分支，他们可以键入：

git 远程修剪源

请注意，-p 参数git fetch -p实际上意味着“修剪”。
无论您选择哪种方式，不存在的远程分支都将从您的本地存储库中删除。

转到 Visual Studio 中的匹配大括号？

Arya

2023-02-18 01:17

Visual Studio 2008 中有没有一种方法可以从右大括号转到左大括号？我发现了很多关于突出显示大括号的内容，但没有找到关于将光标移动到它的内容。

（此问题的 VB.NET 版本：在“If/End If”之间跳转的键盘快捷方式）



解答

我为您找到了这个：在 Visual Studio 中跳转大括号：

将光标放在大括号（您的选择）之前或之后，然后按CTRL+ ]。它适用于圆括号 ( )、方括号 [ ] 和大括号 { }。从现在开始你不需要玩 Where's Waldo? 找到那个支架。

使用上面的快捷方式，您也可以按住SHIFT选择。

在 MacOS 上，使用CMD+ SHIFT+ \。


更多建议

我使用Visual Studio 2008，您可以自定义您想要的快捷方式。

单击菜单 工具 -> 选项 -> 环境 -> 键盘 。然后寻找 Edit.GotoBrace 。

这将告诉您当前为此分配了什么组合键。我 认为 您可以根据需要更改它，但如果Ctrl+]不起作用，它会很有用。

如何枚举 String 类型的枚举？

Arya

2023-02-18 01:16 

enum Suit: String {
    case spades = "♠"
    case hearts = "♥"
    case diamonds = "♦"
    case clubs = "♣"
}

例如，我该怎么做：

for suit in Suit {
    // do something with suit
    print(suit.rawValue)
}

结果示例：

♠
♥
♦
♣



解答

斯威夫特 4.2+

从Swift 4.2（使用 Xcode 10）开始，只需添加协议一致性CaseIterable即可从allCases. 要添加此协议一致性，您只需要在某处编写：

extension Suit: CaseIterable {}

如果枚举是你自己的，你可以直接在声明中指定一致性：

enum Suit: String, CaseIterable { case spades = "♠"; case hearts = "♥"; case diamonds = "♦"; case clubs = "♣" }

然后下面的代码将打印所有可能的值：

Suit.allCases.forEach {
    print($0.rawValue)
}

与早期 Swift 版本（3.x 和 4.x）的兼容性

如果您需要支持 Swift 3.x 或 4.0，您可以通过添加以下代码来模拟 Swift 4.2 实现：

#if !swift(>=4.2)
public protocol CaseIterable {
    associatedtype AllCases: Collection where AllCases.Element == Self
    static var allCases: AllCases { get }
}
extension CaseIterable where Self: Hashable {
    static var allCases: [Self] {
        return [Self](AnySequence { () -> AnyIterator<Self> in
            var raw = 0
            var first: Self?
            return AnyIterator {
                let current = withUnsafeBytes(of: &raw) { $0.load(as: Self.self) }
                if raw == 0 {
                    first = current
                } else if current == first {
                    return nil
                }
                raw += 1
                return current
            }
        })
    }
}
#endif
