CSS职位所需知识

CSS 有几个用于定位元素的属性。默认情况下，所有元素都是position: static. 这意味着元素将根据其在 HTML 结构中的顺序定位，几乎没有例外。

其他position值为relative、absolute、sticky和fixed。通过将元素设置position为这些其他值之一，现在可以使用以下四个属性的组合来定位元素：

  • top
  • right
  • bottom
  • left

换句话说，通过设置position: absolute，我们可以将top: 100px元素添加到距离页面顶部 100 像素的位置。相反，如果我们将bottom: 100px元素设置为距页面底部 100 像素的位置。

这是许多 CSS 新手迷路的地方 -position: absolute有一个参考框架。在上面的示例中，参考系是元素body。position: absolutewithtop: 100px表示元素位于距元素顶部 100 像素的位置body。

position可以通过将父元素的 设置为 以外的任何值position: static来更改位置参考框架或位置上下文。也就是说，我们可以通过给父元素创建一个新的位置上下文：

  • position: relative;
  • position: absolute;
  • position: sticky;
  • position: fixed;

例如，如果<div class="parent">给定一个元素position: relative，则任何子元素都使用<div class="parent">作为它们的位置上下文。如果给定子元素position: absoluteand top: 100px，则该元素将定位在距元素顶部 100 像素的位置<div class="parent">，因为<div class="parent">现在是位置上下文。

另一个 需要注意的因素是 堆叠顺序 ——或者元素在 z 方向上的堆叠方式。这里必须知道的是，元素的堆栈顺序默认情况下是按照它们在 HTML 结构中的顺序的相反顺序定义的。考虑以下示例：

<body>
  <div>Bottom</div>
  <div>Top</div>
</body>

在此示例中，如果两个<div>元素位于页面上的同一位置，则该<div>Top</div>元素将覆盖该<div>Bottom</div>元素。由于在 HTML 结构中<div>Top</div>排在后面，因此<div>Bottom</div>它具有更高的堆叠顺序。

在 JavaScript 中删除 DOM 节点的所有子元素

Arya

2023-02-17 03:51

我将如何着手删除 JavaScript 中 DOM 节点的所有子元素？

假设我有以下（丑陋的）HTML：

<p id="foo">
    <span>hello</span>
    <div>world</div>
</p>

然后我像这样抓住我想要的节点：

var myNode = document.getElementById("foo");

foo我怎么能删除这样的孩子<p id="foo"></p>呢？

我可以这样做吗：

myNode.childNodes = new Array();

或者我应该使用一些组合removeElement？

我希望答案是直截了当的 DOM；如果您还提供了 jQuery 中的答案以及仅 DOM 的答案，则可以加分。



解答

2022+，使用replaceChildren()API​​！

现在可以使用（支持跨浏览器的）replaceChildrenAPI替换所有子项：

container.replaceChildren(...arrayOfNewChildren);

这将同时做：

  • 删除所有现有的孩子，和
  • 在一次操作中追加所有给定的新孩子。

您还可以使用相同的 API 只删除现有的子项，而不替换它们：

container.replaceChildren();

这在 Chrome/Edge 86+、Firefox 78+ 和 Safari 14+ 中得到完全支持。 这是完全指定的行为。这可能比这里提出的任何其他方法 都快 ，因为移除旧孩子和添加新孩子不需要innerHTML, 并且一步而不是多步完成。

如何限制 Oracle 查询在排序后返回的行数？

Arya

2023-02-17 03:50

有没有办法让Oracle查询表现得像包含 MySQLlimit子句一样？

在 MySQL 中，我可以这样做：

select * 
from sometable
order by name
limit 20,10

获得第 21 到第 30 行（跳过前 20 行，给出接下来的 10 行）。行是在 之后选择的order by，所以它实际上是按字母顺序从第 20 个名字开始的。

在 Oracle 中，人们唯一提到的是rownum伪列，但它是 在之前 order by评估的，这意味着：

select * 
from sometable
where rownum <= 10
order by name

将返回一组按名称排序的随机十行，这通常不是我想要的。它还不允许指定偏移量。



解答

您可以为此使用子查询

select *
from  
( select * 
  from emp 
  order by sal desc ) 
where ROWNUM <= 5;

另请查看Oracle/AskTom 上的主题On ROWNUM 和限制结果以获取更多信息。

更新 ：为了限制下限和上限的结果，事情变得更加臃肿

select * from 
( select a.*, ROWNUM rnum from 
  ( <your_query_goes_here, with order by> ) a 
  where ROWNUM <= :MAX_ROW_TO_FETCH )
where rnum  >= :MIN_ROW_TO_FETCH;

（从指定的 AskTom 文章中复制）

更新 2 ：从 Oracle 12c (12.1) 开始，有一种语法可用于限制行或从偏移量开始。

SELECT * 
FROM   sometable
ORDER BY name
OFFSET 20 ROWS FETCH NEXT 10 ROWS ONLY;

有关更多示例，请参见此答案。感谢 Krumia 的提示。


更多建议
