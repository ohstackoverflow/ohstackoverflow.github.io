什么是迭代器、可迭代对象和迭代器？

Arya

2023-02-18 05:05

Python 中的“可迭代”、“迭代器”和“迭代”是什么？它们是如何定义的？



解答

迭代 是一个通用术语，用于一个接一个地获取某物的每一项。任何时候您使用循环（显式或隐式）来遍历一组项目，即迭代。

在 Python 中， iterable 和 iterator 有特定的含义。

可 迭代对象 是一个对象，它有一个__iter__返回 迭代器的 方法，或者定义了一个__getitem__方法，该方法可以采用从零开始的顺序索引（并IndexError在索引不再有效时引发）。所以 可迭代 对象是一个可以从中获取 迭代器的 对象。

迭代 器next是具有(Python 2) 或__next__(Python 3) 方法的对象。

每当你在 Python 中使用for循环，或map，或列表理解等时，都会自动调用该方法从 迭代器next中获取每个项目，从而经历 迭代 过程。 * *

开始学习的好地方是教程的迭代器部分和标准类型页面的迭代器类型部分。了解基础知识后，请尝试Functional Programming HOWTO 的迭代器部分。


更多建议

这是我在教授 Python 课程时使用的解释：

一个可迭代的是：

  • 任何可以循环的东西（即你可以循环一个字符串或文件）或
  • 任何可以出现在 for 循环右侧的东西：for x in iterable: ...或
  • 任何你可以用iter()它调用的东西都会返回一个迭代器：iter(obj)或者
  • __iter__定义返回新迭代器的对象，或者它可能具有__getitem__适合索引查找的方法。

ITERATOR 是一个对象：

  • 具有在迭代过程中记住它所在位置的状态，
  • 用一种__next__方法：
    • 返回迭代中的下一个值
    • 更新状态以指向下一个值
    • 通过提高完成时发出信号StopIteration
  • 这是 自迭代的 （意味着它有一个__iter__返回的方法self）。

笔记：

  • Python 3 中的方法在 Python 2 中__next__拼写next，并且
  • 内置函数next()在传递给它的对象上调用该方法。

例如：

>>> s = 'cat'      # s is an ITERABLE
                   # s is a str object that is immutable
                   # s has no state
                   # s has a __getitem__() method

>>> t = iter(s)    # t is an ITERATOR
                   # t has state (it starts by pointing at the "c"
                   # t has a next() method and an __iter__() method

>>> next(t)        # the next() function returns the next value and advances the state
'c'
>>> next(t)        # the next() function returns the next value and advances
'a'
>>> next(t)        # the next() function returns the next value and advances
't'
>>> next(t)        # next() raises StopIteration to signal that iteration is complete
Traceback (most recent call last):
...
StopIteration

>>> iter(t) is t   # the iterator is self-iterable

在 JavaScript 中递增日期

Arya

2023-02-18 05:04

我需要在 JavaScript 中将日期值增加一天。

例如，我有一个日期值 2010-09-11，我需要将第二天的日期存储在 JavaScript 变量中。

如何将日期增加一天？



解答

三种选择供您选择：

1. 仅使用 JavaScript 的Date对象（无库）：

我之前对 #1 的回答是错误的（它增加了 24 小时，没有考虑到夏令时的转换；Clever Human指出它会在东部时区的 2010 年 11 月 7 日失败）。相反，Jigar 的回答是在没有图书馆的情况下执行此操作的正确方法：

// To do it in local time
var tomorrow = new Date();
tomorrow.setDate(tomorrow.getDate() + 1);

// To do it in UTC
var tomorrow = new Date();
tomorrow.setUTCDate(tomorrow.getUTCDate() + 1);

这甚至适用于一个月（或一年）的最后一天，因为 JavaScript 日期对象对翻转很聪明：

显示代码片段

// (local time)
var lastDayOf2015 = new Date(2015, 11, 31);
console.log("Last day of 2015: " + lastDayOf2015.toISOString());
var nextDay = new Date(+lastDayOf2015);
var dateValue = nextDay.getDate() + 1;
console.log("Setting the 'date' part to " + dateValue);
nextDay.setDate(dateValue);
console.log("Resulting date: " + nextDay.toISOString());

2. 使用MomentJS：

var today = moment();
var tomorrow = moment(today).add(1, 'days');

（注意add修改你调用它的实例，而不是返回一个新实例，所以today.add(1, 'days')会修改today。这就是我们从克隆操作开始的原因var tomorrow = ...。）

3.使用DateJS，但是很久没更新了：

var today = new Date(); // Or Date.today()
var tomorrow = today.add(1).day();


更多建议

var myDate = new Date();

//add a day to the date
myDate.setDate(myDate.getDate() + 1);

在 chrome 调试器/DevTools 面板中冻结屏幕以进行弹出窗口检查？

Arya

2023-02-18 05:04

我正在使用 chrome 检查器尝试分析z-indexTwitter 引导程序弹出窗口，发现它非常令人沮丧......

有没有办法冻结弹出窗口（显示时）以便我可以评估和修改关联的 CSS？

在关联链接上放置固定的“悬停”不会导致弹出窗口出现。



解答

得到它的工作。 这是我的程序：

  1. 浏览到所需的页面
  2. 打开开发控制台 -F12在 Windows/Linux 上或在 macOS 上option+ ⌘+J
  3. Sources在 chrome 检查器中选择选项卡
  4. 在 Web 浏览器窗口中，将鼠标悬停在所需元素上以启动弹出窗口
  5. 在弹出窗口显示时点击F8Windows/Linux（或fn+在 macOS 上）。 如果您单击了实际页面上的任意位置，F8 将不会执行任何操作。 您的最后一次点击需要位于检查器中的某个位置，例如源选项卡F8 __
  6. 转到Elements检查器中的选项卡
  7. 找到您的弹出窗口（它将嵌套在触发元素的 HTML 中）
  8. 享受修改 CSS 的乐趣


更多建议

为了能够检查任何元素，请执行以下操作。即使很难复制悬停状态，这也应该有效：

  • 在控制台中运行以下 javascript。这将在 5 秒内进入调试器。

setTimeout(function(){debugger;}, 5000)

  • 去显示你的元素（通过悬停或其他方式）并等待 Chrome 进入调试器。

  • 现在单击ElementsChrome 检查器中的选项卡，您可以在那里查找您的元素。

  • 您也可以单击该Find Element图标（看起来像一个放大镜），Chrome 会让您检查并通过右键单击它在页面上找到您的元素，然后选择Inspect Element

请注意，此方法与此页面上的其他出色答案略有不同。
