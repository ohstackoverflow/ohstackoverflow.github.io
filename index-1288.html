如何查找和替换 Visual Studio Code 中的所有事件（在所有文件中）？

Arya

2023-02-18 06:01

我不知道如何使用 Visual Studio Code 1.0 版查找和替换不同文件中出现的所有单词。

我觉得这应该是可能的，因为执行 Ctrl + Shift + F 可以让我简单地搜索一个文件夹，但我不知道如何从这里开始。

我查看了各种组合键https://code.visualstudio.com/Docs/customization/keybindings

但我只找到了如何替换打开文件中的所有匹配项 (Ctrl + h)

感谢所有帮助，这让我发疯。



解答

从 vscode 的 1.3 版本开始，这是 可能的

  1. 导航到搜索，单击左侧的图标或：
    • （苹果）cmd + shift + h
    • （个人电脑）ctrl + shift + h
  2. 展开替换
  3. 输入搜索词并替换词
  4. 确认！

如何连接树枝中的字符串

Arya

2023-02-18 06:01

任何人都知道如何连接树枝中的字符串？我想做类似的事情：

{{ concat('http://', app.request.host) }}



解答

这应该可以正常工作：

{{ 'http://' ~ app.request.host }}

要在同一标签中添加过滤器 - 如“trans” - 使用

{{ ('http://' ~ app.request.host) | trans }}

正如Adam Elsodaney 指出的那样，您还可以使用字符串插值，这确实需要双引号字符串：

{{ "http://#{app.request.host}" }}


更多建议

Twig 中还有一个鲜为人知的功能是字符串插值：

{{ "http://#{app.request.host}" }}

将 Unicode 字符串转换为 Python 中的字符串（包含额外符号）

Arya

2023-02-18 06:00

如何将 Unicode 字符串（包含 £ $ 等额外字符）转换为 Python 字符串？



解答

看unicodedata.normalize

title = u"Klüft skräms inför på fédéral électoral große"
import unicodedata
unicodedata.normalize('NFKD', title).encode('ascii', 'ignore')
'Kluft skrams infor pa federal electoral groe'


更多建议

如果不需要翻译非 ASCII 字符，可以使用 encode to ASCII：

>>> a=u"aaaàçççñññ"
>>> type(a)
<type 'unicode'>
>>> a.encode('ascii','ignore')
'aaa'
>>> a.encode('ascii','replace')
'aaa???????'
>>>
