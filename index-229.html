从 Javascript 中的数组中删除空元素

Arya

2023-02-17 01:35

如何从 JavaScript 中的数组中删除空元素？

有没有一种直接的方法，还是我需要遍历它并手动删除它们？



解答

编辑： 这个问题几乎在九年前就得到了回答，当时Array.prototype.

现在，当然，我会推荐你​​使用这个filter方法。

请记住，此方法将返回 一个新数组 ，其中包含通过您提供给它的回调函数的条件的元素。

例如，如果你想删除null或undefined值：

var array = [0, 1, null, 2, "", 3, undefined, 3,,,,,, 4,, 4,, 5,, 6,,,,];

var filtered = array.filter(function (el) {
  return el != null;
});

console.log(filtered);

运行代码片段隐藏结果

这将取决于您认为什么是“空”，例如，如果您正在处理字符串，则上述函数不会删除空字符串的元素。

我看到经常使用的一种典型模式是删除 虚假 元素，其中包括空字符串"", 0, NaN, null,undefined和false。

你可以传递给filter方法，Boolean构造函数，或者在过滤条件函数中返回相同的元素，例如：

var filtered = array.filter(Boolean);

或者

var filtered = array.filter(function(el) { return el; });

在这两种情况下，这是可行的，因为filter第一种情况下的方法将Boolean构造函数作为函数调用，转换值，而在第二种情况下，该filter方法在内部将回调的返回值隐式转换为Boolean.

如果您正在使用稀疏数组，并且您正在尝试摆脱“漏洞”，则可以使用filter传递返回 true 的回调的方法，例如：

var sparseArray = [0, , , 1, , , , , 2, , , , 3],
    cleanArray = sparseArray.filter(function () { return true });

console.log(cleanArray); // [ 0, 1, 2, 3 ]

运行代码片段隐藏结果

旧答案： 不要这样做！

我使用这种方法，扩展原生数组原型：

Array.prototype.clean = function(deleteValue) {
  for (var i = 0; i < this.length; i++) {
    if (this[i] == deleteValue) {         
      this.splice(i, 1);
      i--;
    }
  }
  return this;
};

test = new Array("", "One", "Two", "", "Three", "", "Four").clean("");
test2 = [1, 2,, 3,, 3,,,,,, 4,, 4,, 5,, 6,,,,];
test2.clean(undefined);

或者您可以简单地将现有元素推入其他数组：

// Will remove all falsy values: undefined, null, 0, false, NaN and "" (empty string)
function cleanArray(actual) {
  var newArray = new Array();
  for (var i = 0; i < actual.length; i++) {
    if (actual[i]) {
      newArray.push(actual[i]);
    }
  }
  return newArray;
}

cleanArray([1, 2,, 3,, 3,,,,,, 4,, 4,, 5,, 6,,,,]);

JavaScript 是否有像“range()”这样的方法来在提供的范围内生成一个范围？

Arya

2023-02-17 01:34

在 PHP 中，你可以做...

range(1, 3); // Array(1, 2, 3)
range("A", "C"); // Array("A", "B", "C")

也就是说，有一个函数可以让你通过上下界来获得一个范围内的数字或字符。

为此，JavaScript 有什么内置的东西吗？如果没有，我将如何实施？



解答

它适用于字符和数字，通过可选步骤前进或后退。

var range = function(start, end, step) {
    var range = [];
    var typeofStart = typeof start;
    var typeofEnd = typeof end;

    if (step === 0) {
        throw TypeError("Step cannot be zero.");
    }

    if (typeofStart == "undefined" || typeofEnd == "undefined") {
        throw TypeError("Must pass start and end arguments.");
    } else if (typeofStart != typeofEnd) {
        throw TypeError("Start and end arguments must be of same type.");
    }

    typeof step == "undefined" && (step = 1);

    if (end < start) {
        step = -step;
    }

    if (typeofStart == "number") {

        while (step > 0 ? end >= start : end <= start) {
            range.push(start);
            start += step;
        }

    } else if (typeofStart == "string") {

        if (start.length != 1 || end.length != 1) {
            throw TypeError("Only strings with one character are supported.");
        }

        start = start.charCodeAt(0);
        end = end.charCodeAt(0);

        while (step > 0 ? end >= start : end <= start) {
            range.push(String.fromCharCode(start));
            start += step;
        }

    } else {
        throw TypeError("Only string and number types are supported");
    }

    return range;

}

对于数字，您可以使用 ES6 Array.from()，它现在适用于除 IE 之外的所有东西：

较短的版本：

Array.from({length: 20}, (x, i) => i);

更长的版本：

Array.from(new Array(20), (x, i) => i);​​​​​​

它创建一个从 0 到 19 的数组。这可以进一步缩短为以下形式之一：

Array.from(Array(20).keys());
// or
[...Array(20).keys()];

也可以指定下限和上限，例如：

Array.from(new Array(20), (x, i) => i + *lowerBound*);

一篇更详细地描述这一点的文章：http://www.2ality.com/2014/05/es6-array-methods.html

如何查看只有一个用户提交的 git 日志？

Arya

2023-02-17 01:33

使用时git log，如何按用户过滤以便我只看到来自该用户的提交？



解答

这适用于 gitkgit log和gitk——查看历史记录的两种最常见的方式。
您不需要使用全名：

git log --author="Jon"

将匹配“Jonathan Smith”所做的提交

git log --author=Jon

和

git log --author=Smith

也会工作。如果您不需要任何空格，引号是可选的。

--all 如果您打算搜索所有分支而不仅仅是您的回购中当前提交的祖先，请添加。

您还可以轻松匹配多个作者，因为正则表达式是此过滤器的基础机制。因此，要列出 Jonathan 或 Adam 的提交，您可以这样做：

git log --author="\(Adam\)\|\(Jon\)"

如本问题所述，为了使用正则表达式排除特定作者或一组作者的提交，您可以将否定先行与开关结合使用--perl-regexp：

git log --author='^(?!Adam|Jon).*$' --perl-regexp

或者，您可以使用bash管道排除由 Adam 创作的提交：

git log --format='%H %an' | 
  grep -v Adam | 
  cut -d ' ' -f1 | 
  xargs -n1 git log -1

如果您想排除由 Adam 提交（但不一定是作者）的提交，请替换%an为%cn. 有关此的更多详细信息，请参阅我的博客文章：http://dymitruk.com/blog/2012/07/18/filtering-by-author- name/


更多建议

git log --author="that user"
